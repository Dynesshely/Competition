## 题目描述

假设目前的世界人口有 $x$ 亿，按照每年 $0.1\%$ 的增长速度，$n$ 年后将有多少人？

## 输入格式

一行两个正整数 $x$ 和 $n$，之间有一个空格。其中，$1 \leq x\leq 100, 1\leq n\leq 100$。

## 输出格式

一行一个数，表示答案。以亿为单位，保留到小数点后 $4$ 位。

## 样例 #1

### 样例输入

```
13 10
```

### 样例输出

```
13.1306
```
