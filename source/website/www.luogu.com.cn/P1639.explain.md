# P1639 洛谷 [USACO18FEB] Teleportation B 题解

## 题目链接

- [洛谷 P1639](https://www.luogu.com.cn/problem/P1639)

## 题目大意

在数轴上有一座传送门连接 $x$ 和 $y$ 两点（双向传送）。要将牛粪从 $a$ 运到 $b$，可以灵活使用传送门（也可以不用），求拖拉机需要行驶的最小距离。

## 解题思路

共有三种可能的路线：
1. 直接走：距离为 $|a - b|$。
2. 从 $a$ 走到 $x$，传送至 $y$，再走到 $b$：距离为 $|a - x| + |y - b|$。
3. 从 $a$ 走到 $y$，传送至 $x$，再走到 $b$：距离为 $|a - y| + |x - b|$。

取三者最小值即可。由于距离都是 $0 \sim 100$ 的整数，计算非常简单。

## 代码实现

- 读入 $a, b, x, y$。
- 计算三种方案的距离，取最小值并输出。

## 复杂度分析

- 时间复杂度：$O(1)$
- 空间复杂度：$O(1)$
