# P1103 [洛谷] 书本整理 题解

## 题目链接

- [洛谷 P1103](https://www.luogu.com.cn/problem/P1103)

## 题目大意

有 n 本书，每本书有高度和宽度，高度互不相同。先将书按高度升序排列，然后从中去掉 k 本书，使得剩下的书的不整齐度（相邻两本书宽度差的绝对值之和）最小。求最小不整齐度。

## 解题思路

先按高度排序。问题转化为在 n 本书中选出 m = n - k 本书（保持相对顺序），使得相邻宽度差的绝对值之和最小。设 dp[i][j] 表示以第 i 本书结尾、共选了 j 本书的最小不整齐度。初始化 dp[i][1] = 0（只选一本不产生差值）。转移时枚举前一本的位置 t，有 dp[i][j] = min(dp[t][j-1] + |w[i] - w[t]|)。最后取 dp[i][m] 的最小值作为答案。

## 代码实现

- 按高度升序排序
- m = n - k 表示保留的书籍数量
- dp 数组初始化为无穷大，dp[i][1] = 0
- 三重循环 DP：枚举 j 从 2 到 m，i 从 j 到 n，t 从 j-1 到 i-1
- 答案取 min(dp[i][m]) (i = m..n)

## 复杂度分析

- 时间复杂度：O(n^3)，n <= 100 可接受
- 空间复杂度：O(n^2)
