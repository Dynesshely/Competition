# B2086 不定方程求解 题解
## 题目链接
- [洛谷 B2086](https://www.luogu.com.cn/problem/B2086)
## 题目大意
给定正整数 a, b, c，求不定方程 ax + by = c 的非负整数解 (x, y) 的组数。
## 解题思路
枚举 x 从 0 开始，满足 a*x <= c 时停止。对于每个 x，若 (c - a*x) 能被 b 整除，则找到一个合法解。无需单独枚举 y，因为 y 由整除性唯一确定。
## 代码实现
枚举 x 并使用取模运算判断整除性。由于 a, b, c <= 1000，直接枚举即可通过。
## 复杂度分析
- 时间复杂度 O(c/a)，最坏 O(c)
- 空间复杂度 O(1)
