# P1616 疯狂的采药 题解

## 题目链接

- [洛谷 P1616](https://www.luogu.com.cn/problem/P1616)

## 题目大意

给定总采药时间 t 和 m 种草药，每种草药可以无限次采摘，每种草药有采摘时间 `w[i]` 和价值 `v[i]`。求在不超过总时间的前提下能获得的最大总价值。

## 解题思路

这是一道经典的 **完全背包（Complete Knapsack）** 问题。

完全背包与 0-1 背包的区别在于：每种物品可以取任意多次。因此，在动态规划的内层循环中，我们**正序遍历**容量 j（从 `w[i]` 到 m），使得同一种物品可以被多次选取。

定义 `f[j]` 为总时间不超过 j 时能获得的最大价值。状态转移方程为：

```
f[j] = max(f[j], f[j - w[i]] + v[i])
```

该方程的含义是：对当前草药 i，可以选择不取（`f[j]` 保持不变）或者取一件（`f[j - w[i]] + v[i]`）。由于 j 从小到大遍历，`f[j - w[i]]` 可能已经包含了之前取的该草药，从而实现了无限次选取的效果。

## 代码实现

- **`w[i]` / `v[i]`**：分别存储第 i 种草药的采摘时间和价值。
- **`f[maxm]`**：一维 DP 数组，`maxm` 取 `1e7 + 5`，覆盖最大时间范围。
- **外层循环**：遍历每种草药 i（1 到 n）。
- **内层循环**：对时间 j 从 `w[i]` 到 m 正序 DP。正序保证了同种草药可被多次计入。
- 最后输出 `f[m]` 即为答案。

## 复杂度分析

- 时间复杂度：O(m * t)，其中 m 为草药种类数，t 为总时间。
- 空间复杂度：O(t)，用于一维 DP 数组。
