# P1897 电梯里的尴尬 题解

## 题目链接
- [洛谷 P1897](https://www.luogu.com.cn/problem/P1897)

## 题目大意
计算电梯完成一趟任务的总时间：电梯从 0 层出发，运送 n 个人到各自的目标楼层，最后回到 0 层。向上运行一层 6 秒，向下运行一层 4 秒，每开门一次 5 秒（有人下才开门），每人下电梯 1 秒。

## 解题思路
答案由四部分组成：
1. 上升时间 = 最高楼层 x 6（电梯只需上升到最高需求楼层）
2. 下降时间 = 最高楼层 x 4（从最高层回到 0 层）
3. 开门时间 = 不同楼层数 x 5（同一楼层只开一次门）
4. 下客时间 = 人数 x 1（每人 1 秒）

所以总时间 = 最高楼层 x 10 + 不同楼层数 x 5 + 人数。

## 代码实现
读入楼层并记录最高楼层 maxF，排序后使用 unique 去重得到不同楼层数 uniqueF，代入公式计算输出。

## 复杂度分析
- 时间复杂度: O(n log n)（排序耗时）
- 空间复杂度: O(n)
