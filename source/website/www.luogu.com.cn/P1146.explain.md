# P1146 硬币翻转 题解

## 题目链接

- [洛谷 P1146](https://www.luogu.com.cn/problem/P1146)

## 题目大意

有 N 枚硬币（N 为偶数），全部正面朝上。每次可以翻转任意 N-1 枚硬币，求将所有硬币翻转为反面朝上的最短操作序列，输出字典序最小的方案。

## 解题思路

数学结论：最少操作次数为 N 次。每次翻转 N-1 枚硬币等价于翻转 1 枚硬币后整体取反，但更容易的构造方法为：第 k 次操作时，翻转除第 k 枚以外的所有硬币。这样第 k 枚硬币保持了前 k-1 次未被翻转的状态，最终恰好被翻转 N-1 次（变为反面），而其他硬币都被翻转 N 次（也变为反面）。

## 代码实现

直接按上述构造输出：对于第 k 次操作（k 从 1 到 N），输出一个 N 位的 01 串，第 c 位为 1 表示翻转。第 k 次操作中，当 c < k 时输出 (k-1)&1，否则输出 k&1。

## 复杂度分析

- 时间复杂度：O(N^2)
- 空间复杂度：O(1)
