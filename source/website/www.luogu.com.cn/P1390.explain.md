# P1390 洛谷 公约数的和 题解

## 题目链接
- [洛谷 P1390](https://www.luogu.com.cn/problem/P1390)

## 题目大意

给定 n，求所有满足 1 ≤ i < j ≤ n 的数对 (i, j) 的最大公约数之和：

$$\sum_{i=1}^{n}\sum_{j=i+1}^{n}\gcd(i,j)$$

## 解题思路

利用欧拉函数进行推导。对于每个最大公约数 d，统计有多少对 (i, j) 满足 gcd(i, j) = d。

设 i = d·a，j = d·b，则 gcd(a, b) = 1 且 a < b ≤ ⌊n/d⌋。

对于固定的 d，令 m = ⌊n/d⌋，需要统计满足 1 ≤ a < b ≤ m 且 gcd(a, b) = 1 的数对个数。对于每个 b (2 ≤ b ≤ m)，与 b 互质且小于 b 的 a 的个数即为欧拉函数 φ(b)，因此总数为：

$$\sum_{b=2}^{m} \varphi(b) = \left(\sum_{i=1}^{m} \varphi(i)\right) - 1$$

记 pref[m] = Σ_{i=1}^{m} φ(i)，则答案为：

$$\sum_{d=1}^{n} d \times (\text{pref}[\lfloor n/d \rfloor] - 1)$$

算法步骤：
1. 使用筛法计算 1~n 的欧拉函数 φ。
2. 计算 φ 的前缀和 pref。
3. 枚举 d 从 1 到 n，累加 d × (pref[n/d] - 1) 到答案。

## 代码实现

使用数组 phi 存储欧拉函数值，phi[i] 初始化为 i。对每个 i 从 2 到 n，若 phi[i] == i（即 i 为质数），则对 j 从 i 到 n 步长为 i 进行筛法更新：phi[j] -= phi[j] / i。然后计算 phi 前缀和，最后枚举 d 累加 d × (pref[n/d] - 1)。

## 复杂度分析

- 时间复杂度：O(n log log n)，欧拉函数筛法 O(n log log n)，主循环 O(n)，n ≤ 2×10⁶ 时可以在 1 秒内完成。
- 空间复杂度：O(n)，存储欧拉函数值和前缀和数组。
