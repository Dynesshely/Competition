# B2039 整数大小比较 题解

## 题目链接
- [洛谷 B2039](https://www.luogu.com.cn/problem/B2039)

## 题目大意
输入两个整数 x 和 y，比较它们的大小并输出 `>`、`=` 或 `<`。

## 解题思路
直接比较即可。注意 x 的范围是 `[0, 2^32)` 而 y 的范围是 `[-2^31, 2^31)`，直接用 `int` 可能溢出，因此使用 `long long` 存储两个数后按题目要求输出对应字符。

## 代码实现
使用 `long long` 读入两个整数，通过 `if-else` 判断大小关系，输出 `>`、`=` 或 `<`。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
