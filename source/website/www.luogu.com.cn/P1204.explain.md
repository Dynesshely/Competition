# P1204 [USACO1.2] 挤牛奶 题解

## 题目链接

- [洛谷 P1204](https://www.luogu.com.cn/problem/P1204)

## 题目大意

$N$ 个农民在不同时间段 $[start, end)$ 挤奶。求最长的连续有人挤奶的时间段长度，以及最长的连续无人挤奶的时间段长度。

## 解题思路

区间合并。按开始时间排序后依次处理：
- 若当前区间与已有区间重叠或相邻（`a[i].first <= curEnd`），扩展 `curEnd`。
- 否则产生空隙，更新最长空闲时间和最长挤奶时间，重置区间。

## 代码实现

- `sort(a, a+n)` 按开始时间排序。
- 维护 `curStart, curEnd`，遍历更新 `milking` 和 `idle`。
- 输出 `milking idle`。

## 复杂度分析

- 时间复杂度：$O(N \log N)$，$N \le 5000$。
- 空间复杂度：$O(N)$。
