# P1271 【深基9.例1】选举学生会 题解

## 题目链接
- [洛谷 P1271](https://www.luogu.com.cn/problem/P1271)

## 题目大意
有 n 名候选人（编号 1~n），m 张选票，每张选票上写有一个候选人编号。将所有选票按编号从小到大排序输出。

## 解题思路
由于候选人编号范围较小（1 <= n <= 999），直接使用计数排序。统计每个编号出现的次数，然后按顺序输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;
int cnt[1005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; i++) {
        int x;
        scanf("%d", &x);
        cnt[x]++;
    }
}

void process() {}

void output() {
    for (int i = 1; i <= n; i++)
        for (int j = 0; j < cnt[i]; j++)
            printf("%d ", i);
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + m)。
- 空间复杂度：O(n)。
