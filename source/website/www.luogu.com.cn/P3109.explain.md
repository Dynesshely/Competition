# P3109 [USACO14OPEN] Code Breaking G 题解

## 题目链接
- [洛谷 P3109](https://www.luogu.com.cn/problem/P3109)

## 题目大意
给定一棵 $N$ 个节点的有根树（$1 \le N \le 2 \times 10^4$），每个节点可以填数字 $0 \sim 9$。有 $M$ 个限制（$1 \le M \le 5 \times 10^4$），每个限制指定一个起始节点 $X$（保证深度至少为 $5$）和一个 5 位字符串 $s$，表示从 $X$ 开始向上跳 $5$ 个节点（含 $X$）组成的数字序列不能等于 $s$。

求所有 $10^N$ 种填数方案中，被这些限制排除掉（即至少违反一个限制）的方案数，对 $P = 1234567$ 取模。

## 解题思路

### 问题转化
总方案数为 $10^N$。若能计算出满足所有限制的方案数（即不存在任何禁止序列），则排除数 = $10^N$ - 合法方案数。

将每个限制编码为一个 20 位的二进制模式（5 位数字 $\times$ 4 位/数字），每个数字 $d$ 编码为 $d+1$（范围 $1 \sim 10$，用 0 表示未定义）。

### 树形 DP 设计
对树进行自底向上的 DP。对于节点 $i$，维护一个映射表 $mp[i]$，其中键是"从 $i$ 开始向上 $5$ 个节点可能形成的数字序列编码"，值是该序列在 $i$ 的子树中出现的方案数。

自底向上递推时，将子节点的模式合并到父节点，同时排除违反限制的方案。

### 编码细节
5 位数字 $d_1d_2d_3d_4d_5$（$d_1$ 为起始节点 $X$，$d_5$ 为 $X$ 向上第 4 个祖先）编码为：
$$
code = (d_1) \ll 16 \;|\; (d_2) \ll 12 \;|\; (d_3) \ll 8 \;|\; (d_4) \ll 4 \;|\; d_5
$$
其中每位数字占用 4 比特，范围 $1 \sim 10$。

### 核心算法流程

1. **初始化**：对每个节点 $i$，插入 10 个初始模式 $(d) \ll 16$，表示"节点 $i$ 填数字 $d$，向上尚无更多祖先信息"，每个初始方案数为 $1$。

2. **插入限制**：对每个限制 $(X, s)$，将编码后的模式 $code$ 存入 $mp[X]$，值为 $P-1$（等价于 $-1$ 模 $P$），表示该模式非法。

3. **自底向上合并**（节点从 $n$ 到 $1$）：
   - 将 $mp[i]$ 中的所有模式插入一棵线段树（以编码为下标），用于标记当前已知的所有"合法后缀模式"及其计数。
   - 对于节点 $i$ 的每个子节点 $j$：
     - 将子节点 $j$ 的模式集合从线段树中收集出来，存入 $que$ 数组。
     - 对每个子模式 $p$（计数 $c$），计算 $p$ 的所有"扩展"（即补全为完整的 20 位编码）与当前父节点中已有模式的交集计数。具体地，枚举 $p$ 最低非零 4 比特位（即子节点上最近的未确定祖先），逐步展开以覆盖所有可能的祖先数字组合。
     - 使用差分数组技巧（按 $x$ 坐标排序事件）实现区间批量乘法更新，将子节点的计数乘积累加到线段树上对应的扩展位置。
     - 将乘积累加后的子模式重新插入线段树。
   - 处理完所有子节点后，通过 `doCollect` 将线段树中所有模式及其计数回收到 $mp[i]$ 中。

4. **最终答案**：根节点 $1$ 的模式集合 $mp[1]$ 中，键 $0$ 对应的值即为满足所有限制的合法方案数（模 $P$）。排除数 = $(10^N - \text{合法数}) \bmod P$。

### 线段树的作用
线段树覆盖范围 $[0, M)$（$M = 860000 > 2^{20}$），每个叶子对应一个 20 位编码。支持操作：
- `doIns(c, p)`：在叶子 $c$ 处加 $p$。
- `doMulRange(l, r, p)`：区间 $[l, r]$ 乘 $p$。
- `doAsk(c)`：查询叶子 $c$ 的值。
- `doCollect(f)`：收集所有非零值到映射 $f$ 中。

### 合并原理
当子节点的模式 $p$ 合并到父节点时，$p$ 的最高位是子节点的数字，而父节点的数字应成为新的最高位。$p$ 丢失最低 4 位的"祖先数字"信息后（右移 4 位），与父节点自身的 10 种可能值组合，形成新的 5 位模式。线段树负责高效统计这些组合与现有非法标记的匹配情况。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 20010, M = 860000, MAXN = 2100000, U = 300010, P = 1234567;

int                     n, m, gr[N], nxt[N], sub[M], rgt[M], ans;
char                    buf[9];
map<int, int>           mp[N];
map<int, int>::iterator it;

int  tmp[U], tag[MAXN];
bool has[MAXN];

struct E {
    int x, y;
    E() {}
    E(int x_, int y_) { x = x_, y = y_; }
} que[U], edg[U * 2];

inline bool cmp(const E &a, const E &b) { return a.x < b.x; }

inline void doAdd(int &a, int b) { a = a + b < P ? a + b : a + b - P; }

inline void doMul(int x, int p) { tag[x] = 1LL * tag[x] * p % P; }

inline void doPush(int x) {
    if (tag[x] != 1)
        doMul(x << 1, tag[x]), doMul(x << 1 | 1, tag[x]), tag[x] = 1;
}

void build(int x, int a, int b) {
    if (a == b) return;
    tag[x]  = 1;
    int mid = (a + b) >> 1;
    build(x << 1, a, mid), build(x << 1 | 1, mid + 1, b);
}

void doIns(int x, int a, int b, int c, int p) {
    if (a == b) {
        doAdd(tag[x], p);
        has[x] = !!tag[x];
        return;
    }
    doPush(x);
    int mid = (a + b) >> 1;
    if (c <= mid) doIns(x << 1, a, mid, c, p);
    else doIns(x << 1 | 1, mid + 1, b, c, p);
    has[x] = has[x << 1] | has[x << 1 | 1];
}

void doMulRange(int x, int a, int b, int c, int d, int p) {
    if (c <= a && b <= d) { doMul(x, p); return; }
    doPush(x);
    int mid = (a + b) >> 1;
    if (c <= mid) doMulRange(x << 1, a, mid, c, d, p);
    if (d > mid) doMulRange(x << 1 | 1, mid + 1, b, c, d, p);
}

int doAsk(int x, int a, int b, int c) {
    if (a == b) return tag[x];
    doPush(x);
    int mid = (a + b) >> 1;
    return c <= mid ? doAsk(x << 1, a, mid, c) : doAsk(x << 1 | 1, mid + 1, b, c);
}

void doCollect(int x, int a, int b, map<int, int> &f) {
    if (!has[x]) return;
    if (a == b) {
        if (tag[x])
            doAdd(f[a << 4 & 1048575], tag[x]);
        tag[x] = has[x] = 0;
        return;
    }
    doPush(x);
    int mid = (a + b) >> 1;
    doCollect(x << 1, a, mid, f), doCollect(x << 1 | 1, mid + 1, b, f);
    has[x] = 0;
}

inline void doSolve() {
    int i, j, cnt = 0, s = 0;
    for (i = 0; i < m; ++i) {
        tmp[i] = 0;
        for (j = sub[que[i].x]; ~j; j = sub[j])
            doAdd(tmp[i], doAsk(1, 0, M, j));
        edg[++cnt] = E(que[i].x, que[i].y);
        edg[++cnt] = E(rgt[que[i].x], P - que[i].y);
    }
    sort(edg + 1, edg + cnt + 1, cmp);
    for (i = 1; i <= cnt; ++i) {
        if (i > 1 && edg[i].x > edg[i - 1].x)
            doMulRange(1, 0, M, edg[i - 1].x, edg[i].x - 1, s);
        doAdd(s, edg[i].y);
    }
    for (i = 0; i < m; ++i)
        if (tmp[i])
            doIns(1, 0, M, que[i].x, 1LL * que[i].y * tmp[i] % P);
}

int main() {
    int i, j, x, y;
    scanf("%d%d", &n, &m);
    for (i = 2; i <= n; ++i)
        scanf("%d", &x), nxt[i] = gr[++x], gr[x] = i;
    for (int t = m; t--;) {
        scanf("%d%s", &x, buf);
        for (y = j = 0; j < 5; ++j)
            y = y << 4 | (buf[j] - '0' + 1);
        mp[x + 1][y] = P - 1;
    }
    sub[0] = -1, rgt[0] = M;
    for (int i = 1; i < M; ++i)
        for (int j = 0;; ++j)
            if (i >> (j * 4) & 15) {
                sub[i] = i - (((i >> (j * 4)) & 15) << (j * 4));
                rgt[i] = i + (1 << (j * 4));
                break;
            }
    build(1, 0, M);
    for (int i = n; i; --i) {
        for (int j = 1; j <= 10; ++j)
            mp[i][j << 16] = 1;
        for (it = mp[i].begin(); it != mp[i].end(); ++it)
            doIns(1, 0, M, it->first, it->second);
        for (int j = gr[i]; j; j = nxt[j]) {
            m = 0;
            for (it = mp[j].begin(); it != mp[j].end(); ++it)
                que[m++] = E(it->first, it->second);
            doSolve();
        }
        mp[i].clear();
        doCollect(1, 0, M, mp[i]);
    }
    for (ans = i = 1; i <= n; ++i)
        ans = ans * 10 % P;
    doAdd(ans, P - mp[1][0]);
    return printf("%d", ans), 0;
}
```

## 复杂度分析
- 时间复杂度：$O(N \times K \times \log M)$，其中 $N$ 为节点数，$K$ 为每个节点处理子节点时的模式数量（$K$ 受子节点模式数和完全展开步数影响，在数据范围内可接受），$M = 860000$ 为值域范围。线段树操作均为 $O(\log M)$。
- 空间复杂度：$O(N + MAXN + M)$，其中 $MAXN$ 为线段树节点总数，$M$ 为值域上限。
