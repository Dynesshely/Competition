# P1190 [NOIP 2010 普及组] 接水问题 题解

## 题目链接
- [洛谷 P1190](https://www.luogu.com.cn/problem/P1190)

## 题目大意
有 n 个同学按顺序接水，每个同学接水量为 w_i，有 m 个水龙头，每个龙头每秒供水量为 1。初始时 1~m 号同学各自占用一个龙头同时接水，一旦有人接完，下一个同学立即替换（瞬间完成）。问所有同学都接完水所需的总时间。

## 解题思路
用小根堆（优先队列）模拟 m 个水龙头的当前累计时间。

- 先将前 m 个同学的接水量入堆。
- 对于第 m+1 到第 n 个同学，每次从堆中取出当前累计时间最小的龙头（即最先空闲的龙头），将当前同学的接水量累加上去，再放回堆中。
- 最后，所有同学分配完毕后，堆中的最大值即为总时间（堆中最后一个弹出的元素）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, w[10005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        scanf("%d", &w[i]);
}

void process() {
    priority_queue<int, vector<int>, greater<int>> pq;
    for (int i = 0; i < m; i++)
        pq.push(w[i]);
    for (int i = m; i < n; i++) {
        int t = pq.top();
        pq.pop();
        pq.push(t + w[i]);
    }
    int ans = 0;
    while (!pq.empty()) {
        ans = pq.top();
        pq.pop();
    }
    printf("%d", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log m)，每次堆操作为 O(log m)。
- 空间复杂度：O(m)。
