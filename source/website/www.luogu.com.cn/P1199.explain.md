# P1199 [NOIP 2010 普及组] 三国游戏 题解

## 题目链接
- [洛谷 P1199](https://www.luogu.com.cn/problem/P1199)

## 题目大意
小涵与计算机轮流选择武将，小涵先手。每对武将之间有唯一的默契值。计算机的策略是：每次选择自由武将中与当前小涵已选武将组成默契值最高的那个武将，以破坏小涵的最强组合。双方选完所有武将后，各选一对默契值最高的组合进行比武，高者获胜。问小涵能否必胜，若能，输出她可获得的最大默契值。

## 解题思路
博弈分析 + 贪心。

关键在于：由于计算机总是破坏小涵当前最强的潜在组合，小涵永远无法拿到全局最大的默契值配对吧。但小涵可以拿到每个武将的"次大默契值"中的最大值。

- 对于每个武将 i，找出与其他所有武将的默契值中的最大值和次大值。
- 所有武将的次大值中的最大值即为小涵最终能获得的默契值。
- 可以证明小涵必胜（即总能获得比计算机高的默契值），因为计算机的破坏策略决定了它只能拿到各武将的最大默契值，而小涵总能通过先手优势拿到次大值中的最大值，且该值大于计算机能拿到的任何值（因为每对默契值唯一，次大值中的最大值大于任何最大值中的较小者）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, a[505][505], ans;

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        a[i][i] = -1;
        for (int j = i + 1; j < n; ++j) {
            scanf("%d", &a[i][j]);
            a[j][i] = a[i][j];
        }
    }
}

void process() {
    ans = 0;
    for (int i = 0; i < n; ++i) {
        int max1 = 0, max2 = 0;
        for (int j = 0; j < n; ++j) {
            if (a[i][j] > max1) {
                max2 = max1;
                max1 = a[i][j];
            } else if (a[i][j] > max2)
                max2 = a[i][j];
        }
        if (max2 > ans)
            ans = max2;
    }
}

void output() { printf("1\n%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N^2)。
- 空间复杂度：O(N^2)。
