# P1413 坚果保龄球 题解

## 题目链接
- [洛谷 P1413](https://www.luogu.com.cn/problem/P1413)

## 题目大意
$6 \times 60$ 的棋盘，僵尸从第 $60$ 列出现，每秒向左走 $1$ 步。玩家可以在某行第 $1$ 列放坚果，瞬间碾死该行所有僵尸。每只僵尸出现在第 $P$ 行、时间 $t$，求最少需要多少坚果才能消灭所有僵尸。

## 解题思路
每只僵尸出现的时间为 $t$，从第 $60$ 列走到第 $1$ 列需要 $59$ 秒，因此在时间 $[t, t+59]$ 内它都在棋盘上。若在某行第 $1$ 列放坚果，则该时刻所有在该行且存活时间范围内的僵尸都会被消灭。

问题转化为：对每一行，给定若干时间区间 $[t, t+59]$，求最少选多少个点（放坚果的时刻）使得每个区间至少包含一个点。这是经典的区间选点问题（活动安排问题），贪心策略：按右端点排序，每次选最早结束的区间的右端点。

对 $6$ 行分别做贪心即可。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Interval {
    int l, r;
};

int              n;
vector<Interval> row[7];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        int P, t;
        scanf("%d%d", &P, &t);
        row[P].push_back({t, t + 59});
    }
}

void process() {
    int ans = 0;
    for (int p = 1; p <= 6; ++p) {
        sort(row[p].begin(), row[p].end(), [](const Interval &a, const Interval &b) { return a.r < b.r; });
        int last = -1;
        for (const auto &itv : row[p])
            if (last < itv.l) {
                ++ans;
                last = itv.r;
            }
    }
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n)$，排序复杂度。
- 空间复杂度：$O(n)$。
