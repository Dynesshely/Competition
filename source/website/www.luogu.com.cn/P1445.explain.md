# P1445 [Violet] 樱花 题解

## 题目链接

- [洛谷 P1445](https://www.luogu.com.cn/problem/P1445)

## 题目大意

求方程 $\dfrac{1}{x} + \dfrac{1}{y} = \dfrac{1}{n!}$ 的正整数解 $(x, y)$ 的组数，答案对 $10^9+7$ 取模。数据范围：$1 \le n \le 10^6$。

## 解题思路

对原式进行代数变形：

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n!} \implies \frac{x+y}{xy} = \frac{1}{n!}$$

交叉相乘得 $n!(x+y) = xy$，移项整理：

$$xy - n!x - n!y = 0 \implies (x - n!)(y - n!) = (n!)^2$$

设 $a = x - n!$，$b = y - n!$，则 $a \cdot b = (n!)^2$。因此，每一对正整数因子 $(a, b)$ 对应一组正整数解 $(x, y)$。

所以方程的正整数解组数等于 $(n!)^2$ 的正因子个数。设 $n!$ 的质因数分解为：

$$n! = \prod p_i^{e_i}$$

则 $(n!)^2 = \prod p_i^{2e_i}$，其正因子个数为 $\prod (2e_i + 1)$。

对 $n!$ 做质因数分解，只需筛出 $n$ 以内的所有质数 $p$，对每个质数用勒让德公式计算其指数：

$$e = \sum_{k=1}^{\infty} \left\lfloor \frac{n}{p^k} \right\rfloor$$

最后累乘 $(2e+1) \bmod (10^9+7)$ 即可。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD  = 1000000007;
const int MAXN = 1000000;

int              n;
vector<int>      primes;
bitset<MAXN + 1> isPrime;

void input() { scanf("%d", &n); }

void process() {
    isPrime.set();
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
            for (long long j = 1LL * i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
}

void output() {
    long long ans = 1;
    for (int p : primes) {
        long long e = 0;
        for (long long pk = p; pk <= n; pk *= p)
            e += n / pk;
        ans = ans * (2 * e + 1) % MOD;
    }
    printf("%lld\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：$O(n \log \log n)$，筛法 $O(n \log \log n)$，计算指数部分 $O(\sum_{p \le n} \log_p n) = O(n / \log n \cdot \log n) = O(n)$。
- 空间复杂度：$O(n)$，用于存储筛法标记数组。
