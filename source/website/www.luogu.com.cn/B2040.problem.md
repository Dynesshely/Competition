## 题目描述

判断一个正整数是否是两位数（即大于等于 $10$ 且小于等于 $99$）。

## 输入格式

一个正整数，不超过 $1000$。

## 输出格式

一行。若该正整数是两位数，输出 $1$，否则输出 $0$。

## 样例 #1

### 样例输入

```
54
```

### 样例输出

```
1
```
