# P1322 洛谷 logo语言 题解

## 题目链接
- [洛谷 P1322](https://www.luogu.com.cn/problem/P1322)

## 题目大意
给定一行 Logo 语言命令，海龟在一条直线上爬行。只有三个语句：
- `FD X`：前进 X 个单位
- `BK X`：后退 X 个单位
- `REPEAT N[...]`：重复执行方括号内的命令 N 次

要求输出海龟最终离开原点的距离（即绝对位移）。

## 解题思路
这是一个递归下降解析的问题。将命令字符串视为一个流，通过递归函数 `parse()` 进行解析：
- 遇到 `FD` 则读取数字，累加到当前位置
- 遇到 `BK` 则读取数字，从当前位置减去
- 遇到 `REPEAT` 则读取重复次数，递归解析方括号内的子命令序列，将子命令的位移乘以重复次数后累加
- 遇到 `]` 或字符串末尾则返回

由于命令可以嵌套（`REPEAT` 内部可以包含另一个 `REPEAT`），使用递归处理最自然。最终答案为位移的绝对值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[1000005];
int  n;

int parse(int &i) {
    int pos = 0;
    while (i < n) {
        while (i < n && s[i] == ' ')
            ++i;
        if (i >= n || s[i] == ']')
            break;
        if (s[i] == 'F') {
            i += 2;
            while (i < n && s[i] == ' ')
                ++i;
            int num = 0;
            while (i < n && isdigit(s[i])) {
                num = num * 10 + (s[i] - '0');
                ++i;
            }
            pos += num;
        } else if (s[i] == 'B') {
            i += 2;
            while (i < n && s[i] == ' ')
                ++i;
            int num = 0;
            while (i < n && isdigit(s[i])) {
                num = num * 10 + (s[i] - '0');
                ++i;
            }
            pos -= num;
        } else {
            i += 6;
            while (i < n && s[i] == ' ')
                ++i;
            int times = 0;
            while (i < n && isdigit(s[i])) {
                times = times * 10 + (s[i] - '0');
                ++i;
            }
            while (i < n && s[i] == ' ')
                ++i;
            if (i < n)
                ++i;
            int inner = parse(i);
            while (i < n && s[i] == ' ')
                ++i;
            if (i < n)
                ++i;
            pos += times * inner;
        }
    }
    return pos;
}

int main() {
    fgets(s, sizeof(s), stdin);
    n = strlen(s);
    while (n > 0 && (s[n - 1] == '\n' || s[n - 1] == '\r'))
        s[--n] = '\0';
    int i   = 0;
    int ans = parse(i);
    printf("%d\n", abs(ans));
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为字符串长度，每个字符仅被扫描一次
- 空间复杂度：O(n)，递归栈深度取决于嵌套层数
