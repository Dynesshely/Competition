# P1021 [NOIP 1999 提高组] 邮票面值设计 — 题解

## 题目大意

给定最多可贴邮票数 $N$ 和可设计面值数 $K$（$N + K \le 15$），设计 $K$ 种面值，使得从 $1$ 开始连续能表示的邮资值最大。输出面值方案和最大连续值。

## 解题思路

### DFS 搜索面值组合

面值集合必须包含 $1$（否则无法表示 $1$），其余 $K-1$ 个面值通过 DFS 枚举。

**搜索剪枝**：设当前已知最大连续值为 $\text{cur\_max}$，下一个面值必然在 $[\text{prev} + 1,\ \text{cur\_max} + 1]$ 范围内。因为如果要表示 $\text{cur\_max} + 1$，要么用现有面值组合（但已超出上限），要么需要一个不超过 $\text{cur\_max} + 1$ 的新面值。

### DP 验证连续覆盖范围

对于一组面值，用动态规划计算能从 $1$ 开始连续覆盖到的最大值：

- `dp[v]` = 表示邮资 $v$ 所需的最少邮票张数
- 初始：`dp[0] = 0`，其余为无穷大
- 转移：`dp[v] = min(dp[v - d] + 1)`，对于每个面值 $d \le v$
- 当首次出现 `dp[v] > N` 时停止，$\text{max\_cover} = v - 1$

这本质上是**无限背包**（每种面值无限张），每张邮票重量为面值 $d$，价值为 $1$（张数），背包容量为邮资值，要求总张数不超过 $N$。

### 时间复杂度

- DFS 搜索空间：$K \le 14$（$N \ge 1$），每个面值有至多 $\text{cur\_max} + 1$ 个候选值
- DP 验证：$O(\text{max\_cover} \times K)$，max_cover 通常不超过几百
- 对于 $N+K \le 15$，总搜索量在可行范围内

### 空间复杂度

- DP 数组 $O(\text{max\_cover})$
