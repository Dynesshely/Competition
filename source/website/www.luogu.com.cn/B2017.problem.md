## 题目描述

输入一个除空格以外的可见字符，输出其 ASCII 码。

## 输入格式

一个除空格以外的可见字符。

## 输出格式

一个十进制整数，即该字符的 ASCII 码。

## 样例 #1

### 样例输入

```
A
```

### 样例输出

```
65
```
