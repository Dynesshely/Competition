# B2034 计算 2 的幂 题解

## 题目链接
- [洛谷 B2034](https://www.luogu.com.cn/problem/B2034)

## 题目大意
给定非负整数 $n$（$0 \le n < 31$），计算并输出 $2^n$ 的值。

## 解题思路
直接利用左移运算符 `1 << n` 计算 $2^n$。左移一位等价于乘以 2，因此 `1 << n` 即为 $2^n$。由于 $n < 31$，结果在 32 位有符号整数范围内，不会溢出。相比 `pow()` 函数，位运算避免了浮点数精度问题且效率更高。

## 代码实现
采用 `input()` / `process()` / `output()` 三段式结构。核心计算在 `output()` 中完成：`printf("%d\n", 1 << n)`，process 为空。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
