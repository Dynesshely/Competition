# P1572 计算分数 题解

## 题目链接

- [洛谷 P1572](https://www.luogu.com.cn/problem/P1572)

## 题目大意

给定一个分数加减法表达式（如 `2/1+1/3-1/4`），计算结果并化为最简分数或整数。

## 解题思路

模拟分数加减法过程，逐个读取分数并累加。

利用 `scanf("%d/%d")` 格式串可直接读取 `a/b` 形式的分数，且会自动处理正负号（如 `+1/3` 读作 `1/3`，`-1/4` 读作 `-1/4`）。

**计算过程**：对当前累加结果 $a/b$ 和新分数 $c/d$：
1. 计算 $m = \gcd(b, d)$。
2. 通分：$a = a \times \frac{d}{m}$，$c = c \times \frac{b}{m}$，$b = b \times \frac{d}{m}$。
3. 合并分子：$a = a + c$。
4. 化简：$m = \gcd(a, b)$，$a /= m$，$b /= m$。

**输出处理**：
- 若分母为负，将负号转移到分子上。
- 若分母为 $1$，只输出分子（整数形式）。
- 否则输出 `a/b` 格式。

## 代码实现

- `gcd(int x, int y)`：递归实现欧几里得算法求最大公约数。
- 先用 `scanf("%d/%d", &a, &b)` 读入第一个分数作为初始值。
- `while (scanf("%d/%d", &c, &d) != EOF)` 依次读入后续分数，`scanf` 自动消费前面的 `+`/`-` 符号（符号会作为整数的一部分读入 `c`）。
- 每次循环中：
  - 用 `gcd(b, d)` 求分母的最大公约数。
  - 通分后分子相加，分母相乘除以 gcd。
  - 再对结果分子分母同时除以 gcd 化简。
- 最后处理分母为负的情况，并判断是否输出整数。

## 复杂度分析

- 时间复杂度：$O(n \log V)$，其中 $n$ 为分数个数，每次通分和化简均调用 $\gcd$，复杂度 $O(\log V)$。
- 空间复杂度：$O(1)$，仅使用常数个变量。
