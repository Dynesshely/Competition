# P1373 小a和uim之大逃离 题解

## 题目链接
- [洛谷 P1373](https://www.luogu.com.cn/problem/P1373)

## 题目大意
$n \times m$ 网格，每格有 $0 \sim k$ 的魔液。小 a 和 uim 轮流从任意格子开始，每次向右或向下走一步，小 a 先吸收魔液，uim 后吸收，交替进行，最后一步必须由 uim 吸收。魔瓶容量为 $k$（达到 $k+1$ 清空为 $0$，$k+2$ 为 $1$，依此类推）。求两人最终魔液相等（模 $k+1$ 意义下）的路径方案数模 $10^9+7$。

## 解题思路
动态规划，状态表示为到达 $(i, j)$ 时当前两人魔液差值对 $k+1$ 取模的结果，以及当前轮到谁吸收。

设 $grid[i][j]$ 为魔液值，$modVal = k + 1$。用滚动数组 `prevRow[j][d][0/1]` 和 `curRow[j][d][0/1]` 表示上一行和当前行的 DP 值，其中最后一维 $0$ 表示当前轮是小 a 吸收（小 a 贡献差值增加），$1$ 表示当前轮是 uim 吸收（uim 贡献差值减少）。

转移：
- 每个格子可作为起点（小 a 先取）。
- 从上方或左方转移：若上一步是小 a，则当前步 uim 吸收，差值 $(d - val) \bmod modVal$；若上一步是 uim，则当前步小 a 吸收，差值 $(d + val) \bmod modVal$。
- 最终答案累加所有 `curRow[j][0][1]`（uim 取完最后一步且差值归零）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD = 1000000007;

int n, m, k, modVal, ans;
int grid[805][805];
int prevRow[805][16][2];
int curRow[805][16][2];

void input() {
    scanf("%d%d%d", &n, &m, &k);
    modVal = k + 1;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            scanf("%d", &grid[i][j]);
}

void process() {
    ans = 0;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++)
            for (int d = 0; d < modVal; d++)
                curRow[j][d][0] = curRow[j][d][1] = 0;

        for (int j = 1; j <= m; j++) {
            int val = grid[i][j];
            curRow[j][val][0] = (curRow[j][val][0] + 1) % MOD;

            for (int d = 0; d < modVal; d++) {
                int f0 = prevRow[j][d][0];
                if (f0) {
                    int nd = (d - val + modVal) % modVal;
                    curRow[j][nd][1] = (curRow[j][nd][1] + f0) % MOD;
                }
                int f1 = prevRow[j][d][1];
                if (f1) {
                    int nd = (d + val) % modVal;
                    curRow[j][nd][0] = (curRow[j][nd][0] + f1) % MOD;
                }
            }

            if (j > 1) {
                for (int d = 0; d < modVal; d++) {
                    int f0 = curRow[j - 1][d][0];
                    if (f0) {
                        int nd = (d - val + modVal) % modVal;
                        curRow[j][nd][1] = (curRow[j][nd][1] + f0) % MOD;
                    }
                    int f1 = curRow[j - 1][d][1];
                    if (f1) {
                        int nd = (d + val) % modVal;
                        curRow[j][nd][0] = (curRow[j][nd][0] + f1) % MOD;
                    }
                }
            }

            ans = (ans + curRow[j][0][1]) % MOD;
        }

        for (int j = 1; j <= m; j++)
            for (int d = 0; d < modVal; d++) {
                prevRow[j][d][0] = curRow[j][d][0];
                prevRow[j][d][1] = curRow[j][d][1];
            }
    }
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(nmk)$，其中 $k \le 15$
- 空间复杂度：$O(mk)$，使用滚动数组
