# P1282 [CEOI 1997] 多米诺骨牌 题解

## 题目链接
- [洛谷 P1282](https://www.luogu.com.cn/problem/P1282)

## 题目大意
有 n 个多米诺骨牌，每个骨牌上下各有 1~6 个点，可以旋转 180 度交换上下点数。求最小旋转次数，使得上下两行点数之差的绝对值最小。

## 解题思路
DP 求差值。每个骨牌有两种摆放方式：正放贡献差值 d = a[i] - b[i]，翻转贡献差值 -d。dp[s] 表示达到差值 s 所需的最小旋转次数，用滚动数组优化。遍历差值从 0 开始向两侧扩展，找到第一个可达的差值即为最优，输出对应的最小旋转次数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int OFFSET = 5000;
const int INF    = 0x3f3f3f3f;

int n, a[1005], b[1005];
int dp[10005], ndp[10005];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i)
        scanf("%d%d", &a[i], &b[i]);
}

void process() {
    memset(dp, 0x3f, sizeof(dp));
    dp[OFFSET] = 0;
    for (int i = 0; i < n; ++i) {
        int d = a[i] - b[i];
        memset(ndp, 0x3f, sizeof(ndp));
        for (int s = -OFFSET; s <= OFFSET; ++s) {
            int idx = s + OFFSET;
            if (dp[idx] == INF)
                continue;
            int ns1 = s + d, ns2 = s - d;
            if (ns1 >= -OFFSET && ns1 <= OFFSET)
                ndp[ns1 + OFFSET] = min(ndp[ns1 + OFFSET], dp[idx]);
            if (ns2 >= -OFFSET && ns2 <= OFFSET)
                ndp[ns2 + OFFSET] = min(ndp[ns2 + OFFSET], dp[idx] + 1);
        }
        memcpy(dp, ndp, sizeof(dp));
    }
}

void output() {
    for (int diff = 0; diff <= OFFSET; ++diff) {
        int ans = INF;
        if (dp[OFFSET + diff] < ans)
            ans = dp[OFFSET + diff];
        if (dp[OFFSET - diff] < ans)
            ans = dp[OFFSET - diff];
        if (ans < INF) {
            printf("%d\n", ans);
            return;
        }
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * range)，其中 range 为差值范围。
- 空间复杂度：O(range)。
