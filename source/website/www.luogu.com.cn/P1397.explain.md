# P1397 [NOI2013] 矩阵游戏 题解

## 题目链接
- [洛谷 P1397](https://www.luogu.com.cn/problem/P1397)

## 题目大意
给定递推式：
- F[1][1] = 1
- F[i][j] = a × F[i][j-1] + b（j ≠ 1）
- F[i][1] = c × F[i-1][m] + d（i ≠ 1）

求 F[n][m] mod 10^9+7。其中 n, m 可读入为长度 ≤ 10^6 的大整数（即 n, m 本身可以非常大）。

## 解题思路
首先分析行内递推：F[i][j] 可以看作关于 j 的线性变换，即 F[i][j] = A × F[i][j-1] + B 的形式。具体地：
- 若 a = 1：F[i][j] = F[i][1] + (j-1) × b
- 若 a ≠ 1：F[i][j] = a^(j-1) × F[i][1] + b × (a^(j-1)-1) / (a-1)

可将一行的递推浓缩为一个变换 (A, B)，整行看作一次变换：应用于 F[i][1] 得到 F[i][m]。

同理，行间递推 F[i][1] = c × F[i-1][m] + d 可以看作将上一行的结果经过另一个变换得到当前行第一列。

合并两个变换即可将整个矩阵的递推压缩为从 F[1][1] 到 F[n][m] 的复合变换。

由于 n, m 极大（可达 10^6 位），需要用费马小定理对指数降幂：a^(p-1) ≡ 1 (mod p)，因此指数可以对 MOD-1 取模。

分类讨论 a=1 与 a≠1 两种情况分别推导公式，注意处理逆元。

## 代码实现
将 n, m 按字符串读入，分别对 MOD 和 MOD-1 取模（减 1 处理）。根据 a 是否等于 1 分情况计算行变换和列变换，最终求得 F[n][m]。

## 复杂度分析
- 时间复杂度：O(len(n) + len(m))
- 空间复杂度：O(len(n) + len(m))
