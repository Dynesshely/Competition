# P1111 [洛谷] 修复公路 题解

## 题目链接

- [洛谷 P1111](https://www.luogu.com.cn/problem/P1111)

## 题目大意

N 个村庄，M 条双向公路，每条公路有一个修复完成时间 t。求最早在哪个时刻，任意两个村庄之间可以通车（即整个图连通）。若全部公路修复完毕仍无法连通，输出 -1。

## 解题思路

最小生成树思想（Kruskal）。将所有公路按修复时间升序排序，使用并查集维护连通性。依次加入边，当连通分量个数减为 1 时，最后加入的边的修路时间即为答案。若所有边加入后仍不连通，输出 -1。

## 代码实现

- 读入 N, M 和所有边（存结构体 u, v, t）
- 按 t 排序
- 初始化并查集，cnt = n
- 遍历边，若两端不在同一集合则合并，cnt--，更新 ans = e[i].t
- 若 cnt == 1，输出 ans；否则输出 -1

## 复杂度分析

- 时间复杂度：O(M log M)
- 空间复杂度：O(M)
