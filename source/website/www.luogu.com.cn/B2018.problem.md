## 题目描述

输入一个 ASCII 码，输出对应的字符。

## 输入格式

一个整数，即字符的 ASCII 码，保证存在对应的可见字符。

## 输出格式

一行，包含相应的字符。

## 样例 #1

### 样例输入

```
65
```

### 样例输出

```
A
```

## 提示

保证所有数据 $<128$ 并且 $>0$。
