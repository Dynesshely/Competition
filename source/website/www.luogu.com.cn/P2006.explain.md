# P2006 赵神牛的游戏 题解

## 题目链接
- [洛谷 P2006](https://www.luogu.com.cn/problem/P2006)

## 题目大意
给定 k 点法力值和 m 个技能（每个技能消耗 a_i 法力、造成 b_i 伤害），赵神牛一局只能选一个技能且可重复释放，求能击败体力值为 n 的 boss 的所有技能编号。

## 解题思路
对每个技能单独判断：若 a_i = 0 且 b_i > 0 则无消耗无限放，必定能击败；否则计算最多释放次数 k / a_i，再乘以单次伤害 b_i，若结果 >= n 则该技能可行。最后按升序输出所有可行技能编号或 -1。

## 代码实现
使用 input() / process() / output() 三段式。process 中遍历 m 个技能，判断条件用 long long 避免乘法溢出，可行编号存入 vector，最后统一输出。

## 复杂度分析
- 时间复杂度: O(m)
- 空间复杂度: O(m)
