# P1437 [HNOI2004] 敲砖块 题解

## 题目链接
- [洛谷 P1437](https://www.luogu.com.cn/problem/P1437)

## 题目大意
给定一个 $n$ 层的三角形砖块阵列，每层砖块依次减少一块。敲掉一块砖可得到对应分值，但敲掉第 $i$ 层的第 $j$ 块砖必须先敲掉它左上和右上两块的砖。最多敲 $m$ 块砖，求能获得的最大总分。$n \le 50$，$m \le n(n+1)/2$。

## 解题思路
将三角形顺时针旋转 $45^\circ$，使依赖关系变为列上的前缀依赖，转化为分组背包问题。

把砖块按列处理，每列从上到下敲砖时，下一列的对应前缀也必须被敲掉。

定义 $dp[c][t][k]$ 表示处理到第 $c$ 列，该列敲了 $t$ 块砖，总共敲了 $k$ 块砖时的最大分数。

通过预处理列前缀和 $sum[c][t]$ 表示第 $c$ 列敲前 $t$ 块砖的分数和。转移时枚举当前列敲砖数 $t$ 和之前列的敲砖数，利用 $sm[t][k]$ 数组滚动优化前一列的最优值。

最终答案为所有 $dp$ 值中的最大值。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 1e9;
int       n, m;
int       a[55][55];
int       sum[55][55];
int       dp[2][55][1300];
int       sm[55][1300];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n - i + 1; j++)
            scanf("%d", &a[i][j]);
}

void process() {
    for (int j = 1; j <= n; j++) {
        int c = n - j + 1;
        for (int i = 1; i <= n - j + 1; i++)
            sum[c][i] = sum[c][i - 1] + a[i][j];
    }
    for (int i = 0; i < 2; i++)
        for (int j = 0; j <= n; j++)
            for (int k = 0; k <= m; k++)
                dp[i][j][k] = -INF;
    dp[0][0][0] = 0;
    for (int c = 1; c <= n; c++) {
        int cur = c & 1, pre = cur ^ 1;
        for (int k = 0; k <= m; k++)
            for (int t = 0; t <= n; t++)
                sm[t][k] = -INF;
        for (int k = 0; k <= m; k++)
            for (int t = c - 1; t >= 0; t--)
                sm[t][k] = max(dp[pre][t][k], sm[t + 1][k]);
        for (int k = 0; k <= m; k++)
            for (int t = 0; t <= c && t <= k; t++) {
                int val  = sum[c][t];
                int need = t == 0 ? 0 : t - 1;
                if (sm[need][k - t] > -INF / 2)
                    dp[cur][t][k] = val + sm[need][k - t];
            }
    }
}

void output() {
    int ans = 0;
    for (int k = 0; k <= m; k++)
        for (int t = 0; t <= n; t++)
            ans = max(ans, dp[n & 1][t][k]);
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2 \cdot m)$，$n \le 50$，$m \le 1275$，可接受。
- 空间复杂度：$O(n \cdot m)$ 使用滚动数组优化至 $O(n \cdot m)$。
