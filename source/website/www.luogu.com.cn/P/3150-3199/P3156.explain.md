# P3156 [洛谷] 【深基15.例1】询问学号 题解

## 题目链接
- [洛谷 P3156](https://www.luogu.com.cn/problem/P3156)

## 题目大意
有 n 个学生按顺序进入教室，给出每个学生的学号。m 次询问第 i 个进入教室的学生的学号。

## 解题思路
用数组按顺序存储学号，每次询问直接输出对应下标的元素。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, m;
vector<int> a;

int main() {
    scanf("%d%d", &n, &m);
    a.resize(n);
    for (int i = 0; i < n; ++i)
        scanf("%d", &a[i]);
    while (m--) {
        int q;
        scanf("%d", &q);
        printf("%d\n", a[q - 1]);
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + m)。
- 空间复杂度：O(n)。
