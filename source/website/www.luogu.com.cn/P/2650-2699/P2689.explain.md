# P2689 [洛谷] 东南西北 题解

## 题目链接
- [洛谷 P2689](https://www.luogu.com.cn/problem/P2689)

## 题目大意
给定起点(x1,y1)、终点(x2,y2)和T个时刻的顺风方向（E/S/W/N），每个时刻可选择顺风移动1格或不移动。求到达终点的最少移动步数，无法到达输出-1。

## 解题思路
计算x方向所需步数dx=x2-x1，y方向所需步数dy=y2-y1。统计各方向风的次数。若任意方向所需的步数超过该方向风的次数，则无法到达；否则最少步数为|dx|+|dy|。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int sx, sy, ex, ey, t, ans;
int cnt[4];

void input() {
    scanf("%d%d%d%d%d", &sx, &sy, &ex, &ey, &t);
    for (int i = 0; i < t; ++i) {
        char dir[2];
        scanf("%s", dir);
        if (dir[0] == 'E')
            ++cnt[0];
        else if (dir[0] == 'S')
            ++cnt[1];
        else if (dir[0] == 'W')
            ++cnt[2];
        else
            ++cnt[3];
    }
}

void process() {
    int dx = ex - sx, dy = ey - sy;
    if ((dx > 0 && cnt[0] < dx) || (dx < 0 && cnt[2] < -dx) || (dy > 0 && cnt[3] < dy) || (dy < 0 && cnt[1] < -dy))
        ans = -1;
    else
        ans = abs(dx) + abs(dy);
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(T)
- 空间复杂度：O(1)
