# P2670 [洛谷] 扫雷游戏 题解

## 题目链接
- [洛谷 P2670](https://www.luogu.com.cn/problem/P2670)

## 题目大意
给定n行m列的扫雷地图，*表示地雷，?表示非地雷格。计算每个非地雷格周围的8个方向中地雷的数量。

## 解题思路
遍历每个格子，遇到?时统计其8个方向上的*个数，将?替换为数字字符。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, m;
char g[105][105];
int  dx[] = {-1, -1, -1, 0, 0, 1, 1, 1};
int  dy[] = {-1, 0, 1, -1, 1, -1, 0, 1};

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; ++i)
        scanf("%s", g[i]);
}

void process() {
    for (int i = 0; i < n; ++i)
        for (int j = 0; j < m; ++j)
            if (g[i][j] == '?') {
                int cnt = 0;
                for (int k = 0; k < 8; ++k) {
                    int ni = i + dx[k], nj = j + dy[k];
                    if (ni >= 0 && ni < n && nj >= 0 && nj < m && g[ni][nj] == '*')
                        ++cnt;
                }
                g[i][j] = '0' + cnt;
            }
}

void output() {
    for (int i = 0; i < n; ++i)
        printf("%s\n", g[i]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(nm)
- 空间复杂度：O(nm)
