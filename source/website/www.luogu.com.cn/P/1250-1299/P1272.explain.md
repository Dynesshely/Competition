# P1272 [USACO02FEB] 重建道路 题解

## 题目链接
- [洛谷 P1272](https://www.luogu.com.cn/problem/P1272)

## 题目大意
给定一棵 N 个节点的树，问最少删除多少条边，使得剩余森林中存在一个恰好含 P 个节点的连通块。

## 解题思路
树形 DP。dp[u][k] 表示在以 u 为根的子树中，分离出一个包含 u 且恰好有 k 个节点的连通块所需删除的最少边数。
- 初始化 dp[u][1] = 0（只保留 u 自身，不需要删边）。
- DFS 过程中合并子树：当前已处理了 cur 个节点，尝试从新子树中取 s 个节点加入，转移时需要考虑删除父子边（代价 +1）或保留父子边（代价 + dp[v][s]）。
- 最终答案取 min(dp[1][P], min_{i>=2} dp[i][P] + 1)，因为非根节点还需切断与父节点的边。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 0x3f3f3f3f;

int         n, p;
vector<int> g[155];
int         dp[155][155];
int         sz[155];

void dfs(int u, int par) {
    dp[u][1] = 0;
    int cur  = 1;
    for (int v : g[u]) {
        if (v == par)
            continue;
        dfs(v, u);
        int top = min(cur, p);
        for (int k = top; k >= 1; --k) {
            int old  = dp[u][k];
            dp[u][k] = old + 1;
            int lim  = min(sz[v], p - k);
            for (int s = 1; s <= lim; ++s)
                dp[u][k + s] = min(dp[u][k + s], old + dp[v][s]);
        }
        cur += sz[v];
    }
    sz[u] = cur;
}

void input() {
    scanf("%d%d", &n, &p);
    for (int i = 1; i < n; ++i) {
        int u, v;
        scanf("%d%d", &u, &v);
        g[u].push_back(v);
        g[v].push_back(u);
    }
}

void process() {
    memset(dp, 0x3f, sizeof(dp));
    dfs(1, 0);
}

void output() {
    int ans = dp[1][p];
    for (int i = 2; i <= n; ++i)
        if (dp[i][p] + 1 < ans)
            ans = dp[i][p] + 1;
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * P^2)。
- 空间复杂度：O(N * P)。
