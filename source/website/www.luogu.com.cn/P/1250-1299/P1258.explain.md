# P1258 [洛谷] 小车问题 题解

## 题目大意
甲、乙两人同时从 A 地出发要尽快同时赶到 B 地。有一辆小车（除驾驶员外只能带一人），车速 b 大于步行速度 a。求两人同时到达 B 地的最短时间。

## 解题思路
这是一个经典的数学问题。最优策略为：小车先载一人前进一段距离后放下，让其步行到终点；小车返回接另一人，然后载着另一人赶到终点，使得两人同时到达。

设总距离为 s，则最优时间公式推导如下：
- 设小车载第一人行驶距离为 x，之后返回接第二人。
- 第一人乘车到某点后步行剩余距离；小车返回与第二人相遇，再载第二人到终点。
- 根据两人同时到达的条件，可以解出时间 t 的表达式。

最终公式为：
- 当 b = 0（车不动）时，t = s / a（仅步行）。
- 一般情况下，t = s * (a + 3b) / (b * (b + 3a))。
