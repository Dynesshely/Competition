# P1266 [洛谷] 速度限制 题解

## 题目链接
- [洛谷 P1266](https://www.luogu.com.cn/problem/P1266)

## 题目大意
给定 N 个路口和 M 条有向道路，每条道路有长度 L 和限速 V（V=0 表示限速未知，沿用进入该路前的速度）。初始在路口 0 速度为 70。求从 0 到 D 的最快路线（时间最短），输出经过的路口序列。

## 解题思路
对状态 (路口, 当前速度) 进行 Dijkstra 最短路：

1. 状态定义：dist[u][s] 表示以速度 s 到达路口 u 的最短时间。
2. 初始状态：dist[0][70] = 0。
3. 转移：从 (u, s) 沿道路 e = (u, v, vLimit, len) 到 v：
   - 若 vLimit > 0（有限速），则新速度 ns = vLimit，耗时 t = len / vLimit。
   - 若 vLimit = 0（无限速），则新速度 ns = s，耗时 t = len / s。
   - 新时间 nd = dist[u][s] + t，若 nd < dist[v][ns] 则更新并入队。
4. 最终从所有到达 D 的速度中取最小时间，逆向回溯得到路径。

使用 priority_queue 进行 Dijkstra，prevNode 数组记录每个状态的前驱便于输出路径。
