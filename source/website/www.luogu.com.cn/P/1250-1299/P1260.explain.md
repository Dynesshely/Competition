# P1260 [洛谷] 工程规划 题解

## 题目大意
给定 n 个子任务，以及 m 个形如 T_i - T_j ≤ b 的约束条件（T_i 表示任务 i 的起始时间，均为非负整数）。求一组合法的起始时间，使得所有约束成立且至少有一个任务起始时间为 0。若无解则输出 NO SOLUTION。

## 解题思路
这是一个**差分约束系统**问题。将不等式 T_i - T_j ≤ b 变形为 T_i ≤ T_j + b，这与最短路径中的三角不等式形式一致，因此可以转化为图论模型：

- 从节点 j 到节点 i 连一条权值为 b 的有向边。
- 添加一个超级源点 0，向所有节点连权值为 0 的边（保证所有节点可达）。
- 使用 Bellman-Ford 算法求最短路径。若存在负环则无解；否则 dist[i] 即为一组可行解。
- 最后将所有 dist 减去最小值（使最小值为 0）即得到满足至少一个任务为 0 的解。
