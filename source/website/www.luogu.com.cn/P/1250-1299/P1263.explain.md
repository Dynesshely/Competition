# P1263 [洛谷] Royal guards 题解

## 题目链接
- [洛谷 P1263](https://www.luogu.com.cn/problem/P1263)

## 题目大意
给定一个 m x n 的网格，0 表示空地、1 表示陷阱、2 表示墙。守卫可以放在空地上，但同一行或同一列的两个守卫之间若无墙阻挡则会互相攻击。求最多能放置多少个守卫，并输出方案。

## 解题思路
将网格的行和列按墙分割成连续的段，转化为**二分图最大匹配**问题：

1. **行段编号**：对每行，遇到墙就分段，为每个连续的空地区域分配一个行段编号（hId）。陷阱（1）视为不可放守卫但不断开段。
2. **列段编号**：同样对每列，按墙分段，为每个连续空地区域分配列段编号（vId）。
3. **建图**：对于每个空地格子 (i,j)，其行段编号 hId[i][j] 向列段编号 vId[i][j] 连边。
4. **最大匹配**：运行 Hopcroft-Karp（BFS + DFS 分层增广）求最大匹配，匹配数即为可放置的最大守卫数。
5. **输出方案**：若某空地的行段与列段匹配，则该位置放置守卫。
