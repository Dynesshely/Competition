# P1270 "访问"美术馆 题解

## 题目链接
- [洛谷 P1270](https://www.luogu.com.cn/problem/P1270)

## 题目大意
给定一棵树，根节点为起点，叶子节点是展览室（有一定数量的画），内部节点是走廊分叉点。通过每条走廊需要一定时间，偷一幅画需要 5 秒。给定警察赶到的时间，求在回到起点并留至少 1 秒逃跑的前提下，最多能偷多少幅画。

## 解题思路
树形 DP。递归建树后，对每个节点计算 dp[k] 表示在该节点子树中偷 k 幅画所需的最小时间：
- 叶子节点（展览室）：可以偷 0 到 cnt 幅画，代价为 2*t（来回）+ 5*k。
- 内部节点：先递归处理左右子树，然后用背包合并两个子树的 dp 数组，最后加上经过该走廊的来回时间 2*t。
- 最终答案取根节点 dp 中代价不超过 policeTime-1 的最大 k。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int   t, cnt;
    Node *l, *r;
    Node(int _t, int _c)
        : t(_t)
        , cnt(_c)
        , l(nullptr)
        , r(nullptr) {}
};

Node *read() {
    int ti, cn;
    if (scanf("%d%d", &ti, &cn) != 2)
        return nullptr;
    Node *nd = new Node(ti, cn);
    if (cn == 0) {
        nd->l = read();
        nd->r = read();
    }
    return nd;
}

const int INF = 1e9;
int       budget;

vector<int> dfs(Node *nd) {
    if (nd->cnt > 0) {
        int         maxK = nd->cnt;
        vector<int> dp(maxK + 1, INF);
        dp[0]    = 0;
        int base = 2 * nd->t;
        for (int k = 1; k <= maxK; k++) {
            int val = base + 5 * k;
            if (val <= budget)
                dp[k] = val;
        }
        return dp;
    }
    vector<int> ldp  = dfs(nd->l);
    vector<int> rdp  = dfs(nd->r);
    int         maxK = (int)ldp.size() - 1 + (int)rdp.size() - 1;
    vector<int> dp(maxK + 1, INF);
    dp[0] = 0;
    for (int k1 = 0; k1 < (int)ldp.size(); k1++) {
        if (ldp[k1] >= INF)
            continue;
        for (int k2 = 0; k2 < (int)rdp.size(); k2++) {
            if (rdp[k2] >= INF)
                continue;
            int k   = k1 + k2;
            int val = ldp[k1] + rdp[k2];
            if (val < dp[k])
                dp[k] = val;
        }
    }
    int         add = 2 * nd->t;
    vector<int> res(maxK + 1, INF);
    res[0] = 0;
    for (int k = 1; k <= maxK; k++) {
        if (dp[k] < INF) {
            int val = dp[k] + add;
            if (val <= budget)
                res[k] = val;
        }
    }
    return res;
}

int main() {
    int policeTime;
    scanf("%d", &policeTime);
    budget     = policeTime - 1;
    Node *root = read();
    if (root == nullptr) {
        printf("0\n");
        return 0;
    }
    vector<int> dp  = dfs(root);
    int         ans = 0;
    for (int k = 0; k < (int)dp.size(); k++)
        if (dp[k] <= budget)
            ans = k;
    printf("%d\n", ans);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * P^2)，其中 N 为节点数，P 为总画数。
- 空间复杂度：O(N * P)。
