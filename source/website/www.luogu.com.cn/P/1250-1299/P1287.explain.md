# P1287 盒子与球 题解

## 题目链接
- [洛谷 P1287](https://www.luogu.com.cn/problem/P1287)

## 题目大意
现有 r 个互不相同的盒子和 n 个互不相同的球，要将 n 个球放入 r 个盒子中，不允许有空盒子，求方案数。

## 解题思路
使用容斥原理：先忽略空盒子限制，每个球有 r 种选择，总方案数为 r^n。减去至少有一个空盒的情况，加上至少有两个空盒的情况，依此类推：
ans = sum_{i=0}^{r} (-1)^i * C(r, i) * (r-i)^n
组合数用递推预处理。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, r, ans;

int calcPow(int a, int b) {
    int res = 1;
    while (b) {
        if (b & 1)
            res *= a;
        a *= a;
        b >>= 1;
    }
    return res;
}

void input() { scanf("%d%d", &n, &r); }

void process() {
    int c[11][11] = {0};
    for (int i = 0; i <= r; i++) {
        c[i][0] = c[i][i] = 1;
        for (int j = 1; j < i; j++)
            c[i][j] = c[i - 1][j - 1] + c[i - 1][j];
    }
    for (int i = 0; i <= r; i++) {
        int term = c[r][i] * calcPow(r - i, n);
        if (i & 1)
            ans -= term;
        else
            ans += term;
    }
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(r^2)。
- 空间复杂度：O(r^2)。
