# P1264 [洛谷] K-联赛 题解

## 题目链接
- [洛谷 P1264](https://www.luogu.com.cn/problem/P1264)

## 题目大意
有 n 支球队，每支球队当前已胜 w_i 场、负 d_i 场。第 i 队与第 j 队之间还有 a_{ij} 场比赛未进行（每场分出胜负，无平局）。求所有可能获得冠军（胜场数并列第一）的球队。

## 解题思路
对每支球队 x 分别判断，使用**最大流**验证该队能否夺冠：

1. 计算球队 x 可能的最大胜场数：maxWins = w_x + sum_{j≠x} a_{xj}。
2. 若存在其他球队 y 当前胜场已超过 maxWins，则 x 不可能夺冠。
3. 建图（Dinic 最大流）：
   - 源点 S 向每场未进行的比赛（不涉及 x 的比赛）连边，容量为 a_{ij}（该场比赛数）。
   - 每场比赛向参赛的两支队伍连边，容量为 INF。
   - 每支队伍 y（y≠x）向汇点 T 连边，容量为 maxWins - w_y（y 还能赢的场数上限）。
4. 若最大流等于所有剩余比赛总数 totalRem，则存在一种比赛结果分配使得 x 夺冠。

由于 n ≤ 25，对每支球队都建图跑流在时限内可行。
