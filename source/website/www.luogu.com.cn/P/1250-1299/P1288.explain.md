# P1288 取数游戏 II 题解

## 题目链接
- [洛谷 P1288](https://www.luogu.com.cn/problem/P1288)

## 题目大意
在一个环上，每条边有一个非负整数，至少有一个 0。初始硬币放在一个节点上，两人轮流操作：选择硬币相邻的一条非 0 边，将其上的数减少任意非负整数（至少减 1），然后将硬币移到边的另一端。如果轮到某人时两条相邻边均为 0，则此人输。判断先手是否有必胜策略。

## 解题思路
从起点向左右两个方向分别延伸，计算遇到 0 之前的非零边数量。如果任意一侧的非零边数量为奇数，则先手必胜，否则必败。因为玩家可以控制每次减少的量来迫使对手进入不利局面。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, a[25];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i)
        scanf("%d", &a[i]);
}

void process() {
    int cnt1 = 0, cnt2 = 0;
    for (int i = 0; i < n && a[i] != 0; ++i)
        ++cnt1;
    for (int i = n - 1; i >= 0 && a[i] != 0; --i)
        ++cnt2;
    if (cnt1 % 2 == 1 || cnt2 % 2 == 1)
        printf("YES\n");
    else
        printf("NO\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)。
- 空间复杂度：O(n)。
