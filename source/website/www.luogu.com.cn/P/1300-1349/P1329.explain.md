# P1329 洛谷 数列 题解

## 题目链接
- [洛谷 P1329](https://www.luogu.com.cn/problem/P1329)

## 题目大意
求满足下列条件的数列 a 的个数（对 2^64 取模），并输出前 100 个（或全部）数列：
- a_1 = 0
- |a_i - a_{i+1}| = 1（1 <= i < n）
- 前 n 项和 sum_{i=1}^n a_i = s

## 解题思路
将问题转化为子集和问题。定义相邻项差值 b_i = a_{i+1} - a_i ∈ {-1, 1}，则：
- a_i = sum_{j=1}^{i-1} b_j
- 前 n 项和 S = sum_{i=1}^n sum_{j=1}^{i-1} b_j = sum_{j=1}^{n-1} (n-j) * b_j

令 T = n(n-1)/2，将 b_j ∈ {-1, 1} 映射为 x_j ∈ {0, 1}，其中 b_j = 2*x_j - 1。代入可得：
- S = 2 * sum_{j=1}^{n-1} (n-j) * x_j - T
- 即 sum_{j=1}^{n-1} (n-j) * x_j = (S + T) / 2

权重集合为 {1, 2, ..., n-1}，问题转化为：从该集合中选出子集，使其总和等于 target = (s + T) / 2，每个选择对应一个唯一的数列。

**计数（DP）：**
使用动态规划计算方案数。设 cnt[j][sm] 表示处理到第 j 个权重（即 n-j）时，剩余需要凑出 sm 的方案数。DP 过程从权重 1 到 n-1 依次考虑每个权重选或不选。由于答案需要对 2^64 取模，使用 unsigned long long 自然溢出即可。

**构造（输出前 100 个）：**
从 a_1 = 0 开始 DFS 回溯：
- 尝试 b = -1（即 a_{i+1} = a_i - 1，对应不选该权重）
- 尝试 b = +1（即 a_{i+1} = a_i + 1，对应选择该权重）
利用 DP 的可行性数组 ex 进行剪枝，只搜索可到达目标状态的路径。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef unsigned long long ull;
typedef long long          ll;

int  n;
ll   s;
ull  total, target;
ull  cnt[101][5001];
bool ex[101][5001];
int  outCnt;

void input() {
    if (scanf("%d%lld", &n, &s) != 2)
        n = 0, s = 0;
}

void process() {
    ll totalSigned = (ll)n * (n - 1) / 2;
    if (s < -totalSigned || s > totalSigned) {
        target = 0;
        return;
    }
    ll tmp = s + totalSigned;
    if (tmp % 2 != 0) {
        target = 0;
        return;
    }
    total  = (ull)totalSigned;
    target = (ull)(tmp / 2);

    cnt[n][0] = 1;
    ex[n][0]  = true;

    for (int j = n - 1; j >= 1; j--) {
        int w = n - j;
        for (ull sm = 0; sm <= total; sm++) {
            cnt[j][sm] = cnt[j + 1][sm];
            ex[j][sm]  = ex[j + 1][sm];
            if (sm >= (ull)w) {
                cnt[j][sm] += cnt[j + 1][sm - w];
                if (ex[j + 1][sm - w])
                    ex[j][sm] = true;
            }
        }
    }
}

void gen(int j, ull rem, vector<int> &seq) {
    if (outCnt >= 100)
        return;
    if (j == n) {
        if (rem == 0) {
            for (int i = 0; i < n; i++)
                printf("%d%c", seq[i], i == n - 1 ? '\n' : ' ');
            outCnt++;
        }
        return;
    }
    int w = n - j;
    if (ex[j + 1][rem]) {
        seq.push_back(seq.back() - 1);
        gen(j + 1, rem, seq);
        seq.pop_back();
    }
    if (rem >= (ull)w && ex[j + 1][rem - w]) {
        seq.push_back(seq.back() + 1);
        gen(j + 1, rem - w, seq);
        seq.pop_back();
    }
}

void output() {
    if (target <= total && ex[1][target]) {
        printf("%llu\n", cnt[1][target]);
        vector<int> seq;
        seq.push_back(0);
        outCnt = 0;
        gen(1, target, seq);
    } else {
        printf("0\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n^3)，DP 部分为 O(n * total) = O(n * n^2/2) = O(n^3)，构造部分与输出序列数相关
- 空间复杂度：O(n * total) = O(n^3)，DP 数组存储计数和可行性
