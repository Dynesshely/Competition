## 题目背景

高精度乘法模板题。

## 题目描述

给出两个非负整数，求它们的乘积。

## 输入格式

输入共两行，每行一个非负整数。

## 输出格式

输出一个非负整数表示乘积。

## 样例 #1

### 样例输入

```
1 
2
```

### 样例输出

```
2
```

## 提示

每个非负整数不超过 $10^{2000}$。
