# P1310 [NOIP 2011 普及组] 表达式的值 题解

## 题目链接

- [洛谷 P1310](https://www.luogu.com.cn/problem/P1310)

## 题目大意

给定一个由 `(`, `)`, `+`, `*` 组成的表达式框架（`+` 表示逻辑或 $\oplus$，`*` 表示逻辑与 $\times$），中间空缺的位置需填入 $0$ 或 $1$。求有多少种填法使表达式的值为 $0$，结果对 $10007$ 取模。

## 解题思路

用栈模拟表达式求值，同时 DP 记录每个子表达式结果为 $0$ 和 $1$ 的方案数。

运算符优先级：`*` > `+`。遇到数字位置（即空位）看作一个未知变量，初始状态为 `{ways0=1, ways1=1}`（可填 $0$ 或 $1$）。

运算规则：
- `+`（或运算）：结果为 $0$ 当且仅当两者均为 $0$，即 `ways0 = l0 * r0`；结果为 $1$ 其余情况。
- `*`（与运算）：结果为 $1$ 当且仅当两者均为 $1$，即 `ways1 = l1 * r1`；结果为 $0$ 其余情况。

用操作数栈 `valStk` 和运算符栈 `opStk` 模拟中缀表达式求值，遇到操作数压 `{1,1}`，遇到运算符时先处理栈顶优先级不低于当前运算符的运算。

## 代码实现

- `prec(op)`：返回运算符优先级。
- `applyOp(st, op)`：弹出两个操作数，按运算符规则合并后压栈。
- 主函数中遍历表达式字符，根据括号、运算符和操作数位置进行对应操作。
- 最终栈顶的 `ways0` 即为结果为 $0$ 的方案数。

## 复杂度分析

- 时间复杂度：$O(L)$，每个字符处理一次。
- 空间复杂度：$O(L)$，栈空间。
