# P1914 小书童——凯撒密码 题解

## 题目链接
- [洛谷 P1914](https://www.luogu.com.cn/problem/P1914)

## 题目大意
给定一个由小写字母组成的原文字符串和一个整数 n，将每个字母在字母表中向后移动 n 位（z 的下一个是 a，循环），求出加密后的密码。

## 解题思路
凯撒密码（Caesar cipher）的简单实现。将每个字符减去 `'a'` 映射到 0-25 的范围，加上 n 后对 26 取模，再加回 `'a'` 得到结果。注意先将 n 对 26 取余，因为每 26 位是一个完整循环，不影响结果。

## 代码实现
- `input()` 读入整数 n 和字符串 s。
- `process()` 中先 `n %= 26`，然后遍历每个字符执行 `s[i] = (s[i] - 'a' + n) % 26 + 'a'`。
- `output()` 输出结果字符串。

## 复杂度分析
- 时间复杂度: O(len(s))
- 空间复杂度: O(1)
