# P1232 [NOI2013] 树的计数 题解

## 题目链接

- [洛谷 P1232](https://www.luogu.com.cn/problem/P1232)

## 题目大意

给定一棵树的 BFS 序和 DFS 序，求树高的期望值（精确到 $10^{-3}$）。

## 解题思路

利用 BFS 序和 DFS 序的性质推导哪些层之间必定分段、哪些层之间可能分段。

- BFS 序中相邻两节点若在 DFS 序中不连续（前者的 DFS 位置大于后者），说明它们在不同的 BFS 层，必定是新的分层点。
- DFS 序中相邻两节点若在 BFS 序中不连续（前者 + 1 ≠ 后者），说明它们之间可能有额外的分层点。

用差分数组 `c[]` 标记：必定分层的位置 `ans += 1`，可能分层的位置 `ans += 0.5`。最终 `ans` 即为期望树高。

## 代码实现

- 建立 BFS 编号到 DFS 位置的映射 `d[]` 和 DFS 编号到 BFS 位置的映射 `b[]`。
- 两遍扫描确定必然分层点和可能分层点。
- 差分数组 `c[]` 累积后，`now == 0` 的位置对应 0.5 贡献。

## 复杂度分析

- 时间复杂度：$O(N)$。
- 空间复杂度：$O(N)$。
