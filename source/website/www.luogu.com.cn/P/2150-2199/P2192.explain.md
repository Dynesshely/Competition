# P2192 [洛谷] HXY玩卡片 题解

## 题目链接
- [洛谷 P2192](https://www.luogu.com.cn/problem/P2192)

## 题目大意
给定若干张标有数字 0 或 5 的卡片，选择其中一些排成一列，组成一个能被 90 整除的最大整数（不能有前导 0）。若能组成输出该数，否则输出 -1。

## 解题思路
被 90 整除等价于同时被 9 和 10 整除。被 10 整除要求末位为 0，因此必须至少有一个 0。被 9 整除要求各位数字之和为 9 的倍数，即 5 的个数为 9 的倍数。因此统计 5 的个数 cnt5 和 0 的个数 cnt0。若 cnt0 = 0 则输出 -1。设可用 5 的个数 useCnt5 = cnt5 - cnt5 % 9。若 useCnt5 = 0 则输出 0。否则先输出 useCnt5 个 5，再输出 cnt0 个 0。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, cnt5, cnt0;

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        if (x == 5)
            cnt5++;
        else
            cnt0++;
    }
}

void process() {
    if (cnt0 == 0) {
        printf("-1\n");
        return;
    }
    int useCnt5 = cnt5 - cnt5 % 9;
    if (useCnt5 == 0) {
        printf("0\n");
        return;
    }
    for (int i = 0; i < useCnt5; i++)
        printf("5");
    for (int i = 0; i < cnt0; i++)
        printf("0");
    printf("\n");
}

void output() {}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
