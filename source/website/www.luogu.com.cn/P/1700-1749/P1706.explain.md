# P1706 全排列问题 题解

## 题目链接

- [洛谷 P1706](https://www.luogu.com.cn/problem/P1706)

## 题目大意

给定正整数 n，按照字典序输出 1 到 n 的所有全排列，每个数字占 5 个场宽。

## 解题思路

本题要求输出 n 的全排列（n <= 9），最直接的方法是使用 C++ 标准库中的 `std::next_permutation`。

`next_permutation` 可以将一个序列原地转换为字典序中的下一个排列。当序列已经是字典序最大的排列时（即从大到小有序），它会返回 `false` 并将序列重置为最小的排列（即从小到大有序）。

因此，只需将数组初始化为 `[1, 2, 3, ..., n]`（最小字典序），然后反复调用 `next_permutation` 即可逐次生成所有排列。

## 代码实现

- **动态数组 `ns`**：通过 `new int[n + 1]` 在堆上分配，下标从 1 开始使用。
- **初始化**：将 `ns[i]` 赋值为 i，构成递增初始排列。
- **`do-while` 循环**：先输出当前排列，再调用 `next_permutation` 生成下一个排列。`do-while` 保证了初始排列也能被输出。
- **输出格式**：使用 `printf("%5d", ...)` 实现每个数字保留 5 个场宽。
- **`next_permutation` 的迭代器范围**：`ns + 1` 到 `ns + n + 1`，对应数组下标 [1, n]。

## 复杂度分析

- 时间复杂度：O(n * n!)，全排列共有 n! 个，生成每个排列平均 O(1)，但输出需要 O(n)。
- 空间复杂度：O(n)，仅需存储当前排列。
