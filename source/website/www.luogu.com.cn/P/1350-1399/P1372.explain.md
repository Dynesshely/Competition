# P1372 又是毕业季I 题解

## 题目链接
- [洛谷 P1372](https://www.luogu.com.cn/problem/P1372)

## 题目大意
从 $1 \sim n$ 中选出 $k$ 个不同的数，使得它们的最大公约数最大。输出这个最大公约数。

## 解题思路
选出的 $k$ 个数都可整除 $d$ 时，$\gcd$ 至少为 $d$。要使 $d$ 最大，即 $d \times k \le n$ 的最大整数 $d$，故 $d = n / k$（向下取整），取数 $d, 2d, \dots, kd$ 即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, k, ans;

void input() { scanf("%d%d", &n, &k); }

void process() { ans = n / k; }

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(1)$
- 空间复杂度：$O(1)$
