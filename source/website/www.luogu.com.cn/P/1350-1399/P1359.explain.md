# P1359 洛谷 线性DP 题解

## 题目链接
- [洛谷 P1359](https://www.luogu.com.cn/problem/P1359)

## 题目大意
长江游艇俱乐部在长江上设置了 $n$ 个游艇出租站，游客可在任一出租站租用游艇，并在下游的任何一个出租站归还。已知出租站 $i$ 到 $j$（$i<j$）之间的租金 $r_{i,j}$，求从出租站 $1$ 到出租站 $n$ 所需的最少租金。

## 解题思路
题目等价于在一个有向无环图上求最短路，其中顶点 $1$ 到 $n$ 按编号顺序排列，边 $(i,j)$ 的权值为 $r_{i,j}$。由于 $n\le200$，可以直接使用线性 DP 求解。

设 $dp[i]$ 表示从站 $1$ 到站 $i$ 的最小租金，初始化 $dp[1]=0$，其余为无穷大。对于每个 $i$（$2\le i\le n$），枚举中转站 $j$（$1\le j<i$），状态转移方程为：
$$
dp[i] = \min_{1\le j < i}\big(dp[j] + r_{j,i}\big)
$$
答案为 $dp[n]$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
int r[205][205];
int dp[205];

void input() {
    scanf("%d", &n);
    for (int i = 1; i < n; i++)
        for (int j = i + 1; j <= n; j++)
            scanf("%d", &r[i][j]);
}

void process() {
    memset(dp, 0x3f, sizeof(dp));
    dp[1] = 0;
    for (int i = 2; i <= n; i++)
        for (int j = 1; j < i; j++)
            if (dp[j] + r[j][i] < dp[i])
                dp[i] = dp[j] + r[j][i];
}

void output() { printf("%d\n", dp[n]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2)$，两层循环枚举所有 $i$ 和 $j$。
- 空间复杂度：$O(n^2)$，用于存储租金矩阵 $r$。
