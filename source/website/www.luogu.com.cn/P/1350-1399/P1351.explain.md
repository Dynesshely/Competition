# P1351 洛谷 枚举中间点 题解

## 题目链接
- [洛谷 P1351](https://www.luogu.com.cn/problem/P1351)

## 题目大意
给定一棵 $n$ 个节点的无向连通树，每个节点 $i$ 有权值 $W_i$。定义距离为 2 的有序点对 $(u,v)$ 会产生联合权值 $W_u \times W_v$。求所有有序点对中联合权值的最大值以及所有联合权值之和（对 $10007$ 取模）。

## 解题思路
由于图是一棵树，距离为 2 的两个点必然通过一个中间点相连。因此，枚举每个点 $u$ 作为中间点，考虑其所有邻接点构成的点对 $(v_1, v_2)$。

对于每个中间点 $u$：
- **最大值**：维护邻接点中权值的最大值和次大值，乘积即为以 $u$ 为中间点能产生的最大联合权值。
- **总和**：设所有邻接点的权值和为 $sum$，平方和为 $sumSq$，则有序点对的权值总和为 $sum^2 - sumSq$（因为 $(a+b)^2 = a^2+b^2+2ab$，展开后 $sum^2 - sumSq$ 恰好是所有有序对的乘积之和）。

对所有中间点的结果取全局最大值并累加总和即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n;
vector<int> adj[200005];
int         w[200005];
int         maxAns, sumAns;
const int   MOD = 10007;

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n - 1; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    for (int i = 1; i <= n; i++)
        scanf("%d", &w[i]);
}

void process() {
    maxAns = 0;
    sumAns = 0;
    for (int u = 1; u <= n; u++) {
        if (adj[u].size() < 2)
            continue;
        int max1 = 0, max2 = 0;
        int sum = 0, sumSq = 0;
        for (int v : adj[u]) {
            int val = w[v];
            if (val > max1) {
                max2 = max1;
                max1 = val;
            } else if (val > max2)
                max2 = val;
            sum   = (sum + val) % MOD;
            sumSq = (sumSq + val * val) % MOD;
        }
        maxAns = max(maxAns, max1 * max2);
        sumAns = (sumAns + (sum * sum % MOD - sumSq + MOD)) % MOD;
    }
}

void output() { printf("%d %d\n", maxAns, sumAns); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$，每个节点和每条边恰好遍历一次。
- 空间复杂度：$O(n)$，存储邻接表和权值数组。
