# P1399 [NOI2013] 快餐店 题解

## 题目链接
- [洛谷 P1399](https://www.luogu.com.cn/problem/P1399)

## 题目大意
给定一个 N 个点、N 条边的连通图（基环树），每条边有长度。要求在图中选择一个点（可以是节点，也可以是边上任意位置）开设快餐店，使得该点到最远顾客的距离最小。输出这个最小距离，保留一位小数。

## 解题思路
基环树上的最小化最远距离问题，本质上是在基环树上找"中心"。

1. **找环**：基环树有唯一环，DFS 找出环上的所有节点和边。
2. **计算子树深度**：将环上的节点看作根，DFS 每个根向环外延伸的子树，计算每个子树内的最大深度 dep[i] 和子树内部的直径 tr[i]。
3. **枚举环上断边**：快餐店最优位置要么在某棵子树内（等价于子树直径的一半），要么在环上某条路径上。可以将问题转化为：在环上断一条边，求断边后树上的直径，取所有断边方案中直径的最小值。
4. **前缀/后缀优化**：环上按顺序处理，维护前缀最大 (dep+sum) 和后缀最大 (dep+sub) 等数组，用类似"环上两点最大距离"的 DP 技巧，在 O(m) 内计算所有断边方案下的环上最长路径。
5. **最终答案**：答案为所有情况中最大距离的最小值的一半（因快餐店可以放在路径中点）。注意输出格式保留一位小数。

## 代码实现
DFS 找环，对环上每个节点做子树 DFS 求深度和子树直径。利用四个辅助数组 a, b, c, d 在环上进行 DP 合并。

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
