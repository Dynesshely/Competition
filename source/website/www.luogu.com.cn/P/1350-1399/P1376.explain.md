# P1376 [USACO05MAR] Yogurt factory 机器工厂 题解

## 题目链接
- [洛谷 P1376](https://www.luogu.com.cn/problem/P1376)

## 题目大意
$n$ 周内，第 $i$ 周生产一台机器成本为 $C_i$ 元，每周每台库存保养费为 $S$ 元。第 $i$ 周需交付 $Y_i$ 台机器（可用当周生产或之前存货）。求完成所有订单的最小总成本。

## 解题思路
贪心：第 $i$ 周可用的最低单位成本为 $\min(C_i, \text{前一周最低成本} + S)$。若存货成本（前一周最低成本加保养费）小于本周生产成本，则用存货更优；否则当周生产。

维护 `minCost` 表示到第 $i$ 周时的最低单位成本，每周按 `minCost * Y_i` 累加总成本。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int       n, s;
int       c[10005], y[10005];
long long ans;

void input() {
    scanf("%d%d", &n, &s);
    for (int i = 1; i <= n; ++i)
        scanf("%d%d", &c[i], &y[i]);
}

void process() {
    int minCost = c[1];
    ans = 1ll * minCost * y[1];
    for (int i = 2; i <= n; ++i) {
        if (minCost + s < c[i])
            minCost = minCost + s;
        else
            minCost = c[i];
        ans += 1ll * minCost * y[i];
    }
}

void output() { printf("%lld\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
