# P1352 洛谷 树形DP 题解

## 题目链接
- [洛谷 P1352](https://www.luogu.com.cn/problem/P1352)

## 题目大意
给定一棵 $n$ 个节点的树，每个节点 $i$ 有一个快乐指数 $r_i$。如果某个节点被选中，则其父节点不能同时被选中。求选出的节点集合的最大快乐指数之和。

## 解题思路
经典树形动态规划。以校长（根节点）为起点进行 DFS，设：

- $dp[u][0]$ 表示不选节点 $u$ 时，以 $u$ 为根的子树能获得的最大快乐指数。
- $dp[u][1]$ 表示选节点 $u$ 时，以 $u$ 为根的子树能获得的最大快乐指数。

转移方程：
- 若 $u$ 不选，则其子节点 $v$ 可选可不选：$dp[u][0] = \sum \max(dp[v][0], dp[v][1])$
- 若 $u$ 选，则其子节点 $v$ 都不能选：$dp[u][1] = r_u + \sum dp[v][0]$

最终答案为 $\max(dp[root][0], dp[root][1])$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int maxN = 6005;

int  n, r[maxN], dp[maxN][2], head[maxN], edgeCnt, root;
bool notRoot[maxN];

struct Edge {
    int to, nxt;
} e[maxN];

void addEdge(int u, int v) {
    e[++edgeCnt] = {v, head[u]};
    head[u]      = edgeCnt;
}

void dfs(int u) {
    dp[u][1] = r[u];
    dp[u][0] = 0;
    for (int i = head[u]; i; i = e[i].nxt) {
        int v = e[i].to;
        dfs(v);
        dp[u][1] += dp[v][0];
        dp[u][0] += max(dp[v][0], dp[v][1]);
    }
}

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i)
        scanf("%d", &r[i]);
    for (int i = 1; i < n; ++i) {
        int l, k;
        scanf("%d%d", &l, &k);
        addEdge(k, l);
        notRoot[l] = true;
    }
}

void process() {
    root = 0;
    for (int i = 1; i <= n; ++i)
        if (!notRoot[i]) {
            root = i;
            break;
        }
    dfs(root);
}

void output() { printf("%d\n", max(dp[root][0], dp[root][1])); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$，每个节点恰好遍历一次。
- 空间复杂度：$O(n)$，存储邻接表和 DP 数组。
