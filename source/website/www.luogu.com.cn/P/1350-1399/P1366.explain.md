# P1366 洛谷 有序表的合并 双指针 题解

## 题目链接
- [洛谷 P1366](https://www.luogu.com.cn/problem/P1366)

## 题目大意
给定两个按不降序排序的数列 a 和 b，a 中元素互不相同。求 a 中每个元素在 b 中出现的次数，输出这些次数的按位异或和。

## 解题思路
由于两个数列均已排序，可以使用双指针（归并式）扫描，避免使用哈希表或二分查找。

具体步骤：
1. 用指针 j 指向 a 数组的当前位置。
2. 从左到右扫描 b 数组，对 b 中连续相同的元素进行计数（cnt）。
3. 对于 b 中的每一段相同值 cur：
   - 将 a 指针 j 向后移动，跳过所有小于 cur 的元素（因为 a 递增且不重复）。
   - 如果 a[j] == cur，说明 a 中有这个值，则将 cnt 异或入答案，j 向后移动一位。
4. 输出异或和。

注意：输入量可达 10⁷，需使用快速读入。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef unsigned long long ull;

ull readUll() {
    ull  x = 0;
    char c = getchar();
    while (c < '0' || c > '9')
        c = getchar();
    while (c >= '0' && c <= '9') {
        x = x * 10 + (c - '0');
        c = getchar();
    }
    return x;
}

int readInt() {
    int  x = 0;
    char c = getchar();
    while (c < '0' || c > '9')
        c = getchar();
    while (c >= '0' && c <= '9') {
        x = x * 10 + (c - '0');
        c = getchar();
    }
    return x;
}

int main() {
    int T = readInt();
    while (T--) {
        int         n = readInt(), m = readInt();
        vector<ull> a(n);
        for (int i = 0; i < n; ++i)
            a[i] = readUll();

        int xorSum = 0, j = 0;

        if (m) {
            ull cur = readUll();
            int cnt = 1;
            for (int i = 1; i < m; ++i) {
                ull x = readUll();
                if (x == cur)
                    ++cnt;
                else {
                    while (j < n && a[j] < cur)
                        ++j;
                    if (j < n && a[j] == cur) {
                        xorSum ^= cnt;
                        ++j;
                    }
                    cur = x;
                    cnt = 1;
                }
            }
            while (j < n && a[j] < cur)
                ++j;
            if (j < n && a[j] == cur) {
                xorSum ^= cnt;
                ++j;
            }
        }

        printf("%d\n", xorSum);
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + m)，两数组各扫描一次。
- 空间复杂度：O(n)，存储 a 数组（b 数组边读边处理，无需全部存储）。
