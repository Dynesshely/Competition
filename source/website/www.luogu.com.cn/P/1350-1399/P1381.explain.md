# P1381 单词背诵 题解

## 题目链接
- [洛谷 P1381](https://www.luogu.com.cn/problem/P1381)

## 题目大意
灵梦有 $n$ 个要背的单词，有一篇由 $m$ 个单词构成的文章。找出文章中连续的一段，使得其中包含最多要背的单词种类数（重复只算一个），并在此前提下使该段尽量短。输出最大种类数和最短段长度。

## 解题思路
分为两步：
1. 扫描文章，用哈希表统计要背的单词在文章中出现的种类总数 `total`。
2. 用双指针（滑动窗口）求覆盖全部 `total` 种单词的最短连续段长度：
   - 右指针不断右移扩展窗口，每遇到一个要背的单词就增加计数。
   - 当窗口内包含全部 `total` 种单词时，尝试收缩左指针，更新最短长度。
   - 重复直到右指针到达文章末尾。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int                        n, m, total, l, distinct, ans;
char                       word[15];
unordered_map<string, int> id;
int                        art[100005];
int                        cnt[1005];
bool                       seen[1005];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%s", word);
        id[word] = i;
    }
    scanf("%d", &m);
    for (int i = 0; i < m; i++) {
        scanf("%s", word);
        auto it = id.find(word);
        art[i] = it == id.end() ? -1 : it->second;
    }
}

void process() {
    for (int i = 0; i < m; i++)
        if (art[i] != -1 && !seen[art[i]]) {
            seen[art[i]] = true;
            total++;
        }
    if (!total) {
        ans = 0;
        return;
    }
    ans = m + 1;
    for (int r = 0; r < m; r++) {
        if (art[r] != -1) {
            cnt[art[r]]++;
            if (cnt[art[r]] == 1)
                distinct++;
        }
        while (distinct == total) {
            if (r - l + 1 < ans)
                ans = r - l + 1;
            if (art[l] != -1) {
                cnt[art[l]]--;
                if (cnt[art[l]] == 0)
                    distinct--;
            }
            l++;
        }
    }
}

void output() { printf("%d\n%d\n", total, ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n + m)$，每个单词至多进出滑动窗口一次
- 空间复杂度：$O(n + m)$
