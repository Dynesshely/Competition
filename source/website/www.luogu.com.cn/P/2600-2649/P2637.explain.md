# P2637 [洛谷] 第一次，第二次，成交！Going Once, Going Twice, Gone! B 题解

## 题目链接
- [洛谷 P2637](https://www.luogu.com.cn/problem/P2637)

## 题目大意
FJ有n批干草，m个农夫各出价p_i。FJ设定一个固定售价，出价不低于该价的农夫均可买一批草料。求使总收益最大的最低售价及最大收益。

## 解题思路
将出价排序，枚举每个出价作为售价，统计出价不低于该价的农夫数（不超过n），计算总收益，记录最优解。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, p[1005], bestPrice, maxProfit;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; ++i)
        scanf("%d", &p[i]);
}

void process() {
    sort(p, p + m);
    for (int i = 0; i < m; ++i) {
        int cnt    = min(n, m - i);
        int profit = p[i] * cnt;
        if (profit > maxProfit) {
            maxProfit = profit;
            bestPrice = p[i];
        }
    }
}

void output() { printf("%d %d\n", bestPrice, maxProfit); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(m log m)
- 空间复杂度：O(m)
