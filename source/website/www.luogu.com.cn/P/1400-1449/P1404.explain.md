# P1404 平均数 题解

## 题目链接
- [洛谷 P1404](https://www.luogu.com.cn/problem/P1404)

## 题目大意
给定长度为 $n$ 的数列，求长度 $\ge m$ 的子串的最大平均数，结果乘以 $1000$ 后舍去小数部分取整。

## 解题思路
二分答案（平均值），检查是否存在长度 $\ge m$ 的子串平均值 $\ge mid$。

检查方法：将所有数减去 $mid$，则问题转化为是否存在长度 $\ge m$ 的子串和非负。预处理前缀和 $ps[i] = \sum_{j=1}^{i} (a_j - mid)$，则子串 $[l, r]$ 和为 $ps[r] - ps[l-1]$。遍历 $r$，维护 $ps[0 \sim r-m]$ 的最小值，若 $ps[r] - minPs \ge 0$ 则可行。

为避免浮点二分精度问题，将所有 $a_i$ 乘以 $1000$ 后二分整数。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
int       a[MAXN], n, m;
long long ps[MAXN];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++)
        scanf("%d", &a[i]);
}

bool check(int v) {
    ps[0] = 0;
    for (int i = 1; i <= n; i++)
        ps[i] = ps[i - 1] + 1000LL * a[i] - v;
    long long minPs = 0;
    for (int i = m; i <= n; i++) {
        if (ps[i] >= minPs)
            return true;
        if (ps[i - m + 1] < minPs)
            minPs = ps[i - m + 1];
    }
    return false;
}

void process() {
    int l = 0, r = 2000 * 1000;
    while (l < r) {
        int mid = (l + r + 1) >> 1;
        if (check(mid))
            l = mid;
        else
            r = mid - 1;
    }
    printf("%d\n", l);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log M)$，其中 $M$ 为值域范围。
- 空间复杂度：$O(n)$。
