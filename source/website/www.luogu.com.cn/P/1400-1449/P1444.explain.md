# P1444 [USACO1.3] 虫洞 wormhole 题解

## 题目链接
- [洛谷 P1444](https://www.luogu.com.cn/problem/P1444)

## 题目大意
$n$ 个虫洞在二维平面上，两两配对形成 $n/2$ 对。Bessie 总是沿 $x$ 正方向移动，进入虫洞后会从配对的另一端传出，方向不变。求有多少种配对方案，使得存在某个位置，Bessie 从该位置出发会陷入无限循环。$n \le 12$，$n$ 为偶数。

## 解题思路
DFS 枚举所有配对方案，然后判断是否存在环。

配对枚举：每次找到第一个未配对的虫洞，依次尝试与后面的每个未配对虫洞配对，递归搜索所有 $ (n-1)!! $ 种方案。

环检测：对于给定的配对方案，先按 $y$ 坐标排序（同 $y$ 按 $x$ 排序），计算每个虫洞右侧最近的同 $y$ 虫洞下标（即从该虫洞向右走会到达的虫洞）。然后对每个虫洞作为起点模拟行走，如果出现重复访问则判定存在环。

行走规则：从虫洞 $i$ 出发向右，到达 $rightIdx[i]$（右侧最近的同 $y$ 虫洞），然后跳跃到其配对虫洞 $paired[rightIdx[i]]$，继续向右。若 $rightIdx[i] = -1$（右侧无虫洞）则停止。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Point {
    int x, y;
};

int   n;
Point p[15];
int   rightIdx[15], paired[15];
int   ans;

bool hasCycle() {
    for (int i = 0; i < n; i++) {
        int via[15] = {0};
        int cur     = i;
        while (true) {
            if (rightIdx[cur] == -1)
                break;
            cur = paired[rightIdx[cur]];
            if (via[cur])
                return true;
            via[cur] = 1;
        }
    }
    return false;
}

void dfs(int cnt) {
    if (cnt == n) {
        if (hasCycle())
            ans++;
        return;
    }
    int first = -1;
    for (int i = 0; i < n; i++)
        if (paired[i] == -1) {
            first = i;
            break;
        }
    for (int j = first + 1; j < n; j++)
        if (paired[j] == -1) {
            paired[first] = j;
            paired[j]     = first;
            dfs(cnt + 2);
            paired[first] = -1;
            paired[j]     = -1;
        }
}

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d%d", &p[i].x, &p[i].y);
}

void process() {
    sort(p, p + n, [](const Point &a, const Point &b) {
        if (a.y != b.y)
            return a.y < b.y;
        return a.x < b.x;
    });
    for (int i = 0; i < n; i++) {
        rightIdx[i] = -1;
        for (int j = i + 1; j < n; j++)
            if (p[j].y == p[i].y) {
                rightIdx[i] = j;
                break;
            }
    }
    memset(paired, -1, sizeof(paired));
    ans = 0;
    dfs(0);
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O((n-1)!! \cdot n^2)$，$(n-1)!! = 1 \cdot 3 \cdot 5 \cdots (n-1)$，$n \le 12$ 时最多 $10395$ 种方案。
- 空间复杂度：$O(n)$。
