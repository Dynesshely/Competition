# P1435 [IOI 2000] 回文字串 题解

## 题目链接
- [洛谷 P1435](https://www.luogu.com.cn/problem/P1435)

## 题目大意
给定一个字符串 $s$，通过插入若干字符可将其变为回文串。求最少需要插入的字符数。字符串长度 $l \le 1000$，区分大小写。

## 解题思路
转化为求原串与逆序串的最长公共子序列（LCS）。将一个字符串变成回文串所需的最少插入字符数等于字符串长度减去其与自身逆序的最长公共子序列长度。

原理：将原串 $s$ 与其逆序串 $t = reverse(s)$ 求 LCS，LCS 长度即为原串中已经成回文的部分（最大回文子序列）。需要插入的字符用于补充剩余不对称的部分，因此答案为 $n - LCS$。

DP 状态：$dp[i][j]$ 表示 $s$ 前 $i$ 个字符与 $t$ 前 $j$ 个字符的 LCS 长度。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;

char s[MAXN], t[MAXN];
int  dp[MAXN][MAXN];
int  n;

void input() {
    scanf("%s", s);
    n = strlen(s);
}

void process() {
    for (int i = 0; i < n; i++)
        t[i] = s[n - 1 - i];
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            if (s[i - 1] == t[j - 1])
                dp[i][j] = dp[i - 1][j - 1] + 1;
            else
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
}

void output() { printf("%d\n", n - dp[n][n]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2)$，$n \le 1000$ 时足够快。
- 空间复杂度：$O(n^2)$ 用于 DP 数组，可优化至 $O(n)$。
