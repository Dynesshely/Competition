# P1417 烹调方案 题解

## 题目链接
- [洛谷 P1417](https://www.luogu.com.cn/problem/P1417)

## 题目大意
有 $n$ 件食材，每件食材有三个属性 $a_i, b_i, c_i$。如果在 $t$ 时刻完成第 $i$ 样食材，则得到 $a_i - t \times b_i$ 的美味指数。做第 $i$ 样食材需要 $c_i$ 时间。总时间限制为 $T$，求最大美味指数。

## 解题思路
这是一个01背包的变种，价值与完成时间有关。若直接做01背包，物品的顺序会影响最终价值。

**排序**：考虑两个食材 $i$ 和 $j$，如果先做 $i$ 再做 $j$ 比先做 $j$ 再做 $i$ 更优，则应满足：
$$
a_i + a_j - c_i b_i - (c_i + c_j) b_j \ge a_i + a_j - c_j b_j - (c_i + c_j) b_i
$$
化简得 $c_i b_j \le c_j b_i$，即 $\frac{c_i}{b_i} \le \frac{c_j}{b_j}$。按此比例排序后，再做 01 背包 DP。

**DP**：$dp[t]$ 表示总时间为 $t$ 时的最大美味值，按排序后的顺序做 01 背包。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

struct Item {
    ll a, b, c;
};

bool cmp(const Item &x, const Item &y) { return x.c * y.b < y.c * x.b; }

int  T, n;
Item items[55];
ll   dp[100005];

void input() {
    scanf("%d%d", &T, &n);
    for (int i = 1; i <= n; i++)
        scanf("%lld", &items[i].a);
    for (int i = 1; i <= n; i++)
        scanf("%lld", &items[i].b);
    for (int i = 1; i <= n; i++)
        scanf("%lld", &items[i].c);
}

void process() {
    sort(items + 1, items + n + 1, cmp);
    memset(dp, 0xc0, sizeof(dp));
    dp[0] = 0;
    for (int i = 1; i <= n; i++) {
        ll a = items[i].a, b = items[i].b, c = items[i].c;
        for (int t = T; t >= c; t--)
            dp[t] = max(dp[t], dp[t - c] + a - t * b);
    }
}

void output() {
    ll ans = 0;
    for (int t = 0; t <= T; t++)
        ans = max(ans, dp[t]);
    printf("%lld\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n + nT)$。
- 空间复杂度：$O(T)$。
