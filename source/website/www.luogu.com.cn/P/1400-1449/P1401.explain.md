# P1401 [入门赛 #18] 禁止在 int 乘 int 时不开 long long 题解

## 题目链接

- [洛谷 P1401](https://www.luogu.com.cn/problem/P1401)

## 题目大意

给定两个 int 类型变量 x 和 y 的取值范围，判断 x * y 是否可能超出 int 类型可表示的范围 [−2147483648, 2147483647]。

## 解题思路

x 与 y 的取值范围由四个端点给出：最小值和最大值分别记为 xl, xu 和 yl, yu。乘积的最大值和最小值一定在四个组合中产生：xl * yl, xl * yu, xu * yl, xu * yu。计算出这四个乘积的最小值和最大值，若最小值小于 −2147483648 或最大值大于 2147483647，则可能溢出，需要 long long 类型。

注意使用 long long 类型进行乘法运算以避免中间计算溢出。

## 代码实现

读入 xl, xu, yl, yu，用 long long 计算四个乘积，取最小值和最大值，与 int 范围边界比较后输出相应结果。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
