# P1406 方格填数 题解

## 题目链接
- [洛谷 P1406](https://www.luogu.com.cn/problem/P1406)

## 题目大意
给定 $n \times n$ 个整数，填入 $n \times n$ 的方格矩阵（$1 \le n \le 4$），使得每行、每列、两条对角线上的数字和都相等。数据保证有解，输出字典序最小的方案（按行顺次连接后字典序最小）。

## 解题思路
DFS 搜索所有可能的填数方案，配合剪枝优化：

1. 先将输入数字排序，便于字典序搜索和去重。
2. 每行和 $s$ = 总和 $/ n$。
3. 逐格填充，当填到每行最后一个、每列最后一个以及对角线位置时，检查对应行/列/对角线的剩余和是否与当前数匹配。
4. 跳过相同的数字（若前一个相同数字未使用则跳过），避免重复搜索。

由于 $n \le 4$，搜索空间有限，剪枝后能够快速找到解。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, s;
int  a[20];
int  grid[4][4];
bool used[20];
int  rowSum[4], colSum[4];
int  diag1, diag2;
bool done;

void dfs(int p) {
    if (done)
        return;
    if (p == n * n) {
        if (diag1 == s && diag2 == s)
            done = true;
        return;
    }
    int r = p / n, c = p % n;
    for (int i = 0; i < n * n; i++) {
        if (used[i])
            continue;
        if (i > 0 && a[i] == a[i - 1] && !used[i - 1])
            continue;
        int v = a[i];
        if (c == n - 1) {
            int need = s - rowSum[r];
            if (v != need)
                continue;
        }
        if (r == n - 1) {
            int need = s - colSum[c];
            if (v != need)
                continue;
        }
        if (r == n - 1 && c == 0) {
            int need = s - diag2;
            if (v != need)
                continue;
        }
        if (r == n - 1 && c == n - 1) {
            int need = s - diag1;
            if (v != need)
                continue;
        }
        used[i]    = true;
        grid[r][c] = v;
        rowSum[r] += v;
        colSum[c] += v;
        if (r == c)
            diag1 += v;
        if (r + c == n - 1)
            diag2 += v;
        dfs(p + 1);
        if (done)
            return;
        used[i] = false;
        rowSum[r] -= v;
        colSum[c] -= v;
        if (r == c)
            diag1 -= v;
        if (r + c == n - 1)
            diag2 -= v;
    }
}

int main() {
    scanf("%d", &n);
    int m = n * n, total = 0;
    for (int i = 0; i < m; i++) {
        scanf("%d", &a[i]);
        total += a[i];
    }
    s = total / n;
    sort(a, a + m);
    dfs(0);
    printf("%d\n", s);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++)
            printf("%d%c", grid[i][j], j == n - 1 ? '\n' : ' ');
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O((n^2)!)$ 最坏情况，但剪枝后实际运行很快。
- 空间复杂度：$O(n^2)$。
