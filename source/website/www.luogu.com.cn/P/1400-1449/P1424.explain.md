# P1424 小鱼的航程（改进版） 题解

## 题目链接

- [洛谷 P1424](https://www.luogu.com.cn/problem/P1424)

## 题目大意

小鱼从周 x 开始算起，经过 n 天，工作日每天游泳 250 公里，周末（周六周日）休息。求总共游泳了多少公里。

## 解题思路

将 x 映射为 1~7 的数字表示周一到周日，遍历 n 天，每天计算对应星期几，若为工作日（1~5）则累加 250 公里。星期计算：cur = (x + i - 1) % 7 + 1。

## 代码实现

循环 n 次，每次计算当前日期的星期数，若 cur <= 5 则 ans += 250。最后输出 ans。

## 复杂度分析

- 时间复杂度：O(n)，n 最大为 10^6，在允许范围内。
- 空间复杂度：O(1)。
