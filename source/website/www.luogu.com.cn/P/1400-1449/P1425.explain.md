# P1425 小鱼的游泳时间 题解

## 题目链接

- [洛谷 P1425](https://www.luogu.com.cn/problem/P1425)

## 题目大意

小鱼从当天 a 时 b 分开始游泳，到 c 时 d 分结束，计算游泳时长（小时和分钟）。

## 解题思路

将起止时间均转换为分钟数：start = a * 60 + b, end = c * 60 + d。时间差 diff = end - start，小时数 = diff / 60，分钟数 = diff % 60。

## 代码实现

读入四个整数，计算分钟差后换算为小时和分钟输出。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
