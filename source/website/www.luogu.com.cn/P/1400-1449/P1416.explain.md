# P1416 攻击火星 题解

## 题目链接
- [洛谷 P1416](https://www.luogu.com.cn/problem/P1416)

## 题目大意
需要设计一个 $n$ 个点的无向图（无自环、无重边）。外星人按如下方式攻击：先攻击度为 $0$ 的点（删除），然后攻击度为 $1$ 的点，依此类推直到度为 $n-1$ 的点。度数是动态统计的（删除点后相连点的度 $-1$）。求最多能有多少个点不被攻击。

## 解题思路
目标是构造一个图，使得在按度数从小到大逐轮删除的过程中，尽可能多的点存活下来。

考虑构造一个 $n-1$ 个点的完全图 $K_{n-1}$，再加一个孤立点。则：
- 第 $0$ 轮：删除孤立点（度为 $0$）。
- 第 $1$ 轮：删除 $K_{n-1}$ 中所有点（度为 $n-2$，在一轮中同时删除）。

这样所有点都被删除了。

观察答案模式：当 $n \le 2$ 时答案为 $0$，当 $n \ge 3$ 时答案为 $n-2$。
构造方案：取一个点作为"中心"，将其他 $n-1$ 个点两两相连（形成 $K_{n-1}$），然后断开中心点与部分点的连接使其度为 $n-2$。这样第一轮删掉度为 $0$ 的点（如果有），第二轮删掉度为 $1$ 的点……最终中心点存活。

更简单的构造：将 $n-1$ 个点排成一个环（或链），中心点与它们全部相连。这样 $n-1$ 个外围点度为 $2$，中心点为 $n-1$。依次删除度为 $0,1,2$ 的点后，$n \ge 3$ 时中心点存活，存活数为 $n-2$。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {}

void output() {
    if (n <= 2)
        printf("0\n");
    else
        printf("%d\n", n - 2);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(1)$。
- 空间复杂度：$O(1)$。
