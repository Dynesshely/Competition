# P1433 吃奶酪 题解

## 题目链接
- [洛谷 P1433](https://www.luogu.com.cn/problem/P1433)

## 题目大意
房间里有 $n$ 块奶酪，老鼠从 $(0,0)$ 出发，要将所有奶酪全部吃掉，求最少需要跑多少距离。$n \le 15$，坐标绝对值不超过 $200$。

## 解题思路
状态压缩动态规划。预处理任意两点（包括起点）之间的距离。

定义 $dp[mask][i]$ 表示已经吃掉的奶酪集合为 $mask$，且最后吃的是第 $i$ 块奶酪时的最短距离。

- 初始状态：$dp[1 \ll i][i] = dist(起点, i)$
- 转移：$dp[mask | (1 \ll j)][j] = \min(dp[mask | (1 \ll j)][j], dp[mask][i] + dist(i, j))$，其中 $mask$ 不包含 $j$。
- 最终答案：$\min_{i} dp[full][i]$，其中 $full = (1 \ll n) - 1$。

由于 $n \le 15$，状态数为 $2^{15} \times 15 = 491520$，可以通过。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int    n;
double x[20], y[20], dist[20][20], dp[1 << 15][15];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%lf%lf", &x[i], &y[i]);
}

void process() {
    x[n] = y[n] = 0;
    for (int i = 0; i <= n; i++)
        for (int j = 0; j <= n; j++)
            dist[i][j] = sqrt((x[i] - x[j]) * (x[i] - x[j]) + (y[i] - y[j]) * (y[i] - y[j]));
    for (int mask = 0; mask < (1 << n); mask++)
        for (int i = 0; i < n; i++)
            dp[mask][i] = 1e9;
    for (int i = 0; i < n; i++)
        dp[1 << i][i] = dist[n][i];
    for (int mask = 0; mask < (1 << n); mask++)
        for (int i = 0; i < n; i++)
            if (mask & (1 << i))
                for (int j = 0; j < n; j++)
                    if (!(mask & (1 << j)))
                        dp[mask | (1 << j)][j] = min(dp[mask | (1 << j)][j], dp[mask][i] + dist[i][j]);
}

void output() {
    double ans  = 1e9;
    int    full = (1 << n) - 1;
    for (int i = 0; i < n; i++)
        ans = min(ans, dp[full][i]);
    printf("%.2f\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(2^n \cdot n^2)$，$n \le 15$ 时约 $7.4 \times 10^6$ 次操作。
- 空间复杂度：$O(2^n \cdot n)$ 用于 DP 数组。
