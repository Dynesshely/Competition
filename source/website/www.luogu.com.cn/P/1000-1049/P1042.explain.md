# P1042 [NOIP 2003 普及组] 乒乓球 题解

## 题目链接

- [洛谷 P1042](https://www.luogu.com.cn/problem/P1042)

## 题目大意

给定一串比赛记录，字符 `W` 表示华华得分，`L` 表示对手得分，`E` 表示记录结束。分别按 $11$ 分制和 $21$ 分制输出每局比分。一局结束条件：一方得分 $\ge k$ 且领先对手 $\ge 2$ 分。最后一局若未结束也需输出当前比分。

## 解题思路

**模拟**。先过滤输入（跳过空格和换行，遇 `E` 停止），得到纯净的 `W/L` 序列。然后分别对 $k = 11$ 和 $k = 21$ 处理：

1. 遍历序列，逐字符累计华华和对手的分数。
2. 若一方得分 $\ge k$ **且** 分差 $\ge 2$，则该局结束，输出比分后重置计数器。
3. 遍历结束后，输出当前未完成的比分（即最后一局）。

两套分制的结果之间用空行分隔。特别地，若输入没有任何有效字符（`E` 之前无 `W` 或 `L`），两套分制均输出 `0:0`。

## 代码实现

- `getchar()` 逐字符读取，遇 `W`/`L` 追加到字符串，遇 `E` 停止，无视空格和换行。
- 对 $k=11$ 和 $k=21$ 各遍历一次序列，同一判断逻辑：`(huahua >= k || enemy >= k) && abs(huahua - enemy) >= 2` 时结算本局。
- 每次结算和最终输出后均重置计数器。
- 空输入特殊处理：直接输出 `0:0`。

## 复杂度分析

- 时间复杂度：$O(L)$，$L$ 为有效字符数。两次线性扫描，每次 $O(L)$。
- 空间复杂度：$O(L)$，存储过滤后的字符序列。
