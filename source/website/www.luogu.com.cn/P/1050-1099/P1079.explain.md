# P1079 [NOIP 2012 提高组] Vigenère 密码 题解

## 题目链接

- [洛谷 P1079](https://www.luogu.com.cn/problem/P1079)

## 题目大意

给定密钥 $k$ 和密文 $C$，用 Vigenère 密码解密出明文 $M$。运算 $\operatorname{\circledR}$ 规则：$c_i = m_i \operatorname{\circledR} k_i$，即 $(m_i + k_i) \bmod 26$（字母表循环）。密钥循环使用。

## 解题思路

解密即为加密的逆运算：$m_i = (c_i - k_i + 26) \bmod 26$。

需要注意：
- 运算忽略大小写，但保留明文的大小写形式（与密文字母的大小写保持一致）。
- 密钥循环使用：第 $i$ 个明文字母用密钥第 $i \bmod \text{len}(k)$ 位解密。

## 代码实现

- 遍历密文，对于每个字符 $c_i$：
  - 确定对应的密钥字符 $k_{i \bmod klen}$。
  - 将密钥和密文都转换为 $[0, 25]$ 范围内的数值。
  - 计算明文数值：`pn = (cn - kn + 26) % 26`。
  - 根据密文的大小写输出对应大小写的明文。

## 复杂度分析

- 时间复杂度：$O(L)$，$L \le 1000$。
- 空间复杂度：$O(1)$。
