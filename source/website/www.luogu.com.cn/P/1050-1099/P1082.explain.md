# P1082 [NOIP 2012 提高组] 同余方程 题解

## 题目链接

- [洛谷 P1082](https://www.luogu.com.cn/problem/P1082)

## 题目大意

求关于 $x$ 的同余方程 $ax \equiv 1 \pmod b$ 的最小正整数解。

## 解题思路

$ax \equiv 1 \pmod b$ 等价于 $ax + by = 1$（裴蜀定理）。由于 $\gcd(a, b) = 1$（否则解不存在，但题目保证有解），可用扩展欧几里得算法求出一组解 $(x, y)$，则 $x \bmod b$ 即为最小正整数解。

## 代码实现

```cpp
void exgcd(int a, int b, int &x, int &y) {
    if (b == 0) { x = 1; y = 0; return; }
    exgcd(b, a % b, y, x);
    y -= a / b * x;
}
```

- 调用 `exgcd(a, b, x, y)` 得到特解。
- 最小正整数解：`(x % b + b) % b`。

## 复杂度分析

- 时间复杂度：$O(\log \min(a, b))$。
- 空间复杂度：$O(\log b)$（递归栈）。
