# P1099 [NOIP 2007 提高组] 树网的核 题解

## 题目链接

- [洛谷 P1099](https://www.luogu.com.cn/problem/P1099)

## 题目大意

给定一棵带权树和非负整数 $s$，在树的某条直径上选一段长度不超过 $s$ 的路径 $F$（两端在节点上，可退化为点），使得 $\mathrm{ECC}(F) = \max_{v \in V} D(v, F)$ 最小。$D(v, F)$ 是 $v$ 到路径 $F$ 上任意节点的最小距离。求最小偏心距。

## 解题思路

可以证明：最优的核 $F$ 一定在某条直径上（题目已限定）。算法分为两步。

### 第一步：求直径

两次 DFS 求树的直径：
1. 从任意点（如 $1$）出发 DFS，找到最远点 $A$。
2. 从 $A$ 出发 DFS，找到最远点 $B$。路径 $A \to B$ 即为一条直径。
3. 用 `pre[]` 记录前驱，回溯得到直径节点序列 `diam[0..dc-1]`。

### 第二步：枚举核并计算偏心距

对直径上每个节点 $i$，预处理：
- `dlen[i]`：直径起点到该节点的距离。
- `branch[i]`：以该节点为根、不经过直径上其他节点的子树的最大深度。

枚举直径上的区间 $[l, r]$（满足 `dlen[r] - dlen[l] <= s`），核 $F$ 的偏心距等于以下各项的最大值：

1. **左端点外**：`dlen[l]`（直径左端到 $F$ 左端的距离）
2. **右端点外**：`dlen[dc-1] - dlen[r]`（$F$ 右端到直径右端的距离）
3. **区间内分支**：$\max_{k=l}^{r} branch[k]$
4. **区间左侧分支**（若 $l > 0$）：`dlen[l] + leftMx[l]`，其中 `leftMx[l] = max_{k<l}(branch[k] - dlen[k])`
5. **区间右侧分支**（若 $r < dc-1$）：`rightMx[r] - dlen[r]`，其中 `rightMx[r] = max_{k>r}(branch[k] + dlen[k])`

所有满足条件的区间中，最小偏心距即为答案。

### 预处理优化

`leftMx` 和 `rightMx` 可分别用一次正扫和倒扫 $O(dc)$ 预处理好，避免了区间枚举时的重复计算。区间内分支的 $\max$ 直接暴力 $O(dc)$ 即可（$dc \le n \le 300$）。

## 代码实现

- `dfs1(u, fa)`：计算距离，记录前驱。
- `dfs2(u, fa, mx)`：计算分支子树的最大深度，跳过直径上的节点。
- `rangeMx(l, r)`：计算区间内 `branch[]` 的最大值。
- 枚举所有满足长度约束的区间 $[l, r]$，用预处理数组 $O(1)$ 计算偏心距。

## 复杂度分析

- 时间复杂度：$O(n^2)$（枚举区间 $O(dc^2)$ 乘以 `rangeMx` $O(dc)$），$n \le 300$，实际 $dc \le n$。
- 空间复杂度：$O(n)$。
