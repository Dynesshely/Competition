# P1091 [NOIP 2004 提高组] 合唱队形 题解

## 题目链接

- [洛谷 P1091](https://www.luogu.com.cn/problem/P1091)

## 题目大意

$n$ 位同学排成一排，求最少需要几位同学出列，使得剩下的同学身高先递增后递减（即形成"山峰"形状）。

## 解题思路

等价于求最长"山峰"子序列（LIS + LDS）。

1. 从左往右求以每个位置 $i$ 结尾的最长上升子序列长度 $lis[i]$。
2. 从右往左求以每个位置 $i$ 开头的最长下降子序列长度 $lds[i]$。
3. 以 $i$ 为"峰顶"的同学出列方案中，保留的人数为 $lis[i] + lds[i] - 1$。
4. 答案 = $n - \max_i(lis[i] + lds[i] - 1)$。

## 代码实现

- $lis[i]$：正序 DP，$lis[i] = \max_{j < i, t_j < t_i}(lis[j] + 1)$，初值 $1$。
- $lds[i]$：倒序 DP，$lds[i] = \max_{j > i, t_j < t_i}(lds[j] + 1)$，初值 $1$。
- 求最大山峰长度，输出 $n$ 减去它。

## 复杂度分析

- 时间复杂度：$O(n^2)$，$n \le 100$。
- 空间复杂度：$O(n)$。
