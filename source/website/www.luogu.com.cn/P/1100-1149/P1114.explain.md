# P1114 "非常男女"计划 题解

## 题目链接

- [洛谷 P1114](https://www.luogu.com.cn/problem/P1114)

## 题目大意

给定一个长度为 n 的 0/1 序列（0 表示女生，1 表示男生），求最长的连续子区间，使得区间内 0 和 1 的数量相等。

## 解题思路

将序列中的 0 视为 -1，1 视为 1，则原问题转化为求最长的和为 0 的子区间。计算前缀和数组 pref[i]（前 i 个元素的和），区间 (l, r] 的和为 0 当且仅当 pref[l] = pref[r]。用哈希表（或数组）记录每个前缀和值首次出现的位置，遍历时若当前前缀和之前出现过，则用当前位置与首次出现位置之差更新答案。

## 代码实现

- 0 映射为 -1，1 映射为 1，累加前缀和。
- 用 `firstPos` 数组记录每个前缀和值首次出现的下标（偏移 n 处理负数下标）。
- 初始时 `firstPos[n] = -1` 表示前缀和 0 在位置 -1 出现过。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(n)。
