## 题目描述

对于任意输入的正整数 $n$，请求出具有 $n$ 个不同因子的最小正整数 $m$。

例如：$n=4$，则 $m=6$，因为 $6$ 有 $4$ 个不同整数因子 $1,2,3,6$；而且是最小的有 $4$ 个因子的整数。

## 输入格式

输入有且仅有一行一个整数，代表 $n$。

## 输出格式

输出一行一个整数，代表符合条件的最小正整数 $m$。

## 样例 #1

### 样例输入

```
 4
```

### 样例输出

```
6
```

## 提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1 \leq n \leq 5 \times 10^4$。
