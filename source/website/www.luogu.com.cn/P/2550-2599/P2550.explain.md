# P2550 [洛谷] [AHOI2001] 彩票摇奖 题解

## 题目链接
- [洛谷 P2550](https://www.luogu.com.cn/problem/P2550)

## 题目大意
给定 7 个中奖号码（1~33 互不相同）和 n 张彩票，每张彩票有 7 个号码。统计每张彩票与中奖号码匹配的个数，分别统计特等奖（7 个匹配）到六等奖（1 个匹配）的中奖张数。

## 解题思路
用哈希数组 win 标记中奖号码。遍历每张彩票的 7 个号码，若在 win 中出现则计数加 1。匹配数 cnt 对应奖项为 7 - cnt（特等奖对应 0 号下标，一等奖对应 1 号下标，依此类推），存入结果数组对应位置累加。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, win[35], res[7];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < 7; i++) {
        int x;
        scanf("%d", &x);
        win[x] = 1;
    }
}

void process() {
    for (int i = 0; i < n; i++) {
        int cnt = 0;
        for (int j = 0; j < 7; j++) {
            int x;
            scanf("%d", &x);
            if (win[x])
                cnt++;
        }
        if (cnt)
            res[7 - cnt]++;
    }
}

void output() {
    for (int i = 0; i < 7; i++) {
        if (i)
            printf(" ");
        printf("%d", res[i]);
    }
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * 7)
- 空间复杂度：O(1)
