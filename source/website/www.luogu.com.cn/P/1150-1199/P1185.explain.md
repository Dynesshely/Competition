# P1185 [洛谷] 绘制二叉树 题解

## 题目链接
- [洛谷 P1185](https://www.luogu.com.cn/problem/P1185)

## 题目大意

给定一棵 m 层满二叉树（高度为 m-1），要求绘制其结构图并删除指定的 n 个结点（包括其子树及与父亲的连接）。结点用 `o` 表示，左子树用 `/` 连接，右子树用 `\` 连接。

## 解题思路

模拟构建满二叉树的字符画布局：

1. **行坐标计算**：`row[i]` 表示第 i 层结点的行号。从第 2 层开始递推，非末层结点之间的间距取决于子树的宽度。
2. **列坐标计算**：
   - 第 m 层（叶子层）：每个叶子结点占据固定宽度（4 字符间隔），属于同一父亲的叶子间间隔 3 空格，不同父亲的相邻叶子间间隔 1 空格。
   - 非叶子层：结点的列坐标为左右子结点列坐标的中点。
3. 删除结点时，将 `del[i][j]` 标记为 true，该结点及其子树不被绘制，连接线也跳过。
4. 最后将字符画输出，删除结点处以空格替代。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int m, n;
bool del[11][1024];
int col[11][1024];
int row[11];
char g[800][1600];

void delSub(int layer, int pos) {
    if (layer > m) return;
    del[layer][pos] = true;
    delSub(layer + 1, 2 * pos - 1);
    delSub(layer + 1, 2 * pos);
}

void input() {
    scanf("%d%d", &m, &n);
    for (int i = 0; i < n; i++) {
        int a, b;
        scanf("%d%d", &a, &b);
        delSub(a, b);
    }
}

void process() {
    row[1] = 0;
    for (int i = 2; i <= m; i++)
        if (i == m)
            row[i] = row[i - 1] + 2;
        else
            row[i] = row[i - 1] + 3 * (1 << (m - i - 1));
    int leafCnt = 1 << (m - 1);
    for (int j = 1; j <= leafCnt; j++) {
        int idx = j - 1;
        col[m][j] = (idx / 2) * 6 + (idx % 2) * 4;
    }
    for (int i = m - 1; i >= 1; i--) {
        int nodeCnt = 1 << (i - 1);
        for (int j = 1; j <= nodeCnt; j++)
            col[i][j] = (col[i + 1][2 * j - 1] + col[i + 1][2 * j]) / 2;
    }
    int H = row[m] + 1;
    int W = 6 * (1 << (m - 2)) - 1;
    for (int r = 0; r < H; r++)
        for (int c = 0; c < W; c++)
            g[r][c] = ' ';
    for (int i = 1; i <= m; i++) {
        int nodeCnt = 1 << (i - 1);
        for (int j = 1; j <= nodeCnt; j++) {
            if (del[i][j]) continue;
            int r = row[i], c = col[i][j];
            g[r][c] = 'o';
            if (i < m) {
                int nxt = row[i + 1] - row[i];
                int lc = 2 * j - 1, rc = 2 * j;
                if (!del[i + 1][lc])
                    for (int k = 1; k < nxt; k++)
                        g[r + k][c - k] = '/';
                if (!del[i + 1][rc])
                    for (int k = 1; k < nxt; k++)
                        g[r + k][c + k] = '\\';
            }
        }
    }
}

void output() {
    int H = row[m] + 1;
    int W = 6 * (1 << (m - 2)) - 1;
    for (int r = 0; r < H; r++) {
        for (int c = 0; c < W; c++)
            printf("%c", g[r][c]);
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(2^m)，满二叉树结点数随层数指数增长。
- 空间复杂度：O(2^m * m + H * W)，存储删除标记、行列坐标及字符画网格。
