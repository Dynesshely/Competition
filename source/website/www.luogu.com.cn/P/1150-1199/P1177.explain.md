# P1177 [洛谷] 【模板】排序 题解

## 题目链接
- [洛谷 P1177](https://www.luogu.com.cn/problem/P1177)

## 题目大意

给定 N 个正整数，将它们从小到大排序后输出。

## 解题思路

使用 C++ 标准库中的 `sort` 函数对数组进行升序排序。这是快速排序的模板题，使用 STL 即可通过。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, a[100005];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i)
        scanf("%d", &a[i]);
}

void process() { sort(a, a + n); }

void output() {
    for (int i = 0; i < n; ++i)
        printf("%d ", a[i]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(N log N)，`sort` 基于内省排序（快速排序 + 堆排序 + 插入排序）。
- 空间复杂度：O(log N)，递归调用栈的额外空间。
