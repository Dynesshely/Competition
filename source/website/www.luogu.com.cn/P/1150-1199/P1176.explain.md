# P1176 [洛谷] 路径计数2 题解

## 题目链接
- [洛谷 P1176](https://www.luogu.com.cn/problem/P1176)

## 题目大意

在一个 N x N 的网格中，从左上角 (1,1) 出发，每次只能向下或向右移动一格，求到达右下角 (N,N) 的不同路径数。网格中有 M 个格点有障碍物，无法通行。结果对 100003 取模。

## 解题思路

经典网格 DP，使用一维滚动数组优化空间：

- 设 `dp[j]` 表示从起点到达当前行第 j 列的路径数。
- 按行逐列递推：`dp[j] = (dp[j] + dp[j-1]) % MOD`，其中 `dp[j]` 是来自上方的路径数，`dp[j-1]` 是来自左方的路径数。
- 遇到障碍物时，将 `dp[j]` 置为 0。
- 边界初始化 `dp[1] = 1`。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD = 100003;
const int MAXN = 1000;

int n, m;
bool blocked[MAXN + 2][MAXN + 2];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; i++) {
        int x, y;
        scanf("%d%d", &x, &y);
        blocked[x][y] = true;
    }
}

void process() {
    if (n == 0) {
        printf("0\n");
        return;
    }
    int dp[MAXN + 2] = {0};
    dp[1] = 1;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (blocked[i][j]) {
                dp[j] = 0;
            } else {
                if (j > 1)
                    dp[j] = (dp[j] + dp[j - 1]) % MOD;
            }
        }
    }
    printf("%d\n", dp[n]);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(N^2)，需要遍历整个 N x N 网格。
- 空间复杂度：O(N^2 + M) 用于存储障碍物标记，DP 数组为 O(N)。
