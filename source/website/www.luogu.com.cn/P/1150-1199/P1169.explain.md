# P1169 [ZJOI2007] 棋盘制作 题解

## 题目链接
- [洛谷 P1169](https://www.luogu.com.cn/problem/P1169)

## 题目大意
给定一个 N x M 的 01 矩阵，0 表示白色，1 表示黑色。求最大的正方形和最大的矩形，使得其内部相邻格子颜色交替（即类似国际象棋棋盘的黑白相间）。

## 解题思路
棋盘黑白相间的条件是：对于任意相邻格子 (i,j) 和 (i,j+1) 或 (i+1,j)，它们的颜色不同，等价于在标准的棋盘染色下，位置 (i,j) 的颜色与矩阵中的值相同。因此，可以通过变换将棋盘问题转化为全 1 子矩阵问题。

具体做法：
1. 对矩阵进行变换：若 (i+j) 为奇数，则将该位置颜色取反（1 变 0，0 变 1）。变换后，原问题等价于求全 1 的最大正方形和最大矩形。
2. 求最大全 1 正方形：使用动态规划，`dp[i][j]` 表示以 (i,j) 为右下角的最大全 1 正方形边长。状态转移：若 `mat[i][j] == 1`，则 `dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`。
3. 求最大全 1 矩形：按行处理，维护每个位置向上连续 1 的高度 `height[j]`，然后对每一行使用单调栈计算该行作为底边的最大矩形面积。
4. 将矩阵整体取反（1 变 0，0 变 1），重复步骤 2-3，取两次结果的最大值（因为棋盘染色有两种可能的起点颜色）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;
int a[2005][2005];
int dp[2005][2005];
int height[2005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &a[i][j]);
}

int maxSquare(int mat[][2005], int n, int m) {
    int maxSide = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++) {
            if (mat[i][j] == 1) {
                if (i == 0 || j == 0)
                    dp[i][j] = 1;
                else
                    dp[i][j] = min(min(dp[i - 1][j], dp[i][j - 1]), dp[i - 1][j - 1]) + 1;
                if (dp[i][j] > maxSide)
                    maxSide = dp[i][j];
            } else
                dp[i][j] = 0;
        }
    return maxSide * maxSide;
}

int maxRect(int mat[][2005], int n, int m) {
    memset(height, 0, sizeof(height));
    int maxArea = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++)
            if (mat[i][j] == 1)
                height[j]++;
            else
                height[j] = 0;
        stack<int> st;
        for (int j = 0; j <= m; j++) {
            int h = (j == m) ? 0 : height[j];
            while (!st.empty() && h <= height[st.top()]) {
                int ht = height[st.top()];
                st.pop();
                int w = st.empty() ? j : j - st.top() - 1;
                if (ht * w > maxArea)
                    maxArea = ht * w;
            }
            if (j < m)
                st.push(j);
        }
    }
    return maxArea;
}

void process() {
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            if ((i + j) & 1)
                a[i][j] = 1 - a[i][j];
    int sq1   = maxSquare(a, n, m);
    int rect1 = maxRect(a, n, m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            a[i][j] = 1 - a[i][j];
    int sq2   = maxSquare(a, n, m);
    int rect2 = maxRect(a, n, m);
    printf("%d\n%d\n", max(sq1, sq2), max(rect1, rect2));
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(NM)，其中 N、M 为矩阵的行数和列数。DP 求正方形 O(NM)，单调栈求矩形 O(NM)，两次变换 O(NM)。
- 空间复杂度：O(NM)，存储矩阵及 DP 数组。
