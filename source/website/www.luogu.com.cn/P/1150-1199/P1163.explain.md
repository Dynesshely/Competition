# P1163 银行贷款 题解

## 题目链接
- [洛谷 P1163](https://www.luogu.com.cn/problem/P1163)

## 题目大意
已知贷款原值 w0、每月还款额 w 和还款月数 m，按月计复利，求月利率（百分数形式，精确到 0.1%）。

## 解题思路
使用二分答案求解月利率。月利率必然在 0% 到 300% 之间。

对于给定的月利率 r，每月剩余贷款按如下方式计算：
- 月初欠款额乘以 (1 + r) 产生利息，然后扣除当月还款 w。
- 若最后欠款 <= 0，说明利率偏大；否则偏小。

利用等比数列求和公式可以加速计算：贷款现值 pv = w * (1 - (1 + r)^(-m)) / r。
若 pv > w0，说明利率偏大（同样的还款能覆盖更多本金）；反之偏小。

二分 100 次达到足够精度。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

double w0, w;
int    m;

void input() { scanf("%lf%lf%d", &w0, &w, &m); }

void process() {
    if (w * m <= w0 + 1e-12) {
        printf("0.0");
        return;
    }
    double lo = 0, hi = 3.0;
    for (int i = 0; i < 100; i++) {
        double mid = (lo + hi) / 2;
        double pm  = exp(-m * log(1 + mid));
        double pv  = w * (1 - pm) / mid;
        if (pv > w0)
            lo = mid;
        else
            hi = mid;
    }
    printf("%.1f", (lo + hi) / 2 * 100);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(log(1/eps))，二分固定迭代 100 次。
- 空间复杂度：O(1)。
