# P1198 [JSOI2008] 最大数 题解

## 题目链接
- [洛谷 P1198](https://www.luogu.com.cn/problem/P1198)

## 题目大意
维护一个数列，初始为空。支持两种操作：
- `A n`：将 n 加上最近一次查询的答案 t（初始为 0），对 D 取模后插入到数列末尾。
- `Q L`：查询当前数列中末尾 L 个数的最大值。

## 解题思路
使用线段树。

- 最多 M 次操作，数列长度不会超过 M，可预先开大小为 M 的线段树。
- 维护变量 `len` 表示当前数列长度。
- `A` 操作：计算 `val = (n + lastAns) % D`，在位置 `len + 1` 处单点更新。
- `Q` 操作：查询区间 `[len - L + 1, len]` 的最大值，更新 `lastAns` 并输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int       m, d, len;
long long tree[800005];
long long lastAns;

void update(int node, int l, int r, int pos, long long val) {
    if (l == r) {
        tree[node] = val;
        return;
    }
    int mid = (l + r) >> 1;
    if (pos <= mid)
        update(node << 1, l, mid, pos, val);
    else
        update(node << 1 | 1, mid + 1, r, pos, val);
    tree[node] = max(tree[node << 1], tree[node << 1 | 1]);
}

long long query(int node, int l, int r, int ql, int qr) {
    if (ql <= l && r <= qr)
        return tree[node];
    int       mid = (l + r) >> 1;
    long long res = -1e18;
    if (ql <= mid)
        res = max(res, query(node << 1, l, mid, ql, qr));
    if (qr > mid)
        res = max(res, query(node << 1 | 1, mid + 1, r, ql, qr));
    return res;
}

int main() {
    scanf("%d%d", &m, &d);
    char      op[2];
    long long x;
    for (int i = 0; i < m; ++i) {
        scanf("%s%lld", op, &x);
        if (op[0] == 'A') {
            ++len;
            long long val = (x + lastAns) % d;
            if (val < 0)
                val += d;
            update(1, 1, m, len, val);
        } else {
            lastAns = query(1, 1, m, len - x + 1, len);
            printf("%lld\n", lastAns);
        }
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(M log M)，每次操作 O(log M)。
- 空间复杂度：O(M)。
