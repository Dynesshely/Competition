# P1162 填涂颜色 题解

## 题目链接
- [洛谷 P1162](https://www.luogu.com.cn/problem/P1162)

## 题目大意
在 n x n 的 0/1 方阵中，1 构成一个闭合圈。要求将闭合圈内所有 0 替换为 2。闭合圈内的 0 无法通过上下左右移动到达边界。

## 解题思路
从边界上的所有 0 开始进行 DFS/BFS，标记所有能到达的 0（即闭合圈外的 0）。最后遍历整个方阵，将未被标记的 0 改为 2。

具体步骤：
1. 对四周边界上的 0 分别进行 DFS，标记所有与边界连通的 0。
2. 遍历方阵，将标记过的 0 保持为 0，未标记的 0 改为 2，1 保持不变。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, a[35][35];
bool via[35][35];
int  dx[] = {-1, 0, 1, 0}, dy[] = {0, 1, 0, -1};

void dfs(int x, int y) {
    via[x][y] = true;
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i], ny = y + dy[i];
        if (nx >= 0 && nx < n && ny >= 0 && ny < n && !via[nx][ny] && a[nx][ny] == 0)
            dfs(nx, ny);
    }
}

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            scanf("%d", &a[i][j]);
}

void process() {
    for (int i = 0; i < n; i++) {
        if (a[i][0] == 0 && !via[i][0])
            dfs(i, 0);
        if (a[i][n - 1] == 0 && !via[i][n - 1])
            dfs(i, n - 1);
    }
    for (int j = 0; j < n; j++) {
        if (a[0][j] == 0 && !via[0][j])
            dfs(0, j);
        if (a[n - 1][j] == 0 && !via[n - 1][j])
            dfs(n - 1, j);
    }
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            if (a[i][j] == 0 && !via[i][j])
                a[i][j] = 2;
}

void output() {
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            printf("%d%c", a[i][j], j == n - 1 ? '\n' : ' ');
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n^2)。
- 空间复杂度：O(n^2)。
