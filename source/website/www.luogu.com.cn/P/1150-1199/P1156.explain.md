# P1156 [USACO01OPEN] 垃圾陷阱 题解

## 题目链接
- [洛谷 P1156](https://www.luogu.com.cn/problem/P1156)

## 题目大意
奶牛卡门掉进深度为 D 的垃圾井中。每个垃圾在给定时间 t 被扔下，可以吃掉增加 f 小时生命，或堆放增加 h 高度。初始生命值为 10 小时。若堆放的垃圾总高度 >= D，则卡门可以逃出。求最早逃出时间，若无法逃出则求最长存活时间。

## 解题思路
按时间顺序处理垃圾，使用 DP 求解。

定义 dp[j] 表示在堆放高度为 j 时，卡门能拥有的最大生命值（剩余时间）。初始 dp[0] = 10。

对每个垃圾（按时间排序）：
- 如果当前生命值 dp[j] >= 垃圾掉落时间 t，说明能活到该垃圾落下。
- 可以选择堆放：j + h >= D 则直接输出 t 并结束。
- 否则更新 dp[j + h] = max(dp[j + h], dp[j])。
- 可以选择吃掉：dp[j] += f。

若最终未能逃出，则输出 max(dp[0..D-1]) 作为最长存活时间。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Garbage {
    int t, f, h;
};

int     D, G;
Garbage garb[105];
int     dp[200];

void input() {
    scanf("%d%d", &D, &G);
    for (int i = 0; i < G; i++)
        scanf("%d%d%d", &garb[i].t, &garb[i].f, &garb[i].h);
}

void process() {
    sort(garb, garb + G, [](Garbage a, Garbage b) { return a.t < b.t; });
    memset(dp, 0, sizeof(dp));
    dp[0] = 10;
    for (int i = 0; i < G; i++) {
        int t = garb[i].t, f = garb[i].f, h = garb[i].h;
        for (int j = D - 1; j >= 0; j--) {
            if (dp[j] >= t) {
                int life = dp[j];
                if (j + h >= D) {
                    printf("%d\n", t);
                    return;
                }
                dp[j + h] = max(dp[j + h], life);
                dp[j]     = life + f;
            }
        }
    }
    int ans = 0;
    for (int i = 0; i < D; i++)
        ans = max(ans, dp[i]);
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(G * D)，G <= 100, D <= 100。
- 空间复杂度：O(D)。
