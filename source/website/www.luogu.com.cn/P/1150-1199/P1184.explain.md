# P1184 [洛谷] 高手之在一起 题解

## 题目链接
- [洛谷 P1184](https://www.luogu.com.cn/problem/P1184)

## 题目大意

高手有 n 个方便去的地点。女孩有 m 天的日程安排，每天在一个地点。求高手最多能和她在一起多少天（即女孩所在的地点出现在高手方便去的地点集合中的天数）。

地点名称中可能包含空格。

## 解题思路

用哈希集合（`unordered_set`）存储高手方便去的所有地点，然后逐天判断女孩所在地点是否在集合中，统计匹配的天数。

注意输入可能包含空格，需要使用 `fgets` 读取整行。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, ans;
char buf[1005];
unordered_set<string> via;

void input() {
    scanf("%d%d", &n, &m);
    fgets(buf, sizeof(buf), stdin);
    for (int i = 0; i < n; i++) {
        fgets(buf, sizeof(buf), stdin);
        int len = strlen(buf);
        while (len > 0 && (buf[len - 1] == '\n' || buf[len - 1] == '\r'))
            buf[--len] = 0;
        via.insert(buf);
    }
}

void process() {
    for (int i = 0; i < m; i++) {
        fgets(buf, sizeof(buf), stdin);
        int len = strlen(buf);
        while (len > 0 && (buf[len - 1] == '\n' || buf[len - 1] == '\r'))
            buf[--len] = 0;
        if (via.count(buf))
            ans++;
    }
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(n + m)，插入和查询的平均时间复杂度均为 O(1)。
- 空间复杂度：O(n)，存储方便去的地点集合。
