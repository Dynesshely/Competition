# P2955 [洛谷] Even? Odd? G 题解

## 题目链接
- [洛谷 P2955](https://www.luogu.com.cn/problem/P2955)

## 题目大意
给定 N 个正整数（可达 10^60），判断每个数的奇偶性。

## 解题思路
整数过大无法用内置整数类型存储，用字符串读入，判断末位数字的奇偶性即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  n;
char s[65];

int main() {
    scanf("%d", &n);
    while (n--) {
        scanf("%s", s);
        int len = strlen(s);
        if ((s[len - 1] - '0') & 1)
            puts("odd");
        else
            puts("even");
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * L)，L 为数字字符串长度。
- 空间复杂度：O(1)。
