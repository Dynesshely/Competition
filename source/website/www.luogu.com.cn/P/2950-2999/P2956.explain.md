# P2956 [洛谷] The Robot Plow G 题解

## 题目链接
- [洛谷 P2956](https://www.luogu.com.cn/problem/P2956)

## 题目大意
X*Y 的网格田地，给定 I 个矩形耕地指令（左下角和右上角坐标），求被犁过的不同方格总数。

## 解题思路
开一个布尔数组标记每个格子是否已被犁过，对每条指令遍历矩形区域内所有格子，未犁过的计数并标记。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  x, y, i, ans;
bool via[241][241];

int main() {
    scanf("%d%d%d", &x, &y, &i);
    while (i--) {
        int xll, yll, xur, yur;
        scanf("%d%d%d%d", &xll, &yll, &xur, &yur);
        for (int r = xll; r <= xur; ++r)
            for (int c = yll; c <= yur; ++c)
                if (!via[r][c]) {
                    via[r][c] = true;
                    ++ans;
                }
    }
    printf("%d\n", ans);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(I * X * Y)，最坏 200 * 240 * 240。
- 空间复杂度：O(X * Y)。
