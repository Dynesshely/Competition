# P2907 [洛谷] Roads Around The Farm S 题解

## 题目链接
- [洛谷 P2907](https://www.luogu.com.cn/problem/P2907)

## 题目大意
初始有 N 头牛一起沿路行走，遇到岔路时若可分成大小恰相差 K 的两个非空群组则分裂，否则停止。求最终群组数量。

## 解题思路
递归模拟分裂过程。对于一个大小为 x 的群组，若 x <= K 则无法分裂，返回 1；若 (x - K) 为奇数则无法均分，返回 1；否则分裂为 (x+K)/2 和 (x-K)/2 两部分，递归计算总和。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, k;

int dfs(int x) {
    if (x <= k)
        return 1;
    if ((x - k) & 1)
        return 1;
    return dfs((x + k) / 2) + dfs((x - k) / 2);
}

int main() {
    scanf("%d%d", &n, &k);
    printf("%d\n", dfs(n));
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N)，最坏情况下每次分裂规模减半，递归深度 O(log N)。
- 空间复杂度：O(log N)，递归栈空间。
