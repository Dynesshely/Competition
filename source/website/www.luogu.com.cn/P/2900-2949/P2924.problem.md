## 题目描述

Farmer John 的农场里有 $N$（$5\le N\le250$）个篱笆桩，每个都有独一无二的坐标 $(x_i,y_i)$（$1\le x_i,y_i \le1000$）。他想选择尽量多的篱笆桩来构建他的围栏。这个围栏要美观，所以必须是凸多边形的。那他最多能选多少个呢？

所有的篱笆桩中不存在三点共线。

## 输入格式

第一行一个整数 $N$。

接下来的 $N$ 行，每行包含两个整数 $x_i,y_i$，表示第 $i$ 个篱笆桩的坐标。

## 输出格式

输出一个整数，为构成凸多边形的最大顶点数。

## 样例 #1

### 样例输入

```
6 
5 5 
2 3 
3 2 
1 5 
5 1 
1 1
```

### 样例输出

```
5
```

## 提示

样例构成的图形可以理解为一个正方形，其内部有两个点。
能够围成的最大凸多边形是五边形，其顶点依次为 $(2,3),(3,2),(5,1),(5,5),(1,5)$。

对于 $45\%$ 的数据，保证 $N\le 65$。
