# P2911 [洛谷] Bovine Bones G 题解

## 题目链接
- [洛谷 P2911](https://www.luogu.com.cn/problem/P2911)

## 题目大意
三个骰子分别有 S1、S2、S3 个面，求所有投掷组合中三数之和出现频率最高的值，若有多个输出最小的。

## 解题思路
三重循环枚举所有可能的组合，统计每个和出现的次数，记录出现次数最多的最小和。和的范围为 3 到 S1+S2+S3。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int s1, s2, s3, cnt[81], ans, maxCnt;

int main() {
    scanf("%d%d%d", &s1, &s2, &s3);
    for (int i = 1; i <= s1; ++i)
        for (int j = 1; j <= s2; ++j)
            for (int k = 1; k <= s3; ++k)
                ++cnt[i + j + k];
    for (int i = 3; i <= s1 + s2 + s3; ++i)
        if (cnt[i] > maxCnt) {
            maxCnt = cnt[i];
            ans    = i;
        }
    printf("%d\n", ans);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(S1 * S2 * S3)，不超过 20 * 20 * 40 = 16000。
- 空间复杂度：O(1)，固定大小计数数组。
