# P2934 [USACO09JAN] Safe Travel G 题解

## 题目链接

- [洛谷 P2934](https://www.luogu.com.cn/problem/P2934)

## 题目大意

给定 $N$ 个节点 $M$ 条边的无向带权图，求从每个节点 $i$ 到节点 $1$ 的**次短路径**（不允许与最短路径的最后一条边相同）。

## 解题思路

**Dijkstra + 最短路径树 + DSU**。

1. 以节点 $1$ 为源跑 Dijkstra，得到最短距离 $dis[i]$ 和最短路径树（SPT）。
2. 收集所有**非树边** $(u, v, w)$，计算若经过这条边的路径长度：$dis[u] + w + dis[v]$。
3. 按此长度升序排列非树边。
4. 对于每条非树边 $(u, v)$，它对 SPT 上 $u \to v$ 路径上所有节点的次短路径有贡献。用 DSU 自底向上跳，对尚未确定答案的节点赋值，然后合并。

DSU 的合并方向沿 SPT 向根（dis 更小的方向），每个节点只会被赋值一次。

## 代码实现

- Dijkstra 用优先队列 $O(M \log N)$。
- DSU 维护「下一个未赋值的祖先」，路径压缩加速跳跃。

## 复杂度分析

- 时间复杂度：$O(M \log N + M \alpha(N))$。
- 空间复杂度：$O(N + M)$。
