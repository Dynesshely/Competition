# P1159 排行榜 题解

## 题目链接
- [洛谷 P1159](https://www.luogu.com.cn/problem/P1159)

## 题目大意
有 N 首歌曲，每首歌曲相对于上周排行有 UP（上升）、DOWN（下降）或 SAME（不变）的标记。根据本周排行给出这些标记，推断上周可能的排行榜（即给出一个合法的名次排列）。

## 解题思路
构造一个可行的上周排名：

1. 先处理 SAME 的歌曲，它们上周和本周位置相同，直接放在对应位置。
2. 处理 UP 歌曲：本周排名上升，说明上周排名比本周靠后（数值更大）。将 UP 歌曲从后往前插入到第一个空位。
3. 处理 DOWN 歌曲：本周排名下降，说明上周排名比本周靠前（数值更小）。将 DOWN 歌曲从前往后插入到第一个空位。

这样构造出来的排列一定满足每个歌曲的相对变化要求。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int  n;
char name[105][105];
char stat[105][10];
int  ans[105];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%s%s", name[i], stat[i]);
}

void process() {
    for (int i = 0; i < n; i++)
        ans[i] = -1;
    for (int i = 0; i < n; i++)
        if (stat[i][0] == 'S')
            ans[i] = i;
    vector<pair<int, int>> up, down;
    for (int i = 0; i < n; i++) {
        if (stat[i][0] == 'U')
            up.push_back({i, i});
        if (stat[i][0] == 'D')
            down.push_back({i, i});
    }
    sort(up.rbegin(), up.rend());
    for (auto &p : up) {
        int cur = p.first;
        for (int j = n - 1; j > cur; j--)
            if (ans[j] == -1) {
                ans[j] = cur;
                break;
            }
    }
    sort(down.begin(), down.end());
    for (auto &p : down) {
        int cur = p.first;
        for (int j = 0; j < cur; j++)
            if (ans[j] == -1) {
                ans[j] = cur;
                break;
            }
    }
}

void output() {
    for (int i = 0; i < n; i++)
        printf("%s\n", name[ans[i]]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N^2)，N <= 100。
- 空间复杂度：O(N^2)。
