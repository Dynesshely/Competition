# B2029 大象喝水 题解

## 题目链接
- [洛谷 B2029](https://www.luogu.com.cn/problem/B2029)

## 题目大意
计算一只大象用底面半径为 r 厘米、深 h 厘米的圆柱形小桶至少需要喝多少桶才能凑满 20 升水。

## 解题思路
单桶体积 V = pi * r^2 * h（立方厘米），pi 取 3.14。大象需要 20 升 = 20000 立方厘米。答案即为 ceil(20000 / V)。为避免浮点误差，将 pi 放大 100 倍为 314，总水量也对应放大 100 倍为 2000000，做整数上取整除法即可。

## 代码实现
使用 input(), process(), output() 三段式结构。process() 中计算分母 `den = 314 * r * r * h`，再用 `(2000000 + den - 1) / den` 实现上取整。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
