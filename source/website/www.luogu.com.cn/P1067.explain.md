# P1067 [NOIP 2009 普及组] 多项式输出 题解

## 题目链接

- [洛谷 P1067](https://www.luogu.com.cn/problem/P1067)

## 题目大意

给定一个一元 $n$ 次多项式各项的系数，按规定的格式输出多项式字符串。规则包括：系数为 0 的项省略、系数为正负 1 时省略数字 1、指数为 1 时省略指数、最高次项系数为正时不加 `+` 等。

## 解题思路

从高次项到低次项遍历，对每一项根据系数和指数进行格式化输出。

关键规则：
1. 系数为 0 的项直接跳过。
2. 除了最高次项，系数为正的项前需要加 `+`。
3. 系数为 $+1$ 且指数 $\ge 1$ 时，只输出 `x` 或 `x^b`，省略系数 1。
4. 系数为 $-1$ 且指数 $\ge 1$ 时，输出 `-x` 或 `-x^b`，省略系数 1。
5. 指数为 1 时只输出 `x`，不输出 `^1`。
6. 指数为 0 时（常数项），只输出系数本身。

## 代码实现

- 使用动态数组 `a` 存储系数，下标从 0 到 $n$。
- 使用变量 `z` 从 $n$ 递减到 0 表示当前项的次数。
- `highestPrinted` 标记是否已输出过非零项，用于控制非首项的正号输出。
- 分类讨论：
  - `z == 0`：常数项，直接输出系数（注意正号控制）。
  - `a[i] == 1`：仅输出 `x` 或 `x^z`。
  - `a[i] == -1`：输出 `-x` 或 `-x^z`。
  - 其他情况：输出 `a[i]x` 或 `a[i]x^z`。
- 最后换行并释放动态数组。

> 注意：代码中 `rst = a[i] * pow(x, z)` 本意可能是用于判断该多项式项的值，但实际上是恒等于 $a[i]$（因为 `x=1`），所以仅用来判断系数是否为 0。这里的实现有一种简化 trick：令 `x = 1`，则 `a[i] * 1^z = a[i]`，相当于直接用系数判断。

## 复杂度分析

- 时间复杂度：$O(n)$，遍历一遍所有系数。
- 空间复杂度：$O(n)$，存储系数数组。
