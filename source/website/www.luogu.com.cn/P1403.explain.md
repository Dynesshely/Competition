# P1403 [AHOI2005] 约数研究 题解

## 题目链接
- [洛谷 P1403](https://www.luogu.com.cn/problem/P1403)

## 题目大意
给定 $n$，求 $\sum_{i=1}^{n} f(i)$，其中 $f(i)$ 表示 $i$ 的约数个数。

## 解题思路
将求和式换个角度考虑：对于每个正整数 $d$，它在 $1 \sim n$ 中作为约数出现的次数为 $\lfloor n/d \rfloor$。因此：
$$
\sum_{i=1}^{n} f(i) = \sum_{d=1}^{n} \left\lfloor \frac{n}{d} \right\rfloor
$$
直接 $O(n)$ 累加即可，$n \le 10^6$ 完全可行。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int       n;
long long ans;

void input() { scanf("%d", &n); }

void process() {
    ans = 0;
    for (int i = 1; i <= n; i++)
        ans += n / i;
}

void output() { printf("%lld\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$。
- 空间复杂度：$O(1)$。
