# P1374 进攻幽暗城 题解

## 题目链接
- [洛谷 P1374](https://www.luogu.com.cn/problem/P1374)

## 题目大意
$n \times m$ 网格（$0$ 可通行，$1$ 为障碍），小 A 从起点 $(x_1, y_1)$ 出发，要去终点 $(x_2, y_2)$，每秒可向四个方向走一格或不动。萨尔和恐惧魔王分别有循环移动指令串（$0$ 不动，$1$ 上，$2$ 下，$3$ 左，$4$ 右，出界或撞墙则不动）。小 A 离开萨尔光环（距离 $> d$）不能超过 $s$ 秒。求到达终点的最短时间。

## 解题思路
BFS/DP 模拟。预计算前 $MAXT$ 秒萨尔和恐惧魔王的位置（循环指令串）。使用 `dis[x][y]` 表示到达 $(x, y)$ 时已脱离光环的连续秒数（若 $\le s$ 则为有效状态）。

逐秒 DP 转移：
- 从当前位置向 5 个方向（含不动）尝试移动。
- 计算移动后位置与萨尔下一时刻位置的距离平方。
- 若距离平方 $\le d^2$（在光环内），则脱离计数归零；否则脱离计数 $+1$。
- 若脱离计数 $\le s$，则状态有效。
- 若某时刻可达恐惧魔王位置，则输出当前时间。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n, m, s, d;
char   grid[55][55];
int    sx, sy, ex, ey;
string tMov, dMov;
int    tLen, dLen;
int    dx[] = {0, -1, 1, 0, 0};
int    dy[] = {0, 0, 0, -1, 1};

struct Pos {
    int x, y;
};

Pos movePos(Pos p, char c) {
    int idx = c - '0';
    if (idx == 0)
        return p;
    int nx = p.x + dx[idx];
    int ny = p.y + dy[idx];
    if (nx < 0 || nx >= n || ny < 0 || ny >= m)
        return p;
    if (grid[nx][ny] == '1')
        return p;
    return {nx, ny};
}

void input() {
    scanf("%d%d%d%d", &n, &m, &s, &d);
    for (int i = 0; i < n; i++)
        scanf("%s", grid[i]);
    scanf("%d%d%d%d", &sx, &sy, &ex, &ey);
    sx--; sy--; ex--; ey--;
    cin >> tMov >> dMov;
    tLen = tMov.length();
    dLen = dMov.length();
}

void process() {
    const int MAXT = 200;
    vector<Pos> tPos(MAXT + 1), dPos(MAXT + 1);
    tPos[0] = {sx, sy};
    dPos[0] = {ex, ey};
    for (int i = 0; i < MAXT; i++) {
        tPos[i + 1] = movePos(tPos[i], tMov[i % tLen]);
        dPos[i + 1] = movePos(dPos[i], dMov[i % dLen]);
    }

    const int INF = 1e9;
    vector<vector<int>> dis(n, vector<int>(m, INF));
    dis[sx][sy] = 0;
    if (sx == ex && sy == ey) {
        printf("0\n");
        return;
    }

    int d2 = d * d;
    for (int t = 0; t < MAXT; t++) {
        vector<vector<int>> nd(n, vector<int>(m, INF));
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                if (dis[i][j] == INF) continue;
                for (int dir = 0; dir < 5; dir++) {
                    int nx = i + dx[dir], ny = j + dy[dir];
                    if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
                    if (grid[nx][ny] == '1') continue;
                    int dsq = (nx - tPos[t + 1].x) * (nx - tPos[t + 1].x) + (ny - tPos[t + 1].y) * (ny - tPos[t + 1].y);
                    int nk = (dsq <= d2) ? 0 : (dis[i][j] + 1);
                    if (nk > s) continue;
                    if (nk < nd[nx][ny]) nd[nx][ny] = nk;
                }
            }
        }
        if (nd[dPos[t + 1].x][dPos[t + 1].y] < INF) {
            printf("%d\n", t + 1);
            return;
        }
        dis = move(nd);
    }
    printf("-1\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(MAXT \times n \times m \times 5)$，$MAXT = 200$
- 空间复杂度：$O(nm)$
