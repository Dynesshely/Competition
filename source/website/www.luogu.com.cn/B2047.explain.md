# B2047 分段函数 题解
## 题目链接
- [洛谷 B2047](https://www.luogu.com.cn/problem/B2047)
## 题目大意
给定一个浮点数 x (0 <= x < 20)，计算分段函数 y = f(x) 的值，结果保留三位小数。
## 解题思路
按照 x 所在的三个区间分别计算：x 在 [0,5) 时 y = -x + 2.5；在 [5,10) 时 y = 2 - 1.5(x-3)^2；在 [10,20) 时 y = x/2 - 1.5。直接分支判断即可。
## 代码实现
使用 if-else if-else 判断 x 落在哪个区间，代入对应公式计算并保留三位小数输出。
## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
