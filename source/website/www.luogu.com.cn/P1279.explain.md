# P1279 [CHCI 2002 National Competition #2 Seniors] 字串距离 题解

## 题目链接
- [洛谷 P1279](https://www.luogu.com.cn/problem/P1279)

## 题目大意
定义两个字符串的距离：可以插入任意空格使两串等长，对应位置字符的 ASCII 差的绝对值为距离，空格与字符距离为定值 K，空格与空格距离为 0。求最小距离。

## 解题思路
经典编辑距离 DP。dp[i][j] 表示 A 的前 i 个字符与 B 的前 j 个字符的最小距离：
- 边界：dp[i][0] = i * K（A 全匹配空格），dp[0][j] = j * K。
- 转移：考虑三种情况
  1. A[i] 匹配 B[j]：代价 = |A[i] - B[j]|。
  2. A[i] 匹配空格：代价 = K。
  3. 空格匹配 B[j]：代价 = K。
  取三者最小值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  k;
char a[2010], b[2010];
int  n, m;
int  dp[2010][2010];

void input() {
    scanf("%s%s%d", a + 1, b + 1, &k);
    n = strlen(a + 1);
    m = strlen(b + 1);
}

void process() {
    for (int i = 1; i <= n; i++)
        dp[i][0] = dp[i - 1][0] + k;
    for (int j = 1; j <= m; j++)
        dp[0][j] = dp[0][j - 1] + k;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            dp[i][j] = min(dp[i - 1][j - 1] + abs(a[i] - b[j]), min(dp[i - 1][j] + k, dp[i][j - 1] + k));
}

void output() { printf("%d\n", dp[n][m]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * M)。
- 空间复杂度：O(N * M)。
