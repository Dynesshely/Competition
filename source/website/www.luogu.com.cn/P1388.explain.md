# P1388 洛谷 算式 题解

## 题目链接
- [洛谷 P1388](https://www.luogu.com.cn/problem/P1388)

## 题目大意

给定 n 个数字，在相邻数字之间的 n-1 个空位中放入 k 个乘号和 (n-k-1) 个加号，可以任意加括号改变运算顺序，求表达式的最大可能结果。

## 解题思路

由于 n ≤ 15，数据范围较小，可以采用区间动态规划。

设 `dp[l][r][m]` 表示区间 [l, r] 内使用恰好 m 个乘号时能得到的最大结果。对于区间 [l, r]，枚举分割点 p，将区间分为 [l, p] 和 [p+1, r]，有两种合并方式：

1. **加法合并**：两侧结果相加，消耗 0 个乘号，m = m1 + m2。
2. **乘法合并**：两侧结果相乘，消耗 1 个乘号，m = m1 + m2 + 1。

状态转移为：
- `dp[l][r][m] = max(dp[l][r][m], dp[l][p][m1] + dp[p+1][r][m2])`，其中 m1 + m2 = m
- `dp[l][r][m] = max(dp[l][r][m], dp[l][p][m1] × dp[p+1][r][m2])`，其中 m1 + m2 = m - 1

初始化：`dp[i][i][0] = a[i]`，即单个数字不使用乘号时的结果就是它本身。

最终答案即为 `dp[1][n][k]`。

## 代码实现

使用 dp[l][r][m] 三维数组，初始化 -1 表示不可达。枚举区间长度从小到大，对于每个区间枚举分割点和左右两侧的乘号个数，分别尝试加法和乘法合并。

## 复杂度分析

- 时间复杂度：O(n³·k)，区间枚举 O(n²)，分割点枚举 O(n)，乘号分配 O(k)，n ≤ 15 时完全可行。
- 空间复杂度：O(n²·k)，三维 DP 数组。
