## 题目描述

输入两个整数，比较它们的大小。若 $x>y$ ，输出 `>` ；若 $x=y$ ，输出 `=` ；若 $x<y$，输出 `<` 。

## 输入格式

一行，包含两个整数 $x$ 和 $y$ ，中间用单个空格隔开。 $0 \le x<2^{32}, -2^{31} \le y<2^{31}$ 。

## 输出格式

一个字符。若 $x>y$，输出 `>` ；若 $x=y$ ，输出 `=` ；若 $x<y$ ，输出 `<` ；

## 样例 #1

### 样例输入

```
1000 100
```

### 样例输出

```
>
```
