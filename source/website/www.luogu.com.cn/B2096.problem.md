## 题目描述

给定一个非负整数数组，统计里面每一个数的出现次数。我们只统计到数组里最大的数。

假设 $F_{\max}$（$F_{\max} \le 10^5$）是数组里最大的数，那么我们只统计 $\{0,1,2, \ldots ,F_{\max} \}$ 里每个数出现的次数。

## 输入格式

第一行，一个整数 $n$ 表示数组的大小（$1 \le n \le 10^5$）。

紧接着一行是数组的 $n$ 个元素。

## 输出格式

按顺序输出每个数的出现次数，一行一个数。如果没有出现过，则输出 $0$。

对于例子中的数组，最大的数是 $3$，因此我们只统计 $\{0,1,2,3\}$ 的出现频数。

## 样例 #1

### 样例输入

```
5
1 1 2 3 1
```

### 样例输出

```
0
3
1 
1
```
