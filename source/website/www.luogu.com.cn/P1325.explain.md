# P1325 洛谷 雷达安装 题解

## 题目链接
- [洛谷 P1325](https://www.luogu.com.cn/problem/P1325)

## 题目大意
海岸线为 x 轴，海洋在 x 轴上方，陆地在下方。海上有 n 个岛屿（坐标点），雷达必须安装在陆地上（含海岸线），每个雷达的扫描半径为 d。求覆盖所有岛屿所需的最少雷达数。若不可能覆盖全部岛屿，输出 -1。

## 解题思路
这是经典的区间选点问题（贪心）。

对于每个岛屿 (x, y)，若 y > d 则该岛屿无法被任何雷达覆盖，直接输出 -1。否则，雷达可放置的范围是 x 轴上的一个区间：以岛屿为圆心、d 为半径作圆与 x 轴相交，交点区间为 [x - sqrt(d^2 - y^2), x + sqrt(d^2 - y^2)]。只要雷达安装在该区间内，即可覆盖该岛屿。

问题转化为：给定 n 个区间，在 x 轴上选择尽量少的点，使得每个区间至少包含一个点。

贪心策略：
- 将所有区间按右端点从小到大排序
- 依次遍历区间，若当前区间左端点大于上一个选择的雷达位置，则需要新增一个雷达，并将雷达位置设为当前区间的右端点

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int                  n, d;
double               eps = 1e-6;
pair<double, double> intervals[1005];

void input() {
    scanf("%d%d", &n, &d);
    for (int i = 0; i < n; ++i) {
        int x, y;
        scanf("%d%d", &x, &y);
        if (y > d) {
            printf("-1\n");
            exit(0);
        }
        double t     = sqrt(1.0 * d * d - 1.0 * y * y);
        intervals[i] = {x - t, x + t};
    }
}

void process() {
    sort(intervals, intervals + n, [](const pair<double, double> &a, const pair<double, double> &b) { return a.second < b.second; });
}

void output() {
    int    ans = 0;
    double pos = -1e18;
    for (int i = 0; i < n; ++i) {
        if (intervals[i].first > pos + eps) {
            ++ans;
            pos = intervals[i].second;
        }
    }
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log n)，排序为主
- 空间复杂度：O(n)
