# P1419 寻找段落 题解

## 题目链接
- [洛谷 P1419](https://www.luogu.com.cn/problem/P1419)

## 题目大意
给定长度为 $n$ 的序列 $a$，求长度在 $[S, T]$ 之间的连续子序列的最大平均值，保留三位小数。

## 解题思路
二分答案（平均值），检查是否存在长度在 $[S,T]$ 之间的子串平均值 $\ge mid$。

检查方法：将所有数减去 $mid$，问题转化为是否存在长度在 $[S,T]$ 之间的子串和 $\ge 0$。预处理前缀和 $p[i] = \sum_{j=1}^{i} (a_j - mid)$，则子串 $[l, r]$ 的和为 $p[r] - p[l-1]$。

枚举右端点 $r$，需要检查是否存在左端点 $l$ 满足 $S \le r-l+1 \le T$，即 $r-T \le l-1 \le r-S$。用单调队列维护 $p[l-1]$ 的最小值，即可 $O(n)$ 完成判断。

二分运行固定次数（如 50 次）以保证精度。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int    n, S, T;
double a[100010];
double p[100010];
int    q[100010];

bool check(double x) {
    for (int i = 1; i <= n; i++)
        p[i] = p[i - 1] + a[i] - x;
    int h = 0, t = -1;
    for (int i = S; i <= n; i++) {
        int pos = i - S;
        while (h <= t && p[q[t]] >= p[pos])
            t--;
        q[++t] = pos;
        while (h <= t && q[h] < i - T)
            h++;
        if (h <= t && p[i] - p[q[h]] >= 0)
            return true;
    }
    return false;
}

void input() {
    scanf("%d", &n);
    scanf("%d%d", &S, &T);
    for (int i = 1; i <= n; i++)
        scanf("%lf", &a[i]);
}

void process() {
    double l = -1e4, r = 1e4;
    for (int i = 0; i < 50; i++) {
        double mid = (l + r) / 2;
        if (check(mid))
            l = mid;
        else
            r = mid;
    }
    printf("%.3lf\n", l);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log M)$，二分 50 次，每次 $O(n)$。
- 空间复杂度：$O(n)$。
