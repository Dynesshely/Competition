# P1178 [洛谷] 到天宫做客 题解

## 题目链接
- [洛谷 P1178](https://www.luogu.com.cn/problem/P1178)

## 题目大意

已知一年（4000 年，闰年共 366 天）中有 N 个日期必须待在人间。要求找到最长连续空闲时间段（即没有被占用的连续日期），换算成天上的时间（天上一日，人间一年，即 86400 秒对应人间 366 天），结果四舍五入精确到 1 秒。

## 解题思路

将每个日期转换为该年的第几天（1 ~ 366），然后排序，计算相邻占用日期之间的空闲天数差，以及首尾到边界（第 1 天前、第 366 天后）的空闲天数，取最大值。

最长空闲天数 `maxFree` 与天上的秒数换算公式：

```
ans = round(86400.0 * maxFree / 366.0)
```

即四舍五入到整数。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, ans;
int daysInMonth[13] = {0, 31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
int a[370];

int toDay(int m, int d) {
    int res = d;
    for (int i = 1; i < m; i++)
        res += daysInMonth[i];
    return res;
}

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        int m, d;
        scanf("%d%d", &m, &d);
        a[i] = toDay(m, d);
    }
}

void process() {
    if (n == 0) {
        ans = 86400;
        return;
    }
    sort(a, a + n);
    int maxFree = 0;
    maxFree = max(maxFree, a[0] - 1);
    for (int i = 1; i < n; i++)
        maxFree = max(maxFree, a[i] - a[i - 1] - 1);
    maxFree = max(maxFree, 366 - a[n - 1]);
    ans = (int)(86400.0 * maxFree / 366.0 + 0.5);
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(N log N)，排序的时间复杂度。
- 空间复杂度：O(N)，存储日期数组。
