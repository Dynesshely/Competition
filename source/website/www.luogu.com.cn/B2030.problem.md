## 题目描述

已知线段的两个端点的坐标 `A(Xa,Ya)`，`B(Xb
,Yb)` ，求线段 AB 的长度。

## 输入格式

输入。

共两行。

第一行是两个实数 $Xa，Ya$，即 $A$ 的坐标。

第二行是两个实数 $Xb，Yb$，即 $B$ 的坐标。

输入中所有实数的绝对值均不超过 $10000$。

## 输出格式

输出。

一个实数，即线段 $AB$ 的长度，保留到小数点后 $3$ 位。

## 样例 #1

### 样例输入

```
1 1
2 2
```

### 样例输出

```
1.414
```
