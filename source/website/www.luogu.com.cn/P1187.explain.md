# P1187 [洛谷] 3D模型 题解

## 题目链接
- [洛谷 P1187](https://www.luogu.com.cn/problem/P1187)

## 题目大意

给定一个 n x m 的网格，每个格子上的建筑由若干个 1x1x1 的立方体堆叠而成（高度为 0~9）。求整个建筑模型的表面积（所有外露面的数量）。

## 解题思路

遍历每个非空格子，累加其贡献的表面积：

- 每个高度大于 0 的格子，顶部（1 个面）和底部（1 个面）均贡献表面积。
- 对于四个方向（上下左右）的每个相邻格子，若相邻格子的高度小于当前格子的第 k 层（1 <= k <= 当前高度），则该侧面暴露，贡献 1 个面。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, a[1005][1005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) {
        char s[1005];
        scanf("%s", s + 1);
        for (int j = 1; j <= m; j++)
            a[i][j] = s[j] - '0';
    }
}

void process() {
    int dx[] = {-1, 1, 0, 0};
    int dy[] = {0, 0, -1, 1};
    int ans = 0;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            if (a[i][j] > 0) {
                ans += 2;
                for (int k = 1; k <= a[i][j]; k++)
                    for (int d = 0; d < 4; d++)
                        if (a[i + dx[d]][j + dy[d]] < k)
                            ans++;
            }
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(n * m * max_h * 4)，最坏情况下 n, m <= 1000，max_h <= 9，总操作数约为 3.6e7，在时间限制内可接受。
- 空间复杂度：O(n * m)，存储网格高度。
