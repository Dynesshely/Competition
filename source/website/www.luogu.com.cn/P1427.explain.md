# P1427 小鱼的数字游戏 题解

## 题目链接

- [洛谷 P1427](https://www.luogu.com.cn/problem/P1427)

## 题目大意

输入一串整数，以 0 结束，要求将这串整数（不包括结尾的 0）倒序输出。

## 解题思路

将数字依次读入数组，遇到 0 时停止。然后从最后一个有效元素开始向前遍历，依次输出。

## 代码实现

用数组 a 存储输入，n 记录有效数字个数。循环读入，当读到 0 时 break。之后从 n-1 到 0 倒序输出，用条件运算符控制空格格式。

## 复杂度分析

- 时间复杂度：O(m)，其中 m 为输入数字个数。
- 空间复杂度：O(m)，使用数组存储输入数字。
