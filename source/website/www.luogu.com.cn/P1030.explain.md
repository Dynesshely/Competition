# P1030 [NOIP 2001 普及组] 求先序排列 — 题解

## 题目大意

给定二叉树的中序遍历和后序遍历，求先序遍历。节点数 $\le 8$。

## 解题思路

### 二叉树的三种遍历

| 遍历 | 顺序 |
|------|------|
| 先序（Preorder） | 根 → 左子树 → 右子树 |
| 中序（Inorder） | 左子树 → 根 → 右子树 |
| 后序（Postorder） | 左子树 → 右子树 → 根 |

### 递归构造

后序的**最后一个字符**即是整棵树的**根**。在中序中找到根的位置 $k$，则：
- 中序 $[0, k-1]$ 为左子树，$[k+1, \text{end}]$ 为右子树
- 后序 $[0, k-1]$ 对应左子树的后序，$[k, \text{end}-1]$ 对应右子树的后序

先序输出：先输出根，再递归处理左子树，最后递归处理右子树。

```
build(in, post):
    if empty: return
    root = post[-1]
    print(root)          // 先序：先输出根
    k = in.find(root)
    build(in[0:k], post[0:k])         // 左子树
    build(in[k+1:], post[k:-1])       // 右子树
```

### 时间复杂度

- $O(n^2)$，其中 $n \le 8$，可忽略不计

### 空间复杂度

- 递归栈 $O(n)$
