# P3081 [USACO13MAR] Hill Walk G 题解

## 题目链接
- [洛谷 P3081](https://www.luogu.com.cn/problem/P3081)

## 题目大意
有 $N$ 座山，每座山由笛卡尔坐标系中的一条线段 $(x_1, y_1) \to (x_2, y_2)$ 表示（$x_1 < x_2$，$y_1 < y_2$）。线段互不相交，第一座山从 $(0,0)$ 开始。Bessie 从山 $1$ 出发，沿山走到终点后垂直下落。若降落在另一座山上则继续行走，否则落到 $y = -\infty$。求 Bessie 经过的山的总数。

## 解题思路

### 问题转化
将山视为平面上的线段，Bessie 的行走路线是一个折线：沿山行走（从左到右），到右端点后垂直下落，落到下一个山的线段上再继续行走。

关键观察：对于每座山 $i$，其右端点 $(x_2, y_2)$ 在垂直向下方向第一个碰到的山就是下一座山。由于线段互不相交，这个关系是确定的。

### 算法框架
1. 将山 $2 \sim N$ 按左端点 $x_1$ 升序排序。
2. 用 Li Chao 线段树动态维护"活跃"的山（即当前可能被降落的山）。Li Chao 树支持插入一条线段，并查询在某个 $x$ 坐标处的最大 $y$ 值。
3. 从山 $1$ 开始模拟：
   - 当前山 $i$ 的右端点为 $(x_2, y_2)$。
   - 将所有左端点 $x_1 \le x_2$ 且尚未插入的山加入 Li Chao 树。
   - 查询 Li Chao 树在 $x_2$ 处的最大 $y$ 值。由于所有山都是"下方"的候选，返回最大值对应的山编号即为下一个落点。
   - 如果查询不到结果（返回值为 $-1$），则 Bessie 落入深渊，过程结束。
4. 每访问一座山计数器加一。

### Li Chao 线段树细节
- 每座山是一条线段，存入 Li Chao 树时按直线方程计算：$y = kx + b$，其中 $k = (y_1 - y_2) / (x_1 - x_2)$，$b = y_1 - k \cdot x_1$。
- 对于垂直线段（$x_1 = x_2$）特殊处理。
- 比较函数 `isBetter` 判断在某个 $x$ 处哪个线段更高；为避免精度问题使用 `eps`，且编号大的山优先（方便后续处理）。

### 正确性
由于山互不相交且排序后满足偏序关系，Li Chao 树可以高效地维护在当前 $x$ 坐标下的最高山，从而找到 Bessie 的下一个落脚点。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long     ll;
typedef long double   ld;
typedef pair<ld, int> pdi;

const int maxN = 5000005, maxH = 10000005, maxV = 2000000005;
const ld  eps = 1e-9;

struct line {
    int x1, y1, x2, y2;
    line(int x1 = 0, int y1 = -1, int x2 = 0, int y2 = -1)
        : x1(x1), y1(y1), x2(x2), y2(y2) {}
    ld val(int x) const {
        if (x1 == x2)
            return x == x1 ? max(y1, y2) : -1;
        ld k = (y1 - y2) / (ld)(x1 - x2);
        ld b = y1 - k * x1;
        return k * x + b;
    }
} a[maxN];

int tot, lc[maxH], rc[maxH], bst[maxH];

ld sub(int x, int y, int p) { return a[x].val(p) - a[y].val(p); }

bool isBetter(int x, int y, int p) {
    if (!x) return 0;
    if (!y) return 1;
    return sub(x, y, p) > eps || (sub(x, y, p) > -eps && x > y);
}

void updNode(int &u, int l, int r, int x) {
    if (!u) u = ++tot;
    int m = (l + r) >> 1;
    if (isBetter(x, bst[u], m)) swap(bst[u], x);
    if (isBetter(x, bst[u], l)) updNode(lc[u], l, m, x);
    if (isBetter(x, bst[u], r)) updNode(rc[u], m + 1, r, x);
}

void updSeg(int &u, int l, int r, int s, int t, int x) {
    if (t < l || s > r) return;
    if (s <= l && r <= t) return updNode(u, l, r, x);
    if (!u) u = ++tot;
    int m = (l + r) >> 1;
    updSeg(lc[u], l, m, s, t, x);
    updSeg(rc[u], m + 1, r, s, t, x);
}

pdi maxPair(pdi x, pdi y) {
    if (x.first - y.first > eps) return x;
    return y;
}

pdi query(int u, int l, int r, int s) {
    if (s < l || s > r || !u) return make_pair(-1, 0);
    pdi ret(a[bst[u]].val(s), bst[u]);
    if (l == r) return ret;
    int m = (l + r) >> 1;
    return maxPair(ret, maxPair(query(lc[u], l, m, s), query(rc[u], m + 1, r, s)));
}

int n, root;

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) {
        int x1, y1, x2, y2;
        scanf("%d%d%d%d", &x1, &y1, &x2, &y2);
        a[i] = line(x1, y1, x2, y2);
    }
}

void process() {
    sort(a + 2, a + n + 1, [](const line &x, const line &y) { return x.x1 < y.x1; });
    int p = 2, i = 1, ans = 0;
    do {
        ans++;
        if (p <= i) p = i + 1;
        while (p <= n && a[i].x2 >= a[p].x1) {
            if (a[i].y2 >= a[p].val(a[i].x2))
                updSeg(root, 0, maxV, a[p].x1, a[p].x2 - 1, p);
            p++;
        }
        i = query(root, 0, maxV, a[i].x2).second;
    } while (i);
    printf("%d\n", ans);
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(N \log V)$，其中 $N$ 为山的数量，$V$ 为坐标值域（$0 \sim 2 \times 10^9$），每次 Li Chao 树的插入和查询均为 $O(\log V)$。
- 空间复杂度：$O(N + H)$，其中 $H$ 为 Li Chao 树的节点数。
