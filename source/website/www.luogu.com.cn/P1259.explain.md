# P1259 [洛谷] 黑白棋子的移动 题解

## 题目链接

## 题目大意
有 2n 个棋子排成一行，左边 n 个白子（o），右边 n 个黑子（*），最右边有两个空位（--）。每次必须同时移动相邻的两个棋子到空位上（可以左移或右移，不能调换两个棋子的左右顺序），要求最终排成黑白相间的形式（即 o*o*o*...--）。

## 解题思路
采用**递归分治**策略。对于 n > 4 的情况：
1. 将中间相邻的一对棋子（第 n 和 n+1 个）移到最右边的空位。
2. 将最左边的一对黑子移到刚空出的位置。
3. 这样原问题规模减少 1，递归处理 n-1 的情况。

基础情况 n = 4 有固定的操作序列：
- (3, 7, 1, 6, 0) 一系列移动即可完成。

每次移动后输出当前棋子的排列状态。
