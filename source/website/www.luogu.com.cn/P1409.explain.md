# P1409 骰子 题解

## 题目链接
- [洛谷 P1409](https://www.luogu.com.cn/problem/P1409)

## 题目大意
$n$ 个人排成一排，你排在第 $m$ 个。每轮队首的人投一次骰子：
- 掷到 $1$：队首获胜。
- 掷到 $2,4,6$：队首排到队尾。
- 掷到 $3,5$：队首出队。

队列只剩一人时自动获胜。求你获胜的概率。

## 解题思路
设 $f_{n,m}$ 为 $n$ 个人中第 $m$ 个获胜的概率。考虑第一轮的结果：

- 掷到 $1$（概率 $1/6$）：队首获胜，若 $m=1$ 则获胜，概率为 $0$。
- 掷到 $2,4,6$（概率 $3/6=1/2$）：队首移到队尾。$n$ 人中第 $m$ 个变为 $n$ 人中第 $m-1$ 个（$m>1$），若 $m=1$ 则变为第 $n$ 个。
- 掷到 $3,5$（概率 $2/6=1/3$）：队首出队。$n$ 人中第 $m$ 个变为 $n-1$ 人中第 $m-1$ 个（$m>1$），若 $m=1$ 则不可能获胜。

因此可以列出 DP 方程：

$$
f_{n,1} = \frac{1}{6} \cdot 1 + \frac{1}{2} \cdot f_{n,n} + \frac{1}{3} \cdot 0
$$

$$
f_{n,m} = \frac{1}{6} \cdot 0 + \frac{1}{2} \cdot f_{n,m-1} + \frac{1}{3} \cdot f_{n-1,m-1} \quad (m>1)
$$

化简得到递推式，可以从 $n=1$ 开始递推到 $n$。利用前缀和优化可将复杂度降至 $O(n^2)$。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int    n, m;
double prob[1001][1001];
double pow2[1001];

void input() { scanf("%d%d", &n, &m); }

void process() {
    pow2[0] = 1.0;
    for (int i = 1; i <= n; i++)
        pow2[i] = pow2[i - 1] * 0.5;
    prob[1][1] = 1.0;
    for (int cur = 2; cur <= n; cur++) {
        double *g = prob[cur - 1];
        double *f = prob[cur];
        double  sum2[1001];
        sum2[1] = 0.0;
        for (int k = 2; k <= cur; k++)
            sum2[k] = g[k - 1] + 0.5 * sum2[k - 1];
        f[1] = (1.0 / 6.0) * (1.0 + sum2[cur]) / (1.0 - pow2[cur]);
        for (int k = 2; k <= cur; k++)
            f[k] = pow2[k - 1] * f[1] + (1.0 / 3.0) * sum2[k];
    }
}

void output() { printf("%.9lf\n", prob[n][m]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2)$。
- 空间复杂度：$O(n^2)$。
