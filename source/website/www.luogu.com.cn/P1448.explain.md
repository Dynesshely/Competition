# P1448 [NOI1998] 围巾裁剪 题解

## 题目链接

- [洛谷 P1448](https://www.luogu.com.cn/problem/P1448)

## 题目大意

给定一个边被均匀分为 $N$ 段的正三角形围巾，划分为 $N^2$ 个面积为 $1$ 的小正三角形单元。其中 $M$ 个单元被蛀虫咬坏（不可使用）。要求沿单元边界裁剪出两块与原来形状相同（均为正三角形）的小围巾，且每块小围巾内不包含任何坏单元。求两块小围巾面积之和的最大值。数据范围：$N \le 100$。

## 解题思路

正三角形围巾被划分为 $N$ 行，第 $i$ 行有 $2i-1$ 个小三角形，其中 $i$ 个朝上（$\Delta$）、$i-1$ 个朝下（$\nabla$）。坐标 $(X, Y)$ 表示第 $X$ 行第 $Y$ 个单元。

首先需要预处理每个位置作为顶点时能构成的最大合法正三角形边长。由于围巾是正三角形，裁剪必须沿边界，所以可裁剪出的三角形朝向与原围巾一致。

经典做法是将问题分解为三次旋转处理（旋转 $0^\circ$、$120^\circ$、$240^\circ$），每次处理某个固定方向上的最大三角形：

1. **make()**：自底向上 DP，对每个朝上的单元 $(i, 2j-1)$，若它本身可用，则它作为顶点的最大三角形边长取决于它下方三个相邻单元的最小值加 $1$。对朝下的单元 $(i, 2j)$ 类似地从上往下 DP。
2. **ask()**：枚举切割行 $i$，将围巾分为上下两部分。分别计算上半部分和下半部分中能裁剪出的最大正三角形边长，更新答案。
3. **splay()**：将围巾旋转 $120^\circ$，重复上述过程，从而覆盖所有可能的三角形朝向和切割方向。

通过三次旋转和枚举切割位置，可以找到所有可能的合法裁剪方案，取面积和的最大值。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 105;
int       n, m, ans, a[MAXN][MAXN * 2], b[MAXN][MAXN * 2];

void make() {
    for (int i = 1; i <= n; i++)
        if (a[n][2 * i - 1] != -1)
            a[n][2 * i - 1] = 1;
    for (int i = n - 1; i >= 1; i--)
        for (int j = 1; j <= i; j++)
            if (a[i][2 * j - 1] != -1) {
                a[i][2 * j - 1] = 1;
                if (a[i + 1][2 * j - 1] != -1 && a[i + 1][2 * j] != -1 && a[i + 1][2 * j + 1] != -1)
                    a[i][2 * j - 1] = min(a[i + 1][2 * j - 1], a[i + 1][2 * j + 1]) + 1;
            }
    for (int i = 2; i <= n; i++)
        for (int j = 1; j <= i - 1; j++)
            if (a[i][2 * j] != -1) {
                a[i][2 * j] = 1;
                if (j != 1 && j != i - 1 && a[i - 1][2 * (j - 1) + 1] != -1 && a[i - 1][2 * (j - 1)] != -1 && a[i - 1][2 * j] != -1)
                    a[i][2 * j] = min(a[i - 1][2 * (j - 1)], a[i - 1][2 * j]) + 1;
            }
}

void ask() {
    for (int i = 1; i < n; i++) {
        int max1 = 0, max2 = 0;
        for (int j = 1; j <= i; j++) {
            for (int k = 1; k <= j; k++)
                max1 = max(max1, min(i - j + 1, a[j][2 * k - 1]));
            for (int k = 1; k <= j - 1; k++)
                max1 = max(max1, a[j][2 * k]);
        }
        for (int j = i + 1; j <= n; j++) {
            for (int k = 1; k <= j; k++)
                max2 = max(max2, a[j][2 * k - 1]);
            for (int k = 1; k <= j - 1; k++)
                max2 = max(max2, min(j - i, a[j][2 * k]));
        }
        if (max1 != 0 && max2 != 0)
            ans = max(ans, max1 * max1 + max2 * max2);
    }
}

void splay() {
    memset(b, 0, sizeof(b));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++)
            if (a[i][2 * j - 1] == -1)
                b[n - i + j][(n - i) * 2 + 1] = -1;
        for (int j = 1; j <= i - 1; j++)
            if (a[i][2 * j] == -1)
                b[n - i + j + 1][(n - i) * 2 + 2] = -1;
    }
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= 2 * i - 1; j++)
            a[i][j] = b[i][j];
}

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < m; i++) {
        int x, y;
        scanf("%d%d", &x, &y);
        a[x][y] = -1;
    }
}

void process() {
    for (int t = 0; t < 3; t++) {
        make();
        ask();
        splay();
    }
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：$O(N^3)$。三次旋转（$t=3$），每次枚举切割位置 $O(N)$，在每个切割位置扫描上半和下半的最大三角形 $O(N^2)$，总复杂度 $O(N^3)$。$N \le 100$ 时可接受。
- 空间复杂度：$O(N^2)$，存储三角网格状态。
