# P1343 地震逃生 题解

## 题目链接
- [洛谷 P1343](https://www.luogu.com.cn/problem/P1343)

## 题目大意
给定一个有 $n$ 个点 $m$ 条边的**有向图**，每条边有一个容量 $c$（最多能容纳的学生数）。$x$ 名学生从 $1$ 号点（教室）逃往 $n$ 号点（安全地带），必须成批出发，每批学生同时从起点出发同时到达终点，且只有一批全部到达后下一批才能出发。求每批最多能运出多少学生，以及总共需要多少批。

## 解题思路
本题是一个**最大流**问题。将每条边的容量视为学生通行上限，则每批最多能运出的学生数等于从 $1$ 到 $n$ 的最大流。

使用 **Dinic 算法**求最大流：
1. 建图，添加有向边及反向边（容量为 $0$）。
2. 反复进行 BFS 分层 + DFS 增广，直到无法增广为止。

得到最大流 $perBatch$ 后：
- 若 $perBatch = 0$，说明无法到达 $n$ 号点，输出 `Orz Ni Jinan Saint Cow!`。
- 否则输出 $perBatch$ 和所需批数 $\lceil x / perBatch \rceil$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int       MAXN = 205;
const long long INF  = 1LL << 60;

struct Edge {
    int       v, rev;
    long long cap;
};

vector<Edge> g[MAXN];
int          level[MAXN], it[MAXN];
int          n, m, x;

void addEdge(int u, int v, long long c) {
    g[u].push_back({v, (int)g[v].size(), c});
    g[v].push_back({u, (int)g[u].size() - 1, 0});
}

bool bfs(int s, int t) {
    memset(level, -1, sizeof(level));
    queue<int> q;
    level[s] = 0;
    q.push(s);
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        for (auto &e : g[u])
            if (e.cap > 0 && level[e.v] < 0) {
                level[e.v] = level[u] + 1;
                q.push(e.v);
            }
    }
    return level[t] >= 0;
}

long long dfs(int u, int t, long long f) {
    if (u == t)
        return f;
    for (int &i = it[u]; i < (int)g[u].size(); i++) {
        Edge &e = g[u][i];
        if (e.cap > 0 && level[u] < level[e.v]) {
            long long d = dfs(e.v, t, min(f, e.cap));
            if (d > 0) {
                e.cap -= d;
                g[e.v][e.rev].cap += d;
                return d;
            }
        }
    }
    return 0;
}

long long maxFlow(int s, int t) {
    long long flow = 0;
    while (bfs(s, t)) {
        memset(it, 0, sizeof(it));
        long long f;
        while ((f = dfs(s, t, INF)) > 0)
            flow += f;
    }
    return flow;
}

void input() { scanf("%d%d%d", &n, &m, &x); }

void process() {
    for (int i = 0; i < m; i++) {
        int a, b, c;
        scanf("%d%d%d", &a, &b, &c);
        addEdge(a, b, c);
    }
    long long perBatch = maxFlow(1, n);
    if (perBatch == 0)
        printf("Orz Ni Jinan Saint Cow!\n");
    else
        printf("%lld %lld\n", perBatch, (x + perBatch - 1) / perBatch);
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2 m)$，Dinic 算法在一般情况下的复杂度，实际在 $n \le 200, m \le 2000$ 的范围内运行迅速。
- 空间复杂度：$O(n+m)$，存储图的邻接表。
