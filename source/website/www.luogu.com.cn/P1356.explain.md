# P1356 洛谷 模意义下 DP 题解

## 题目链接
- [洛谷 P1356](https://www.luogu.com.cn/problem/P1356)

## 题目大意
给定一个整数数列，可以在每两个整数之间任意放置 `+` 或 `-` 号构成表达式。判断是否存在一种符号放置方式，使得表达式的值能被 k 整除。有多组测试数据。

## 解题思路
采用模意义下的动态规划。设 `dp[j]` 表示处理到当前数时，能否得到模 k 余 j 的值。

1. 初始化：第一个数 a[0] 的模 k 值为 `(a[0] % k + k) % k`，标记 `dp` 数组对应位置为 true。
2. 递推：对于后续每个数 a[i]，根据当前所有可达余数 j，分别加上和减去 a[i] 后取模，得到新的可达余数 `(j + a[i]) % k` 和 `(j - a[i] + k) % k`，标记到辅助数组 `nd` 中。
3. 滚动更新 `dp` 数组，重复直到处理完所有数。
4. 最终检查 `dp[0]` 是否为 true，若为 true 则输出 `Divisible`，否则输出 `Not divisible`。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, k;
int  a[10005];
bool dp[105], nd[105];

void input() {
    scanf("%d%d", &n, &k);
    for (int i = 0; i < n; ++i)
        scanf("%d", &a[i]);
}

void process() {
    for (int i = 0; i < k; ++i)
        dp[i] = false;
    dp[(a[0] % k + k) % k] = true;
    for (int i = 1; i < n; ++i) {
        for (int j = 0; j < k; ++j)
            nd[j] = false;
        for (int j = 0; j < k; ++j)
            if (dp[j]) {
                nd[((j + a[i]) % k + k) % k] = true;
                nd[((j - a[i]) % k + k) % k] = true;
            }
        for (int j = 0; j < k; ++j)
            dp[j] = nd[j];
    }
}

void output() {
    if (dp[0])
        printf("Divisible\n");
    else
        printf("Not divisible\n");
}

int main() {
    int m;
    scanf("%d", &m);
    while (m--) {
        input();
        process();
        output();
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(M * n * k)，其中 M 为数据组数，n 为数列长度，k 为模数。
- 空间复杂度：O(k)，使用两个长度为 k 的布尔数组进行滚动 DP。
