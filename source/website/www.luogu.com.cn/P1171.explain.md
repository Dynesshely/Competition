# P1171 售货员的难题 题解

## 题目链接
- [洛谷 P1171](https://www.luogu.com.cn/problem/P1171)

## 题目大意
有 n 个村庄，售货员从村庄 1 出发，每个村庄恰好到访一次，最后返回村庄 1。各村庄之间的单向路程已知，求最短的总路程。

## 解题思路
这是一个经典的旅行商问题（TSP），n <= 20，因此可以使用状态压缩动态规划（Held-Karp 算法）求解。

定义状态：
- `dp[mask][u]` 表示当前已访问的村庄集合为 `mask`（二进制表示，第 i 位为 1 表示已访问），且当前位于村庄 u 时的最短路径长度。

初始化：
- `dp[1][0] = 0`，只访问了村庄 1（下标从 0 开始），当前位置为村庄 0。

状态转移：
- 对于当前状态 `(mask, u)`，枚举下一个未访问的村庄 v，更新 `dp[mask | (1<<v)][v] = min(dp[mask | (1<<v)][v], dp[mask][u] + dist[u][v])`。

最终答案：
- 遍历所有可能的最后村庄 u（u != 0），取 `dp[(1<<n)-1][u] + dist[u][0]` 的最小值，即从 u 返回村庄 1 的最短路径。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, dist[20][20], dp[1 << 20][20];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            scanf("%d", &dist[i][j]);
}

void process() {
    memset(dp, 0x3f, sizeof(dp));
    dp[1][0] = 0;
    for (int mask = 1; mask < (1 << n); mask++) {
        if (!(mask & 1))
            continue;
        for (int u = 0; u < n; u++) {
            if (!(mask >> u & 1))
                continue;
            int cur = dp[mask][u];
            if (cur == 0x3f3f3f3f)
                continue;
            for (int v = 0; v < n; v++) {
                if (mask >> v & 1)
                    continue;
                int nmask = mask | (1 << v);
                int nv    = cur + dist[u][v];
                if (nv < dp[nmask][v])
                    dp[nmask][v] = nv;
            }
        }
    }
    int full = (1 << n) - 1;
    int ans  = 0x3f3f3f3f;
    for (int u = 1; u < n; u++) {
        int val = dp[full][u] + dist[u][0];
        if (val < ans)
            ans = val;
    }
    printf("%d\n", ans);
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n^2 * 2^n)，枚举所有状态和转移，n <= 20 在可接受范围内。
- 空间复杂度：O(n * 2^n)，存储 DP 表。
