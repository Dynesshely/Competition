# P1836 数页码 题解

## 题目链接

- [洛谷 P1836](https://www.luogu.com.cn/problem/P1836)

## 题目大意

一本书从第 1 页编号到第 n 页。求所有页码中每个数字的数码之和（例如第 123 页的数码和为 1+2+3=6）。

## 解题思路

本题本质是求 1 到 n 所有整数的十进制数码之和。

一个高效的解决方式是使用数位 DP（Digit DP）或数学公式，但本题提供的代码采用了**最直接的暴力枚举**方法：

- 对 i 从 1 到 n 逐一处理。
- 对每个 i，将其转换为字符串，遍历每个字符并转换为数字后累加。

由于 n 最大可达 1e9，暴力枚举 O(n * log n) 在时间限制内可能较大，但代码实现简单直观。

## 代码实现

- **`cal(n)`**：辅助函数，计算单个整数 n 的数码和。
  - 使用 `ostringstream os` 将 n 转换为字符串。
  - 遍历字符串的每一位，将字符转换为数字（`ns[i] - '0'`）累加到 `ans`。
- **主循环**：从 1 到 n 依次调用 `cal(i)` 并累加结果。
- 最后输出总和。

## 复杂度分析

- 时间复杂度：O(n * log n)，n 最大为 1e9，log n 表示数字的位数（最大为 10）。
- 空间复杂度：O(log n)，仅需存储当前数字的字符串表示。
