# P2762 太空飞行计划问题 题解

## 题目链接

- [洛谷 P2762](https://www.luogu.com.cn/problem/P2762)

## 题目大意

有 m 个实验和 n 个仪器，每个实验需要特定的仪器集合并带来收益，每个仪器有购置成本。求选择哪些实验和仪器能使净收益（实验收入 - 仪器成本）最大。

## 解题思路

本题是经典的 **最大权闭合子图** 问题，使用 **最小割（最大流）** 求解。

**建模方法**：
1. 建立源点 S 和汇点 T。
2. 源点 S 向每个实验节点连边，容量为该实验的收益（收入）。
3. 每个实验节点向其所需的仪器节点连边，容量为 INF（保证最小割不会切断这些边）。
4. 每个仪器节点向汇点 T 连边，容量为该仪器的成本。
5. 求该网络的最大流（即最小割）。

**结果输出**：
- 在残量网络中，从源点 S 出发能到达的实验即为要进行的实验，能到达的仪器即为要配置的仪器。
- 最大净收益 = 所有实验收益之和 - 最小割容量。

**原理**：最小割将节点划分为 S 集和 T 集。若实验在 T 集（即未被选中），其收益被割掉；若仪器在 S 集（即被选中），其成本被割掉。最小化割集等价于最大化净收益。

## 代码实现

- **快速 IO**（`namespace io`）：手写快读快写，使用 1MB 缓冲区，支持 `read<T>()` 泛型读入和 `write()` 输出。
- **Dinic 模板类**（`template<size_t maxn>`）：封装了 BFS 分层和 DFS 多路增广的标准 Dinic 最大流算法。
  - `addEdge(from, to, cap)`：添加正向边和反向边，`real` 标记区分正反。
  - `BFS()`：从源点 S 分层，`vis` 同时记录可达性用于最后输出选择方案。
  - `DFS(p, rest)`：多路增广，使用 `cur` 数组（当前弧优化）加速。
  - `maxFlow(S, T)`：反复 BFS + DFS 直到无法增广。
- **建图**：读入 n（实验数）和 m（仪器数），S = n + m + 1，T = n + m + 2。按上述建模方法建图。
- **实验仪器读取**：使用 `getline(cin, str)` 和 `stringstream` 读取每行不定数量的仪器编号。
- **结果输出**：遍历检查 `dinic.vis[]`，输出 S 集可达的实验和仪器节点，计算净收益并输出。

## 复杂度分析

- 时间复杂度：O(V^2 E)（Dinic 一般情况快于该上界），其中 V = n + m + 2，E 包括所有实验-仪器边。
- 空间复杂度：O(V + E)，邻接表存图。
