# P1052 [NOIP 2005 提高组] 过河 题解

## 题目链接

- [洛谷 P1052](https://www.luogu.com.cn/problem/P1052)

## 题目大意

在长度为 $L$ 的独木桥上有 $M$ 个石子，青蛙从起点 $0$ 出发，每次跳跃距离在 $[S, T]$ 之间。青蛙到达或跳过终点 $L$ 即为过河。求过河过程中最少需要踩到的石子数。

## 解题思路

### 朴素 DP

定义 $dp[i]$ 表示跳到位置 $i$ 时踩到的最少石子数。转移方程：

$$dp[i] = \min_{S \le j \le T}\{dp[i - j] + stone[i]\}$$

其中 $stone[i]$ 表示位置 $i$ 是否有石子。但 $L \le 10^9$，直接 DP 不可行。

### 路径压缩（离散化）

注意到 $M \le 100$ 而 $S, T \le 10$。对于两个石子之间的空隙，当空隙足够大时，DP 的行为会出现周期性质。具体地，若两石子间距超过 $S \times T$，可以将其压缩为 $S \times T$，因为所有大于等于 $S \times T$ 的距离都是可达的（由裴蜀定理，当步长集合包含互质的两数时，充分大以后的所有距离均可达）。

因此算法步骤：
1. 将石子坐标排序。
2. 对相邻石子的间距进行压缩：`compressed[i] = compressed[i-1] + min(gap, S*T)`。
3. 在新坐标系上 DP，范围从 $0$ 到 $compressed[M] + \min(L - a[M], S \times T)$。
4. 注意特判 $S = T$ 的情况，此时跳跃距离固定，直接检查每个石子坐标是否被 $S$ 整除即可。

### 特判：$S = T$

当 $S = T$ 时，青蛙每次跳跃距离固定。此时只需统计有多少个石子的位置能被 $S$ 整除，这些石子必然被踩到。

## 代码实现

- 读入 $L, S, T, M$ 后排序石子数组 `a[]`。
- 若 $S = T$，统计 `a[i] % S == 0` 的个数并输出。
- 否则进行路径压缩：`b[1] = min(a[1], threshold)`，之后 `b[i] = b[i-1] + min(a[i] - a[i-1], threshold)`。
- 压缩后终点 `compressedL = b[M] + min(L - a[M], threshold)`。
- 在新坐标上 DP：`dp[0] = 0`，其余初始化为 `0x3f3f3f3f`。转移时枚举步长 $[S, T]$，若到达或越过 `compressedL` 则更新答案。

## 复杂度分析

- 时间复杂度：$O(M \log M + M \times S \times T \times (T - S + 1))$，排序加上压缩后的 DP，由于 $S, T \le 10$，DP 状态数约为 $100 \times 100 = 10^4$ 级别。
- 空间复杂度：$O(S \times T \times M)$，即压缩后的坐标范围。
