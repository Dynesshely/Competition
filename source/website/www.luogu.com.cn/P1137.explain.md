# P1137 旅行计划 题解

## 题目链接

- [洛谷 P1137](https://www.luogu.com.cn/problem/P1137)

## 题目大意

给定 N 个城市和 M 条有向边（从西到东），从任意城市出发，只能沿东向移动，以城市 i 为终点。求以每个城市为终点时最多能游览的城市数量。

## 解题思路

拓扑排序 + DP。因为边从西到东保证无环，构成 DAG。按拓扑序处理，对于每个节点 u，遍历其出边 v，用 dp[u] + 1 更新 dp[v] 的最大值。初始每个城市的 dp 值为 1（自身）。

## 代码实现

统计入度，将入度为 0 的节点入队。依次取出队首节点 u，更新其所有后继节点 v 的 dp 值，当 v 的入度减为 0 时入队。

## 复杂度分析

- 时间复杂度：O(N + M)
- 空间复杂度：O(N + M)
