# P1384 幸运数与排列 题解

## 题目链接
- [洛谷 P1384](https://www.luogu.com.cn/problem/P1384)

## 题目大意
定义幸运数为仅由数字 4 和 7 构成的数。求在 1 到 n 的全排列中字典序第 k 小的排列里，有多少个数满足：该数本身是幸运数，且它在排列中的位置编号也是幸运数。若不存在第 k 小的排列则输出 -1。n, k ≤ 1e9。

## 解题思路
n 最大可达 1e9，无法直接构造全排列。观察到：当 n 较大时，只有最后若干位会参与排列变化，前缀部分保持 1,2,3,... 的顺序排列。

**Cantor 展开**：给定 k，可以求出最小的 m 使得 m! ≥ k，那么前 n-m 位保持不动，后 m 位进行第 k 小排列（1-indexed）。

具体步骤：
1. 预处理阶乘 fact[0..13]（13! > 1e9）。
2. 若 n ≤ 12 且 k > n!，输出 -1。
3. 找到最小的 m 使 fact[m] ≥ k；若 m > n，输出 -1。
4. 前缀长度 prefixLen = n - m。在前缀 [1, prefixLen] 中，所有幸运数都在自己的位置上，直接计数。
5. 对后 m 个位置，使用 Cantor 展开（逆康托展开）构造第 k 小排列的这 m 位。
6. 对于每一位 i（0-indexed），位置 pos = prefixLen + i + 1，值 val = prefixLen + perm[i]。若 isLucky(pos) && isLucky(val)，答案 +1。

幸运数的生成：通过 DFS 递归生成 ≤ limit 的所有幸运数（仅含 4 和 7）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int               n;
long long         k;
long long         fact[15];
vector<long long> lucky;

void genLucky(long long limit, long long cur) {
    if (cur > limit)
        return;
    if (cur > 0)
        lucky.push_back(cur);
    genLucky(limit, cur * 10 + 4);
    genLucky(limit, cur * 10 + 7);
}

bool isLucky(long long x) {
    while (x > 0) {
        int d = x % 10;
        if (d != 4 && d != 7)
            return false;
        x /= 10;
    }
    return true;
}

void input() { scanf("%d%lld", &n, &k); }

void process() {
    fact[0] = 1;
    for (int i = 1; i <= 13; i++)
        fact[i] = fact[i - 1] * i;
    if (n <= 12 && k > fact[n]) {
        printf("-1");
        return;
    }
    int m = 0;
    for (int i = 1; i <= 13; i++)
        if (fact[i] >= k) {
            m = i;
            break;
        }
    if (m > n) {
        printf("-1");
        return;
    }
    int prefixLen = n - m;
    lucky.clear();
    genLucky(prefixLen, 0);
    int         ans = (int)lucky.size();
    vector<int> nums(m);
    iota(nums.begin(), nums.end(), 1);
    vector<int> perm(m);
    long long   k0 = k - 1;
    for (int i = 0; i < m; i++) {
        int f   = m - 1 - i;
        int idx = (int)(k0 / fact[f]);
        perm[i] = nums[idx];
        nums.erase(nums.begin() + idx);
        k0 %= fact[f];
    }
    for (int i = 0; i < m; i++) {
        long long pos = (long long)prefixLen + i + 1;
        long long val = (long long)prefixLen + perm[i];
        if (isLucky(pos) && isLucky(val))
            ans++;
    }
    printf("%d", ans);
}

void output() {}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(m^2 + log_{10} n)，m 为末尾参与排列的位数（m ≤ 13），逆康托展开时有 O(m^2) 的数组删除操作，幸运数生成与位数成正比。
- 空间复杂度：O(m + lucky_cnt)。
