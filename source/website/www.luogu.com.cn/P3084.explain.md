# P3084 [USACO13OPEN] Photo G 题解

## 题目链接
- [洛谷 P3084](https://www.luogu.com.cn/problem/P3084)

## 题目大意
有 N（N <= 2e5）头牛排成一排，Farmer John 拍了 M（M <= 1e5）张照片，每张照片覆盖一段连续区间 [a_i, b_i]，且每张照片中恰好有一头斑点牛。求整个牛群中最多可能有多少头斑点牛，若无解输出 -1。

## 解题思路
对每张照片 (x, y) 提取两条约束：
1. 区间 [x, y] 内恰好有一头斑点牛，所以前一头斑点牛必须在 x 之前（即位置 <= x-1），这限制了 r[y]。
2. 区间 [x, y] 内有一头斑点牛，所以当前位置 y+1 的前一头斑点牛至少为 x，这限制了 l[y+1]。

通过扫描线传播约束：
- r[i] = min(r[i], r[i+1])：从右向左保证 r 单调不增
- l[i] = max(l[i], l[i-1])：从左向右保证 l 单调不减

DP 定义：f[i] 表示前 i 头牛中，第 i 头是斑点牛时的最大斑点牛数量。转移：
f[i] = max{ f[j] + 1 | l[i] <= j <= r[i] }

使用单调队列优化 DP 转移：
- 维护一个双端队列，按 f[j] 递减排列。
- 扫描 i 从 1 到 n+1（n+1 为虚拟终点），将满足 j <= r[i] 的 j 加入队列，弹出小于 l[i] 的队首。
- 队列非空时 f[i] = f[队首] + 1，否则 f[i] = -1 表示不可行。

最终答案为 f[n+1] - 1（减去虚拟终点），若 f[n+1] = -1 则输出 -1。

## 代码实现
先读入照片约束更新 l 和 r 数组，再传播约束使其合法。用单调队列优化 DP 求最大可放置斑点牛数。

## 复杂度分析
- 时间复杂度：O(N + M)
- 空间复杂度：O(N)
