# P1083 [NOIP 2012 提高组] 借教室 题解

## 题目链接

- [洛谷 P1083](https://www.luogu.com.cn/problem/P1083)

## 题目大意

有 $n$ 天，每天有 $r_i$ 个教室可用。$m$ 个订单按顺序处理，第 $j$ 个订单在 $[s_j, t_j]$ 天内每天需要 $d_j$ 个教室。找到第一个无法满足的订单编号，若全部满足输出 $0$。

## 解题思路

**二分答案 + 差分数组。**

订单是否可满足具有单调性：若前 $mid$ 个订单能满足，则前 $mid-1$ 个也能满足。二分第一个无法满足的订单编号。

对固定的 $mid$，用差分数组模拟前 $mid$ 个订单的教室占用：
- `diff[s] += d`，`diff[t+1] -= d`。
- 前缀和还原每天的占用，若某天超出 $r_i$ 则不可满足。

## 代码实现

- 二分 $mid \in [1, m]$。
- `check(mid)`：初始化差分数组，累加前 $mid$ 个订单，前缀和检查。
- 若 `check(mid)` 成功，`lo = mid + 1`；否则 `ans = mid, hi = mid - 1`。
- 最后根据 `ans` 输出。

## 复杂度分析

- 时间复杂度：$O((n+m) \log m)$，$n, m \le 10^6$。
- 空间复杂度：$O(n + m)$。
