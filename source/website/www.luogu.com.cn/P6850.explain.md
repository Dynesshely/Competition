# P6850 NOI 题解

## 题目链接

- [洛谷 P6850](https://www.luogu.com.cn/problem/P6850)

## 题目大意

输入 9 个整数，分别表示笔试通过题数、六道题（D1T1~D2T3）的分数、是否为 A 类选手、集训队分数线。计算总分（笔试基础 50 + 笔试通过题数 + 六题分数 + A 类加分 5 分），判断是否达到分数线，达到输出 `AKIOI`，否则输出 `AFO`。

## 解题思路

这是一道简单的条件判断题，按照题面给出的 NOI 计分规则计算总分：

1. 笔试基础 50 分，加上每通过一题 1 分（即笔试得分 = 50 + a）。
2. 总分 = 笔试得分 + 六题得分（b + c + d + e + f + g）。
3. 如果是 A 类选手（h == 1），总分额外加 5 分。
4. 总分 $\ge$ 集训队分数线 i 则输出 `AKIOI`，否则 `AFO`。

## 代码实现

- 变量 `a` 到 `i` 分别对应输入的 9 个整数。
- 条件表达式 `50 + a + b + c + d + e + f + g + (h == 1 ? 5 : 0) >= i` 直接判断是否进队。
- 使用 `scanf` / `printf` 进行标准输入输出。

## 复杂度分析

- 时间复杂度：$O(1)$。
- 空间复杂度：$O(1)$。
