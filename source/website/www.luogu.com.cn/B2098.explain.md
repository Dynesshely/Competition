# B2098 [洛谷] 整数去重 题解

## 题目链接
- [洛谷 B2098](https://www.luogu.com.cn/problem/B2098)

## 题目大意
对序列去重，保留每个数第一次出现的位置，按原顺序输出去重后的序列。

## 解题思路
值域很小（10~100），用数组标记是否已出现过。遍历输入，首次出现的数输出并标记。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, via[105], first;

void input() { scanf("%d", &n); }

void process() {
    for (int i = 1; i <= n; i++) {
        int x;
        scanf("%d", &x);
        if (!via[x]) {
            via[x] = 1;
            if (first) printf(" ");
            printf("%d", x);
            first = 1;
        }
    }
    printf("\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(V)，V 为值域大小
