## 题目背景

mxj 的启发。

## 题目描述

禽兽患传染病了。一个禽兽会每轮传染 $x$ 个禽兽。试问 $n$ 轮传染后有多少禽兽被传染？

## 输入格式

两个非负整数 $x$ 和 $n$。

## 输出格式

一个整数，即被传染的禽兽数。

## 样例 #1

### 样例输入

```
10 2
```

### 样例输出

```
121
```

## 提示

保证 $x,n$ 以及答案都在 $64$ 位无符号整数的表示范围内。
