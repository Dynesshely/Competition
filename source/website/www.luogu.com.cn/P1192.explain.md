# P1192 台阶问题 题解

## 题目链接
- [洛谷 P1192](https://www.luogu.com.cn/problem/P1192)

## 题目大意
有 N 级台阶，一开始在底部（第 0 级），每次可以向上迈 1~K 级台阶。求到达第 N 级台阶的不同方式总数，结果对 100003 取模。

## 解题思路
简单动态规划。

- 设 dp[i] 表示到达第 i 级台阶的方案数，dp[0] = 1。
- 转移：dp[i] = sum(dp[i - j])，其中 1 <= j <= K 且 i - j >= 0。
- 最终答案即为 dp[N] % 100003。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD = 100003;

int         n, k;
vector<int> dp;

void input() { scanf("%d%d", &n, &k); }

void process() {
    dp.resize(n + 1);
    dp[0] = 1;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= k && i - j >= 0; j++)
            dp[i] = (dp[i] + dp[i - j]) % MOD;
}

void output() { printf("%d\n", dp[n]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * K)。
- 空间复杂度：O(N)。
