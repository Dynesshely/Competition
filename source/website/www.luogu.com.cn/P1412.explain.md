# P1412 经营与开发 题解

## 题目链接
- [洛谷 P1412](https://www.luogu.com.cn/problem/P1412)

## 题目大意
飞船按顺序经过 $n$ 个星球，初始钻头能力值为 $w$。每个星球分为两类：
- 资源型（$type=1$）：若开采，获得 $a_i \times p$ 金钱，之后钻头损耗 $k\%$。
- 维修型（$type=2$）：若维修，支付 $b_i \times p$ 金钱，之后钻头修复 $c\%$。

金钱可透支，求最大总收入。

## 解题思路
从后往前动态规划。设 $dp$ 表示从当前星球出发到结束能获得的最大"收益系数"（即每个单位钻头能力能带来的最大收入），则：

- 从后向前遍历，$dp = \max(dp, \text{当前星球操作后的收益} + \text{后续的收益})$。
- 对于资源型星球：若开采，则获得 $a_i$（乘当前 $p$ 为 $a_i \cdot p$），然后 $p$ 变为 $p \cdot (1-0.01k)$，后续收益为 $dp \cdot (1-0.01k)$。所以 $dp = \max(dp, a_i + (1-0.01k) \cdot dp)$。
- 对于维修型星球：若维修，则支付 $b_i$（乘当前 $p$ 为 $b_i \cdot p$），然后 $p$ 变为 $p \cdot (1+0.01c)$，后续收益为 $dp \cdot (1+0.01c)$。所以 $dp = \max(dp, -b_i + (1+0.01c) \cdot dp)$。

最终答案为 $w \times dp$。

由于收益系数与当前 $p$ 成正比，从后向前 DP 避免了正推时 $p$ 不断变化带来的复杂性。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, k, c, w;
int typ[100005];
int val[100005];

void input() {
    scanf("%d%d%d%d", &n, &k, &c, &w);
    for (int i = 0; i < n; i++)
        scanf("%d%d", &typ[i], &val[i]);
}

void process() {
    double kRate = 1.0 - 0.01 * k;
    double cRate = 1.0 + 0.01 * c;
    double dp    = 0.0;
    for (int i = n - 1; i >= 0; i--) {
        if (typ[i] == 1)
            dp = max(dp, val[i] + kRate * dp);
        else
            dp = max(dp, -val[i] + cRate * dp);
    }
    printf("%.2f\n", w * dp);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$。
- 空间复杂度：$O(n)$。
