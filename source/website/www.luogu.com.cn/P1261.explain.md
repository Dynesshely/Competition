# P1261 [洛谷] 服务器储存信息问题 题解

## 题目链接
- [洛谷 P1261](https://www.luogu.com.cn/problem/P1261)

## 题目大意
n 台服务器构成一个无向连通图，每台服务器有重要程度 rank（1-10）。服务器 v 对服务器 w 感兴趣当且仅当不存在服务器 u 使得 r(u) > r(w) 且 δ(v,u) ≤ δ(v,w)。定义 B(v) 为 v 感兴趣的服务器集合，求所有 |B(v)| 之和。

## 解题思路
利用 rank 范围很小（1-10）这一关键性质，设计高效的算法：

1. **预处理**：对每个 rank 值 r，计算所有节点到 rank = r 的节点的最近距离（多源 Dijkstra），记录在 best[r][v] 中。
2. **计算限制距离**：定义 limit[r][v] = min_{k > r} best[k][v]，表示从 v 出发到 rank 大于 r 的节点的最近距离。这可以通过从大到小递推得到。
3. **统计答案**：对每个节点 s（rank = r），以 s 为源点运行 Dijkstra，但利用 limit 进行剪枝：当当前距离 d ≥ limit[r][v] 时，v 不可达（因为存在 rank 更高的节点更近），无需继续扩展。这样保证了总复杂度不超过 O(30n)。
   - 对于最高 rank 的节点，它会对所有节点感兴趣（因为不存在更高 rank 的节点），直接累加。
