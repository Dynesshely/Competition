# P1226 快速幂 题解

## 题目链接

- [洛谷 P1226](https://www.luogu.com.cn/problem/P1226)

## 题目大意

求 $a^b \bmod p$。

## 解题思路

快速幂（二进制取幂）。将指数 $b$ 按二进制分解，每次底数自乘、指数右移。遇 1 则乘入结果。时间复杂度 $O(\log b)$。

## 代码实现

- `base = a % p, exp = b, ans = 1`。
- 循环：若 `exp & 1` 则 `ans = ans * base % p`；`base = base * base % p`；`exp >>= 1`。
- 按格式输出 `a^b mod p=ans`。

## 复杂度分析

- 时间复杂度：$O(\log b)$。
- 空间复杂度：$O(1)$。
