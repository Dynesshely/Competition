## 题目描述

给定 $3$ 个整数 $a,b,c$，计算表达式 $(a+b)/c$ 的值，`/` 是整除运算。

## 输入格式

输入仅一行，包括三个整数 $a,b,c$，数与数之间以一个空格分开。

## 输出格式

输出一行，即表达式的值。

## 样例 #1

### 样例输入

```
1 1 3
```

### 样例输出

```
0
```

## 提示

对于 $100 \%$ 的数据，$-{10}^4 < a, b, c < {10}^4$，$c \ne 0$。
