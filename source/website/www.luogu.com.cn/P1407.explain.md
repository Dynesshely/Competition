# P1407 [国家集训队] 稳定婚姻 题解

## 题目链接
- [洛谷 P1407](https://www.luogu.com.cn/problem/P1407)

## 题目大意
给定 $n$ 对夫妻的婚姻状况，以及 $m$ 对曾经交往过的情侣关系。若一对夫妻离婚后，所有人最终仍能重新组合成 $n$ 对情侣（通过旧情复燃的连锁反应），则称这段婚姻是不安全的，否则是安全的。判断每对婚姻是否安全。

## 解题思路
将问题建模为有向图，利用强连通分量（Tarjan 算法）求解：

- 对每对夫妻 $i$，建边 $G_i \rightarrow B_i$（妻子到丈夫）；
- 对每对曾经的情侣 $(G_j, B_i)$，建边 $B_i \rightarrow G_j$（丈夫到曾经的恋人）。

若某对夫妻中的两人在同一个强连通分量中，则他们离婚后可以通过 SCC 内的环重新配对，这段婚姻不安全；否则安全。

原理：若 $G_i$ 和 $B_i$ 在同一个 SCC 中，说明存在一条交替路径（夫妻边和情侣边交替）形成环，使得离婚后所有人可以重新配对。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int              n, m;
map<string, int> id;
vector<int>      g[8005];
int              wife[4005], husband[4005];
int              dfn[8005], low[8005], scc[8005], stk[8005], top, idx, sccCnt;
bool             inStk[8005];

void tarjan(int u) {
    dfn[u] = low[u] = ++idx;
    stk[++top]      = u;
    inStk[u]        = true;
    for (int v : g[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (inStk[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }
    if (low[u] == dfn[u]) {
        ++sccCnt;
        int v;
        do {
            v        = stk[top--];
            inStk[v] = false;
            scc[v]   = sccCnt;
        } while (v != u);
    }
}

void input() {
    scanf("%d", &n);
    char girl[16], boy[16];
    for (int i = 1; i <= n; ++i) {
        scanf("%s%s", girl, boy);
        if (!id.count(girl))
            id[girl] = id.size();
        if (!id.count(boy))
            id[boy] = id.size();
        wife[i]    = id[girl];
        husband[i] = id[boy];
        g[wife[i]].push_back(husband[i]);
    }
    scanf("%d", &m);
    for (int i = 1; i <= m; ++i) {
        scanf("%s%s", girl, boy);
        int u = id[girl], v = id[boy];
        g[v].push_back(u);
    }
}

void process() {
    int tot = id.size();
    for (int i = 0; i < tot; ++i)
        if (!dfn[i])
            tarjan(i);
}

void output() {
    for (int i = 1; i <= n; ++i) {
        if (scc[wife[i]] == scc[husband[i]])
            printf("Unsafe\n");
        else
            printf("Safe\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n + m)$，Tarjan 算法线性遍历图。
- 空间复杂度：$O(n + m)$。
