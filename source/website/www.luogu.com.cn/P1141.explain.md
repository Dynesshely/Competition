# P1141 01迷宫 题解

## 题目链接

- [洛谷 P1141](https://www.luogu.com.cn/problem/P1141)

## 题目大意

给定一个 n x n 的 01 矩阵，位于 0 的格子可以移动到相邻的 1 格子，位于 1 的格子可以移动到相邻的 0 格子。有 m 个询问，每次询问从某个格子出发能到达多少个格子（含自身）。

## 解题思路

连通分量预处理。将可以互达的格子划分为同一连通分量，使用 BFS/DFS 染色。每个连通分量的大小即为该分量中所有格子的答案。对于每个询问 O(1) 输出所在分量的大小。

## 代码实现

BFS 遍历整个矩阵，对每个未访问的格子进行 BFS，扩展时要求相邻格子的字符值不同。使用 comp 数组标记每个格子所属的连通分量编号，compSz 数组记录每个分量的大小。

## 复杂度分析

- 时间复杂度：O(n^2 + m)
- 空间复杂度：O(n^2)
