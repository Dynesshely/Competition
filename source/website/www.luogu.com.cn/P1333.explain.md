# P1333 洛谷 瑞瑞的木棍 题解

## 题目链接
- [洛谷 P1333](https://www.luogu.com.cn/problem/P1333)

## 题目大意
给定若干根木棍，每根木棍两端各有一种颜色（用单词表示）。将这些木棍首尾相连排成一条线，要求相邻的两端颜色相同。问是否存在一种排列方式，将全部木棍连接成一条完整的线。

## 解题思路
本题等价于判断一个无向图是否存在欧拉通路（Eulerian trail）：
- 将每种颜色视为一个顶点，每根木棍视为一条连接两端颜色的无向边。
- 欧拉通路存在的充要条件：
  1. 该图是连通的（度非零的顶点属于同一个连通分量）。
  2. 度数为奇数的顶点个数为 $0$ 或 $2$。

实现步骤：
1. 使用哈希表（`unordered_map`）将颜色字符串映射为顶点编号（动态添加）。
2. 统计每个顶点的度数。
3. 使用并查集维护连通性。
4. 检查奇数度顶点个数是否为 $0$ 或 $2$，以及所有非零度顶点是否属于同一连通分量。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

unordered_map<string, int> mp;
vector<int>                deg, fa, sz;
int                        n;
bool                       ans;

int find(int x) {
    while (fa[x] != x) {
        fa[x] = fa[fa[x]];
        x     = fa[x];
    }
    return x;
}

void input() {
    char a[15], b[15];
    n = 0;
    while (scanf("%s %s", a, b) == 2) {
        string sa(a), sb(b);
        if (!mp.count(sa)) {
            mp[sa] = n++;
            deg.push_back(0);
            fa.push_back(n - 1);
            sz.push_back(1);
        }
        if (!mp.count(sb)) {
            mp[sb] = n++;
            deg.push_back(0);
            fa.push_back(n - 1);
            sz.push_back(1);
        }
        int u = mp[sa], v = mp[sb];
        deg[u]++;
        deg[v]++;
        int fu = find(u), fv = find(v);
        if (fu != fv) {
            if (sz[fu] < sz[fv])
                swap(fu, fv);
            fa[fv] = fu;
            sz[fu] += sz[fv];
        }
    }
}

void process() {
    if (!n) {
        ans = true;
        return;
    }
    int odd = 0;
    for (int i = 0; i < n; i++)
        if (deg[i] & 1)
            odd++;
    if (odd != 0 && odd != 2) {
        ans = false;
        return;
    }
    int root = -1;
    for (int i = 0; i < n; i++) {
        if (!deg[i])
            continue;
        int fi = find(i);
        if (root == -1)
            root = fi;
        else if (fi != root) {
            ans = false;
            return;
        }
    }
    ans = true;
}

void output() { printf("%s\n", ans ? "Possible" : "Impossible"); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(N \log^{*} N)$，其中 $N$ 为颜色种类数，主要来自并查集操作。
- 空间复杂度：$O(N)$，用于存储哈希表、度数、并查集等。
