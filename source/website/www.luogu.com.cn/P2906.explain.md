# P2906 [USACO08OPEN] Cow Neighborhoods G 题解

## 题目链接

- [洛谷 P2906](https://www.luogu.com.cn/problem/P2906)

## 题目大意

$N$ 头奶牛，每头坐标 $(X_i, Y_i)$。若两头奶牛曼哈顿距离 $\le C$，则它们互为邻居。邻居关系具有传递性，形成"奶牛社区"。求社区数量和最大社区的奶牛数。

## 解题思路

### 坐标变换

曼哈顿距离 $|X_i - X_j| + |Y_i - Y_j| \le C$ 难以直接处理。通过坐标变换：

$$x_i = X_i + Y_i,\quad y_i = X_i - Y_i$$

则曼哈顿距离条件变为 $\max(|x_i - x_j|, |y_i - y_j|) \le C$（切比雪夫距离），等价于 $|x_i - x_j| \le C$ 且 $|y_i - y_j| \le C$。

### 扫描线 + 并查集

1. 按 $x$ 坐标升序排序所有奶牛。
2. 双指针维护滑动窗口：窗口内所有牛的 $x$ 坐标差 $\le C$。
3. 对窗口内的牛，用 `multiset` 按 $y$ 坐标维护，查找 $y$ 坐标最近的邻居（`lower_bound`），若 $|y|$ 差 $\le C$ 则用并查集合并。
4. 最终统计连通块数量和最大块大小。

## 代码实现

- 坐标变换后排序。
- `multiset` 插入哨兵 $\pm\infty$。
- 滑动窗口：左指针 $l$ 随 $i$ 右移，将 $x$ 差 $> C$ 的牛从 multiset 移除。
- 对当前牛 $i$，查找 $y$ 最近的上下邻居，若距离 $\le C$ 则 `merge`。
- 统计：`find(i) == i` 计数连通块，`sz[find(i)]` 统计块大小。

## 复杂度分析

- 时间复杂度：$O(N \log N)$。
- 空间复杂度：$O(N)$。
