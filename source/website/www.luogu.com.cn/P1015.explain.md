# P1015 [NOIP 1999 普及组] 回文数 题解

## 题目链接

- [洛谷 P1015](https://www.luogu.com.cn/problem/P1015)

## 题目大意

给定一个 $N$ 进制数 $M$（$N \le 16$，$M$ 的长度 $\le 100$），每次操作：将 $M$ 与其反转相加（按 $N$ 进制运算）。若在 $30$ 步内变为回文数则输出步数，否则输出 `Impossible!`。

## 解题思路

**模拟 + N 进制高精度加法**。用 `vector<int>` 存储 $N$ 进制数的每一位。每次操作：

1. 求反转数 `b = reverse(a)`。
2. 逐位相加 `a[i] + b[i] + carry`，结果模 $N$ 存为当前位，进位向下一位传播。
3. 若最终有进位则补一位。
4. 检查结果是否回文。

若 $30$ 步内回文则输出 `STEP=步数`，否则 `Impossible!`。

## 代码实现

- 读入时将字符转数字：`0-9` → $0\sim9$，`A-F` → $10\sim15$。
- 加法时数组 `a` 高位在前（方便反转），逐位从低位开始运算。
- 回文检查：比较 `a[i]` 和 `a[len-1-i]`。

## 复杂度分析

- 时间复杂度：$O(L \cdot steps)$，$L \le 100$（可能因进位增长到 $\sim 130$），$steps \le 30$。
- 空间复杂度：$O(L)$。
