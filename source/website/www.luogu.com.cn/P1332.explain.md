# P1332 洛谷 血色先锋队 题解

## 题目链接
- [洛谷 P1332](https://www.luogu.com.cn/problem/P1332)

## 题目大意
给定一个 $n \times m$ 的矩阵，有 $a$ 个感染源（初始被感染的位置），感染每过一个小时会向上下左右四个方向扩散一格。有 $b$ 个领主，需要输出每个领主所在位置被感染的时间。如果某个位置就是感染源，则感染时间为 $0$。

## 解题思路
典型的多源 BFS 最短路问题：
1. 将所有感染源作为 BFS 的初始节点，距离设为 $0$ 并入队。
2. 执行 BFS，每次向四个方向扩展，更新未访问位置的距离（即感染时间）。
3. 对于每个领主的询问，直接输出从最近感染源到该位置的距离。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int                   n, m, a, b;
int                   dx[] = {0, 0, 1, -1};
int                   dy[] = {1, -1, 0, 0};
int                   dist[505][505];
queue<pair<int, int>> q;

void input() {
    scanf("%d%d%d%d", &n, &m, &a, &b);
    memset(dist, -1, sizeof(dist));
    for (int i = 0; i < a; i++) {
        int x, y;
        scanf("%d%d", &x, &y);
        dist[x][y] = 0;
        q.push({x, y});
    }
}

void process() {
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i], ny = y + dy[i];
            if (nx < 1 || nx > n || ny < 1 || ny > m)
                continue;
            if (dist[nx][ny] != -1)
                continue;
            dist[nx][ny] = dist[x][y] + 1;
            q.push({nx, ny});
        }
    }
}

void output() {
    for (int i = 0; i < b; i++) {
        int x, y;
        scanf("%d%d", &x, &y);
        printf("%d\n", dist[x][y]);
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \times m)$，BFS 遍历整个矩阵。
- 空间复杂度：$O(n \times m)$，用于存储距离矩阵和队列。
