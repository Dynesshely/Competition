# P1371 NOI元丹 题解

## 题目链接
- [洛谷 P1371](https://www.luogu.com.cn/problem/P1371)

## 题目大意
给定一个只含 `N`、`O`、`I` 三种字符的字符串，求其中 `NOI` 子序列（不要求连续）的个数。允许在任意位置插入一个额外字符（`N`、`O` 或 `I` 任选），使得操作后 `NOI` 子序列的数量最大化。

## 解题思路
先计算原串中 `NOI` 子序列的数量：
- 从左到右扫描，维护前缀中 `N` 的数量 `preN[i]`。
- 从右到左扫描，维护后缀中 `I` 的数量 `sufI[i]`。
- 对每个 `O`，以它为中间的 `NOI` 子序列数为 `preN[i-1] * sufI[i+1]`，累加即得原串答案。

考虑插入一个字符的最大增益：
- 若在开头插入 `N`：新增的 `NOI` 子序列数为插入后整个串中 `OI` 子序列数，即原串的 `OI` 子序列总数 `cntOI`。
- 若在末尾插入 `I`：新增的 `NOI` 子序列数为原串的 `NO` 子序列总数 `cntNO`。
- 若在某位置插入 `O`：在位置 $i$ 插入时，新增数为该位置前的 `N` 数量乘后的 `I` 数量，即 `preN[i-1] * sufI[i]`，取最大值。

取上述增益的最大值加到原串答案上即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int       n;
char      s[100005];
long long preN[100005], sufI[100005];
long long ans, bestGain;

void input() { scanf("%d%s", &n, s + 1); }

void process() {
    preN[0] = 0;
    for (int i = 1; i <= n; ++i)
        preN[i] = preN[i - 1] + (s[i] == 'N');
    sufI[n + 1] = 0;
    for (int i = n; i >= 1; --i)
        sufI[i] = sufI[i + 1] + (s[i] == 'I');
    long long cntN = 0, cntNO = 0;
    for (int i = 1; i <= n; ++i)
        if (s[i] == 'N')
            ++cntN;
        else if (s[i] == 'O') {
            ans += preN[i - 1] * sufI[i + 1];
            cntNO += cntN;
        }
    long long cntI = 0, cntOI = 0;
    for (int i = n; i >= 1; --i)
        if (s[i] == 'I')
            ++cntI;
        else if (s[i] == 'O')
            cntOI += cntI;
    bestGain = cntOI;
    if (cntNO > bestGain)
        bestGain = cntNO;
    for (int i = 1; i <= n + 1; ++i) {
        long long cur = preN[i - 1] * sufI[i];
        if (cur > bestGain)
            bestGain = cur;
    }
    ans += bestGain;
}

void output() { printf("%lld\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
