# P1450 [HAOI2008] 硬币购物 题解

## 题目链接

- [洛谷 P1450](https://www.luogu.com.cn/problem/P1450)

## 题目大意

共有 $4$ 种硬币，面值分别为 $c_1, c_2, c_3, c_4$。有 $n$ 次购买，每次购买给定每种硬币的数量上限 $d_i$ 和目标金额 $s$，求使用不超过数量上限的硬币凑出 $s$ 的方案数。数据范围：$c_i, d_i, s \le 10^5$，$n \le 1000$。

## 解题思路

直接对每组询问做多重背包会超时。利用 $4$ 种硬币这一较小的规模，采用 **完全背包预处理 + 容斥原理**。

**步骤一：完全背包预处理**

先忽略每种硬币的数量限制，求出用这 $4$ 种硬币凑出任意金额 $j$ 的方案数 $f[j]$（完全背包）：

$$f[0] = 1,\quad f[j] = \sum_{i=1}^4 f[j - c_i]\ (j \ge c_i)$$

**步骤二：容斥原理**

对于一次询问 $(d_1, d_2, d_3, d_4, s)$，我们求的是每个硬币使用不超过 $d_i$ 的方案数。用容斥原理从总方案中减去超过限制的方案：

$$\text{ans} = \sum_{\text{mask}=0}^{15} (-1)^{\text{bitcount(mask)}} \cdot f\!\left(s - \sum_{i \in \text{mask}} (d_i + 1) \cdot c_i\right)$$

其中 $\text{mask}$ 枚举哪些硬币超过限制（强制先用 $d_i+1$ 枚，剩余任意）。若剩余金额非负则累加，根据超过限制的硬币个数奇偶性决定加减符号（奇减偶加）。

枚举所有 $2^4 = 16$ 个子集，每次 $O(1)$ 计算，单次询问 $O(16)$。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int       c[5], d[5], n, s;
long long f[100005];

void input() { scanf("%d%d%d%d%d", &c[1], &c[2], &c[3], &c[4], &n); }

void process() {
    f[0] = 1;
    for (int i = 1; i <= 4; i++)
        for (int j = c[i]; j <= 100000; j++)
            f[j] += f[j - c[i]];
}

void output() {
    while (n--) {
        scanf("%d%d%d%d%d", &d[1], &d[2], &d[3], &d[4], &s);
        long long ans = 0;
        for (int mask = 0; mask < 16; mask++) {
            int       bits = __builtin_popcount(mask);
            long long sum  = s;
            for (int i = 0; i < 4; i++)
                if (mask >> i & 1)
                    sum -= 1LL * (d[i + 1] + 1) * c[i + 1];
            if (sum >= 0) {
                if (bits & 1)
                    ans -= f[sum];
                else
                    ans += f[sum];
            }
        }
        printf("%lld\n", ans);
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：预处理完全背包 $O(4 \cdot \max s) = O(4 \times 10^5)$，每次询问 $O(16)$，$n$ 次询问 $O(16n) = O(1.6 \times 10^4)$，总复杂度约 $O(4 \times 10^5)$。
- 空间复杂度：$O(\max s) = O(10^5)$，预处理的 DP 数组。
