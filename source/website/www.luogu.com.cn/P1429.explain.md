# P1429 平面最近点对（加强版） 题解

## 题目链接
- [洛谷 P1429](https://www.luogu.com.cn/problem/P1429)

## 题目大意
给定平面上 $n$ 个点，找出所有点对中的最短距离，精确到小数点后 $4$ 位。$n \le 200000$，坐标范围 $0 \le x,y \le 10^9$。

## 解题思路
采用分治法求解平面最近点对问题。将所有点按 $x$ 坐标排序后递归处理：

- 递归终止条件：区间内点数 $\le 2$ 时直接计算距离。
- 分治过程：取中点将点集分为左右两部分，递归得到两侧的最小距离 $d$。
- 合并处理：筛选出距中线距离小于 $d$ 的点，按 $y$ 坐标排序后，对每个点检查与其 $y$ 坐标差小于 $d$ 的点并更新距离，利用鸽巢原理保证每个点最多检查 $6$ 个点。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Point {
    int x, y;
};

Point p[200005], tmp[200005];
int   n;

double dist(const Point &a, const Point &b) {
    long long dx = a.x - b.x, dy = a.y - b.y;
    return sqrt(dx * dx + dy * dy);
}

bool cmpX(const Point &a, const Point &b) { return a.x < b.x; }

bool cmpY(const Point &a, const Point &b) { return a.y < b.y; }

double solve(int l, int r) {
    if (l >= r)
        return 1e100;
    if (r - l == 1)
        return dist(p[l], p[r]);
    int    mid  = (l + r) / 2;
    int    midX = p[mid].x;
    double d    = min(solve(l, mid), solve(mid + 1, r));
    int    cnt  = 0;
    for (int i = l; i <= r; i++)
        if (abs(p[i].x - midX) < d)
            tmp[cnt++] = p[i];
    sort(tmp, tmp + cnt, cmpY);
    for (int i = 0; i < cnt; i++)
        for (int j = i + 1; j < cnt && tmp[j].y - tmp[i].y < d; j++)
            d = min(d, dist(tmp[i], tmp[j]));
    return d;
}

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d%d", &p[i].x, &p[i].y);
}

void process() { sort(p, p + n, cmpX); }

void output() { printf("%.4lf\n", solve(0, n - 1)); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log^2 n)$（排序 $O(n \log n)$，分治合并时按 $y$ 排序 $O(n \log n)$，总复杂度 $O(n \log^2 n)$），可优化至 $O(n \log n)$ 归并排序。
- 空间复杂度：$O(n)$ 用于存储临时数组。
