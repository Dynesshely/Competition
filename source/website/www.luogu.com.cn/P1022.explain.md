# P1022 [NOIP 2000 普及组] 计算器的改良 题解

## 题目链接

- [洛谷 P1022](https://www.luogu.com.cn/problem/P1022)

## 题目大意

输入一个合法的一元一次方程（含整数、小写字母及 `+`、`-`、`=`），求解方程，结果精确到小数点后三位。

## 解题思路

解析字符串方程，将未知数的系数和常数项分别归并到等式两侧，然后计算 $x = \frac{\text{常数项}}{\text{未知数系数}}$。

核心思想：
1. 将方程按左侧和右侧分开解析，每个"段"用 `section` 结构体表示，记录该段是否为未知数项、是否为负数。
2. 遇到 `+`、`-`、`=` 时切分当前段并开始新段。
3. 解析完成后，分别累加左侧和右侧的常数项、未知数系数。
4. 最终 $x = \frac{\text{右侧常数} - \text{左侧常数}}{\text{左侧系数} - \text{右侧系数}}$。

## 代码实现

关键数据结构 `section`：
- `prefixes`: 存储该段数值的各位数字（从高位到低位）。
- `isX`: 标记该段是否为未知数项。
- `isNagative`: 标记该段是否为负数。
- `getValue()`: 将 `prefixes` 转换为整数，返回带符号的数值。

解析流程（`main` 函数）：
- 使用指针 `cs` 指向当前处理的向量（左侧 `left` 或右侧 `right`）。
- 遍历字符串：`+` 切段、`-` 切段并设置负标记、`=` 切换到右侧、数字追加到 `prefixes`、字母标记为未知数。
- 使用 `skipFirstNegative` 变量处理方程开头即为负数的情况（如 `-5+12y=0`），此时不应切出空段。
- 解析完成后分别计算左右两侧的常数和系数，解出未知数。
- 使用 `printf("%c=%.3f\n", unknown, result)` 格式化输出，精确到三位小数。

> 注意：代码中 `getValue()` 使用 `pow(10, p)` 逐位还原数字，实现较为复杂。更简洁的方式可以是直接处理字符串数值。

## 复杂度分析

- 时间复杂度：$O(L)$，其中 $L$ 为方程字符串长度，每个字符处理一次。
- 空间复杂度：$O(L)$，存储所有解析出的段结构体。
