# P3064 [USACO12DEC] Gangs of Istanbull/Cowstantinople G 题解

## 题目链接
- [洛谷 P3064](https://www.luogu.com.cn/problem/P3064)

## 题目大意
有 $N$ 头奶牛分属 $M$ 个帮派，帮派 $1$ 是 Bessie 所在的帮派。每分钟有一头奶牛进入牧场，入场规则如下：
- 若牧场为空，则该奶牛所属帮派控制牧场。
- 若牧场已被该帮派控制，则奶牛直接吃草。
- 否则，新来奶牛与当前控制帮派的一头奶牛对峙，两头奶牛同时离开牧场。

Bessie 知道每个帮派的奶牛数量，希望冲突结束后帮派 $1$ 控制牧场。判断是否可能，若可能则输出：
1. `YES`
2. 最后留在牧场上的帮派 $1$ 奶牛的最大数量
3. 达到该数量的字典序最小的奶牛入场顺序

## 解题思路

### 可行性判断
将帮派 $1$ 视为"我方"，其余帮派视为"敌方"。每一头敌方奶牛最多可以抵消一头我方奶牛（通过对峙同时离场）。因此，我方奶牛数量必须严格大于"必要抵消量"。

设 `tot` 为敌方奶牛总数，`a[mx]` 为敌方最大帮派的成员数。必要抵消量 `least` 为：
$$
least = \max(a[mx] - (tot - a[mx]),\; tot \bmod 2)
$$

- $a[mx] - (tot - a[mx])$：最大敌方帮派与其他帮派的人数差。若最大帮派人数超过其余帮派总和，则我方需要额外派人抵消其超出部分，否则该帮派会一直控制牧场。
- $tot \bmod 2$：最终牧场中必定保留的帮派 $1$ 奶牛数量要与敌方总数奇偶性匹配（每次对峙双方各减一，奇偶性不变）。

若 $a[1] \le least$ 则不可能，输出 `NO`；否则最大保留数为 $a[1] - least$。

### 构造字典序最小序列
构造思路是贪心地让编号小的帮派尽量早出现（因为字典序比较的是出场顺序）。策略如下：

1. **前置保护阶段**：当最大敌方帮派仍可能在未来占据优势时，先派我方奶牛入场作为缓冲。
2. **奇偶调整**：若当前总剩余奶牛数的奇偶性需要调整，再派一头我方奶牛。
3. **消耗阶段**：交替派出敌方奶牛，优先消耗最大帮派的奶牛以减少威胁。当需要时，整批派出一整个帮派的全部奶牛（`curCnt` 机制）以最小化出现的帮派编号。
4. **收尾阶段**：派出剩余的我方奶牛和最后一组敌方奶牛。

每一步的决策依据 `doCalc()` 重新计算的 `tot`、`mx`、`nxt` 来动态调整，确保最终牧场由帮派 $1$ 控制且保留数最大。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int a[105], n, m, nxt, tot, mx, cur, curCnt;

void doCalc() {
    nxt = tot = mx = 0;
    for (int i = 2; i <= m; ++i) {
        if (a[i] <= 0)
            continue;
        if (!nxt)
            nxt = i;
        tot += a[i];
        if (!mx || a[i] > a[mx])
            mx = i;
    }
}

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; ++i)
        scanf("%d", &a[i]);
    doCalc();
    int least = max(a[mx] - (tot - a[mx]), tot % 2);
    if (a[1] <= least) {
        printf("NO\n");
        return 0;
    }
    printf("YES\n%d\n", a[1] - least);
    while (a[mx] > curCnt + tot - a[mx]) {
        printf("1\n");
        --a[1];
        cur = 1;
        ++curCnt;
    }
    if ((curCnt + tot) % 2) {
        printf("1\n");
        --a[1];
        cur = 1;
        ++curCnt;
    }
    while (tot) {
        if (curCnt) {
            if (a[mx] > curCnt + tot - a[mx] - 2) {
                printf("%d\n", mx);
                --a[mx];
            } else {
                printf("%d\n", nxt);
                --a[nxt];
            }
            --curCnt;
            if (!curCnt)
                cur = 0;
        } else {
            cur    = nxt;
            curCnt = a[nxt];
            while (a[nxt]--)
                printf("%d\n", nxt);
        }
        doCalc();
        if (tot == a[mx] && (!curCnt || cur == 1))
            break;
    }
    while (a[1]--)
        printf("1\n");
    while (a[nxt]--)
        printf("%d\n", nxt);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(NM)$，其中 $N$ 为奶牛总数，$M$ 为帮派数，每次迭代都需要 $O(M)$ 重新计算统计信息。
- 空间复杂度：$O(M)$。
