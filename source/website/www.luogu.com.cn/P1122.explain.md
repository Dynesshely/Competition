# P1122 最大子树和 题解

## 题目链接

- [洛谷 P1122](https://www.luogu.com.cn/problem/P1122)

## 题目大意

给定一棵 n 个节点的树，每个节点有一个权值（可能为负）。可以进行若干次"修剪"操作：每次去掉一条边，扔掉其中一棵子树。求经过若干次修剪后剩下的连通子图的最大权值和（可以一步也不修剪）。

## 解题思路

树形 DP。任选一个根（如 1），进行一次 DFS。对于每个节点 u，设 dp[u] 表示以 u 为根的子树中，选取包含 u 的连通块所能获得的最大权值和。转移时，对于每个子节点 v，若 dp[v] > 0 则累加（因为剪掉负收益的子树更优），否则丢弃。即 dp[u] = a[u] + sum(max(0, dp[v]))。全局答案取所有 dp[u] 的最大值。

## 代码实现

- 邻接表建图，从节点 1 开始 DFS（传入父节点参数避免回走）。
- 递归计算子节点的 dp 值，若为正则累加到当前节点。
- 遍历所有节点取 dp 最大值作为答案。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(n)。
