# P1365 洛谷 WJMZBMR打osu! / Easy 期望DP 题解

## 题目链接
- [洛谷 P1365](https://www.luogu.com.cn/problem/P1365)

## 题目大意
给定一个长度为 n 的字符串，由 `o`、`x`、`?` 三种字符组成。`o` 表示本次点击成功（combo 长度增加 1），`x` 表示失败（combo 清零），`?` 表示有 50% 概率成功、50% 概率失败。连续 a 次成功可得到 a² 分。求期望得分。

## 解题思路
使用期望 DP，维护当前期望 combo 长度 len 和期望总得分 ans，线性扫描字符串：

- 遇到 `o`：combo 长度加 1，新增得分 = (len+1)² - len² = 2×len + 1，故 ans += 2×len + 1，len = len + 1。
- 遇到 `x`：combo 清零，len = 0，得分不变。
- 遇到 `?`：有 1/2 概率为 `o`，1/2 概率为 `x`。
  - 期望新增得分 = (1/2) × (2×len + 1) + (1/2) × 0 = (2×len + 1) / 2。
  - 期望新 combo 长度 = (1/2) × (len + 1) + (1/2) × 0 = (len + 1) / 2。

最终输出 ans。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n;
char        s[300010];
long double ans;

void input() { scanf("%d%s", &n, &s); }

void process() {
    long double len = 0;
    ans             = 0;
    for (int i = 0; i < n; i++) {
        if (s[i] == 'o') {
            ans += 2 * len + 1;
            len = len + 1;
        } else if (s[i] == 'x') {
            len = 0;
        } else {
            ans += (2 * len + 1) / 2;
            len = (len + 1) / 2;
        }
    }
}

void output() { printf("%.4Lf\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，一次线性扫描。
- 空间复杂度：O(1)，仅使用常数个变量（不计输入字符串存储）。
