# P1327 洛谷 数列排序 题解

## 题目链接
- [洛谷 P1327](https://www.luogu.com.cn/problem/P1327)

## 题目大意
给定一个互不相同的数列 a，每次可以交换任意两个数，求将数列从小到大排序所需的最少交换次数。

## 解题思路
这是一个排列环（置换循环）问题。

将数列与排序后的目标位置建立映射关系。对于原数列的每个位置，它当前存放的值应该被放到排序后的对应位置，由此形成若干个置换环。每个长度为 L 的环需要 L-1 次交换才能还原。

具体做法：
1. 将原数组的值与下标记录为 pair 并排序，得到每个元素的目标位置映射数组 `to`
2. 遍历每个位置 i，若尚未访问且 to[i] != i（即不在正确位置），则沿着 to 指针遍历整个环，统计环长度 len
3. 该环需要 len - 1 次交换，累加所有环的 (len - 1) 即为答案

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, a[100005], to[100005];
bool via[100005];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i)
        scanf("%d", &a[i]);
}

void process() {
    vector<pair<int, int>> p(n);
    for (int i = 0; i < n; ++i)
        p[i] = {a[i], i};
    sort(p.begin(), p.end());
    for (int i = 0; i < n; ++i)
        to[p[i].second] = i;
}

void output() {
    int ans = 0;
    for (int i = 0; i < n; ++i) {
        if (via[i] || to[i] == i)
            continue;
        int len = 0, cur = i;
        while (!via[cur]) {
            via[cur] = true;
            cur      = to[cur];
            ++len;
        }
        ans += len - 1;
    }
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log n)，排序为主
- 空间复杂度：O(n)
