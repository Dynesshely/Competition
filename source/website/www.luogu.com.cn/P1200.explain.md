# P1200 [USACO1.1] 你的飞碟在这儿 题解

## 题目链接

- [洛谷 P1200](https://www.luogu.com.cn/problem/P1200)

## 题目大意

给定两个长度不超过 $6$ 的大写字母字符串，分别代表彗星名和小组名。将每个字母转为数字（A=1, B=2, ..., Z=26），求所有字母数字之积模 $47$。若两串结果相等输出 `GO`，否则输出 `STAY`。

## 解题思路

简单模拟。遍历字符串，将字母转为 `ch - 'A' + 1`，累乘并对 $47$ 取模。比较两个结果。

## 代码实现

- `calc(s)`：遍历字符串 $s$，`res = 1`，`res = res * (s[i] - 'A' + 1) % 47`。
- 读入两个字符串，调用 `calc` 比较，输出 `GO` 或 `STAY`。

## 复杂度分析

- 时间复杂度：$O(|s_1| + |s_2|)$，字符串长 $\le 6$。
- 空间复杂度：$O(1)$。
