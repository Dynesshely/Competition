# P1440 求m区间内的最小值 题解

## 题目链接
- [洛谷 P1440](https://www.luogu.com.cn/problem/P1440)

## 题目大意
给定一个 $n$ 项的数列，对于每个 $i$，求 $a_i$ 之前 $m$ 个数（即 $a_{i-m}$ 到 $a_{i-1}$）的最小值。若前面不足 $m$ 个数则从 $a_1$ 开始取；若前面没有数则输出 $0$。$n \le 2 \times 10^6$。

## 解题思路
单调队列维护滑动窗口最小值。遍历数列时，队列中存储元素下标，保证队列对应的 $a$ 值单调递增。

对于每个位置 $i$：
1. 若队列不为空，队首元素即为前 $m$ 个数的最小值（输出 $a[que[head]]$），否则输出 $0$。
2. 移除超出窗口范围的下标（$que[head] \le i - m$）。
3. 移除队列中所有 $a$ 值大于等于 $a[i]$ 的元素（维护单调性）。
4. 将 $i$ 入队。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 2000005;

int n, m, a[MAXN], que[MAXN], head, tail;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++i)
        scanf("%d", &a[i]);
}

void process() {
    head = 1, tail = 0;
    for (int i = 1; i <= n; ++i) {
        if (head <= tail)
            printf("%d\n", a[que[head]]);
        else
            printf("0\n");
        while (head <= tail && que[head] <= i - m)
            ++head;
        while (head <= tail && a[que[tail]] >= a[i])
            --tail;
        que[++tail] = i;
    }
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$，每个元素入队出队最多一次。
- 空间复杂度：$O(n)$ 用于数列和队列。
