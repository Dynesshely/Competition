# P1432 倒水问题 题解

## 题目链接
- [洛谷 P1432](https://www.luogu.com.cn/problem/P1432)

## 题目大意
给定两个水壶 $A$ 和 $B$，容量分别为 $C_a, C_b$，目标水量 $N$。有 6 种操作：装满 A/B、倒空 A/B、将 A 倒入 B 或将 B 倒入 A。求使水壶 $B$ 中恰好有 $N$ 单位水的最少操作步骤数及方案。$T \le 30$，$C_a \le C_b \le 1000$，$C_a$ 与 $C_b$ 互质。

## 解题思路
BFS 搜索状态空间。每个状态由 $(a, b)$ 表示，分别表示两个水壶中的水量。从 $(0, 0)$ 开始 BFS，每次应用 6 种操作生成新状态，直到水壶 $B$ 中的水量为 $N$。

BFS 保证找到的路径是最短操作序列。记录每个状态的前驱状态和执行的操作编号，最后回溯还原步骤。

6 种操作编号：
1. fill A
2. fill B
3. empty A
4. empty B
5. pour B A（将 B 倒入 A 直到 A 满或 B 空）
6. pour A B（将 A 倒入 B 直到 B 满或 A 空）

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int         Ca, Cb, N;
vector<int> ops;

struct State {
    int a, b, op, prev;
};

void input() { scanf("%d%d%d", &Ca, &Cb, &N); }

void process() {
    ops.clear();
    int           cols  = Cb + 1;
    int           total = (Ca + 1) * cols;
    vector<bool>  vis(total, false);
    vector<State> states;
    states.reserve(total);
    queue<int> q;

    auto addState = [&](int a, int b, int op, int prev) {
        int id = a * cols + b;
        if (vis[id])
            return;
        vis[id] = true;
        states.push_back({a, b, op, prev});
        q.push(states.size() - 1);
    };

    addState(0, 0, 0, -1);
    int targetIdx = -1;

    while (!q.empty()) {
        int idx = q.front();
        q.pop();
        int a = states[idx].a, b = states[idx].b;

        if (b == N) {
            targetIdx = idx;
            break;
        }

        addState(Ca, b, 1, idx);
        addState(a, Cb, 2, idx);
        addState(0, b, 3, idx);
        addState(a, 0, 4, idx);

        int pour = min(b, Ca - a);
        if (pour > 0)
            addState(a + pour, b - pour, 5, idx);

        pour = min(a, Cb - b);
        if (pour > 0)
            addState(a - pour, b + pour, 6, idx);
    }

    for (int i = targetIdx; i != 0; i = states[i].prev)
        ops.push_back(states[i].op);
    reverse(ops.begin(), ops.end());
}

void output() {
    printf("%d", (int)ops.size());
    for (int op : ops)
        printf(" %d", op);
    printf("\n");
}

int main() {
    int T;
    scanf("%d", &T);
    while (T--) {
        input();
        process();
        output();
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(T \cdot C_a \cdot C_b)$，状态空间不超过 $(C_a + 1) \times (C_b + 1)$，BFS 遍历所有状态。
- 空间复杂度：$O(C_a \cdot C_b)$ 用于存储访问标记和状态数组。
