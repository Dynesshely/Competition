# P1028 [NOIP 2001 普及组] 数的计算 — 题解

## 题目大意

给定正整数 $n$，按如下规则构造数列：
1. 只有一个数 $n$ 的数列是合法的
2. 在合法数列末尾可追加一个不超过最后一项一半的正整数

求所有合法数列的个数。$n \le 1000$。

## 解题思路

### 递推

设 $f(i)$ 表示以 $i$ 开头的合法数列个数。

$$f(i) = 1 + \sum_{j=1}^{\lfloor i/2 \rfloor} f(j)$$

其中 $1$ 代表仅有 $[i]$ 自身的数列，求和部分代表以 $i$ 开头、第二个数为 $j$（$j \le i/2$）的所有数列。

### 实现

从 $1$ 到 $n$ 递推计算 $f(i)$。对于每个 $i$，枚举 $j$ 从 $1$ 到 $i/2$ 累加 $f(j)$。

也可以优化为 $O(n)$：维护前缀和 `sum[i] = sum[i-1] + f[i]`，则 $f(i) = 1 + \text{sum}[i/2]$。

### 时间复杂度

- 直接递推：$O(n^2)$，$n \le 1000$ 完全可行
- 前缀和优化：$O(n)$

### 空间复杂度

- $O(n)$
