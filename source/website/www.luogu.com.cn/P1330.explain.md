# P1330 洛谷 封锁阳光大学 题解

## 题目链接
- [洛谷 P1330](https://www.luogu.com.cn/problem/P1330)

## 题目大意
给定一张由 $n$ 个点 $m$ 条边构成的无向图，需要在点上放置河蟹来封锁道路。每只河蟹封锁一个点，与该点相连的所有边均被封锁。但河蟹不能放置在相邻的两个点上（否则会发生冲突）。问最少需要多少只河蟹才能封锁所有道路，如果无法实现则输出 `Impossible`。

## 解题思路
问题转化为：选择尽量少的顶点，使得每条边至少有一个端点被选中，且没有一条边的两个端点同时被选中。这本质上是在求二分图的最小顶点覆盖，但此处要求的是每条边至少一端被选且两端不能同时选，等价于对每个连通分量做二分图染色，选择染色数量较少的那种颜色。

具体做法：
1. 对图进行 BFS 染色，将顶点交替染成 0 和 1。
2. 如果某条边的两端颜色相同，说明图不是二分图，输出 `Impossible`。
3. 否则，对于每个连通分量，取两种颜色中较少的那种颜色数量，累加到答案中。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, m, ans;
vector<int> g[10005];
int         col[10005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; ++i) {
        int u, v;
        scanf("%d%d", &u, &v);
        g[u].push_back(v);
        g[v].push_back(u);
    }
}

bool process() {
    memset(col, -1, sizeof(col));
    for (int s = 1; s <= n; ++s) {
        if (col[s] != -1)
            continue;
        col[s] = 0;
        queue<int> q;
        q.push(s);
        int  cnt[2] = {1, 0};
        bool ok     = true;
        while (!q.empty()) {
            int u = q.front();
            q.pop();
            for (int v : g[u]) {
                if (col[v] == -1) {
                    col[v] = col[u] ^ 1;
                    ++cnt[col[v]];
                    q.push(v);
                } else if (col[v] == col[u]) {
                    ok = false;
                }
            }
        }
        if (!ok)
            return false;
        ans += min(cnt[0], cnt[1]);
    }
    return true;
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    if (!process())
        printf("Impossible\n");
    else
        output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n + m)$，每个顶点和每条边均被遍历一次。
- 空间复杂度：$O(n + m)$，用于存储邻接表和颜色数组。
