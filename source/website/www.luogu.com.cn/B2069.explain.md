# B2069 求分数序列和 题解

## 题目链接
- [洛谷 B2069](https://www.luogu.com.cn/problem/B2069)

## 题目大意
给定递推关系 `q_{i+1}=q_i+p_i, p_{i+1}=q_i`，初始 `p1=1, q1=2`，求前 n 项分数 `q_i/p_i` 之和，精确到小数点后 4 位。

## 解题思路
直接模拟递推过程，用两个整型变量维护当前分子 q 和分母 p，每轮累加 `q/p` 到答案中，再按递推式更新下一项的值。

## 代码实现
- 使用 `double` 累加和，`printf("%.4lf")` 控制输出精度。
- 分子分母增长很快，但题目限制 n <= 30，`int` 足够存储中间值。

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
