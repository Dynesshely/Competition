# B2085 第 n 小的质数 题解
## 题目链接
- [洛谷 B2085](https://www.luogu.com.cn/problem/B2085)
## 题目大意
给定正整数 n（n <= 30000），求正整数中第 n 小的质数。
## 解题思路
使用埃拉托色尼筛法预处理质数表，筛到足够大的上限即可。第 30000 个质数为 350377，因此筛到 400000 一定能覆盖。
## 代码实现
用 vector<bool> 标记质数，线性扫描并计数，当计数达到 n 时直接输出并返回。筛法从 i*i 开始标记以减少重复。
## 复杂度分析
- 时间复杂度：O(lim log log lim)，其中 lim = 400000
- 空间复杂度：O(lim)
