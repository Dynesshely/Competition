# B2024 输出浮点数 题解

## 题目链接
- [洛谷 B2024](https://www.luogu.com.cn/problem/B2024)

## 题目大意
读入一个双精度浮点数，分别以 `%f`、`%.5f`、`%e` 和 `%g` 四种格式输出，每个占一行。

## 解题思路
直接使用 C 标准库的 `printf` 格式化输出即可。四种格式的区别：
- `%f`：默认6位小数的十进制浮点表示。
- `%.5f`：保留5位小数的十进制浮点表示。
- `%e`：科学记数法（形如 `1.234568e+01`），默认6位小数。
- `%g`：根据数值自动选择 `%f` 或 `%e` 中较短的表示，并去除尾随零。

## 代码实现
三段式结构：`input()` 用 `scanf("%lf", &x)` 读入，`output()` 中四次 `printf` 依次输出四行。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
