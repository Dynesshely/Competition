# P1019 [NOIP 2000 提高组] 单词接龙 — 题解

## 题目大意

给定 $n$（$n \le 20$）个单词和一个起始字母，将单词接成一条最长的"龙"。连接规则：

1. 相邻单词的重叠部分（前一单词的后缀 = 后一单词的前缀）合并为一个
2. 相邻两部分不能存在包含关系（重叠长度不能等于任一单词的全长）
3. 每个单词最多使用 2 次

求以给定字母开头的最长"龙"的长度。

## 解题思路

### 预处理重叠关系

对于每一对单词 $(i, j)$，计算 `overlap[i][j]`，即单词 $i$ 的末尾与单词 $j$ 的开头**最小**的重合长度：

```cpp
for len = 1 to min(len_i, len_j):
    if i.substr(len_i - len) == j.substr(0, len):
        return len   // 返回第一个（最小）匹配
return 0             // 无重叠
```

合法的连接条件：`overlap > 0` 且 `overlap < len_i` 且 `overlap < len_j`（排除包含关系）。

### DFS + 回溯

从所有首字母匹配的单词出发，进行深度优先搜索：

```
dfs(last_word, current_length):
    ans = max(ans, current_length)
    for each word i:
        if cnt[i] < 2 and can_connect(last, i):
            cnt[i]++
            dfs(i, current_length + len[i] - overlap[last][i])
            cnt[i]--
```

每次连接新单词时，龙的长度增加 `len_new - overlap`。

### 时间复杂度

- 预处理重叠关系：$O(n^2 \times L^2)$，其中 $L$ 为单词长度（通常很短）
- DFS 搜索空间：每个单词最多用 2 次，最坏情况探索大量组合，但 $n \le 20$ 时可通过

### 空间复杂度

- $O(n^2)$ 存储重叠矩阵
