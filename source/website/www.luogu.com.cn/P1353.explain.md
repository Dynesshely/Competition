# P1353 洛谷 动态规划 题解

## 题目链接
- [洛谷 P1353](https://www.luogu.com.cn/problem/P1353)

## 题目大意
贝西进行 $n$ 分钟的晨跑。在第 $i$ 分钟，她可以选择跑步（跑 $d_i$ 米，疲劳度 $+1$）或休息（疲劳度 $-1$）。疲劳度不能超过 $m$，且休息时必须休息到疲劳度为 $0$ 才能再次跑步。晨跑结束时疲劳度必须为 $0$。求最大跑步距离。

## 解题思路
动态规划。设 $dp[i][j]$ 表示第 $i$ 分钟结束时疲劳度为 $j$ 的最大跑步距离，初始化为 $-1$（不可达），$dp[0][0] = 0$。

对于每个状态 $(i, j)$：
1. **跑步**：若 $j < m$，则 $dp[i+1][j+1] = \max(dp[i+1][j+1], dp[i][j] + d_{i+1})$
2. **休息（疲劳度为 0）**：若 $j = 0$，则 $dp[i+1][0] = \max(dp[i+1][0], dp[i][0])$
3. **休息（疲劳度 > 0）**：若 $j > 0$ 且 $i+j \le n$，则直接休息到疲劳度为 0，$dp[i+j][0] = \max(dp[i+j][0], dp[i][j])$

最终答案为 $dp[n][0]$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, d[10010], dp[10010][510];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++)
        scanf("%d", &d[i]);
}

void process() {
    memset(dp, -1, sizeof(dp));
    dp[0][0] = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j <= m; j++) {
            if (dp[i][j] < 0)
                continue;
            if (j < m && dp[i][j] + d[i + 1] > dp[i + 1][j + 1])
                dp[i + 1][j + 1] = dp[i][j] + d[i + 1];
            if (j == 0 && dp[i][0] > dp[i + 1][0])
                dp[i + 1][0] = dp[i][0];
            if (j > 0 && i + j <= n && dp[i][j] > dp[i + j][0])
                dp[i + j][0] = dp[i][j];
        }
}

void output() { printf("%d\n", dp[n][0]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \times m)$，枚举分钟和疲劳度的所有状态。
- 空间复杂度：$O(n \times m)$，存储 DP 数组。
