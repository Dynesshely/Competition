# P1439 两个排列的最长公共子序列 题解

## 题目链接
- [洛谷 P1439](https://www.luogu.com.cn/problem/P1439)

## 题目大意
给出 $1, 2, \ldots, n$ 的两个排列 $P_1$ 和 $P_2$，求它们的最长公共子序列（LCS）长度。$n \le 10^5$。

## 解题思路
传统的 $O(n^2)$ LCS DP 无法处理 $n = 10^5$ 的规模。由于输入是两个排列（元素互不相同且均为 $1 \sim n$），可以将问题转化为 LIS（最长上升子序列）。

映射：记录 $P_1$ 中每个值的位置 $pos[val]$，然后将 $P_2$ 中的每个值替换为它在 $P_1$ 中的位置索引。得到新数组 $arr$ 后，$P_1$ 和 $P_2$ 的 LCS 长度就等于 $arr$ 的 LIS 长度。

这是因为 $P_1$ 和 $P_2$ 的公共子序列对应 $P_2$ 中的元素在 $P_1$ 中的位置序列是递增的。

LIS 使用 $O(n \log n)$ 的二分法求解。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
int p1[100005], p2[100005], pos[100005], arr[100005];

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) {
        scanf("%d", &p1[i]);
        pos[p1[i]] = i;
    }
    for (int i = 1; i <= n; ++i) {
        scanf("%d", &p2[i]);
        arr[i] = pos[p2[i]];
    }
}

void process() {
    vector<int> tail;
    for (int i = 1; i <= n; ++i) {
        int  x  = arr[i];
        auto it = lower_bound(tail.begin(), tail.end(), x);
        if (it == tail.end())
            tail.push_back(x);
        else
            *it = x;
    }
    printf("%d\n", (int)tail.size());
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n)$，LIS 二分 $O(\log n)$ 每次。
- 空间复杂度：$O(n)$ 用于存储数组。
