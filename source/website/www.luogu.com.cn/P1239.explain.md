# P1239 数字统计 题解

## 题目链接

- [洛谷 P1239](https://www.luogu.com.cn/problem/P1239)

## 题目大意

统计 $1 \sim N$ 中每个数字 $0 \sim 9$ 各出现多少次。

## 解题思路

逐位统计法。对每一位（个、十、百...），计算该位上每个数字出现的次数。

设当前位因子为 $factor$（$1,10,100,\ldots$）：
- `higher = n / (factor*10)`：高位
- `cur = (n/factor) % 10`：当前位
- `lower = n % factor`：低位

对数字 $d$（$1 \sim 9$）：贡献 = `higher * factor + (cur>d ? factor : cur==d ? lower+1 : 0)`。$0$ 特殊处理（排除前导零）。

## 代码实现

- 外层循环 $factor$ 从 $1$ 到 $\le n$。
- 内层对 $d=1..9$ 和 $d=0$ 分别计算。

## 复杂度分析

- 时间复杂度：$O(\log N)$。
- 空间复杂度：$O(1)$。
