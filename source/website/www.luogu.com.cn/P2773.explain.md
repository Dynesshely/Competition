# P2773 漂亮字串 题解

## 题目链接

- [洛谷 P2773](https://www.luogu.com.cn/problem/P2773)

## 题目大意

用 O 和 X 构造字符串，要求：O 的总数不超过 countO，X 的总数不超过 countX，连续 O 的长度不超过 maxO，连续 X 的长度不超过 maxX。求满足条件的最长字符串长度。

## 解题思路

本题是 **数学构造** 题，通过分类讨论直接计算答案，无需 DP 或搜索。

**关键观察**：每个 O 都可以作为分隔符将 X 的连续段分开。用 countO 个 O 最多可以隔出 countO+1 个 X 的连续段，每段长度不超过 B = min(countX, maxX)。同理，用 countX 个 X 最多可以隔出 countX+1 个 O 的连续段，每段长度不超过 A = min(countO, maxO)。

**分类讨论**：
1. 若 maxO = 0（不能放 O），则只能放 X，答案为 min(countX, maxX)。
2. 若 maxX = 0（不能放 X），则只能放 O，答案为 min(countO, maxO)。
3. 若 X 过多：`(countO + 1) * B < countX`，说明即使每段 X 都取最大长度 B，也无法用完所有 X，答案为 `(countO + 1) * B + countO`（全部 O 加上尽可能多的 X）。
4. 若 O 过多：`(countX + 1) * A < countO`，类似，答案为 `(countX + 1) * A + countX`。
5. 否则，O 和 X 都能全部用上，答案为 countO + countX。

## 代码实现

- `long long X, Y, A, B`：X = countO（O 的总数），Y = countX（X 的总数），A = maxO（连续 O 的限制），B = maxX（连续 X 的限制）。
- `min(long long x, long long y)`：手写取最小值函数（避免重名问题）。
- 读入后，先执行 `A = min(X, A)` 和 `B = min(Y, B)`，将每段最大长度限制在可用总数以内。
- 然后按上述四种情况依次判断，直接输出答案。
- 使用 `while (scanf(...) != EOF)` 循环处理多组测试数据。

## 复杂度分析

- 时间复杂度：O(1)，每组数据常数时间计算。
- 空间复杂度：O(1)，仅使用几个变量。
