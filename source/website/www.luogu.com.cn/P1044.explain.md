# P1044 [NOIP 2003 普及组] 栈 题解

## 题目链接

- [洛谷 P1044](https://www.luogu.com.cn/problem/P1044)

## 题目大意

给定一个入栈序列 $1,2,\dots,n$，求所有可能的出栈序列总数。$n \le 18$。

## 解题思路

经典的**卡特兰数**问题。$n$ 个元素的出栈序列数等于第 $n$ 个卡特兰数：

$$C_n = \frac{1}{n+1} \binom{2n}{n}$$

递推公式（更便于计算）：

$$C_0 = 1,\quad C_{n+1} = C_n \cdot \frac{4n+2}{n+2}$$

本题 $n \le 18$，结果不超过 32 位整数范围（$C_{18} = 477638700$），直接循环递推即可。

## 代码实现

- 使用 `long long` 存储卡特兰数。
- `C = C * (4*i - 2) / (i + 1)` 迭代 $n$ 次。

## 复杂度分析

- 时间复杂度：$O(n)$。
- 空间复杂度：$O(1)$。
