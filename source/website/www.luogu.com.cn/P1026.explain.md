# P1026 [NOIP 2001 提高组] 统计单词个数 — 题解

## 题目大意

给定一个长度不超过 $200$ 的字符串，将其分成 $k$ 段（$k \le 40$），每段中包含的单词个数之和最大。字典包含不超过 $6$ 个单词。单词可以部分重叠，但每个单词的首字母不能重复使用。

## 解题思路

### 预处理：最短匹配

对于字符串的每个起始位置 $i$，在字典中查找能够匹配的单词，记录**最短**的结束位置 `min_end[i]`（最短是为了避免阻塞后续单词的首字母）。

### 预处理：区间单词计数

计算 `cnt[l][r]`：子串 $s[l..r]$（1-indexed）中包含的单词个数。对于区间内每个起始位置 $i$，若 `min_end[i] \le r`（即该单词完全位于区间内），则计数 $+1$。

### 动态规划

设 $dp[i][j]$ 表示前 $i$ 个字符分成 $j$ 段的最大单词数。

$$dp[i][j] = \max_{j-1 \le m < i} \big( dp[m][j-1] + cnt[m+1][i] \big)$$

**初始化**：$dp[i][1] = cnt[1][i]$。

### 时间复杂度

- 预处理最短匹配：$O(n \times |\text{dict}| \times L)$
- 预处理 $cnt$：$O(n^3)$，但 $n \le 200$ 下可接受
- DP：$O(n^2 k)$

### 空间复杂度

- $O(n^2 + nk)$
