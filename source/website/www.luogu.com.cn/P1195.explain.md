# P1195 口袋的天空 题解

## 题目链接
- [洛谷 P1195](https://www.luogu.com.cn/problem/P1195)

## 题目大意
有 N 朵云，M 条边表示云朵间可以连边，每条边有代价 L。要将所有云朵连接成 K 个棉花糖（即 K 个连通分量），每个棉花糖至少包含一朵云。求最小总代价，若无法实现则输出 "No Answer"。

## 解题思路
Kruskal 最小生成树变体。

- 初始时每朵云自成一个连通分量，共 N 个分量。
- 按边权从小到大排序，依次尝试合并两个不同的分量。
- 每成功合并一次，连通分量数减一，累加边权。
- 当连通分量数减至 K 时停止，输出总代价。
- 若所有边处理完后连通分量数仍大于 K，则无解。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Edge {
    int x, y, w;
};

bool cmp(const Edge &a, const Edge &b) { return a.w < b.w; }

int parent[1005], rnk[1005];

int find(int u) {
    while (parent[u] != u) {
        parent[u] = parent[parent[u]];
        u         = parent[u];
    }
    return u;
}

void unite(int u, int v) {
    u = find(u);
    v = find(v);
    if (u == v)
        return;
    if (rnk[u] < rnk[v])
        parent[u] = v;
    else if (rnk[u] > rnk[v])
        parent[v] = u;
    else {
        parent[v] = u;
        rnk[u]++;
    }
}

int  n, m, k;
Edge edges[10005];

void input() {
    scanf("%d%d%d", &n, &m, &k);
    for (int i = 0; i < m; i++)
        scanf("%d%d%d", &edges[i].x, &edges[i].y, &edges[i].w);
}

void process() {
    for (int i = 1; i <= n; i++) {
        parent[i] = i;
        rnk[i]    = 0;
    }
    sort(edges, edges + m, cmp);
    int cost = 0, cnt = n;
    for (int i = 0; i < m && cnt > k; i++) {
        if (find(edges[i].x) != find(edges[i].y)) {
            unite(edges[i].x, edges[i].y);
            cost += edges[i].w;
            cnt--;
        }
    }
    if (cnt != k)
        printf("No Answer\n");
    else
        printf("%d\n", cost);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(M log M)，排序为主。
- 空间复杂度：O(N + M)。
