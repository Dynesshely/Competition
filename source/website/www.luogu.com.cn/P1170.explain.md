# P1170 兔八哥与猎人 题解

## 题目链接
- [洛谷 P1170](https://www.luogu.com.cn/problem/P1170)

## 题目大意
在一个 M x N 的果园中，兔子和猎人各在一棵果树上。如果猎人与兔子所在位置的连线上没有其它果树，则猎人能看到兔子，兔子不安全。现给定多组猎人和兔子的坐标，判断兔子是否安全。

## 解题思路
将果树视为整点坐标。两个整点之间的线段上存在其他整点当且仅当两个坐标差的最大公约数大于 1。若 gcd(|dx|, |dy|) > 1，说明连线上存在其它果树（整点），猎人无法看到兔子，兔子安全；若 gcd = 1，说明连线上没有其它果树，猎人可以直接看到兔子，兔子不安全。

因此对每组坐标，计算横纵坐标差的最大公约数：
- gcd = 1：输出 `no`（不安全）。
- gcd > 1：输出 `yes`（安全）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, ax, ay, bx, by;

void input() { scanf("%d", &n); }

void process() {
    while (n--) {
        scanf("%d%d%d%d", &ax, &ay, &bx, &by);
        int dx = abs(ax - bx);
        int dy = abs(ay - by);
        if (__gcd(dx, dy) == 1)
            printf("no\n");
        else
            printf("yes\n");
    }
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log V)，其中 n 为数据组数，log V 为 gcd 计算的时间。
- 空间复杂度：O(1)，仅使用常数个变量。
