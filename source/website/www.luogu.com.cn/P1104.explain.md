# P1104 [洛谷] 生日 题解

## 题目链接

- [洛谷 P1104](https://www.luogu.com.cn/problem/P1104)

## 题目大意

给定 n 个同学的姓名和出生年月日，按年龄从大到小排序输出姓名。年龄相同（生日相同）时，输入靠后的同学先输出。

## 解题思路

自定义排序结构体，按年、月、日升序排列（越早出生年龄越大），若生日完全相同则按下标降序排列（后输入的先输出）。

## 代码实现

- 定义结构体 `Stu` 存储姓名、年、月、日、输入序号
- 使用 `sort` 配合 lambda 表达式自定义比较规则
- 依次输出排序后的姓名

## 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
