# P1346 [CHCI 2002 Regional Competition Juniors] 电车 题解

## 题目链接
- [洛谷 P1346](https://www.luogu.com.cn/problem/P1346)

## 题目大意
给定一个 N 个路口的有向图，每个路口连接若干条轨道，默认指向第一条轨道。电车从一个路口出发，沿着当前开关指向的轨道行驶无需切换，若要走其他轨道则需要切换一次开关。求从路口 A 到路口 B 所需的最少切换次数，若无法到达则输出 -1。

## 解题思路
将每个路口作为图中的一个结点。对于路口 i，其连接的第一条轨道对应的边权为 0（默认方向，无需切换），其余轨道对应的边权为 1（需要切换）。这样问题转化为单源最短路，使用 Dijkstra 算法求解从 A 到 B 的最短路径即可。若 B 不可达则输出 -1。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int                    n, a, b;
vector<pair<int, int>> adj[105];
int                    dist[105];
bool                   via[105];

void input() {
    scanf("%d%d%d", &n, &a, &b);
    for (int i = 1; i <= n; i++) {
        int k;
        scanf("%d", &k);
        for (int j = 1; j <= k; j++) {
            int v;
            scanf("%d", &v);
            adj[i].push_back({v, j == 1 ? 0 : 1});
        }
    }
}

void process() {
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    memset(dist, 0x3f, sizeof(dist));
    dist[a] = 0;
    pq.push({0, a});
    while (!pq.empty()) {
        int d = pq.top().first, u = pq.top().second;
        pq.pop();
        if (via[u])
            continue;
        via[u] = true;
        for (auto &e : adj[u]) {
            int v = e.first, w = e.second;
            if (dist[v] > d + w) {
                dist[v] = d + w;
                pq.push({dist[v], v});
            }
        }
    }
}

void output() {
    if (dist[b] == 0x3f3f3f3f)
        printf("-1\n");
    else
        printf("%d\n", dist[b]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O((N + K) log N)，其中 K 为所有路口的轨道总数
- 空间复杂度：O(N + K)
