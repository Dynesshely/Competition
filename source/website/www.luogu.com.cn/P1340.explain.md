# P1340 [IOI 2003] 兽径管理 题解

## 题目链接
- [洛谷 P1340](https://www.luogu.com.cn/problem/P1340)

## 题目大意
有 $N$ 个草地，每周新发现一条兽径（无向边）。每周需要在当前已知的所有兽径中选择若干条，使得所有草地连通，且总长度最小。如果无法连通则输出 $-1$，否则输出最小生成树的总长度。共处理 $W$ 周。

## 解题思路
题目要求每加入一条边后输出当前图的最小生成树（MST）权值。

由于 $N \le 200$，$W \le 6000$，直接在每轮对已有边跑 Kruskal 是可行的。每条边加入时，维护一个按边权升序排列的边集（插入排序思想），然后每次对整个边集跑一遍 Kruskal 求 MST 权值。若连通性不足（$cnt < N-1$）则输出 $-1$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

struct edge {
    int u, v, w;
};

int find(vector<int> &fa, int x) { return fa[x] == x ? x : fa[x] = find(fa, fa[x]); }

int main() {
    int n, w;
    scanf("%d%d", &n, &w);
    vector<edge> edges;
    edges.reserve(w);
    for (int i = 0; i < w; ++i) {
        int u, v, wgt;
        scanf("%d%d%d", &u, &v, &wgt);
        edge ne = {u, v, wgt};
        auto it = edges.begin();
        while (it != edges.end() && it->w < wgt)
            ++it;
        edges.insert(it, ne);
        vector<int> fa(n + 1);
        for (int j = 1; j <= n; ++j)
            fa[j] = j;
        int cnt = 0, sum = 0;
        for (const auto &e : edges) {
            int fu = find(fa, e.u), fv = find(fa, e.v);
            if (fu != fv) {
                fa[fu] = fv;
                sum += e.w;
                if (++cnt == n - 1)
                    break;
            }
        }
        if (cnt == n - 1)
            printf("%d\n", sum);
        else
            printf("-1\n");
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(W \cdot (W + N \log N))$，每轮插入需要 $O(W)$ 查找位置，Kruskal 需要 $O(W \alpha(N))$ 合并。由于 $W \le 6000$，总操作在可接受范围内。
- 空间复杂度：$O(W)$，存储所有已知边。
