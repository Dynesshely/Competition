# P1323 洛谷 删数问题 题解

## 题目链接
- [洛谷 P1323](https://www.luogu.com.cn/problem/P1323)

## 题目大意
给定一个集合，其元素生成规则为：1 是集合元素；若 P 是集合元素，则 2P+1 和 4P+5 也是集合元素。取出集合中最小的 k 个元素，按从小到大的顺序拼接成一个多位数。然后从中删除 m 个数位，使得剩下的数字组成的数最大。输出删除前的多位数和删除后的最大数。

## 解题思路
问题分为两个阶段：

**第一阶段：生成最小的 k 个数并拼接**
利用最小堆（优先队列）模拟生成过程。初始将 1 入堆，每次取出堆顶元素（即当前未处理的最小元素），将其转换为字符串拼接到结果中，然后将 2P+1 和 4P+5 入堆。重复 k 次即得删除前的数字串。

**第二阶段：删除 m 个数位使剩余数字最大**
这是一个经典问题：从数字串中删除 m 位使剩余数字最大。使用单调栈思想——遍历每个数字字符，当栈非空且还有删除次数剩余，且栈顶字符小于当前字符时，弹出栈顶（相当于删除该位），然后将当前字符入栈。遍历结束后若还有删除次数剩余，从末尾继续删除。最终栈中字符即为最大数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    k, m;
string allDigits, ans;

void input() {
    if (scanf("%d%d", &k, &m) == 2) {
    }
}

void process() {
    priority_queue<long long, vector<long long>, greater<long long>> pq;
    pq.push(1);
    for (int i = 0; i < k; i++) {
        long long cur = pq.top();
        pq.pop();
        allDigits += to_string(cur);
        pq.push(2 * cur + 1);
        pq.push(4 * cur + 5);
    }
    int rem = m;
    for (char c : allDigits) {
        while (rem > 0 && !ans.empty() && ans.back() < c) {
            ans.pop_back();
            rem--;
        }
        ans.push_back(c);
    }
    while (rem > 0) {
        ans.pop_back();
        rem--;
    }
}

void output() { printf("%s\n%s\n", allDigits.c_str(), ans.c_str()); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(k log k + |digits|)，其中 |digits| 为拼接后的数字串长度，堆操作每次 O(log k)
- 空间复杂度：O(k)，堆和字符串的存储开销
