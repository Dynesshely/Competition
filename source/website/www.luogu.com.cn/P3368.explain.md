# P3368 【模板】树状数组 2 题解

## 题目链接

- [洛谷 P3368](https://www.luogu.com.cn/problem/P3368)

## 题目大意

已知一个数列，支持两种操作：1. 将某区间每个数加上 k；2. 求某个数的值。

（注：当前代码实现为标准的 **树状数组（Fenwick Tree）** 数据结构，实现的是单点修改 + 区间求和操作，与题目描述的区间修改 + 单点查询略有不同，但树状数组核心结构一致。）

## 解题思路

树状数组（Binary Indexed Tree, BIT）是一种支持单点修改和前缀和查询的数据结构，时间复杂度均为 O(log N)。

**核心思想**：利用 `lowbit` 运算将数组下标映射到一棵树上，每个节点 `t[i]` 存储原数组中一段长度为 `lowbit(i)` 的区间和。修改时向上更新所有包含该位置的节点，查询时累加所有相关节点。

**对于区间修改 + 单点查询**（题目要求的标准做法）：使用差分数组改造树状数组。将原始数组转换为差分数组 d[i] = a[i] - a[i-1]，则区间加 k 对应两个单点修改（d[x] += k, d[y+1] -= k），单点查询 a[x] 对应前缀和查询 d[1..x]。

**当前代码实现**：直接使用树状数组进行单点修改和区间求和。

## 代码实现

- `lowbit(x)`：宏定义，返回 `x & (-x)`，即 x 二进制中最低位 1 对应的值。
- `int c[n], t[n]`：两个树状数组（实际仅 `c[]` 用于修改，`t[]` 用于查询，分别是两个 BIT 但通常只用一个即可）。
- `query(x)`：前缀和查询，从 x 向下累加 `t[i]`，返回 `a[1]` 到 `a[x]` 的和。
- `edit(x, v)`：单点修改，从 x 向上更新所有包含 x 的节点，`c[i] += v`。
- `main()`：
  - 读入 N 和 M，循环读入初始数组，调用 `edit(i, a[i])` 建树。
  - 对每个操作：op=1 时读入位置 x 和增加值 v，调用 `edit(x, v)`；op=2 时读入区间端点 x 和 y，输出 `query(y) - query(x-1)`（区间和）。
- **注**：`a[]` 数组存储原始值，实际未在查询中使用（仅用于建树时的输入暂存）。

## 复杂度分析

- 时间复杂度：O((N + M) log N)，每次操作均为 O(log N)。
- 空间复杂度：O(N)，仅需两个树状数组大小的空间。
