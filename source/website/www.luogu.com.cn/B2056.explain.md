# B2056 求整数的和与均值 题解
## 题目链接
- [洛谷 B2056](https://www.luogu.com.cn/problem/B2056)
## 题目大意
读入 n 个整数，输出它们的和与均值，均值保留到小数点后 5 位。
## 解题思路
边读入边累加求和，最后除以 n 得到均值。注意求和结果可能较大，使用 int 足够（n <= 10000，|ai| <= 10000，和的范围在 [-1e8, 1e8] 内）。
## 代码实现
使用三段式结构 input/process/output。process 中循环读入累加，最后将 sum 转为 double 再除以 n 得到浮点结果。输出时用 `%.5lf` 保留五位小数。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
