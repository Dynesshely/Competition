# P1102 [洛谷] A-B 数对 题解

## 题目链接

- [洛谷 P1102](https://www.luogu.com.cn/problem/P1102)

## 题目大意

给定 N 个正整数和一个正整数 C，统计满足 A - B = C 的数对 (A, B) 的个数。相同数字出现在不同位置算不同的数对。

## 解题思路

使用哈希表（`unordered_map`）统计每个数出现的次数。遍历每个不同的数 x，若 x + C 也在哈希表中，则答案累加 `cnt[x] * cnt[x + C]`。注意结果可能很大，需使用 `long long`。

## 代码实现

- 读入 N, C 和所有数，用 `unordered_map<int, int>` 统计频次
- 遍历 map，对每个键值对 `(x, cnt)`，查找 `x + c` 是否存在
- 若存在，答案加上 `cnt[x] * cnt[x + c]`
- 输出答案

## 复杂度分析

- 时间复杂度：O(N)
- 空间复杂度：O(N)
