# P16683 盆栽 题解

## 题目链接

- [洛谷 P16683](https://www.luogu.com.cn/problem/P16683)

## 题目大意

$n$ 盆花，每盆美丽度为 $a_i$。$m$ 次询问，每次给定魔法值 $k$。可选择一个区间 $[l, r]$ 将所有 $a_i$ 异或 $k$（或不操作）。求操作后美丽度总和的最小值和最大值。

## 解题思路

对单点 $i$，异或后的变化量 $\Delta_i = (a_i \oplus k) - a_i$。

选择区间 $[l, r]$ 操作，总和变化 $= \sum_{i=l}^r \Delta_i$。因此：

- **最大总和** $=$ 原总和 $+ \max(0, \Delta$ 的最大子段和$)$
- **最小总和** $=$ 原总和 $+ \min(0, \Delta$ 的最小子段和$)$

每询问用 Kadane 算法 $O(n)$ 扫描 $\Delta$ 数组：维护 `cur_max` 和 `cur_min`，分别记录以当前位置结尾的最大/小子段和，同步更新全局 `best_max` 和 `best_min`。

$n, m \le 5000$，$O(nm) = 2.5 \times 10^7$，在 C++ 中可行。

## 代码实现

- 预计算原总和 `total`。
- 每询问遍历 $a$，计算 $\Delta_i$，Kadane 更新。
- 答案需用 `long long` 防止溢出（$a_i < 2^{30}$，$n \le 5000$）。

## 复杂度分析

- 时间复杂度：$O(nm)$，约 $2.5 \times 10^7$。
- 空间复杂度：$O(n + m)$。
