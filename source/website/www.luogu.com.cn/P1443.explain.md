# P1443 马的遍历 题解

## 题目链接
- [洛谷 P1443](https://www.luogu.com.cn/problem/P1443)

## 题目大意
在一个 $n \times m$ 的棋盘上，马从点 $(x, y)$ 出发。计算马按日字走法到达棋盘上每个点的最少步数，不能到达则输出 $-1$。$n, m \le 400$。

## 解题思路
BFS 求单源最短路。马有 8 个日字方向走法，从起点开始 BFS，逐层扩展，首次访问到某点时的步数即为最少步数。

使用二维数组 $dis$ 存储距离，初始化为 $-1$。起点距离为 $0$，BFS 过程中每步将 8 个可行方向中未访问的点入队，距离加 $1$。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, x, y;
int dis[405][405];
int dx[] = {-2, -2, -1, -1, 1, 1, 2, 2};
int dy[] = {-1, 1, -2, 2, -2, 2, -1, 1};

void input() { scanf("%d%d%d%d", &n, &m, &x, &y); }

void process() {
    memset(dis, -1, sizeof(dis));
    dis[x][y] = 0;
    queue<pair<int, int>> q;
    q.push({x, y});
    while (!q.empty()) {
        int cx = q.front().first, cy = q.front().second;
        q.pop();
        for (int i = 0; i < 8; i++) {
            int nx = cx + dx[i], ny = cy + dy[i];
            if (nx >= 1 && nx <= n && ny >= 1 && ny <= m && dis[nx][ny] == -1) {
                dis[nx][ny] = dis[cx][cy] + 1;
                q.push({nx, ny});
            }
        }
    }
}

void output() {
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++)
            printf("%d ", dis[i][j]);
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \cdot m)$，每个点最多访问一次，每次扩展 8 个方向。
- 空间复杂度：$O(n \cdot m)$ 用于棋盘距离数组。
