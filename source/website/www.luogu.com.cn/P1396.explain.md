# P1396 营救 题解

## 题目链接
- [洛谷 P1396](https://www.luogu.com.cn/problem/P1396)

## 题目大意
给定一个 n 个节点、m 条边的无向图，每条边有一个拥挤度。求从 s 到 t 的一条路径，使得路径上边的拥挤度最大值最小。输出该最小值。

## 解题思路
最小瓶颈路问题。将所有边按拥挤度从小到大排序，使用并查集（DSU）依次加入每条边，每次加入后检查 s 与 t 是否连通。当 s 和 t 首次连通时，当前加入的边的拥挤度即为答案。

该算法本质是 Kruskal 求 MST 的过程，利用了最小生成树上的 s→t 路径即为最小瓶颈路的性质。

## 代码实现
边排序后依次加入 DSU，检查 s 和 t 的连通性。

## 复杂度分析
- 时间复杂度：O(m log m)（排序）
- 空间复杂度：O(n + m)
