# P1324 洛谷 矩形分割 题解

## 题目链接
- [洛谷 P1324](https://www.luogu.com.cn/problem/P1324)

## 题目大意
将一块 N x M 的木板沿格线切割成 1x1 的小方块。每次只能选择一条横线或竖线切割，切割代价各不相同。切割一次后木板分成两块，后续需要分别切割。求最小总代价。

## 解题思路
这是一个经典的贪心问题。切割过程类似于哈夫曼编码的思想——每次选择代价最大的切割线先切，可以最小化总代价。

具体地：
- 将横线切割代价和竖线切割代价分别从大到小排序
- 维护当前横向块数 rCnt 和纵向块数 cCnt，初始均为 1
- 每次选择当前剩余的最大代价线：若选横线，该刀会经过所有 cCnt 个纵向块，代价乘以 cCnt；选竖线同理，代价乘以 rCnt
- 选完后对应块数加 1

由于代价大的线越早切，被乘的次数越少，因此按代价从大到小切是最优的。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, m;
vector<int> rowCost, colCost;

void input() {
    scanf("%d%d", &n, &m);
    rowCost.resize(n - 1);
    colCost.resize(m - 1);
    for (int i = 0; i < n - 1; ++i)
        scanf("%d", &rowCost[i]);
    for (int i = 0; i < m - 1; ++i)
        scanf("%d", &colCost[i]);
}

long long process() {
    sort(rowCost.begin(), rowCost.end(), greater<int>());
    sort(colCost.begin(), colCost.end(), greater<int>());
    long long ans = 0;
    int       i = 0, j = 0, rCnt = 1, cCnt = 1;
    while (i < n - 1 || j < m - 1) {
        if (j == m - 1 || (i < n - 1 && rowCost[i] > colCost[j])) {
            ans += (long long)rowCost[i] * cCnt;
            ++i;
            ++rCnt;
        } else {
            ans += (long long)colCost[j] * rCnt;
            ++j;
            ++cCnt;
        }
    }
    return ans;
}

void output(long long res) { printf("%lld\n", res); }

int main() {
    input();
    output(process());
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O((N+M) log(N+M))，排序为主
- 空间复杂度：O(N+M)
