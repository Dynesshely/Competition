## 题目描述

输入一个正整数 $n$，求正整数范围中第 $n$ 小的质数。

## 输入格式

一个不超过 $30000$ 的正整数 $n$。

## 输出格式

第 $n$ 小的质数。

## 样例 #1

### 样例输入

```
10
```

### 样例输出

```
29
```
