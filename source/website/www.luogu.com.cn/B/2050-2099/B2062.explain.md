# B2062 乘方计算 题解
## 题目链接
- [洛谷 B2062](https://www.luogu.com.cn/problem/B2062)
## 题目大意
给定整数 a 和正整数 n，计算 a^n。结果绝对值不超过 10^6。
## 解题思路
直接模拟乘法过程：初始化结果为 1，循环 n 次每次乘 a。注意 a 可能为负整数，奇数幂保留符号，偶数幂为正。
## 代码实现
使用 input/process/output 三段式结构。res 初始化为 1，for 循环累乘 n 次，最后输出结果。全程使用 int 运算。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
