# B2099 [洛谷] 矩阵交换行 题解

## 题目链接
- [洛谷 B2099](https://www.luogu.com.cn/problem/B2099)

## 题目大意
给定 5×5 矩阵，将第 n 行和第 m 行交换后输出。

## 解题思路
读入矩阵和两个行号，逐列 swap 两行元素，最后按格式输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a[6][6], r1, r2;

void input() {
    for (int i = 1; i <= 5; i++)
        for (int j = 1; j <= 5; j++) scanf("%d", &a[i][j]);
    scanf("%d%d", &r1, &r2);
}

void process() {
    for (int j = 1; j <= 5; j++) swap(a[r1][j], a[r2][j]);
    for (int i = 1; i <= 5; i++)
        for (int j = 1; j <= 5; j++)
            printf("%d%c", a[i][j], j == 5 ? '\n' : ' ');
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)，矩阵固定 5×5
- 空间复杂度：O(1)
