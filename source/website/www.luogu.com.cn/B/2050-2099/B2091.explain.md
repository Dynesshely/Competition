# B2091 向量点积计算 题解
## 题目链接
- [洛谷 B2091](https://www.luogu.com.cn/problem/B2091)
## 题目大意
给定两个 n 维整数向量 a 和 b，计算它们的点积（对应分量乘积之和）。
## 解题思路
直接按定义模拟：读入 n 和两个数组，循环累加 a[i] * b[i] 并输出结果。
## 代码实现
采用 input() / process() / output() 三段式结构。数组大小开 1005 保证 n <= 1000 不越界。循环内直接累乘累加即可。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
