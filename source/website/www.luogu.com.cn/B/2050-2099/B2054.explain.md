# B2054 求平均年龄 题解
## 题目链接
- [洛谷 B2054](https://www.luogu.com.cn/problem/B2054)
## 题目大意
给定 n 名学生的年龄（15-25 的整数），计算并输出他们的平均年龄，保留两位小数。
## 解题思路
读入 n 后累加所有年龄得到 sum，平均 age = sum / n，用 `printf("%.2f")` 输出即可满足误差要求。
## 代码实现
- 用 `double` 存储平均值，累加时用 `int` 避免精度损失，计算时转 `double`。
- 输出格式 `%.2f` 自动保留两位小数。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
