## 题目描述

输出一个整数数列中不与最大数相同的数字之和。

## 输入格式

输入分为两行：

第一行为 $N$($N$ 为接下来数的个数，$N \le 100$)；

第二行为 $N$ 个整数，数与数之间以一个空格分开，每个整数的范围是 $-1000,000$ 到 $1000,000$。

## 输出格式

输出为 $N$ 个数中除去最大数其余数字之和。

## 样例 #1

### 样例输入

```
3
1 2 3
```

### 样例输出

```
3
```
