# B2093 查找特定的值 题解
## 题目链接
- [洛谷 B2093](https://www.luogu.com.cn/problem/B2093)
## 题目大意
给定一个长度为 n 的序列（下标从 0 开始）和一个整数 x，输出 x 在序列中第一次出现的位置，若不存在则输出 -1。
## 解题思路
从头到尾线性扫描数组，一旦找到等于 x 的元素就记录下标并立即跳出循环；若扫描结束仍未找到则下标保持 -1。
## 代码实现
将读入、处理分成两个函数。处理时用一个变量存答案，匹配到第一个 x 后立即 break 即可。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)（存储序列），O(1) 辅助空间
