# B2055 均值 题解

## 题目链接
- [洛谷 B2055](https://www.luogu.com.cn/problem/B2055)

## 题目大意
给定样本容量 n 和 n 个浮点数，计算它们的算术平均数。

## 解题思路
累加所有样本值后除以 n 即为均值。注意浮点数精度要求，相对误差或绝对误差不超过 1e-3 即算正确。

## 代码实现
- 使用 `double` 类型存储和与均值以避免精度损失。
- 按题意输出保留四位小数，使用 `printf("%.4f")`。
- 数据范围小（n <= 100），直接线性累加即可。

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
