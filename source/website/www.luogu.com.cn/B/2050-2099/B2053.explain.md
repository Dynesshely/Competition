# B2053 求一元二次方程 题解
## 题目链接
- [洛谷 B2053](https://www.luogu.com.cn/problem/B2053)
## 题目大意
给定三个浮点数 a, b, c，求一元二次方程 ax^2+bx+c=0 的根，精确到小数点后 5 位。若无实根输出 `No answer!`。
## 解题思路
计算判别式 d = b^2 - 4ac，分三类讨论：d < 0 无实根；|d| < 1e-9 有唯一实根 x = -b/(2a)；d >= 1e-9 有两个实根，代入求根公式并保证 x1 <= x2 后输出。
## 代码实现
用 `double` 处理浮点数，通过 `fabs(d) < 1e-9` 判断判别式是否为零，避免浮点误差。两个不等根通过 `swap` 排序后输出。
## 复杂度分析
- 时间复杂度 O(1)
- 空间复杂度 O(1)
