# B2097 [洛谷] 最长平台 题解

## 题目链接
- [洛谷 B2097](https://www.luogu.com.cn/problem/B2097)

## 题目大意
找出数组中最长连续相等段的长度。

## 解题思路
遍历数组，用 cur 记录当前平台长度。若当前元素等于前一个，cur 加 1；否则更新答案并将 cur 重置为 1。最后再取一次最大值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, a[105];

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) scanf("%d", &a[i]);
}

void process() {
    int ans = 0, cur = 1;
    for (int i = 2; i <= n; i++)
        if (a[i] == a[i - 1]) ++cur;
        else ans = max(ans, cur), cur = 1;
    ans = max(ans, cur);
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
