# B2096 [洛谷] 直方图 题解

## 题目链接
- [洛谷 B2096](https://www.luogu.com.cn/problem/B2096)

## 题目大意
给定 n 个非负整数，统计 0 到最大值之间每个数的出现次数，按顺序每行一个输出。

## 解题思路
边读入边计数，同时记录最大值。最后从 0 循环到最大值，输出每个数的计数值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXV = 100005;
int n, cnt[MAXV], maxV;

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        ++cnt[x];
        if (x > maxV) maxV = x;
    }
}

void process() {
    for (int i = 0; i <= maxV; i++) printf("%d\n", cnt[i]);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + maxV)
- 空间复杂度：O(maxV)
