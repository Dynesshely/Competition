## 题目描述

两个整数 $a$ 和 $b$ 分别作为分子和分母，即分数 $\dfrac{a}{b}$，求它的浮点数值（双精度浮点数，保留小数点后 $9$ 位）。

## 输入格式

输入仅一行，包括两个整数 $a$ 和 $b$。

## 输出格式

输出也仅一行，分数 $\dfrac{a}{b}$ 的浮点数值（双精度浮点数，保留小数点后 $9$ 位）。

## 样例 #1

### 样例输入

```
5 7
```

### 样例输出

```
0.714285714
```

## 提示

对于 $100 \%$ 的数据，$1 \le a, b \le {10}^9$。
