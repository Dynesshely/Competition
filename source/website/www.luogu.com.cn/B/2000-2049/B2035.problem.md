## 题目描述

给定一个整数 $N$，判断其正负。如果 $N>0$, 输出 `positive` ; 如果 $N=0$, 输出 `zero` ; 如果 $N<0,$ 输出 `negative`。

## 输入格式

一个整数 $N(-10^9 \le N \le 10^9)$。

## 输出格式

如果 $N>0$， 输出 `positive`; 

如果 $N=0$, 输出 `zero`； 

如果 $N<0$， 输出 `negative`。

## 样例 #1

### 样例输入

```
95
```

### 样例输出

```
positive
```
