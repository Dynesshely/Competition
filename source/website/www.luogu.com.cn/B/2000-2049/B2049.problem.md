## 题目描述

输入三个整数，输出最大的数。

## 输入格式

输入为一行，包含三个整数，数与数之间以一个空格分开。

## 输出格式

输出一行，包含一个整数，即最大的整数。

## 样例 #1

### 样例输入

```
10 20 56
```

### 样例输出

```
56
```

## 提示

### 数据规模与约定

对于全部的测试点，保证输入的整数均在 32 位有符号整型 (`int`/`long int`) 范围内。
