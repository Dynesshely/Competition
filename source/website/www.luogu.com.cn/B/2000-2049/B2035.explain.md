# B2035 判断数正负 题解

## 题目链接
- [洛谷 B2035](https://www.luogu.com.cn/problem/B2035)

## 题目大意
给定一个整数 N，判断其是正数、零还是负数，并输出对应的英文单词。

## 解题思路
直接使用 if-else 分支结构判断 N 与 0 的大小关系即可。N 的范围在 int 内，使用 `int` 类型存储。

## 代码实现
采用三段式结构：`input()` 读入整数，`process()` 为空（无需预处理），`output()` 中用 `if / else if / else` 判断符号并输出 `positive`、`zero` 或 `negative`。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
