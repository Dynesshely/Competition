# B2009 计算 (a+b)/c 的值 题解

## 题目链接
- [洛谷 B2009](https://www.luogu.com.cn/problem/B2009)

## 题目大意
给定三个整数 a, b, c，计算 (a+b)/c 的整除结果。

## 解题思路
直接按照题目公式计算即可。使用三个 int 变量存储输入，计算 (a+b)/c 并输出。需要注意 c 不为 0。

## 代码实现
采用 input(), process(), output() 三段式结构。核心运算为整数除法 (a+b)/c，C++ 中整数除法自动截断小数部分，符合整除题意。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
