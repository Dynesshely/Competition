# B2010 带余除法 题解

## 题目链接
- [洛谷 B2010](https://www.luogu.com.cn/problem/B2010)

## 题目大意
给定被除数 a 和除数 b（b 非零），求整数商（a / b 的整数部分）和余数（a % b）。

## 解题思路
直接使用 C++ 的整除运算符 `/` 和取余运算符 `%` 即可得到结果。无需处理特殊情况，数据范围保证 a 在 [0, 100]、b 在 [1, 100] 内。

## 代码实现
读入两个整数 a, b，输出 `a / b` 和 `a % b`，中间用空格隔开。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
