# B2037 奇偶数判断 题解

## 题目链接
- [洛谷 B2037](https://www.luogu.com.cn/problem/B2037)

## 题目大意
给定一个整数 n，判断其奇偶性：奇数输出 `odd`，偶数输出 `even`。

## 解题思路
利用位运算判断奇偶：整数的最低位为 1 则是奇数，为 0 则是偶数。通过 `n & 1` 即可取得最低位，比取模运算效率更高。

## 代码实现
采用 input / process / output 三段式结构。核心判断在 output 中完成：`if (n & 1)` 为真时输出 `odd`，否则输出 `even`。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
