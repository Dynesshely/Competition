# B2030 计算线段长度 题解

## 题目大意
给定平面上两个点 A(Xa, Ya) 和 B(Xb, Yb)，求线段 AB 的长度，结果保留三位小数。

## 解题思路
直接套用两点间欧几里得距离公式：距离 = sqrt((Xb - Xa)^2 + (Yb - Ya)^2)。题目为入门题，直接计算输出即可。

采用三段式结构：`input()` 读入四个实数，`process()` 用 `sqrt` 计算距离，`output()` 用 `printf("%.3lf")` 保留三位小数输出。
