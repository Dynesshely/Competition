# B2042 判断一个数能否同时被 3 和 5 整除 题解

## 题目链接
- [洛谷 B2042](https://www.luogu.com.cn/problem/B2042)

## 题目大意
给定一个整数 n，判断其是否能同时被 3 和 5 整除，输出 YES 或 NO。

## 解题思路
由于 3 和 5 互质，一个数能同时被两者整除等价于能被它们的乘积 15 整除。因此只需判断 n % 15 == 0 即可。

## 代码实现
使用三段式结构。input() 读入整数 n，process() 中用取模运算 n % 15 判断并输出结果。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
