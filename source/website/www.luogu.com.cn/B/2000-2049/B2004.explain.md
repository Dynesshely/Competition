# B2004 对齐输出 题解

## 题目链接
- [洛谷 B2004](https://www.luogu.com.cn/problem/B2004)

## 题目大意
读入三个整数，按每个整数占 8 个字符宽度右对齐输出，整数之间以一个空格分隔。

## 解题思路
使用 C 语言的 `printf` 格式化输出即可。`%Nd` 表示将整数按十进制右对齐输出，宽度为 N 个字符。`%8d` 即满足每个整数占 8 个字符宽度的要求。

## 代码实现
采用 `input()` / `process()` / `output()` 三段式结构。核心是 `output()` 中的 `printf("%8d %8d %8d\n", a, b, c);`。

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
