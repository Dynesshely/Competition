## 题目描述

将一个三位数反向输出，例如输入 $358$，反向输出 $853$。

## 输入格式

一个三位数 $n$。

## 输出格式

反向输出 $n$。

## 样例 #1

### 样例输入

```
100
```

### 样例输出

```
001
```

## 样例 #2

### 样例输入

```
001
```

### 样例输出

```
100
```

## 样例 #3

### 样例输入

```
678
```

### 样例输出

```
876
```
