## 题目描述

输入一个双精度浮点数，将其向零舍入到整数。说明：向零舍入的含义是，正数向下舍入，负数向上舍入。

## 输入格式

一个双精度浮点数 $x$。

## 输出格式

一个整数，即向零舍入到整数的结果。

## 样例 #1

### 样例输入

```
2.3
```

### 样例输出

```
2
```

## 提示

$-10^{15} \le x \le 10^{15}$。
