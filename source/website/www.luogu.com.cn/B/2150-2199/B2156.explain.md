# B2156 [洛谷] 最长单词 2 题解

## 题目链接
- [洛谷 B2156](https://www.luogu.com.cn/problem/B2156)

## 题目大意
给定一个以 `.` 结尾的英文句子，单词之间用空格分隔，求句中最长的单词。若有多个最长单词，输出第一个。

## 解题思路
使用 strtok 以空格和句点为分隔符分割单词，遍历每个单词记录最长长度及对应的单词。由于句子已保证以 `.` 结尾，分割时可同时用空格和句点作为分隔符。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[510], word[510], ans[510];

void input() { cin.getline(s, 505); }

void process() {
    int   maxLen = 0;
    char *token  = strtok(s, " .");
    while (token != NULL) {
        int curLen = strlen(token);
        if (curLen > maxLen) {
            maxLen = curLen;
            strcpy(ans, token);
        }
        token = strtok(NULL, " .");
    }
}

void output() { printf("%s\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(L)，L 为句子长度
- 空间复杂度：O(L)
