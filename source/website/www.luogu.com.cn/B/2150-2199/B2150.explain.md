# B2150 [洛谷] 判断闰年 题解

## 题目链接
- [洛谷 B2150](https://www.luogu.com.cn/problem/B2150)

## 题目大意
给定一个公元年份 a（0 < a < 3000），判断是否为闰年。闰年条件：能被 400 整除，或者能被 4 整除但不能被 100 整除。是闰年输出 Y，否则输出 N。

## 解题思路
直接按闰年定义进行条件判断即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a;

void input() { scanf("%d", &a); }

void process() {
    if (a % 400 == 0 || (a % 4 == 0 && a % 100 != 0))
        printf("Y\n");
    else
        printf("N\n");
}

void output() {}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
