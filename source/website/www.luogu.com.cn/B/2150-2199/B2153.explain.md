# B2153 [洛谷] 求阶乘的和 题解

## 题目链接
- [洛谷 B2153](https://www.luogu.com.cn/problem/B2153)

## 题目大意
给定正整数 n（1 < n < 12），求 1! + 2! + 3! + ... + n! 的和。

## 解题思路
用变量 fact 记录当前阶乘值，每次乘以 i 得到 i!，累加到 sum 中。由于 n 较小，直接循环计算即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, sum;

void input() { scanf("%d", &n); }

void process() {
    int fact = 1;
    for (int i = 1; i <= n; i++) {
        fact *= i;
        sum += fact;
    }
}

void output() { printf("%d\n", sum); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
