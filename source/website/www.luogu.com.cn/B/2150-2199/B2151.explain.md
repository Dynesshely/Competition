# B2151 [洛谷] 正常血压 题解

## 题目链接
- [洛谷 B2151](https://www.luogu.com.cn/problem/B2151)

## 题目大意
给定 n 组收缩压和舒张压数据，正常血压条件：收缩压在 90~140 之间且舒张压在 60~90 之间（含端点）。求连续正常血压的最长小时数。

## 解题思路
遍历每组数据，若当前血压正常则连续计数 cur 加 1，否则更新最大连续值 ans 并将 cur 置 0。遍历结束后再更新一次 ans 取最大值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, ans, cur;

void input() { scanf("%d", &n); }

void process() {
    for (int i = 0; i < n; i++) {
        int s, d;
        scanf("%d%d", &s, &d);
        if (s >= 90 && s <= 140 && d >= 60 && d <= 90)
            cur++;
        else {
            if (cur > ans)
                ans = cur;
            cur = 0;
        }
    }
    if (cur > ans)
        ans = cur;
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
