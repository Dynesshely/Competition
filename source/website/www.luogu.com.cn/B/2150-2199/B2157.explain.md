# B2157 [洛谷] 短信计费 题解

## 题目链接
- [洛谷 B2157](https://www.luogu.com.cn/problem/B2157)

## 题目大意
短信每条 0.1 元，每条短信最多 70 个字。超过 70 个字则按每 70 字一条分割成多条发送。给定 n 条短信的字数，计算当月短信总资费，精确到小数点后 1 位。

## 解题思路
每条短信的计费条数为 (字数 + 69) / 70（向上取整），乘以 0.1 元累加即可。注意使用浮点数或整数运算后转为浮点数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n;
double total;

void input() { scanf("%d", &n); }

void process() {
    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        total += (x + 69) / 70 * 0.1;
    }
}

void output() { printf("%.1lf\n", total); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
