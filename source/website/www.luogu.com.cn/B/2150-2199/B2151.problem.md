## 题目描述

监护室每小时测量一次病人的血压，若收缩压在 $90\sim 140$ 之间并且舒张压在 $60\sim 90$ 之间(包含端点值)则称之为正常，现给出某病人若干次测量的血压值，计算病人保持正常血压的最长小时数。

## 输入格式

第一行为一个正整数 $n(n<100)$，其后有 $n$ 行，每行 $2$ 个正整数，分别为一次测量的收缩压和舒张压。

## 输出格式

输出仅一行，血压连续正常的最长小时数。

## 样例 #1

### 样例输入

```
4
100 80
90 50
120 60
140 90
```

### 样例输出

```
2
```
