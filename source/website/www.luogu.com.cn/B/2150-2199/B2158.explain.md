# B2158 [洛谷] 谁考了第 k 名 题解

## 题目链接
- [洛谷 B2158](https://www.luogu.com.cn/problem/B2158)

## 题目大意
给定 n 名学生的学号和成绩（各不相同），求第 k 名学生的学号和成绩。按成绩从高到低排名。

## 解题思路
将学号和成绩存入两个数组，按成绩从高到低进行冒泡排序，交换成绩时同步交换学号。排序后输出第 k 个元素即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n, k;
int    id[105];
double sc[105];

void input() {
    scanf("%d%d", &n, &k);
    for (int i = 0; i < n; i++)
        scanf("%d%lf", &id[i], &sc[i]);
}

void process() {
    for (int i = 0; i < n - 1; i++)
        for (int j = 0; j < n - 1 - i; j++)
            if (sc[j] < sc[j + 1]) {
                swap(sc[j], sc[j + 1]);
                swap(id[j], id[j + 1]);
            }
}

void output() { printf("%d %g\n", id[k - 1], sc[k - 1]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n^2)
- 空间复杂度：O(n)
