# B2154 [洛谷] 数 1 的个数 题解

## 题目链接
- [洛谷 B2154](https://www.luogu.com.cn/problem/B2154)

## 题目大意
给定十进制正整数 n（1 <= n <= 10000），写下从 1 到 n 的所有整数，数一下其中数字 1 出现的总次数。

## 解题思路
编写函数 countOne 统计单个整数中数字 1 的个数：不断取末位判断是否为 1，然后除以 10。对 1 到 n 每个数累加结果即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, ans;

void input() { scanf("%d", &n); }

int countOne(int x) {
    int cnt = 0;
    while (x) {
        if (x % 10 == 1)
            cnt++;
        x /= 10;
    }
    return cnt;
}

void process() {
    for (int i = 1; i <= n; i++)
        ans += countOne(i);
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(1)
