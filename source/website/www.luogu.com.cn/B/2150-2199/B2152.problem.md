## 题目描述

给定一个整数 $n$，要求从个位开始分离出它的每一位数字。从个位开始按照从低位到高位的顺序依次输出每一位数字。

## 输入格式

输入一个整数 $n$ （$1 \le n \le 100\,000\,000$）。

## 输出格式

从个位开始按照从低位到高位的顺序依次输出每一位数字。数字之间以一个空格分开。

## 样例 #1

### 样例输入

```
123
```

### 样例输出

```
3 2 1
```
