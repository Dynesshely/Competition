# B2155 [洛谷] 合法 C 标识符 题解

## 题目链接
- [洛谷 B2155](https://www.luogu.com.cn/problem/B2155)

## 题目大意
给定一个不含空白符的字符串（长度 <= 20），判断是否为 C 语言合法标识符。要求：只包含字母、数字和下划线，且不能以数字开头。合法输出 yes，否则输出 no。

## 解题思路
先判断首位是否为字母或下划线，若不是则直接返回 no。再遍历其余字符，判断每个字符是否为字母、数字或下划线，若存在非法字符则返回 no。全部通过输出 yes。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[25];
bool ok = true;

void input() { scanf("%s", s); }

void process() {
    int len = strlen(s);
    if (!((s[0] >= 'a' && s[0] <= 'z') || (s[0] >= 'A' && s[0] <= 'Z') || s[0] == '_')) {
        ok = false;
        return;
    }
    for (int i = 1; i < len; i++)
        if (!((s[i] >= 'a' && s[i] <= 'z') || (s[i] >= 'A' && s[i] <= 'Z') || (s[i] >= '0' && s[i] <= '9') || s[i] == '_')) {
            ok = false;
            return;
        }
}

void output() { printf("%s\n", ok ? "yes" : "no"); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(L)，L 为字符串长度
- 空间复杂度：O(1)
