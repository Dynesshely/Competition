# B2612 [洛谷] 简单的分苹果 3 题解

## 题目链接
- [洛谷 B2612](https://www.luogu.com.cn/problem/B2612)

## 题目大意
分苹果，每人3个剩11个，每人4个少1个。求人数和苹果数。

## 解题思路
设人数为x，苹果数为y。y=3x+11=4x-1，解得x=12，y=47。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d %d\n", 12, 47);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
