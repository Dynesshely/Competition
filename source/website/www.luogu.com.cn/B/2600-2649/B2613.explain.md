# B2613 [洛谷] 打字速度 题解

## 题目链接
- [洛谷 B2613](https://www.luogu.com.cn/problem/B2613)

## 题目大意
小A每分钟120字，UIM每分钟80字，UIM先打12分钟。求小A开始打字后多久能赶上UIM。

## 解题思路
设t分钟后赶上，120t=80(12+t)，解得t=24。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d\n", 24);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
