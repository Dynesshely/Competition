# B2606 [洛谷] 圆的计算 题解

## 题目链接
- [洛谷 B2606](https://www.luogu.com.cn/problem/B2606)

## 题目大意
给定半径r=5，π=3.141593，求圆的周长、面积和球体积。

## 解题思路
代入公式计算：周长c=2πr，面积a=πr²，体积v=4/3πr³。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double pi = 3.141593, r = 5;
    double c = 2 * pi * r;
    double a = pi * r * r;
    double v = 4.0 / 3.0 * pi * r * r * r;
    printf("%g\n%g\n%g\n", c, a, v);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
