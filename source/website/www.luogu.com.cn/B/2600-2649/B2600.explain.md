# B2600 [洛谷] 简单的分苹果 题解

## 题目链接
- [洛谷 B2600](https://www.luogu.com.cn/problem/B2600)

## 题目大意
10个苹果，小A拿2个，UIM拿4个，八尾勇拿剩下的。求小A和UIM共拿多少个，八尾勇拿多少个。

## 解题思路
直接计算：2+4=6，10-2-4=4。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d %d\n", 2 + 4, 10 - 2 - 4);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
