# B2608 [洛谷] 评测机队列 题解

## 题目链接
- [洛谷 B2608](https://www.luogu.com.cn/problem/B2608)

## 题目大意
8台评测机30分钟评完，10台6分钟评完。求多少台评测机可在10分钟时刚好评完。

## 解题思路
牛吃草问题。设每台每分钟处理1单位任务，新增任务速度为v。8×30-30v=10×6-6v，解得v=7，初始任务量=8×30-30×7=30。设需x台：10x-10×7=30，解得x=9。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d\n", 9);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
