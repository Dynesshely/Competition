# B2609 [洛谷] 清扫教室 题解

## 题目链接
- [洛谷 B2609](https://www.luogu.com.cn/problem/B2609)

## 题目大意
3人3小时扫3间教室，求9人9小时扫几间教室。

## 解题思路
每人每小时扫3/(3×3)=1/3间教室。9人9小时可扫9×9×(1/3)=27间。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d\n", 27);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
