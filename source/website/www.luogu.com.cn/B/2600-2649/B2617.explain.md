# B2617 [洛谷] 英文字母 题解

## 题目链接
- [洛谷 B2617](https://www.luogu.com.cn/problem/B2617)

## 题目大意
求M是第几个字母，以及第18个大写字母是什么。

## 解题思路
M是第13个字母（A=1），第18个字母是R。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("13\nR\n");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
