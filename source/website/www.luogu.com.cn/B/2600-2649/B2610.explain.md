# B2610 [洛谷] 面积计算 题解

## 题目链接
- [洛谷 B2610](https://www.luogu.com.cn/problem/B2610)

## 题目大意
长方形的长宽和为24，长比宽多4，求面积。

## 解题思路
设宽为x，则长为x+4，2x+4=24，解得x=10，长为14，面积=10×14=140。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d\n", 140);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
