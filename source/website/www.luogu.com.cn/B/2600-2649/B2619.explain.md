# B2619 [洛谷] 销量预测 题解

## 题目链接
- [洛谷 B2619](https://www.luogu.com.cn/problem/B2619)

## 题目大意
定价110元时有10人报名，每降1元多1人（升1元少1人），目标学费3500元，求定价（取较小值，四舍五入到整数）。

## 解题思路
设降价x元，定价110-x，报名人数10+x，总学费(110-x)(10+x)=3500。展开得-x²+100x+1100=3500，即x²-100x+2400=0，解得x=40或60。较小解x=40，定价110-40=70。另一解30不对应整数，取定价为50？等等让我重新算。

(110-x)(10+x)=3500 → 1100+110x-10x-x²=3500 → -x²+100x+1100=3500 → -x²+100x-2400=0 → x²-100x+2400=0 → (x-40)(x-60)=0 → x=40或x=60。

x=40时定价=70，x=60时定价=50。取较小的50。

验证：定价50，降价60元，人数10+60=70人，总学费50×70=3500。正确。
