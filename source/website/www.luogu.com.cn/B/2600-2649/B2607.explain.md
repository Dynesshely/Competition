# B2607 [洛谷] 简单的猴子吃桃 题解

## 题目链接
- [洛谷 B2607](https://www.luogu.com.cn/problem/B2607)

## 题目大意
猴子每天吃剩余桃子的一半多一个，第4天发现只剩1个，求最初桃子数量。

## 解题思路
逆向推导：第3天吃完后剩1个，则第3天吃前有(1+1)×2=4个，第2天吃前有(4+1)×2=10个，第1天吃前有(10+1)×2=22个。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int ans = 1;
    for (int i = 0; i < 3; ++i)
        ans = (ans + 1) * 2;
    printf("%d\n", ans);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
