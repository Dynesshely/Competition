# B2616 [洛谷] 跑步 题解

## 题目链接
- [洛谷 B2616](https://www.luogu.com.cn/problem/B2616)

## 题目大意
小A速度5m/s，八尾勇速度8m/s，八尾勇在小A后面100m同时起跑，求八尾勇追上小A所需时间。

## 解题思路
相对速度为8-5=3m/s，距离100m，时间=100/3≈33.3333秒。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << 100.0 / 3 << endl;
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
