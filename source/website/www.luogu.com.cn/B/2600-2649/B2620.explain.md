# B2620 [洛谷] 数字比较 题解

## 题目链接
- [洛谷 B2620](https://www.luogu.com.cn/problem/B2620)

## 题目大意
输入两个整数a和b，输出a>b、a<=b、a!=b三个比较结果（真为1，假为0）。

## 解题思路
直接使用关系运算符比较并输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d %d %d\n", a > b, a <= b, a != b);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
