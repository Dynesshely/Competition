# B2611 [洛谷] 竞赛得分 题解

## 题目链接
- [洛谷 B2611](https://www.luogu.com.cn/problem/B2611)

## 题目大意
小A和UIM得分和为480，UIM得分是小A的1.4倍，求各自得分。

## 解题思路
设小A得x分，则UIM得1.4x分，x+1.4x=480，解得x=200，UIM=280。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d %d\n", 200, 280);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
