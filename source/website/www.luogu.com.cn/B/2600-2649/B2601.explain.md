# B2601 [洛谷] 简单的分苹果 2 题解

## 题目链接
- [洛谷 B2601](https://www.luogu.com.cn/problem/B2601)

## 题目大意
14个苹果均分给4名同学，求每位同学分得几个、一共分出去几个、放回冰箱几个。

## 解题思路
整除与取模运算：14/4=3，14/4*4=12，14%4=2。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d\n%d\n%d\n", 14 / 4, 14 / 4 * 4, 14 % 4);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
