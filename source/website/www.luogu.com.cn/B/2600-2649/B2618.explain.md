# B2618 [洛谷] 玩橡皮泥 题解

## 题目链接
- [洛谷 B2618](https://www.luogu.com.cn/problem/B2618)

## 题目大意
两块球形橡皮泥半径分别为4和10，揉在一起做成正方体，求正方体棱长（向下取整）。

## 解题思路
总体积=4/3π×4³+4/3π×10³，正方体体积等于总体积，棱长a=体积的立方根，向下取整。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double pi = 3.141593;
    double v  = 4.0 / 3.0 * pi * (4 * 4 * 4 + 10 * 10 * 10);
    double a  = pow(v, 1.0 / 3.0);
    printf("%d\n", (int)a);
    return 0;
}
```

- 时间复杂度：O(1)
- 空间复杂度：O(1)
