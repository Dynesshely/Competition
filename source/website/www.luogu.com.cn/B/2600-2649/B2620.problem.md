## 题目描述

输入两个整数 $a$ 和 $b$（其绝对值不超过 $10000$），想知道：

1. $a$ 是否大于 $b$？
2. $a$ 是否小于或等于 $b$？
3. $a$ 是否不等于 $b$？

## 输入格式

输入两个整数 a 和 b。

## 输出格式

输出 3 个整数，用空格隔开。对于每一个询问，如果成立（条件为真）输出 1，否则输出 0。

## 样例 #1

### 样例输入

```
2 3
```

### 样例输出

```
0 1 1
```
