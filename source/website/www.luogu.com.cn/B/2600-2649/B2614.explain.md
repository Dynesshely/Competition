# B2614 [洛谷] 鸡兔共笼 题解

## 题目链接
- [洛谷 B2614](https://www.luogu.com.cn/problem/B2614)

## 题目大意
鸡兔同笼，头共35，脚共94。求鸡和兔各几只。

## 解题思路
设鸡x只，兔y只。x+y=35，2x+4y=94。解得x=23，y=12。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    printf("%d %d\n", 12, 23);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
