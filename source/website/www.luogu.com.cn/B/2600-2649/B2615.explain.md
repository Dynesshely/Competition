# B2615 [洛谷] 定期存款 题解

## 题目链接
- [洛谷 B2615](https://www.luogu.com.cn/problem/B2615)

## 题目大意
小A每年存一年期3.5%，连本带利续存5年。UIM直接存五年定期4%，本金均为10000元。求5年后两人各有多少钱。

## 解题思路
小A：10000×(1+0.035)^5；UIM：10000×(1+0.04×5)。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << 10000 * pow(1.035, 5) << ' ' << 10000 * (1 + 0.04 * 5) << endl;
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
