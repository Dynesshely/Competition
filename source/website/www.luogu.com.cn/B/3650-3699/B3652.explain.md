# B3652 [洛谷] 【语言月赛202208】渡荆门送别 题解

## 题目链接
- [洛谷 B3652](https://www.luogu.com.cn/problem/B3652)

## 题目大意
给定 n 个数，第一行输出每个数比最大值矮多少（max - a_i），第二行输出每个数比最小值高多少（a_i - min）。

## 解题思路
先遍历一遍找出最大值和最小值，再遍历一次依次计算差值输出。注意 a_i 最大 10^18，使用 long long。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int               n;
vector<long long> a;
long long         mx, mn;

int main() {
    scanf("%d", &n);
    a.resize(n);
    mx = 0;
    mn = 1e18 + 1;
    for (int i = 0; i < n; ++i) {
        scanf("%lld", &a[i]);
        if (a[i] > mx)
            mx = a[i];
        if (a[i] < mn)
            mn = a[i];
    }
    for (int i = 0; i < n; ++i)
        printf("%lld ", mx - a[i]);
    printf("\n");
    for (int i = 0; i < n; ++i)
        printf("%lld ", a[i] - mn);
    printf("\n");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)。
- 空间复杂度：O(n)。
