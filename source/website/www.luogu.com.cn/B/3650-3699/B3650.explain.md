# B3650 [洛谷] 【语言月赛202208】求和 题解

## 题目链接
- [洛谷 B3650](https://www.luogu.com.cn/problem/B3650)

## 题目大意
输入 n，每行输出 1 到 i 的累加和（i = 1..n）。

## 解题思路
使用 long long 维护前缀和，每次累加后输出。注意 n 最大 10^7，避免使用 endl 导致 TLE。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

int main() {
    scanf("%d", &n);
    long long s = 0;
    for (int i = 1; i <= n; ++i) {
        s += i;
        printf("%lld\n", s);
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)。
- 空间复杂度：O(1)。
