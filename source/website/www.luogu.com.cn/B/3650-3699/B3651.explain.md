# B3651 [洛谷] 【语言月赛202208】数组调整 题解

## 题目链接
- [洛谷 B3651](https://www.luogu.com.cn/problem/B3651)

## 题目大意
给定长度为 n 的数组，将第 k 个数改为 -a_k，求修改后数组所有数的和。

## 解题思路
读入所有数并累加求和，同时记录第 k 个数。最终和减去两倍的 a_k 即为答案。注意使用 long long。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int       n, k;
long long sum;

int main() {
    scanf("%d%d", &n, &k);
    for (int i = 1; i <= n; ++i) {
        int a;
        scanf("%d", &a);
        sum += a;
        if (i == k)
            sum -= 2LL * a;
    }
    printf("%lld\n", sum);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)。
- 空间复杂度：O(1)。
