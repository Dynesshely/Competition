## 题目描述

求 $X$，$Y$ 之间的素数个数（包括 $X$ 和 $Y$）。

## 输入格式

两个整数 $X$ 和 $Y$（$1 \le X,Y \le 1.1 \times 10^6$）。

## 输出格式

输出一个整数，表示 $X, Y$ 之间的素数个数（包括 $X$ 和 $Y$）。

## 样例 #1

### 样例输入

```
1 100
```

### 样例输出

```
25
```
