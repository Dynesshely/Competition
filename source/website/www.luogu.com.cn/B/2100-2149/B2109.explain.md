# B2109 [洛谷] 统计数字字符个数 题解

## 题目链接
- [洛谷 B2109](https://www.luogu.com.cn/problem/B2109)

## 题目大意
输入一行字符，统计其中数字字符（'0'~'9'）的个数。

## 解题思路
用 fgets 读入一整行字符串，遍历每个字符，判断其 ASCII 值是否在 '0' 到 '9' 之间，若在则计数器加一。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[260];
int  cnt;

void input() { fgets(s, 260, stdin); }

void process() {
    for (int i = 0; s[i]; i++)
        if (s[i] >= '0' && s[i] <= '9')
            cnt++;
}

void output() { printf("%d\n", cnt); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为字符串长度
- 空间复杂度：O(1)
