# B2131 甲流病人初筛 题解

## 题目链接
- [洛谷 B2131](https://www.luogu.com.cn/problem/B2131)

## 题目大意
给定 n 个病人的姓名、体温和是否咳嗽（1 表示咳嗽），筛选出体温 ≥ 37.5 且咳嗽的病人，按输入顺序输出姓名，最后输出人数。

## 解题思路
读入所有病人信息，遍历检查体温是否 ≥ 37.5 且咳嗽标志为 1，满足条件则输出姓名并计数，最后输出总人数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int   n;
char  names[205][10];
float temps[205];
int   cough[205];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%s%f%d", names[i], &temps[i], &cough[i]);
}

void process() {
    int cnt = 0;
    for (int i = 0; i < n; i++)
        if (temps[i] >= 37.5f && cough[i] == 1) {
            printf("%s\n", names[i]);
            cnt++;
        }
    printf("%d\n", cnt);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
