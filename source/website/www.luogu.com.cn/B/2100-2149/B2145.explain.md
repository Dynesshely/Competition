# B2145 洛谷 digit 函数 题解

## 题目链接
- [洛谷 B2145](https://www.luogu.com.cn/problem/B2145)

## 题目大意
给定正整数 n（n <= 1e9）和 k（k <= 10），输出 n 从右边数第 k 个数字。

## 解题思路
通过循环先去掉右边 k-1 位数字（n /= 10 重复 k-1 次），然后取 n 对 10 的余数即为所求数字。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, k;

int digit(int n, int k) {
    for (int i = 1; i < k; i++)
        n /= 10;
    return n % 10;
}

void input() { scanf("%d%d", &n, &k); }

void process() { printf("%d\n", digit(n, k)); }

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(k)
- 空间复杂度：O(1)
