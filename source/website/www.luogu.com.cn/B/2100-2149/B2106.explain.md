# B2106 [洛谷] 矩阵转置 题解

## 题目链接
- [洛谷 B2106](https://www.luogu.com.cn/problem/B2106)

## 题目大意
给定一个 n 行 m 列的矩阵 A，输出它的转置矩阵 A^T，即将原矩阵的行列互换。

## 解题思路
转置矩阵的第 i 行第 j 列元素等于原矩阵的第 j 行第 i 列元素。直接按列优先遍历原矩阵，先枚举列再枚举行，输出 a[j][i] 即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a[105][105], n, m;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &a[i][j]);
}

void process() {}

void output() {
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++)
            printf("%d ", a[j][i]);
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * m)
- 空间复杂度：O(n * m)
