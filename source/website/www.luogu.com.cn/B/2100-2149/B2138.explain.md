# B2138 洛谷 最大质因子序列 题解

## 题目链接
- [洛谷 B2138](https://www.luogu.com.cn/problem/B2138)

## 题目大意
给定正整数 m, n（2 <= m <= n <= 5000），依次输出 m 到 n 之间每个数的最大质因子。若某数本身是质数，则输出该数自身。

## 解题思路
对区间内每个数，通过试除法分解质因数，记录每次整除时的因子作为当前最大质因子。循环结束后若剩余部分大于 1，则其本身即为最大质因子。结果用逗号分隔输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int m, n;

int maxPrimeFactor(int x) {
    int maxF = 1, tmp = x;
    for (int i = 2; i * i <= tmp; i++)
        while (tmp % i == 0) {
            maxF = i;
            tmp /= i;
        }
    if (tmp > 1)
        maxF = tmp;
    return maxF;
}

void input() { scanf("%d%d", &m, &n); }

void process() {
    for (int i = m; i <= n; i++) {
        printf("%d", maxPrimeFactor(i));
        if (i < n)
            printf(",");
    }
    printf("\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O((n-m) sqrt(n))
- 空间复杂度：O(1)
