# B2132 素数对 题解

## 题目链接
- [洛谷 B2132](https://www.luogu.com.cn/problem/B2132)

## 题目大意
找出所有不超过 n 的素数对（相差为 2 的两个素数），每对一行，若没有则输出 empty。

## 解题思路
先用埃拉托色尼筛法求出所有 ≤ n 的素数，然后遍历 2 到 n-2，若 i 和 i+2 均为素数则输出。若遍历完未找到任何素数对则输出 empty。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++)
        if (isPrime[i])
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
    bool found = false;
    for (int i = 2; i + 2 <= n; i++)
        if (isPrime[i] && isPrime[i + 2]) {
            printf("%d %d\n", i, i + 2);
            found = true;
        }
    if (!found)
        printf("empty\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log log n)
- 空间复杂度：O(n)
