# B2104 [洛谷] 矩阵加法 题解

## 题目链接
- [洛谷 B2104](https://www.luogu.com.cn/problem/B2104)

## 题目大意
输入两个 n×m 矩阵 A 和 B，输出 A+B。

## 解题思路
读入 A 存入数组，读入 B 时直接加到 A 对应位置，最后输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, a[105][105], b;

void input() { scanf("%d%d", &n, &m); }

void process() {
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) scanf("%d", &a[i][j]);
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            scanf("%d", &b);
            a[i][j] += b;
        }
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            printf("%d%c", a[i][j], j == m ? '\n' : ' ');
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(nm)
- 空间复杂度：O(nm)
