# B2148 洛谷 再求 f(x,n) 题解

## 题目链接
- [洛谷 B2148](https://www.luogu.com.cn/problem/B2148)

## 题目大意
计算 f(x, n) = x / (n + x / ((n-1) + x / ((n-2) + ... + x / (1 + x)...)))，结果保留两位小数。

## 解题思路
定义递归函数 doSolve(x, n)，当 n == 1 时返回 x / (1 + x)，否则返回 x / (n + doSolve(x, n-1))。从内向外逐层计算。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

double x;
int    n;

double doSolve(double x, int n) {
    if (n == 1)
        return x / (1 + x);
    return x / (n + doSolve(x, n - 1));
}

void input() { scanf("%lf%d", &x, &n); }

void process() { printf("%.2f\n", doSolve(x, n)); }

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)（递归栈空间）
