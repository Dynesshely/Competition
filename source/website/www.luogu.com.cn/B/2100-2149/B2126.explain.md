# B2126 连续出现的字符 题解

## 题目链接
- [洛谷 B2126](https://www.luogu.com.cn/problem/B2126)

## 题目大意
给定整数 k 和一个仅包含大小写字母的字符串，找出第一个连续出现次数不低于 k 次的字符。若不存在则输出 `No`。

## 解题思路
遍历字符串，统计连续相同字符的出现次数，当计数达到 k 时立即输出该字符；若遍历结束仍未找到，输出 `No`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  k;
char s[1005];

void input() { scanf("%d%s", &k, s); }

void process() {
    int len = strlen(s);
    int cnt = 1;
    if (cnt >= k) {
        printf("%c\n", s[0]);
        return;
    }
    for (int i = 1; i < len; i++) {
        if (s[i] == s[i - 1])
            cnt++;
        else
            cnt = 1;
        if (cnt >= k) {
            printf("%c\n", s[i]);
            return;
        }
    }
    printf("No\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为字符串长度。
- 空间复杂度：O(n)。
