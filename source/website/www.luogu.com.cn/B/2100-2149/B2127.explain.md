# B2127 求正整数 2 和 n 之间的完全数 题解

## 题目链接
- [洛谷 B2127](https://www.luogu.com.cn/problem/B2127)

## 题目大意
求正整数 2 到 n 之间的所有完全数。完全数是指所有比它小的因数之和等于它本身的自然数，如 6 = 1 + 2 + 3。

## 解题思路
枚举 2 到 n 的每个数，判断其是否为完全数。判断时枚举到 sqrt(x)，找出所有真因数并求和，若和等于 x 则输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

bool isPerfect(int x) {
    int sum = 0;
    for (int i = 1; i * i <= x; i++)
        if (x % i == 0) {
            if (i != x)
                sum += i;
            int j = x / i;
            if (j != x && j != i)
                sum += j;
        }
    return sum == x;
}

void process() {
    for (int i = 2; i <= n; i++)
        if (isPerfect(i))
            printf("%d\n", i);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n sqrt n)
- 空间复杂度：O(1)
