# B2117 [洛谷] 整理药名 题解

## 题目链接
- [洛谷 B2117](https://www.luogu.com.cn/problem/B2117)

## 题目大意
给定 n 个药品名，将其整理成规范格式：第一个字符如果是字母则大写，其余字母小写，非字母字符不变。

## 解题思路
对每个单词，先将首字母转为大写（若为小写），再将剩余字母中的大写转为小写。直接遍历每个单词的每个字符进行条件判断即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  n;
char w[105][25];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%s", w[i]);
}

void process() {
    for (int i = 0; i < n; i++) {
        int len = strlen(w[i]);
        if (w[i][0] >= 'a' && w[i][0] <= 'z')
            w[i][0] = w[i][0] - 'a' + 'A';
        for (int j = 1; j < len; j++)
            if (w[i][j] >= 'A' && w[i][j] <= 'Z')
                w[i][j] = w[i][j] - 'A' + 'a';
    }
}

void output() {
    for (int i = 0; i < n; i++)
        printf("%s\n", w[i]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * len)，n 为单词数，len 为单词平均长度
- 空间复杂度：O(n * len)
