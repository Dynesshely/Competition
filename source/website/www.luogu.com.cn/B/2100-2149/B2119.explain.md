# B2119 [洛谷] 删除单词后缀 题解

## 题目链接
- [洛谷 B2119](https://www.luogu.com.cn/problem/B2119)

## 题目大意
给定一个单词，若其以 `er`、`ly` 或 `ing` 结尾，则删除该后缀，否则保持不变。题目保证删除后缀后的单词长度不为 0。

## 解题思路
先计算单词长度，依次检查是否以 `ing`、`ly`、`er` 结尾。通过判断末尾几个字符是否匹配来确定后缀，匹配时在对应位置置字符串结束符 `\0` 即可删除后缀。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[35];

void input() { scanf("%s", s); }

void process() {
    int len = strlen(s);
    if (len >= 3 && s[len - 1] == 'g' && s[len - 2] == 'n' && s[len - 3] == 'i')
        s[len - 3] = '\0';
    else if (len >= 2 && s[len - 1] == 'y' && s[len - 2] == 'l')
        s[len - 2] = '\0';
    else if (len >= 2 && s[len - 1] == 'r' && s[len - 2] == 'e')
        s[len - 2] = '\0';
}

void output() { printf("%s\n", s); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为单词长度
- 空间复杂度：O(1)
