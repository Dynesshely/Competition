# B2107 [洛谷] 图像旋转 题解

## 题目链接
- [洛谷 B2107](https://www.luogu.com.cn/problem/B2107)

## 题目大意
给定一个 n 行 m 列的黑白图像，将其顺时针旋转 90 度后输出。

## 解题思路
顺时针旋转 90 度后，原图第 i 行第 j 列的元素会移动到第 j 行第 n-1-i 列。输出时按新图的坐标枚举，即第 i 行第 j 列输出原图第 n-1-j 行第 i 列的元素。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a[105][105], n, m;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &a[i][j]);
}

void process() {}

void output() {
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++)
            printf("%d ", a[n - 1 - j][i]);
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * m)
- 空间复杂度：O(n * m)
