# B2116 [洛谷] 加密的病历单 题解

## 题目链接
- [洛谷 B2116](https://www.luogu.com.cn/problem/B2116)

## 题目大意
给定加密后的字符串，解密规则为加密过程的逆操作：先将大小写反转，再将字符串逆序，最后将每个字母循环右移 3 位（即用前一个字母替代，但这里是解密所以是加密的逆操作）。

## 解题思路
加密过程为：循环左移 3 位 -> 逆序存储 -> 大小写反转。因此解密需反向操作：先大小写反转，再逆序，最后循环右移 3 位（即每个字母减 3，处理边界环绕）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[55];
int  len;

void input() {
    scanf("%s", s);
    len = strlen(s);
}

void process() {
    for (int i = 0; i < len; i++)
        if (s[i] >= 'A' && s[i] <= 'Z')
            s[i] = s[i] - 'A' + 'a';
        else
            s[i] = s[i] - 'a' + 'A';
    for (int i = 0; i < len / 2; i++) {
        char t         = s[i];
        s[i]           = s[len - 1 - i];
        s[len - 1 - i] = t;
    }
    for (int i = 0; i < len; i++)
        if (s[i] >= 'a' && s[i] <= 'z')
            s[i] = (s[i] - 'a' + 3) % 26 + 'a';
        else
            s[i] = (s[i] - 'A' + 3) % 26 + 'A';
}

void output() { printf("%s\n", s); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为字符串长度
- 空间复杂度：O(1)
