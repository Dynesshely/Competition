## 题目描述

两位正整数的简单算术运算（只考虑整数运算），算术运算为：

`+`，加法运算；

`-`，减法运算；

`*`，乘法运算；

`/`，整除运算；

`%`，取余运算。

算术表达式的格式为（运算符前后可能有空格）：

运算数运算符运算数。

请输出相应的结果。

## 输入格式

一行算术表达式。

## 输出格式

整型算数运算的结果（结果值不一定为 $2$ 位数，可能多于 $2$ 位或少于 $2$ 位）。

## 样例 #1

### 样例输入

```
32+64
```

### 样例输出

```
96
```
