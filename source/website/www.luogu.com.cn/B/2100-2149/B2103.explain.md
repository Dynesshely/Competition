# B2103 [洛谷] 图像相似度 题解

## 题目链接
- [洛谷 B2103](https://www.luogu.com.cn/problem/B2103)

## 题目大意
给定两幅 m×n 的 0-1 图像，计算相同像素点占总像素点数的百分比，精确到小数两位。

## 解题思路
先读入第一幅图像存入数组，再读入第二幅图像逐元素比较，统计相同位置像素相等的个数。输出 `same / (m*n) * 100`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int m, n, a[105][105], b, same;

void input() { scanf("%d%d", &m, &n); }

void process() {
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) scanf("%d", &a[i][j]);
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            scanf("%d", &b);
            if (a[i][j] == b) ++same;
        }
    printf("%.2f\n", 100.0 * same / (m * n));
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(mn)
- 空间复杂度：O(mn)
