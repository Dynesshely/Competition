## 题目描述

输入一个字符串，输出该字符串是否回文。回文是指顺读和倒读都一样的字符串。

## 输入格式

输入一行字符串，长度小于 $100$。

## 输出格式

如果字符串是回文，输出 `yes`；否则，输出 `no`。

## 样例 #1

### 样例输入

```
abcdedcba
```

### 样例输出

```
yes
```
