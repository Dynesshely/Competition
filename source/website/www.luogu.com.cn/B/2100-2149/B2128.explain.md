# B2128 素数个数 题解

## 题目链接
- [洛谷 B2128](https://www.luogu.com.cn/problem/B2128)

## 题目大意
给定正整数 n（n ≤ 50000），求 2 到 n 之间有多少个素数。

## 解题思路
使用埃拉托色尼筛法标记所有合数，然后统计剩余素数个数。从 2 开始，若当前数是素数，则将其倍数全部标记为合数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++)
        if (isPrime[i])
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
    int cnt = 0;
    for (int i = 2; i <= n; i++)
        if (isPrime[i])
            cnt++;
    printf("%d\n", cnt);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n log log n)
- 空间复杂度：O(n)
