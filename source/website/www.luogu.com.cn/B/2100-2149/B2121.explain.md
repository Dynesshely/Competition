# B2121 最长最短单词 题解

## 题目链接
- [洛谷 B2121](https://www.luogu.com.cn/problem/B2121)

## 题目大意
输入一行句子，单词由连续的字母构成，空格、逗号和句号为间隔。输出第一个最长的单词和第一个最短的单词。

## 解题思路
遍历字符串，用 `isalpha` 判断字母来提取单词。记录当前单词长度，若大于已记录的最大长度则更新最长单词，若小于已记录的最小长度则更新最短单词。遍历结束后输出结果。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[20005];
char cur[105];
char longest[105], shortest[105];
int  maxLen, minLen = 105;

void input() { fgets(s, 20005, stdin); }

void process() {
    int len = strlen(s);
    if (s[len - 1] == '\n')
        s[--len] = '\0';
    int idx = 0;
    for (int i = 0; i <= len; i++) {
        if (isalpha(s[i])) {
            cur[idx++] = s[i];
        } else if (idx > 0) {
            cur[idx] = '\0';
            if (idx > maxLen) {
                maxLen = idx;
                strcpy(longest, cur);
            }
            if (idx < minLen) {
                minLen = idx;
                strcpy(shortest, cur);
            }
            idx = 0;
        }
    }
}

void output() { printf("%s\n%s\n", longest, shortest); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为句子长度。
- 空间复杂度：O(n)。
