## 题目描述

给定一个只包含小写字母的字符串，请你找到第一个仅出现一次的字符。如果没有，输出 `no`。

## 输入格式

一个字符串，长度小于 $1100$。

## 输出格式

输出第一个仅出现一次的字符，若没有则输出 `no`。

## 样例 #1

### 样例输入

```
abcabd
```

### 样例输出

```
c
```

## 样例 #2

### 样例输入

```
aabbcc
```

### 样例输出

```
no
```
