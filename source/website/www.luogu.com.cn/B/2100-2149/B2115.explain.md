# B2115 [洛谷] 密码翻译 题解

## 题目链接
- [洛谷 B2115](https://www.luogu.com.cn/problem/B2115)

## 题目大意
给定一个加密后的字符串，解密规则：将每个字母用其前一个字母替代（a 变 z，A 变 Z，b 变 a，B 变 A，以此类推），非字母字符不变。

## 解题思路
遍历字符串，对每个字符判断：若是小写字母，则减 1（a 特殊处理为 z）；若是大写字母同理处理；非字母字符保持不变。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[10010];

void input() { fgets(s, 10010, stdin); }

void process() {
    for (int i = 0; s[i]; i++) {
        if (s[i] >= 'a' && s[i] <= 'z')
            s[i] = (s[i] - 'a' - 1 + 26) % 26 + 'a';
        else if (s[i] >= 'A' && s[i] <= 'Z')
            s[i] = (s[i] - 'A' - 1 + 26) % 26 + 'A';
    }
}

void output() { printf("%s", s); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为字符串长度
- 空间复杂度：O(1)
