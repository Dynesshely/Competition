# B2137 洛谷 判决素数个数 题解

## 题目链接
- [洛谷 B2137](https://www.luogu.com.cn/problem/B2137)

## 题目大意
给定两个整数 X 和 Y（1 <= X, Y <= 1.1e6），求 [X, Y] 区间内（含端点）素数的个数。

## 解题思路
数据范围较大，使用埃氏筛法预处理所有素数。先令 X <= Y，然后用布尔数组标记合数，最后统计区间内的素数个数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1100000;
bool      isComp[MAXN + 1];
int       x, y;

void input() { scanf("%d%d", &x, &y); }

void process() {
    if (x > y)
        swap(x, y);
    isComp[0] = isComp[1] = true;
    for (int i = 2; i * i <= y; i++)
        if (!isComp[i])
            for (int j = i * i; j <= y; j += i)
                isComp[j] = true;
    int cnt = 0;
    for (int i = x; i <= y; i++)
        if (!isComp[i])
            cnt++;
    printf("%d\n", cnt);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(y log log y)
- 空间复杂度：O(y)
