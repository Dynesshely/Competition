# B2122 单词翻转 题解

## 题目链接
- [洛谷 B2122](https://www.luogu.com.cn/problem/B2122)

## 题目大意
输入一个由空格分隔单词的句子，将每个单词反转后按行输出。

## 解题思路
遍历字符串，遇到空格或字符串结尾时，将当前单词的字符逆序输出，然后换行。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[105];

void input() { fgets(s, 105, stdin); }

void process() {
    int len = strlen(s);
    if (s[len - 1] == '\n')
        s[--len] = '\0';
    int start = 0;
    for (int i = 0; i <= len; i++) {
        if (s[i] == ' ' || s[i] == '\0') {
            for (int j = i - 1; j >= start; j--)
                putchar(s[j]);
            putchar('\n');
            start = i + 1;
        }
    }
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为句子长度。
- 空间复杂度：O(n)。
