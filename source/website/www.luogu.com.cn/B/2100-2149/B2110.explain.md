# B2110 [洛谷] 找第一个只出现一次的字符 题解

## 题目链接
- [洛谷 B2110](https://www.luogu.com.cn/problem/B2110)

## 题目大意
给定一个只包含小写字母的字符串，找出第一个仅出现一次的字符。若没有则输出 `no`。

## 解题思路
用一个长度为 26 的频次数组记录每个字母出现的次数。第一遍遍历统计所有字符频次，第二遍遍历找到第一个频次为 1 的字符输出。若遍历完仍未找到则输出 `no`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[1100];
int  freq[26];

void input() { scanf("%s", s); }

void process() {
    for (int i = 0; s[i]; i++)
        freq[s[i] - 'a']++;
}

void output() {
    for (int i = 0; s[i]; i++)
        if (freq[s[i] - 'a'] == 1) {
            printf("%c\n", s[i]);
            return;
        }
    printf("no\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为字符串长度
- 空间复杂度：O(1)
