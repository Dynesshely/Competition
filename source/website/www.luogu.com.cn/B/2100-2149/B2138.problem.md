## 题目描述

任意输入两个正整数 $m,n$（$2\le m\le n\le5000$），依次输出 $m$ 到 $n$ 之间每个数的最大质因子（包括 $m$ 和 $n$；如果某个数本身是质数，则输出这个数自身）。

## 输入格式

一行，包含两个正整数 $m$ 和 $n$，其间以单个空格间隔。

## 输出格式

一行，每个整数的最大质因子，以逗号间隔。

## 样例 #1

### 样例输入

```
5 10
```

### 样例输出

```
5,3,7,2,3,5
```
