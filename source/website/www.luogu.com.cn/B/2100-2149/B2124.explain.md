# B2124 判断字符串是否为回文 题解

## 题目链接
- [洛谷 B2124](https://www.luogu.com.cn/problem/B2124)

## 题目大意
输入一个字符串，判断它是否为回文串（顺读和倒读相同），输出 `yes` 或 `no`。

## 解题思路
从字符串两端向中间比较字符，若出现不相等的情况则不是回文；否则是回文。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[105];

void input() { scanf("%s", s); }

void process() {
    int len = strlen(s);
    for (int i = 0; i < len / 2; i++)
        if (s[i] != s[len - 1 - i]) {
            printf("no\n");
            return;
        }
    printf("yes\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为字符串长度。
- 空间复杂度：O(n)。
