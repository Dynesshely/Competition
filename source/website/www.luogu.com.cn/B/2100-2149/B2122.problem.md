## 题目描述

小明同学写单词的时候喜欢反着写，比如 `hello` 他会写成 `olleh`。给出小明同学写的一个句子，请你将所有的单词复原。

## 输入格式

共一行，一个字符串表示句子，单词之间以空格分隔。

## 输出格式

每个单词一行。

## 样例 #1

### 样例输入

```
olleh dlrow
```

### 样例输出

```
hello
world
```

## 提示

句子长度不超过 $100$，仅包含小写字母和空格，不存在多余的空格。
