# B2142 洛谷 求 1+2+3+...+N 的值 题解

## 题目链接
- [洛谷 B2142](https://www.luogu.com.cn/problem/B2142)

## 题目大意
给定正整数 N（N <= 200），用递归方法求 1 + 2 + 3 + ... + N 的值。

## 解题思路
定义递归函数 doSum(x)，当 x == 1 时返回 1，否则返回 x + doSum(x - 1)。递归深度不超过 N，符合题目要求。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

int doSum(int x) {
    if (x == 1)
        return 1;
    return x + doSum(x - 1);
}

void input() { scanf("%d", &n); }

void process() { printf("%d\n", doSum(n)); }

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N)
- 空间复杂度：O(N)（递归栈空间）
