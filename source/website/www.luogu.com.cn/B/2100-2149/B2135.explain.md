# B2135 洛谷 单词替换 题解

## 题目链接
- [洛谷 B2135](https://www.luogu.com.cn/problem/B2135)

## 题目大意
给定一个由空格分隔的字符串 s（长度 <= 200），以及两个单词 a 和 b。将 s 中所有等于 a 的单词替换为 b，输出替换后的字符串。

## 解题思路
按空格分割字符串 s 中的单词，对每个单词判断是否与 a 相等。若相等则输出 b，否则输出原单词。单词之间用空格分隔。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[205], a[105], b[105], w[105];

void input() {
    fgets(s, sizeof(s), stdin);
    int sl = strlen(s);
    while (sl > 0 && (s[sl - 1] == '\n' || s[sl - 1] == '\r'))
        s[--sl] = '\0';
    scanf("%s", a);
    scanf("%s", b);
}

void process() {
    int len = strlen(s), wi = 0, first = 1;
    for (int i = 0; i <= len; i++)
        if (s[i] == ' ' || s[i] == '\0') {
            w[wi]     = '\0';
            int match = (strcmp(w, a) == 0);
            if (!first)
                printf(" ");
            printf("%s", match ? b : w);
            first = 0;
            wi    = 0;
        } else
            w[wi++] = s[i];
    printf("\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(L)，其中 L 为字符串长度
- 空间复杂度：O(L)
