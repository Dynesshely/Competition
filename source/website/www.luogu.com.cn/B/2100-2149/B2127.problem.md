## 题目描述

求正整数 $2$ 和 $n$ 之间的完全数（一行一个数）。

完全数：对于一个自然数，所有比它小的所有因数之和，等于它本身，它就是个完全数。如 $6=1+2+3$。

## 输入格式

输入 $n(n \le 10000)$。

## 输出格式

一行一个数，按由小到大的顺序。

## 样例 #1

### 样例输入

```
10
```

### 样例输出

```
6
```

## 样例 #2

### 样例输入

```
100
```

### 样例输出

```
6
28
```
