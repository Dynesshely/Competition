## 题目描述

在程序中定义一函数 $digit(n,k)$，它能分离出整数 $n$ 从右边数第 $k$ 个数字。

## 输入格式

正整数 $n$ 和 $k$。

## 输出格式

一个数字。

## 样例 #1

### 样例输入

```
31859 3
```

### 样例输出

```
8
```

## 提示

$n \le 10^9$。

$k \le 10$。
