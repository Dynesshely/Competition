# B2102 [洛谷] 计算鞍点 题解

## 题目链接
- [洛谷 B2102](https://www.luogu.com.cn/problem/B2102)

## 题目大意
在 5×5 矩阵中寻找鞍点——该元素是所在行的最大值且是所在列的最小值。

## 解题思路
预先计算每行最大值和每列最小值，再遍历矩阵检查是否同时满足两个条件。若找到则输出，否则输出 `not found`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a[6][6], rowMax[6], colMin[6];

void input() {
    for (int i = 1; i <= 5; i++)
        for (int j = 1; j <= 5; j++) scanf("%d", &a[i][j]);
}

void process() {
    for (int i = 1; i <= 5; i++) {
        rowMax[i] = a[i][1];
        for (int j = 2; j <= 5; j++)
            if (a[i][j] > rowMax[i]) rowMax[i] = a[i][j];
    }
    for (int j = 1; j <= 5; j++) {
        colMin[j] = a[1][j];
        for (int i = 2; i <= 5; i++)
            if (a[i][j] < colMin[j]) colMin[j] = a[i][j];
    }
    for (int i = 1; i <= 5; i++)
        for (int j = 1; j <= 5; j++)
            if (a[i][j] == rowMax[i] && a[i][j] == colMin[j]) {
                printf("%d %d %d\n", i, j, a[i][j]);
                return;
            }
    printf("not found\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)，矩阵固定 5×5
- 空间复杂度：O(1)
