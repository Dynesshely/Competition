# B2112 [洛谷] 石头剪子布 题解

## 题目链接
- [洛谷 B2112](https://www.luogu.com.cn/problem/B2112)

## 题目大意
给定 N 次石头剪子布游戏的选择，输出每次游戏的结果：Player1 胜、Player2 胜或平局。

## 解题思路
石头 (Rock) 胜剪刀 (Scissors)，剪刀胜布 (Paper)，布胜石头。对于每局游戏，先判断平局，再判断 Player1 的三种获胜情况，其余情况均为 Player2 获胜。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {
    char s1[15], s2[15];
    for (int i = 0; i < n; i++) {
        scanf("%s%s", s1, s2);
        if (strcmp(s1, s2) == 0)
            printf("Tie\n");
        else if ((strcmp(s1, "Rock") == 0 && strcmp(s2, "Scissors") == 0) || (strcmp(s1, "Scissors") == 0 && strcmp(s2, "Paper") == 0) || (strcmp(s1, "Paper") == 0 && strcmp(s2, "Rock") == 0))
            printf("Player1\n");
        else
            printf("Player2\n");
    }
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N)
- 空间复杂度：O(1)
