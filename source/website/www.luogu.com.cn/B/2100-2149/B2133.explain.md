# B2133 我家的门牌号 题解

## 题目链接
- [洛谷 B2133](https://www.luogu.com.cn/problem/B2133)

## 题目大意
门牌号从 1 开始顺序编号。设我家门牌号为 x，共有 total 家，满足其余各家门牌号之和减去我家门牌号的两倍等于 n。求 x 和 total，数据保证唯一解。

## 解题思路
设总共有 total 家，则门牌号总和为 total(total+1)/2。其余各家之和为总和减去 x，依题意有 (总和 - x) - 2x = n，即 总和 - 3x = n，变形得 总和 - n = 3x。枚举 total 从 1 开始，令 diff = total(total+1)/2 - n，若 diff > 0 且 diff 能被 3 整除，则 x = diff/3，检查 x 是否在 [1, total] 范围内，满足即输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {
    for (int total = 1;; total++) {
        int sum = total * (total + 1) / 2;
        if (sum > n) {
            int diff = sum - n;
            if (diff % 3 == 0) {
                int my = diff / 3;
                if (my >= 1 && my <= total) {
                    printf("%d %d\n", my, total);
                    return;
                }
            }
        }
    }
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(sqrt(n))
- 空间复杂度：O(1)
