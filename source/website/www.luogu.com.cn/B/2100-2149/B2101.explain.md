# B2101 [洛谷] 计算矩阵边缘元素之和 题解

## 题目链接
- [洛谷 B2101](https://www.luogu.com.cn/problem/B2101)

## 题目大意
求矩阵边缘元素（第一行、最后一行、第一列、最后一列）之和。

## 解题思路
边读入边判断：若当前元素位于边缘（行=1 或 m，列=1 或 n），累加到答案。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int m, n, x, sum;

void input() { scanf("%d%d", &m, &n); }

void process() {
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            scanf("%d", &x);
            if (i == 1 || i == m || j == 1 || j == n) sum += x;
        }
    printf("%d\n", sum);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(mn)
- 空间复杂度：O(1)
