## 题目描述

编程求 $2\sim n$（$n$ 为大于等于 $2$ 的正整数）中有多少个素数。

## 输入格式

输入 $n(2 \le n \le 50000)$。

## 输出格式

输出一行，表示 $2\sim n$ 中的素数个数。

## 样例 #1

### 样例输入

```
10
```

### 样例输出

```
4
```

## 提示

对于所有数据，保证 $2 \le n \le 50000$。
