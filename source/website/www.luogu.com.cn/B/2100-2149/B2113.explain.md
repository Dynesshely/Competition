# B2113 [洛谷] 输出亲朋字符串 题解

## 题目链接
- [洛谷 B2113](https://www.luogu.com.cn/problem/B2113)

## 题目大意
给定字符串 s，定义亲朋字符串：第 i 个字符为 s[i] 与 s[i+1] 的 ASCII 值之和对应的字符，最后一个字符由 s 的最后一个字符与第一个字符之和得到。

## 解题思路
遍历字符串，对于每个位置 i，计算 s[i] + s[(i+1) % len] 的 ASCII 值之和并直接输出对应字符。利用取模运算处理最后一个字符与第一个字符相加的情况。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[110];
int  len;

void input() {
    fgets(s, 110, stdin);
    len = strlen(s);
    while (len > 0 && (s[len - 1] == '\n' || s[len - 1] == '\r'))
        s[--len] = '\0';
}

void process() {}

void output() {
    for (int i = 0; i < len; i++)
        printf("%c", s[i] + s[(i + 1) % len]);
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为字符串长度
- 空间复杂度：O(1)
