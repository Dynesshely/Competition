# B2123 字符串 p 型编码 题解

## 题目链接
- [洛谷 B2123](https://www.luogu.com.cn/problem/B2123)

## 题目大意
给定一个纯数字字符串，输出其 p 型编码串：将连续相同数字的个数和该数字依次拼接。

## 解题思路
遍历字符串，统计连续相同字符的个数。当遇到不同字符时，输出个数和当前字符，然后重置计数器。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[1005];

void input() { scanf("%s", s); }

void process() {
    int len = strlen(s);
    int cnt = 1;
    for (int i = 1; i <= len; i++) {
        if (s[i] == s[i - 1])
            cnt++;
        else {
            printf("%d%c", cnt, s[i - 1]);
            cnt = 1;
        }
    }
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为字符串长度。
- 空间复杂度：O(n)。
