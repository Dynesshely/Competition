## 题目描述

已知 $f(x,n)=\sqrt{n+\sqrt{(n-1)+\sqrt{(n-2)+\sqrt{...+2+\sqrt{1+x}}}}}$。

计算 $f(x,n)$ 的值。

## 输入格式

输入实数 $x$ 和正整数 $n$。

## 输出格式

函数值，保留两位小数。

## 样例 #1

### 样例输入

```
4.2 10
```

### 样例输出

```
3.68
```

## 提示

#### 数据范围

对于 $100\%$ 的数据，$1 \le x,n \le 10$。
