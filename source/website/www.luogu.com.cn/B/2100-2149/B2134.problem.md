## 题目描述

两个质数的和是 $S$，它们的积最大是多少？

## 输入格式

一个不大于 $10000$ 的正整数 $S$，为两个质数的和。

## 输出格式

一个整数，为两个质数的最大乘积。数据保证有解。

## 样例 #1

### 样例输入

```
50
```

### 样例输出

```
589
```
