# B2118 [洛谷] 验证子串 题解

## 题目链接
- [洛谷 B2118](https://www.luogu.com.cn/problem/B2118)

## 题目大意
输入两个字符串，判断其中一个是否为另一个的子串，输出对应的判断结果。

## 解题思路
使用 C 标准库函数 strstr 判断子串关系。先判断 s1 是否为 s2 的子串，再判断 s2 是否为 s1 的子串，若都不满足则输出 `No substring`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s1[35], s2[35];

void input() { scanf("%s%s", s1, s2); }

void process() {
    if (strstr(s2, s1))
        printf("%s is substring of %s\n", s1, s2);
    else if (strstr(s1, s2))
        printf("%s is substring of %s\n", s2, s1);
    else
        printf("No substring\n");
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len1 * len2)，strstr 最坏情况下为 O(n*m)
- 空间复杂度：O(1)
