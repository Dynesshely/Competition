## 题目描述

输入一行字符，统计出其中数字字符的个数。

## 输入格式

一行字符串，总长度不超过 $255$。

## 输出格式

输出为 $1$ 行，输出字符串里面数字字符的个数。

## 样例 #1

### 样例输入

```
Today is 2021-03-27
```

### 样例输出

```
8
```
