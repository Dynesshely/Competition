# B2114 [洛谷] 配对碱基链 题解

## 题目链接
- [洛谷 B2114](https://www.luogu.com.cn/problem/B2114)

## 题目大意
根据一条 DNA 单链碱基序列，输出其互补链。碱基配对规则：A 与 T 配对，G 与 C 配对。

## 解题思路
遍历输入字符串，对每个字符进行映射转换：A 变 T，T 变 A，G 变 C，C 变 G，依次输出即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

char s[260];

void input() { scanf("%s", s); }

void process() {}

void output() {
    for (int i = 0; s[i]; i++) {
        if (s[i] == 'A')
            printf("T");
        else if (s[i] == 'T')
            printf("A");
        else if (s[i] == 'G')
            printf("C");
        else if (s[i] == 'C')
            printf("G");
    }
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为碱基序列长度
- 空间复杂度：O(1)
