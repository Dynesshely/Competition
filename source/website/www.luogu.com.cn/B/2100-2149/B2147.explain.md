# B2147 洛谷 求 f(x,n) 题解

## 题目链接
- [洛谷 B2147](https://www.luogu.com.cn/problem/B2147)

## 题目大意
计算 f(x, n) = sqrt(n + sqrt((n-1) + sqrt((n-2) + ... + sqrt(2 + sqrt(1 + x))...)))，结果保留两位小数。

## 解题思路
定义递归函数 doSolve(x, n)，当 n == 1 时返回 sqrt(1 + x)，否则返回 sqrt(n + doSolve(x, n-1))。从内向外逐层计算。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

double x;
int    n;

double doSolve(double x, int n) {
    if (n == 1)
        return sqrt(1 + x);
    return sqrt(n + doSolve(x, n - 1));
}

void input() { scanf("%lf%d", &x, &n); }

void process() { printf("%.2f\n", doSolve(x, n)); }

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)（递归栈空间）
