# B2134 洛谷 质数的和与积 题解

## 题目链接
- [洛谷 B2134](https://www.luogu.com.cn/problem/B2134)

## 题目大意
给定一个正整数 S（S <= 10000），S 可以表示为两个质数的和。求这两个质数的最大可能乘积。

## 解题思路
枚举其中一个质数 i 从 2 到 S/2，判断 i 和 S-i 是否均为质数。若是，计算乘积并维护最大值。质数判定采用试除法，时间复杂度 O(sqrt(S))。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int s;

void input() { scanf("%d", &s); }

bool isPrime(int x) {
    if (x < 2)
        return false;
    for (int i = 2; i * i <= x; i++)
        if (x % i == 0)
            return false;
    return true;
}

void process() {
    int maxProd = 0;
    for (int i = 2; i <= s / 2; i++)
        if (isPrime(i) && isPrime(s - i)) {
            int prod = i * (s - i);
            if (prod > maxProd)
                maxProd = prod;
        }
    printf("%d\n", maxProd);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(S sqrt(S))
- 空间复杂度：O(1)
