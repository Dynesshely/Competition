# B2108 [洛谷] 图像模糊处理 题解

## 题目链接
- [洛谷 B2108](https://www.luogu.com.cn/problem/B2108)

## 题目大意
给定 n 行 m 列的图像各像素点灰度值，进行模糊处理：四周最外侧像素点灰度值不变，中间各像素点新灰度值为该像素点及其上下左右四个相邻像素点原灰度值的平均（四舍五入到最接近的整数）。

## 解题思路
新建一个矩阵 b 存放结果。遍历每个像素点，若是边界点则直接复制原值；否则计算该点及其上下左右共五个点的灰度值之和，四舍五入取平均。四舍五入可用 `(sum + 2) / 5` 实现整数除法。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a[105][105], b[105][105], n, m;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &a[i][j]);
}

void process() {
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++) {
            if (i == 0 || i == n - 1 || j == 0 || j == m - 1)
                b[i][j] = a[i][j];
            else {
                int sum = a[i][j] + a[i - 1][j] + a[i + 1][j] + a[i][j - 1] + a[i][j + 1];
                b[i][j] = (sum + 2) / 5;
            }
        }
}

void output() {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++)
            printf("%d ", b[i][j]);
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * m)
- 空间复杂度：O(n * m)
