# B2111 [洛谷] 基因相关性 题解

## 题目链接
- [洛谷 B2111](https://www.luogu.com.cn/problem/B2111)

## 题目大意
比较两条长度相同的 DNA 序列，计算相同位置碱基相同的比例，若该比例大于等于给定阈值则判定为相关，输出 `yes`，否则输出 `no`。

## 解题思路
读入阈值和两条 DNA 序列，遍历序列逐位比较，统计相同位置的个数。将相同个数除以总长度得到比例，与阈值比较后输出结果。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

double threshold;
char   s1[510], s2[510];

void input() {
    scanf("%lf", &threshold);
    scanf("%s%s", s1, s2);
}

void process() {}

void output() {
    int n = strlen(s1), cnt = 0;
    for (int i = 0; i < n; i++)
        if (s1[i] == s2[i])
            cnt++;
    if ((double)cnt / n >= threshold)
        printf("yes\n");
    else
        printf("no\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(len)，len 为 DNA 序列长度
- 空间复杂度：O(1)
