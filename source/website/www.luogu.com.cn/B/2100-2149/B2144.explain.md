# B2144 洛谷 阿克曼（Ackermann）函数 题解

## 题目链接
- [洛谷 B2144](https://www.luogu.com.cn/problem/B2144)

## 题目大意
计算阿克曼函数 A(m, n) 的值，其中 m <= 3，n <= 10。函数定义为：
- A(0, n) = n + 1
- A(m, 0) = A(m-1, 1)
- A(m, n) = A(m-1, A(m, n-1))

## 解题思路
直接按照定义实现递归函数即可。由于 m 和 n 的范围很小，递归深度有限，无需优化。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int m, n;

int ack(int m, int n) {
    if (m == 0)
        return n + 1;
    if (n == 0)
        return ack(m - 1, 1);
    return ack(m - 1, ack(m, n - 1));
}

void input() { scanf("%d%d", &m, &n); }

void process() { printf("%d\n", ack(m, n)); }

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：由递归展开次数决定，m 很小时为 O(2^m) 级别
- 空间复杂度：O(m)（递归栈空间）
