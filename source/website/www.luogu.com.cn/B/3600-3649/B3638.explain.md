# B3638 [洛谷] T1 三角形面积 题解

## 题目链接
- [洛谷 B3638](https://www.luogu.com.cn/problem/B3638)

## 题目大意
给定平面直角坐标系上三个整点坐标，求围成的三角形面积，保证答案为整数。

## 解题思路
使用鞋带公式（行列式法）计算面积：S = |x1(y2-y3) + x2(y3-y1) + x3(y1-y2)| / 2。避免使用浮点运算。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int xA, yA, xB, yB, xC, yC;

int main() {
    scanf("%d%d%d%d%d%d", &xA, &yA, &xB, &yB, &xC, &yC);
    int area = abs(xA * (yB - yC) + xB * (yC - yA) + xC * (yA - yB));
    printf("%d\n", area / 2);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
