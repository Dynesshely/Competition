# B3639 [洛谷] T2 点亮灯笼 题解

## 题目链接
- [洛谷 B3639](https://www.luogu.com.cn/problem/B3639)

## 题目大意
n 个灯笼环形摆放，初始全灭。按下第 x 个按钮会反转 x 及其相邻两个灯笼的状态。给定 m 次操作，输出最终状态。

## 解题思路
用数组模拟灯笼状态，每次操作对 x 及其左右邻居（环形处理）进行异或 1 反转。最后依次输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, a[1005];

int main() {
    scanf("%d%d", &n, &m);
    while (m--) {
        int x;
        scanf("%d", &x);
        a[x] ^= 1;
        int l = x == 1 ? n : x - 1;
        int r = x == n ? 1 : x + 1;
        a[l] ^= 1;
        a[r] ^= 1;
    }
    for (int i = 1; i <= n; ++i)
        printf("%d ", a[i]);
    printf("\n");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + m)。
- 空间复杂度：O(n)。
