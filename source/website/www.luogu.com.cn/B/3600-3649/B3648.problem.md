## 题目描述

小 A 今年 $x$ 岁。他刚学了英语，想告诉其他人：`I am x years old.`

## 输入格式

输入一个数 $x(2\le x \le 100)$，代表小 A 的年龄。

## 输出格式

输出一句话。

## 样例 #1

### 样例输入

```
2
```

### 样例输出

```
I am 2 years old.
```

## 样例 #2

### 样例输入

```
5
```

### 样例输出

```
I am 5 years old.
```
