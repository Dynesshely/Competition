# B3648 [洛谷] 【语言月赛202208】你几岁了 题解

## 题目链接
- [洛谷 B3648](https://www.luogu.com.cn/problem/B3648)

## 题目大意
输入年龄 x，输出 "I am x years old."。

## 解题思路
读入整数后直接格式化输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int x;

int main() {
    scanf("%d", &x);
    printf("I am %d years old.\n", x);
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
