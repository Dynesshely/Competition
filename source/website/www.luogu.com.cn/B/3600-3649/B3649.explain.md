# B3649 [洛谷] 【语言月赛202208】大小比较 题解

## 题目链接
- [洛谷 B3649](https://www.luogu.com.cn/problem/B3649)

## 题目大意
给定整数 a 和 b，若 a <= b 输出 "YE5"，否则输出 "N0"。

## 解题思路
直接比较后按条件输出对应字符串。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a, b;

int main() {
    scanf("%d%d", &a, &b);
    if (a <= b)
        puts("YE5");
    else
        puts("N0");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
