# P1284 [USACO02FEB] 三角形牧场 题解

## 题目链接
- [洛谷 P1284](https://www.luogu.com.cn/problem/P1284)

## 题目大意
给定 n 块木板，每块有长度 l_i，要求所有木板都用上，拼成一个三角形。求最大可能面积（乘以 100 后舍尾），若无法构成三角形输出 -1。

## 解题思路
用 DP 找出所有可能的边长方组合。dp[i][j] 表示能否用部分木板拼出长度 i 和 j 的两条边（第三条边为 total - i - j）。使用 bitset 优化二维可行性 DP。对于每个可行的 (a, b, c) 且满足三角形不等式，用海伦公式计算面积取最大值。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n, len[45], total;
double ans = -1;

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        scanf("%d", &len[i]);
        total += len[i];
    }
}

void process() {
    const int                maxS = 1600;
    vector<bitset<maxS + 1>> dp(maxS + 1);
    dp[0][0] = 1;
    for (int idx = 0; idx < n; ++idx) {
        int                      x   = len[idx];
        vector<bitset<maxS + 1>> pre = dp;
        for (int i = total; i >= x; --i)
            dp[i] |= pre[i - x];
        for (int i = 0; i <= total; ++i)
            dp[i] |= pre[i] << x;
    }
    for (int a = 1; a <= total; ++a) {
        for (int b = 1; b <= total; ++b) {
            if (!dp[a][b])
                continue;
            int c = total - a - b;
            if (c <= 0)
                continue;
            if (a + b <= c || a + c <= b || b + c <= a)
                continue;
            double s    = total / 2.0;
            double area = sqrt(s * (s - a) * (s - b) * (s - c));
            if (area > ans)
                ans = area;
        }
    }
}

void output() {
    if (ans < 0)
        printf("-1\n");
    else
        printf("%d\n", (int)(ans * 100 + 1e-9));
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n * total^2 / 64)。
- 空间复杂度：O(total^2 / 64)。
