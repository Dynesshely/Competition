# P1430 序列取数 题解

## 题目链接
- [洛谷 P1430](https://www.luogu.com.cn/problem/P1430)

## 题目大意
给定一个长为 $n$ 的整数序列，A 和 B 轮流取数（A 先手）。每个人可以从序列的左端或右端取若干个数（至少一个），但不能两端都取。所有数取完后，统计各自取数总和作为得分。两人都足够聪明，求 A 的最终得分。$T \le 100$ 组数据，$n \le 1000$。

## 解题思路
这是一个博弈 DP 问题。定义 $dp[l][r]$ 表示在区间 $[l,r]$ 上先手能获得的最大分数减去后手分数的值（即净胜分），但直接转移复杂度为 $O(n^3)$。

优化思路：观察到转移方程为
$$dp[l][r] = \max\left( \sum_{i=l}^k a_i - dp[k+1][r],\; \sum_{i=k}^r a_i - dp[l][k-1] \right)$$
通过维护 $L[l][r] = \min(dp[l][r], 0)$ 和 $R[l][r] = \min(dp[l][r], 0)$ 辅助数组，可以在 $O(n^2)$ 内完成转移。

具体实现中，对于每个右端点 $r$，从左到左扫描左端点 $l$，动态维护 $L$（左侧的 dp 最小值）和 $R$（右侧的 dp 最小值），使得每个区间 $[l,r]$ 的 dp 值可在 $O(1)$ 内计算。

最终 A 的得分等于 $(总分数 + dp[1][n]) / 2$，但代码中直接输出 dp 值作为先手得分，通过交替取数的净胜分间接计算。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int                T;
vector<int>        ns;
vector<vector<ll>> arrays;
vector<ll>         answers;

void input() {
    scanf("%d", &T);
    ns.resize(T);
    arrays.resize(T);
    for (int i = 0; i < T; ++i) {
        int n;
        scanf("%d", &n);
        ns[i] = n;
        arrays[i].resize(n);
        for (int j = 0; j < n; ++j)
            scanf("%lld", &arrays[i][j]);
    }
}

void process() {
    answers.resize(T);
    for (int t = 0; t < T; ++t) {
        int        n = ns[t];
        vector<ll> a(n + 1, 0);
        vector<ll> pref(n + 1, 0);
        for (int i = 1; i <= n; ++i) {
            a[i]    = arrays[t][i - 1];
            pref[i] = pref[i - 1] + a[i];
        }
        vector<ll> R(n + 2, 0);
        for (int r = 1; r <= n; ++r) {
            ll L = 0;
            for (int l = r; l >= 1; --l) {
                ll dpVal;
                if (l == r)
                    dpVal = a[l];
                else
                    dpVal = (pref[r] - pref[l - 1]) - min(L, R[l]);
                if (dpVal < L)
                    L = dpVal;
                if (dpVal < R[l])
                    R[l] = dpVal;
                if (l == 1 && r == n)
                    answers[t] = dpVal;
            }
        }
    }
}

void output() {
    for (int i = 0; i < T; ++i)
        printf("%lld\n", answers[i]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(T \cdot n^2)$，对于每组数据 $n \le 1000$ 可以接受。
- 空间复杂度：$O(n)$ 用于存储辅助数组。
