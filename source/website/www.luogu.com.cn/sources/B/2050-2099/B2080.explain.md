# B2080 计算多项式的值 题解
## 题目链接
- [洛谷 B2080](https://www.luogu.com.cn/problem/B2080)
## 题目大意
给定浮点数 $x$ 和正整数 $n$，计算多项式 $x^n + x^{n-1} + \cdots + x + 1$ 的值，结果保留两位小数。
## 解题思路
多项式共 $n+1$ 项，从常数项 $1$ 开始，每次乘以 $x$ 得到下一项，累加即可。注意 $x \neq 1$ 且 $n \le 10^6$，直接循环不会超时。
## 代码实现
- 用 `cur` 记录当前项，初始为 $1$，每轮 `cur *= x` 并累加到 `ans`。
- 使用 `double` 类型存储，输出格式为 `%.2lf`。
- 采用三段式结构：`input()`, `process()`, `output()`。
## 复杂度分析
- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$
