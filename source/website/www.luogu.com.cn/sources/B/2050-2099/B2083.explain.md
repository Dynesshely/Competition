# B2083 画矩形 题解

## 题目链接
- [洛谷 B2083](https://www.luogu.com.cn/problem/B2083)

## 题目大意
给定高 `a`、宽 `b`、填充字符 `c` 及空心/实心标志 `f`，输出对应矩形。

## 解题思路
双层循环逐行逐列输出。实心时全部填字符 `c`；空心时仅第一行、最后一行、第一列、最后一列输出 `c`，其余位置输出空格。

## 代码实现
- 利用 `putchar` 单字符输出，避免字符串拼接开销。
- 空心判断条件 `i == 0 || i == a-1 || j == 0 || j == b-1` 简洁覆盖边界。

## 复杂度分析
- 时间复杂度：`O(a * b)`，遍历整个矩形区域。
- 空间复杂度：`O(1)`，仅使用常量个变量。
