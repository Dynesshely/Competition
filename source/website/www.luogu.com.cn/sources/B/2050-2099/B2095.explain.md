# B2095 白细胞计数 题解
## 题目链接
- [洛谷 B2095](https://www.luogu.com.cn/problem/B2095)
## 题目大意
给定 n 个浮点数样本，去掉一个最大值和一个最小值后，计算剩余 n-2 个样本的平均值以及所有有效样本与该平均值之差的绝对值的最大值。
## 解题思路
先遍历一遍数组找到最大值和最小值的下标，再遍历剩余元素求和并计算平均值，最后遍历有效样本更新最大绝对偏差。三步遍历完成后按要求保留两位小数输出。
## 代码实现
使用 `double` 存储浮点数，分别记录最值下标而非直接剔除最值，避免了排序修改原顺序。用 `fabs` 计算绝对值，`%.2lf` 格式化输出。
## 复杂度分析
- 时间复杂度 O(n)
- 空间复杂度 O(n)
