# B2075 幂的末尾 题解
## 题目链接
- [洛谷 B2075](https://www.luogu.com.cn/problem/B2075)
## 题目大意
给定正整数 a 和 b，求幂 a^b 的末三位数字，不足三位高位补零输出。
## 解题思路
末三位数字等价于 a^b mod 1000。由于 b ≤ 10000，直接循环 b 次累乘取模即可，无需快速幂。输出时使用 `%03d` 保证高位补零。
## 代码实现
- 循环内每次 `res = res * a % 1000` 保持结果只保留末三位。
- 通过 `printf("%03d\n", res)` 输出，自动处理前导零。
## 复杂度分析
- 时间复杂度：O(b)
- 空间复杂度：O(1)
