## 题目描述

给出一个整数 $a$ 和一个正整数 $n$，求乘方 $a^n$。

## 输入格式

一行，包含两个整数 $a$ 和 $n$。$-1000000 \le a \le 1000000$，$1 \le n \le 10000$。

## 输出格式

一个整数，即乘方结果。题目保证最终结果的绝对值不超过 $1000000$。

## 样例 #1

### 样例输入

```
2 3
```

### 样例输出

```
8
```
