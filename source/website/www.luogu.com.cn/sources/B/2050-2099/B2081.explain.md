# B2081 与 7 无关的数 题解
## 题目链接
- [洛谷 B2081](https://www.luogu.com.cn/problem/B2081)
## 题目大意
给定正整数 $n$，求 $[1,n]$ 中所有既不能被 $7$ 整除、十进制表示中也不含数字 $7$ 的正整数的平方和。
## 解题思路
枚举 $1$ 到 $n$ 每个数，判断是否被 $7$ 整除以及是否含有数字 $7$。若均不满足，则累加其平方。数字 $7$ 的判定通过不断取末位检查实现。
## 代码实现
- `has7(x)` 循环取 `x % 10` 检查末位是否为 $7$，然后 `x /= 10` 去掉末位。
- 流程按 `input()` / `process()` / `output()` 三段式组织。
## 复杂度分析
- 时间复杂度：$O(n \log n)$，其中 $\log n$ 为数字 $7$ 的判定开销（$n<100$，实际极小）。
- 空间复杂度：$O(1)$，仅使用常数额外空间。
