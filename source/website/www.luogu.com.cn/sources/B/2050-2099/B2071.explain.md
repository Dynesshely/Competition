# B2071 余数相同问题 题解
## 题目链接
- [洛谷 B2071](https://www.luogu.com.cn/problem/B2071)
## 题目大意
给定三个正整数 a, b, c，求大于 1 的最小整数 x，使得 a, b, c 除以 x 的余数相同。
## 解题思路
若 a ≡ b ≡ c (mod x)，则 x 能整除 |a-b|、|a-c|、|b-c|，即 x 是三者最大公约数的大于 1 的最小因子。求出 d = gcd(|a-b|, |a-c|, |b-c|)，若 d < 2 则答案为 2，否则找 d 的最小质因子即可。
## 代码实现
- 计算三个差值的最大公约数，用绝对值处理。
- 若 d ≤ 1 直接输出 2（题目保证有解）。
- 试除 [2, √d] 找最小因子，未找到则 d 本身为质数答案即为 d。
## 复杂度分析
- 时间复杂度：O(√d)（最坏 √10⁶ ≈ 1000）
- 空间复杂度：O(1)
