# B2089 数组逆序重存放 题解
## 题目链接
- [洛谷 B2089](https://www.luogu.com.cn/problem/B2089)
## 题目大意
输入 n 和一串长度为 n 的整数，将这 n 个数逆序输出。
## 解题思路
使用双指针交换法，从数组两端向中间遍历，交换对应位置的元素，实现原地逆序。
## 代码实现
- 采用经典的 input / process / output 三段式结构。
- process 中循环 n/2 次，用 swap 交换首尾对称元素。
- 输出时在元素之间填充空格，末尾换行。
## 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)（不计输入存储）
