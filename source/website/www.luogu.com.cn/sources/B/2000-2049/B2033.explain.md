# B2033 A*B 问题 题解

## 题目链接
- [洛谷 B2033](https://www.luogu.com.cn/problem/B2033)

## 题目大意
输入两个正整数 A 和 B（1 <= A, B <= 50000），输出它们的乘积。

## 解题思路
A 和 B 最大均为 50000，乘积最大为 2.5e9，超出 32 位 int 的范围（约 2.1e9），因此需要使用 64 位整数类型（`long long`）存储结果。在计算时通过乘以 `1ll` 将表达式提升为 `long long` 运算，避免 int 溢出。

## 代码实现
三段式结构：`input()` 读入两个 int，`process()` 中用 `1ll * a * b` 计算乘积并赋值给 `long long` 变量，`output()` 用 `%lld` 输出。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
