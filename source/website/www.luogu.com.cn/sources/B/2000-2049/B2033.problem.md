## 题目描述

输入两个正整数 $A$ 和 $B$，求 $A \times B$ 的值。注意乘积的范围和数据类型的选择。

## 输入格式

一行，包含两个正整数 $A$ 和 $B$，中间用单个空格隔开。$1 \le A,B \le 50000$。

## 输出格式

一个整数，即 $A \times B$ 的值。

## 样例 #1

### 样例输入

```
3 4
```

### 样例输出

```
12
```

## 样例 #2

### 样例输入

```
36 18
```

### 样例输出

```
648
```
