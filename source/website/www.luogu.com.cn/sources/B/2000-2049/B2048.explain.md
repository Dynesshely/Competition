# B2048 计算邮资 题解
## 题目链接
- [洛谷 B2048](https://www.luogu.com.cn/problem/B2048)
## 题目大意
给定邮件重量和是否加急，计算总邮费：1000g以内8元，超出部分每500g加4元（不足500g按500g计），加急再加5元。
## 解题思路
基础费用为8元。对超重部分用 `(x - 1000 + 499) / 500` 向上取整算出超重档数，每档4元。最后根据字符判断是否加收5元。
## 代码实现
使用 `(x - 1000 + 499) / 500 * 4` 避免浮点运算，直接在 `process()` 中计算并输出。
## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
