# B2015 计算并联电阻的阻值 题解

## 题目链接
- [洛谷 B2015](https://www.luogu.com.cn/problem/B2015)

## 题目大意
给定两个电阻的阻值 \(r_1, r_2\)，根据并联电阻公式 \(R = \dfrac{1}{\frac{1}{r_1} + \frac{1}{r_2}}\) 计算并联后的总阻抗，结果保留两位小数。

## 解题思路
直接套用并联电阻公式。将输入的两个实数代入 \(R = 1 / (1/r_1 + 1/r_2)\) 计算，再用格式化输出保留两位小数即可。

## 代码实现
- `main()` 中直接完成输入、计算、输出三部分。
- 使用 `double` 存储实数，`scanf` / `printf` 进行读写。
- 输出格式 `%.2lf` 保留两位小数。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
