# B2046 骑车与走路 题解
## 题目链接
- [洛谷 B2046](https://www.luogu.com.cn/problem/B2046)
## 题目大意
给定一段距离，计算步行（1.2 m/s）和骑车（3.0 m/s，但需额外 50s 固定开销）各自所需时间，比较哪个更快。
## 解题思路
步行时间 = n / 1.2，骑车时间 = 27 + 23 + n / 3.0。直接比较两个浮点数即可。
## 代码实现
输入一个整数 n，用 double 计算两种方式的时间并输出 Bike / Walk / All。
## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
