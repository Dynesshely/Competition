# B2032 等差数列末项计算 题解

## 题目链接
- [洛谷 B2032](https://www.luogu.com.cn/problem/B2032)

## 题目大意
给定等差数列的前两项 a1, a2 以及项数 n，求该数列的第 n 项的值。

## 解题思路
等差数列的公差 d = a2 - a1，通项公式为 an = a1 + (n-1)*d，直接代入计算即可。

## 代码实现
- `input()` 读入 a1, a2, n。
- `process()` 计算公差 d 和末项 ans。
- `output()` 输出 ans。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
