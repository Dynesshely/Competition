## 题目描述

求两个整数的和。

## 输入格式

一行，两个用空格隔开的整数。

## 输出格式

两个整数的和。

## 样例 #1

### 样例输入

```
1 2
```

### 样例输出

```
3
```

## 样例 #2

### 样例输入

```
10230 21312
```

### 样例输出

```
31542
```

## 提示

对于 $100\%$ 的数据，输入的整数在 $[1, 4 \times {10}^{18}]$ 内。
