## 题目描述

判断一个数 $n$ 能否同时被 $3$ 和 $5$ 整除。

## 输入格式

输入一行，包含一个整数 $n$。

## 输出格式

输出一行，如果能同时被 $3$ 和 $5$ 整除输出 `YES`，否则输出 `NO`。

## 样例 #1

### 样例输入

```
15
```

### 样例输出

```
YES
```

## 提示

$-10^9<n<10^9$
