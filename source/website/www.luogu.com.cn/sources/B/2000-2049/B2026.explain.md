# B2026 计算浮点数相除的余 题解

## 题目链接
- [洛谷 B2026](https://www.luogu.com.cn/problem/B2026)

## 题目大意
给定两个双精度浮点数 a 和 b，计算 a 除以 b 的余数 r，满足 a = k * b + r，其中 k 为整数且 0 <= r < b。

## 解题思路
浮点数取模不能直接使用 C++ 的 % 运算符（仅适用于整数），因此利用 `<cmath>` 中的 `fmod` 函数直接计算浮点数余数，再按格式输出即可。

## 代码实现
使用 `input()` / `process()` / `output()` 三段式结构。核心调用 `fmod(a, b)` 返回双精度浮点余数，输出保留 4 位小数。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
