# B2036 输出绝对值 题解

## 题目链接
- [洛谷 B2036](https://www.luogu.com.cn/problem/B2036)

## 题目大意
输入一个浮点数 n，输出它的绝对值，结果保留两位小数。

## 解题思路
直接调用 C++ 标准库中的 `fabs()` 函数计算浮点数的绝对值，再按格式输出。对于入门题目无需额外逻辑。

## 代码实现
`input()` 用 `scanf` 读入 double 类型，`process()` 用 `fabs(n)` 取绝对值，`output()` 用 `printf("%.2lf")` 保留两位小数输出。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
