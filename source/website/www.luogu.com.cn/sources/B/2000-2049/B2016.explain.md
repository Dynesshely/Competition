# B2016 浮点数向零舍入 题解

## 题目链接
- [洛谷 B2016](https://www.luogu.com.cn/problem/B2016)

## 题目大意
输入一个双精度浮点数，将其向零舍入到整数（正数向下舍入，负数向上舍入）并输出。

## 解题思路
C++ 中直接将浮点数强制转换为整型会截断小数部分，恰好实现向零舍入的效果。因此直接将读入的 `double` 强制转为 `long long` 即可。

## 代码实现
三段式结构：`input()` 读入 `double` 型变量 `x`，`process()` 用 `(long long)x` 截断赋值给 `ans`，`output()` 输出 `ans`。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
