# B2044 有一门课不及格的学生 题解

## 题目链接
- [洛谷 B2044](https://www.luogu.com.cn/problem/B2044)

## 题目大意
给定一名学生的语文、数学、英语三科成绩（0~100），判断是否恰好有一科不及格（小于 60 分），是则输出 1，否则输出 0。

## 解题思路
依次判断每科成绩是否小于 60，统计不及格科目数。若计数恰好为 1 则输出 1，否则输出 0。在 C++ 中，布尔表达式 `(a < 60)` 的结果可隐式转换为整数（true 为 1，false 为 0），因此可直接累加三个条件得到不及格科目数。

## 代码实现
核心代码为 `(a < 60) + (b < 60) + (c < 60)` 统计不及格科目数，然后通过三目运算符 `cnt == 1 ? 1 : 0` 输出结果。

## 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
