# B2038 奇偶 ASCII 值判断 题解

## 题目链接
- [洛谷 B2038](https://www.luogu.com.cn/problem/B2038)

## 题目大意
输入一个字符，判断其 ASCII 码值的奇偶性，若为奇数输出 `YES`，否则输出 `NO`。

## 解题思路
字符在 C/C++ 中以整型（ASCII 码）存储，可以直接对字符变量进行模 2 运算。若 `ch % 2 == 1` 则 ASCII 值为奇数，否则为偶数。

## 代码实现
采用 `input()` / `process()` / `output()` 三段式结构。`scanf("%c", &ch)` 读入一个字符，`output()` 中直接判断 `ch % 2 == 1` 并输出对应结果。

## 复杂度分析
- 时间复杂度: O(1)
- 空间复杂度: O(1)
