# P1034 [NOIP 2002 提高组] 矩形覆盖 — 题解

## 题目大意

平面上有 $n$ 个点，需要用 $k$ 个边平行于坐标轴的矩形全部覆盖。矩形之间必须完全分开（边与顶点不可重合）。求最小的矩形面积之和。$n \le 50$，$k \le 4$。

## 解题思路

### DFS + 剪枝

将点按 $x$ 坐标排序后，依次将每个点分配到某个矩形中，追踪每个矩形当前的边界框。

**搜索过程**：
```
dfs(idx, rects, used):
    if idx == n: 更新最优答案
    if 当前面积和 ≥ best: 剪枝
    对于点 pts[idx]:
        尝试加入已有的 used 个矩形中的某一个
            若加入后不与任何其他矩形重叠，递归
        若 used < k: 创建新矩形包含该点
```

### 矩形维护

每个矩形维护其边界 $[x_1, x_2] \times [y_1, y_2]$。加入新点时扩展边界框。两个矩形重叠当且仅当在 $x$ 和 $y$ 方向都有交集（不含仅边/顶点接触）。

面积计算：若矩形覆盖了至少一个点，面积为 $(x_2 - x_1) \times (y_2 - y_1)$；空矩形面积为 $0$。覆盖单个点或共线点的矩形面积为 $0$（符合题意）。

### 时间复杂度

- 理论上搜索空间可达 $k^n$，但面积剪枝（一旦超过已知最优解即停止）在实践中非常有效
- $n \le 50$，$k \le 4$，题目数据范围内可过

### 空间复杂度

- $O(k)$ 用于存储矩形状态，加上递归栈
