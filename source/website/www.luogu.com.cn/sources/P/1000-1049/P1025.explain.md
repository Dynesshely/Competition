# P1025 [NOIP 2001 提高组] 数的划分 — 题解

## 题目大意

将整数 $n$ 分成 $k$ 份（每份不为空），顺序无关（即 $1+1+5$ 与 $5+1+1$ 视为同一种）。求不同分法的数量。$6 < n \le 200$，$2 \le k \le 6$。

## 解题思路

### 动态规划

设 $dp[i][j]$ 表示将 $i$ 分成 $j$ 份的方案数。

**状态转移**（经典整数划分 DP）：

$$dp[i][j] = dp[i-1][j-1] + dp[i-j][j]$$

**含义**：
- $dp[i-1][j-1]$：从 $i$ 中分出一个 $1$ 作为单独的一份，剩下的 $i-1$ 分成 $j-1$ 份
- $dp[i-j][j]$：先给每份放一个 $1$（保证每份非空），再把剩下的 $i-j$ 分成 $j$ 份

**初始状态**：$dp[0][0] = 1$。

### 正确性说明

这种方式天然保证了"顺序无关"，因为 DP 始终按非递减方式生成划分：`dp[i-1][j-1]` 的操作等价于追加一个值为 $1$ 的份（最小可能的份值），`dp[i-j][j]` 的操作等价于给每个已有份加 $1$（保持相对大小不变）。

### 时间复杂度

- $O(nk)$，$n \le 200$，$k \le 6$

### 空间复杂度

- $O(nk)$
