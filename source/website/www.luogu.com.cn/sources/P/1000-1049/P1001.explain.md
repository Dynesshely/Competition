# P1001 A+B Problem 题解

## 题目链接

- [洛谷 P1001](https://www.luogu.com.cn/problem/P1001)

## 题目大意

输入两个整数 $a, b$，输出它们的和。

## 解题思路

这是一道经典的入门题目，旨在让选手熟悉输入输出操作。题目要求直接计算两个整数的和并输出，没有任何额外的算法难度。数据范围在 $[-10^9, 10^9]$ 之间，使用 `int` 类型即可满足。

## 代码实现

代码直接使用 C++ 标准库：

- `cin >> a >> b` 从标准输入读取两个整数。
- `cout << a + b << endl` 输出它们的和并换行。
- 使用 `bits/stdc++.h` 头文件包含所有标准库，适合竞赛环境使用。

## 复杂度分析

- 时间复杂度：$O(1)$，仅执行一次加法和一次输出。
- 空间复杂度：$O(1)$，仅使用两个整型变量。
