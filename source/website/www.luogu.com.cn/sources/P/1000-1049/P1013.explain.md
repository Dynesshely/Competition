# P1013 [NOIP 1998 提高组] 进制位 题解

## 题目链接

- [洛谷 P1013](https://www.luogu.com.cn/problem/P1013)

## 题目大意

给定一张 $n \times n$ 的加法表，表中字母代表某进制下的数字。表格给出任意两个单数字之和的表示（可能为一位或两位）。要求推导出每个字母对应的数字以及该进制的基数。若无解则输出 `ERROR!`。

## 解题思路

**暴力枚举**。$n \le 9$，字母数 $m = n-1 \le 8$。枚举 $m!$ 种字母到数字的映射（数字取 $0 \sim m-1$），对每种映射检查加法表是否一致，同时确定进制。

检查规则：
- **单位数结果**：$val(a) + val(b) = val(\text{结果})$ 且 $val(\text{结果}) < base$。
- **两位数结果** "XY"：进位符 $X$ 必须对应数字 $1$，且 $val(a) + val(b) = base + val(Y)$。
- 所有两位数结果必须导出**相同的** $base$。

由于题目保证至多一组解，一旦找到即可输出。

## 代码实现

- 用 `next_permutation` 枚举所有排列。
- 对每种排列，先验证 0 和 1 的存在性，再扫描全表同时确定 base 和验证一致性。
- 最后再验证所有单数字结果不越界（$< base$）。

## 复杂度分析

- 时间复杂度：$O(m! \times n^2)$，$m \le 8$ 时 $8! \times 81 \approx 3.3 \times 10^7$。
- 空间复杂度：$O(n^2)$。
