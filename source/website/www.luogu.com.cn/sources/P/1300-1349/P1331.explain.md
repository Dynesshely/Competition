# P1331 洛谷 海战 题解

## 题目链接
- [洛谷 P1331](https://www.luogu.com.cn/problem/P1331)

## 题目大意
给定一个 $R \times C$ 的棋盘，`#` 表示船只的一部分，`.` 表示水。每艘船必须是矩形（方形）且船与船之间不能相邻（即上下或左右相邻的 `#` 属于同一艘船才算合法）。判断船只的摆放是否合法，若合法则输出船只数量，否则输出 `Bad placement.`。

## 解题思路
1. 遍历棋盘，对每一个未访问的 `#` 进行 BFS/DFS 搜索，标记出整个连通分量。
2. 在搜索过程中记录该连通分量的边界（最小行、最大行、最小列、最大列）。
3. 在边界矩形区域内检查是否所有格子都是 `#`，若存在 `.` 则说明该船只不是矩形（可能是 L 形等），非法。
4. 若所有连通分量均为矩形，则输出船只总数；否则输出 `Bad placement.`。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
char      grid[MAXN][MAXN];
bool      via[MAXN][MAXN];
int       r, c, ships;
int       dx[] = {0, 0, 1, -1};
int       dy[] = {1, -1, 0, 0};

void input() {
    scanf("%d%d", &r, &c);
    for (int i = 1; i <= r; i++)
        scanf("%s", grid[i] + 1);
}

bool isRect(int sr, int sc) {
    int                   minR = sr, maxR = sr, minC = sc, maxC = sc;
    queue<pair<int, int>> q;
    q.push({sr, sc});
    via[sr][sc] = true;
    while (!q.empty()) {
        int x = q.front().first, y = q.front().second;
        q.pop();
        minR = min(minR, x);
        maxR = max(maxR, x);
        minC = min(minC, y);
        maxC = max(maxC, y);
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i], ny = y + dy[i];
            if (nx >= 1 && nx <= r && ny >= 1 && ny <= c && !via[nx][ny] && grid[nx][ny] == '#') {
                via[nx][ny] = true;
                q.push({nx, ny});
            }
        }
    }
    for (int i = minR; i <= maxR; i++)
        for (int j = minC; j <= maxC; j++)
            if (grid[i][j] != '#')
                return false;
    return true;
}

void process() {
    ships = 0;
    memset(via, 0, sizeof(via));
    for (int i = 1; i <= r; i++)
        for (int j = 1; j <= c; j++)
            if (grid[i][j] == '#' && !via[i][j]) {
                if (!isRect(i, j)) {
                    ships = -1;
                    return;
                }
                ships++;
            }
}

void output() {
    if (ships == -1)
        printf("Bad placement.\n");
    else
        printf("There are %d ships.\n", ships);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(R \times C)$，每个格子至多被访问一次。
- 空间复杂度：$O(R \times C)$，用于存储棋盘和访问标记。
