## 题目描述

输入一串二叉树，输出其前序遍历。

## 输入格式

第一行为二叉树的节点数 $n$。($1 \leq n \leq 26$)

后面 $n$ 行，第一个字母为节点，后两个字母分别为其左右儿子。特别地，数据保证第一行读入的节点必为根节点。

空节点用 `*` 表示。

## 输出格式

二叉树的前序遍历。

## 样例 #1

### 样例输入

```
6
abc
bdi
cj*
d**
i**
j**
```

### 样例输出

```
abdicj
```
