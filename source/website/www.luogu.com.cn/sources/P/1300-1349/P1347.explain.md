# P1347 [ECNA 2001] 排序 题解

## 题目链接
- [洛谷 P1347](https://www.luogu.com.cn/problem/P1347)

## 题目大意
给定 n 个不同的大写字母（A 到第 n 个字母）和 m 个形如 `A<B` 的关系，要求依次考虑每个关系，判断：
- 若根据前 x 个关系能确定所有元素的全序，则输出 `Sorted sequence determined after x relations: yyy...y.`
- 若根据前 x 个关系发现存在矛盾（环），则输出 `Inconsistency found after x relations.`
- 若处理完所有关系仍无法确定全序，则输出 `Sorted sequence cannot be determined.`

注意：确定顺序后即可结束，不必考虑后续出现矛盾的情况。

## 解题思路
逐个添加关系，每次添加后对新图执行拓扑排序。三种情况对应：
1. **矛盾（环）**：如果拓扑排序得到的序列长度小于当前图中结点数，说明存在环。
2. **全序确定**：如果拓扑排序序列长度等于 n，且每步队列中始终只有一个元素（唯一确定），则全序已确定。
3. **无法确定**：处理完所有关系后仍无法满足以上两种情况。

使用邻接表建图，inDeg 数组记录入度。每次添加一条关系后重新拓扑排序，检查上述条件。注意去除重复关系。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, m;
vector<int> adj[26];
int         inDeg[26];
bool        edge[26][26];

int main() {
    scanf("%d%d", &n, &m);
    memset(inDeg, 0, sizeof(inDeg));
    memset(edge, 0, sizeof(edge));

    for (int i = 1; i <= m; i++) {
        char buf[4];
        scanf("%s", buf);
        int u = buf[0] - 'A';
        int v = buf[2] - 'A';

        if (!edge[u][v]) {
            edge[u][v] = true;
            adj[u].push_back(v);
            inDeg[v]++;
        }

        int tmpInDeg[26];
        memcpy(tmpInDeg, inDeg, sizeof(inDeg));

        queue<int> q;
        for (int j = 0; j < n; j++)
            if (tmpInDeg[j] == 0)
                q.push(j);

        vector<int> order;
        bool        unique = true;

        while (!q.empty()) {
            if (q.size() > 1)
                unique = false;
            int cur = q.front();
            q.pop();
            order.push_back(cur);
            for (int nxt : adj[cur])
                if (--tmpInDeg[nxt] == 0)
                    q.push(nxt);
        }

        if ((int)order.size() < n) {
            printf("Inconsistency found after %d relations.\n", i);
            return 0;
        }

        if (unique) {
            printf("Sorted sequence determined after %d relations: ", i);
            for (int v : order)
                putchar('A' + v);
            printf(".\n");
            return 0;
        }
    }

    printf("Sorted sequence cannot be determined.\n");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(mn + m^2)，每次拓扑排序 O(n + 边数)，最坏边数 O(m)
- 空间复杂度：O(n^2)
