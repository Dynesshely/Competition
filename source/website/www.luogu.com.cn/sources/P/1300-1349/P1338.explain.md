# P1338 [JSOI2008] 末日的传说 题解

## 题目链接
- [洛谷 P1338](https://www.luogu.com.cn/problem/P1338)

## 题目大意
给定 $n$ 和 $m$，构造一个 $1$ 到 $n$ 的排列，使得该排列的逆序对数量恰好为 $m$，并且在所有满足条件的排列中字典序最小。

## 解题思路
本题要求构造字典序最小且逆序对数为 $m$ 的排列。核心思路是从左到右逐位确定每个数字，贪心地选择尽可能小的数，同时保证后面能凑出恰好 $m$ 个逆序对。

关键观察：对于一个长度为 $r$ 的序列，最多能产生 $\frac{r(r-1)}{2}$ 个逆序对（即完全逆序时）。

从左到右确定第 $p$ 位时，设剩余数字个数为 $r = n-p+1$。剩余数字的最大逆序对数为 $maxInv = \frac{(r-1)(r-2)}{2}$（当前位取最小值时，后面 $r-1$ 个数字能产生的最大逆序对数）。

- 如果 $m > maxInv$，说明取当前最小值不够，需要将第 $p$ 位设为某个较大的数 $p+k$，其中 $k = m - maxInv$。这样当前位贡献了 $k$ 个逆序对（与后面比它小的数），剩下的 $m - k$ 个逆序对由后面 $r-1$ 个数完全逆序产生（即为 $maxInv$）。然后输出剩余数即可结束。
- 如果 $m \le maxInv$，则第 $p$ 位取当前最小值，并继续处理下一位。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;

void input() { scanf("%d%d", &n, &m); }

void process() {
    int p = 1;
    while (true) {
        int       r      = n - p + 1;
        long long maxInv = 1LL * (r - 1) * (r - 2) / 2;
        if (m > maxInv) {
            int k = m - maxInv;
            printf("%d ", p + k);
            for (int x = n; x > p + k; x--)
                printf("%d ", x);
            for (int x = p + k - 1; x >= p; x--)
                printf("%d%c", x, x == p ? '\n' : ' ');
            return;
        }
        printf("%d%c", p, p == n ? '\n' : ' ');
        p++;
        if (p > n)
            return;
    }
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$，每个位置最多一次判定即可确定。
- 空间复杂度：$O(1)$，仅使用常数个变量。
