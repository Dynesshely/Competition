# P1348 Couple number 题解

## 题目链接
- [洛谷 P1348](https://www.luogu.com.cn/problem/P1348)

## 题目大意
定义一个整数 n 为 Couple number 当且仅当存在两个自然数 x, y 使得 n = x^2 - y^2。给定区间 [a, b]，求该区间内 Couple number 的个数。

## 解题思路
n = x^2 - y^2 = (x - y)(x + y)。设 u = x - y，v = x + y，则 u 和 v 同奇偶（因为 x = (v+u)/2, y = (v-u)/2 需为自然数，故 u, v 同奇偶），且 u <= v，u, v 同号。反过来，对于任意满足 u, v 同奇偶且 u >= 0, v >= u 的整数对，都能构造出 x, y。

n 能表示为两自然数平方差，当且仅当：
- n 是奇数，或
- n 能被 4 整除

因为：
- 若 n 为奇数，可令 u = 1, v = n，两者同奇偶（1 和 n 均为奇数），能构造。
- 若 n 为 4 的倍数，可令 u = 2, v = n/2，两者均为偶数，能构造。
- 若 n 模 4 余 2，则 n = 2k，任何分解 n = u * v 中 u, v 必然一奇一偶，无法同奇偶，故不可构造。

因此，问题转化为统计 [a, b] 中奇数和 4 的倍数的个数。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int a, b;

void input() { scanf("%d%d", &a, &b); }

int process() {
    int firstOdd = a;
    if (firstOdd % 2 == 0)
        firstOdd++;
    int lastOdd = b;
    if (lastOdd % 2 == 0)
        lastOdd--;
    int oddCnt = 0;
    if (firstOdd <= b)
        oddCnt = (lastOdd - firstOdd) / 2 + 1;

    int r         = ((a % 4) + 4) % 4;
    int firstFour = a;
    if (r != 0)
        firstFour += (4 - r);
    int fourCnt = 0;
    if (firstFour <= b) {
        int lastFour = b - ((b % 4 + 4) % 4);
        fourCnt      = (lastFour - firstFour) / 4 + 1;
    }

    return oddCnt + fourCnt;
}

void output() { printf("%d\n", process()); }

int main() {
    input();
    output();
    return 0;
}
```

- 空间复杂度：O(1)
