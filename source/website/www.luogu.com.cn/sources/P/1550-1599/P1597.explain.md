# P1597 洛谷 语句解析 题解

## 题目链接

- [洛谷 P1597](https://www.luogu.com.cn/problem/P1597)

## 题目大意

给定一段长度不超过 $255$ 的 PASCAL 语言代码，只包含对变量 $a, b, c$ 的赋值语句。每条语句格式为 `[变量]:=[变量或一位整数];`。未赋值的变量值为 $0$，输出最终 $a, b, c$ 的值。

## 解题思路

由于代码格式固定，可以直接逐字符解析。遍历字符串，当遇到字母 `a`、`b` 或 `c` 时，说明这是一个赋值语句的左值。向后跳过 `:=` 三个字符，然后读取右值：如果是一个数字字符，直接转换为数值；如果是一个变量字母，则取该变量当前的值。将值赋给左值对应的变量。最终输出三个变量的值。

## 代码实现

- 用 `val[0]`、`val[1]`、`val[2]` 分别表示 $a, b, c$ 的值。
- 遍历字符串，遇到 `a`~`c` 时用 `t = s[i] - 'a'` 得到左值索引，`i += 3` 跳过 `:=`。
- 取 `s[i]` 判断是数字（`'0' ~ '9'`）还是变量（`'a' ~ 'c'`），相应赋值。

## 复杂度分析

- 时间复杂度：$O(L)$，其中 $L$ 为字符串长度。
- 空间复杂度：$O(L)$
