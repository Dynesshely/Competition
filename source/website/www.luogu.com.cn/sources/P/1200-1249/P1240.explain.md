# P1240 诸侯安置 题解

## 题目链接

- [洛谷 P1240](https://www.luogu.com.cn/problem/P1240)

## 题目大意

在一个菱形网格上放置 $k$ 个诸侯，要求任意两个不在同一行、同一列、同一对角线。求方案数模 504。

## 解题思路

将菱形转换为阶梯形：$2n-1$ 行，奇数行有 $i$ 个有效位置（$i$ 为行号），偶数行有 $i-1$ 个。问题转化为在阶梯形上选 $k$ 个互不攻击位置的方案数。

用 DP：$f[i][j]$ 表示前 $i$ 行放置 $j$ 个的方案数。转移：
$$f[i][j] = f[i-1][j] + f[i-1][j-1] \times (len[i] - j + 1)$$

即第 $i$ 行不放 $+$ 第 $i$ 行放一个（有 $len[i]-(j-1)$ 个可选位置）。

## 代码实现

- 预处理 `len[i]`：奇数 $i$ 为 $i$，偶数 $i$ 为 $i-1$。
- DP 初值 $f[i][0] = 1$，递推至 $f[2n-1][k]$。

## 复杂度分析

- 时间复杂度：$O(NK)$。
- 空间复杂度：$O(NK)$。
