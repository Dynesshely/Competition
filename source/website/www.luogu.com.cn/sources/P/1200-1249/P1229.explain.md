# P1229 遍历问题 题解

## 题目链接

- [洛谷 P1229](https://www.luogu.com.cn/problem/P1229)

## 题目大意

给定二叉树的前序遍历和后序遍历，求可能的中序遍历有多少种。

## 解题思路

前序：根 → 左 → 右；后序：左 → 右 → 根。前序第二个字符是左子树的根。在后序中找到它，确定左子树大小。递归处理左右子树。

关键：当左子树或右子树为空时，有两种可能（缺少左孩子和缺少右孩子在此处无法区分）。因此每当出现仅有一个子树的情况，答案乘 2。

## 代码实现

- `dfs(l1, r1, l2, r2)`：递归处理前序 $[l1,r1]$ 和后序 $[l2,r2]$。
- 取 `leftRoot = pre[l1+1]`，在后序中定位。
- 若左/右子树为空，结果乘 2。

## 复杂度分析

- 时间复杂度：$O(N^2)$（每次在后序中搜索），$N \le 100$。
- 空间复杂度：$O(N)$ 递归栈。
