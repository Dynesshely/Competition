# P3086 [USACO13OPEN] Figure Eight G 题解

## 题目链接
- [洛谷 P3086](https://www.luogu.com.cn/problem/P3086)

## 题目大意
在 $N \times N$（$5 \le N \le 300$）的网格中，`.` 表示完美区域，`*` 表示瑕疵。定义数字"8"由两个矩形上下拼接而成：
- 上矩形和下矩形的内部至少包含一个格子（宽高均至少为 $3$）。
- 上矩形的底边是下矩形顶边的一个子集（即两矩形共享一行，上矩形不宽于下矩形且水平居中在其范围内）。
- 两个矩形的**边框**必须全部由 `.` 组成（内部可以包含 `*`）。
- 美学分数 = 上矩形围成面积 $\times$ 下矩形围成面积（面积以矩形内部格数计算，即宽 $\times$ 高，不含边框）。

求能刻出的最大美学分数，若无法刻出则输出 $-1$。

## 解题思路

### 预处理
定义两个辅助数组：
- `up[i][j]`：从 $(i,j)$ 向上连续的 `.` 个数（包含自身）。
- `lef[i][j]`：从 $(i,j)$ 向左连续的 `.` 个数（包含自身）。

这两个数组可以 $O(N^2)$ 递推得到。

### 问题拆分
"8"由两个共享同一行（记为 $h$）的矩形构成。因此，对于每个可能的分割行 $h$ 和列区间 $[l, r]$，可以分别考虑以上矩形和下矩形的最大可能面积。

### 上矩形 DP
记 $dp[h][l][r]$ 表示以行 $h$ 为底边、列区间 $[l, r]$ 为上底边的上矩形的最大围成面积。

对于固定的 $(l, r)$，从底行 $h$ 向上寻找最高的可行顶行 $t$，满足：
- $(t, l)$ 到 $(h, l)$ 的左边框全为 `.`，即 $up[h][l] \ge h - t + 1$。
- $(t, r)$ 到 $(h, r)$ 的右边框全为 `.`，即 $up[h][r] \ge h - t + 1$。
- $(l, t)$ 到 $(r, t)$ 的上边框全为 `.`，即 $lef[t][r] \ge r - l + 1$。
- $(l, h)$ 到 $(r, h)$ 的下边框全为 `.`，即 $lef[h][r] \ge r - l + 1$。

顶行 $t$ 从底部向上搜索，利用 `up` 和 `lef` 可以快速检查可行性。面积计算为内部面积：
$$
dp[h][l][r] = (r - l - 1) \times (h - t - 1)
$$

### 水平方向扩展优化
在同一个底行 $h$ 上，较宽的矩形可能包含较窄的有效矩形。为简化后续计算，对每行的 $dp$ 值沿水平方向取前缀/后缀最大值：
$$
dp[h][l][r] = \max(dp[h][l][r],\; dp[h][l][r-1]) \\
dp[h][l][r] = \max(dp[h][l][r],\; dp[h][l+1][r])
$$

这样 $dp[h][l][r]$ 表示列区间 $[l, r]$ 内能容纳的最大上矩形面积。

### 下矩形与面积乘积
对于每个列区间 $[l, r]$，从最底行 $n$ 向上扫描行 $h$，寻找下矩形最远的底行 $t$（下矩形的下边框），使得下半部分也能构成一个矩形。

对每个合法的 $(h, l, r)$，下矩形面积 = $(r - l - 1) \times (t - h - 1)$，总分数为：
$$
ans = \max(ans,\; (r - l - 1) \times (t - h - 1) \times dp[h][l][r])
$$

其中 $dp[h][l][r]$ 是以 $h$ 为上矩形底边的最大上矩形面积。

### 边界处理
若最终 $ans$ 为 $0$（即无法构成任何"8"），则输出 $-1$。注意所有矩形的宽高至少为 $3$，因此 $r - l - 1 \ge 2$（即 $r - l \ge 3$），高度类似。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 305;

int  n, up[N][N], lef[N][N], ans, dp[N][N][N];
char grid[N][N];

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) {
        scanf("%s", grid[i] + 1);
        for (int j = 1; j <= n; ++j)
            if (grid[i][j] == '.')
                up[i][j] = up[i - 1][j] + 1, lef[i][j] = lef[i][j - 1] + 1;
    }
    for (int l = 1; l <= n; ++l)
        for (int r = l + 2; r <= n; ++r) {
            int t = 1;
            for (int h = 1; h <= n; ++h) {
                if (l < r - lef[h][r] + 1)
                    continue;
                while ((t < h - min(up[h][l], up[h][r]) + 1 || l <= r - lef[t][r] + 1) && t + 2 <= h)
                    ++t;
                if (t + 2 <= h)
                    dp[h][l][r] = (r - l - 1) * (h - t - 1);
            }
        }
    for (int i = 1; i <= n; ++i)
        for (int len = 4; len <= n; ++len) {
            for (int l = 1; l + len - 1 <= n; ++l)
                dp[i][l][l + len - 1] = max(dp[i][l][l + len - 1], dp[i][l][l + len - 2]);
            for (int r = n; r - len + 1 >= 1; --r)
                dp[i][r - len + 1][r] = max(dp[i][r - len + 1][r], dp[i][r - len + 2][r]);
        }
    for (int l = 1; l <= n; ++l)
        for (int r = l + 2; r <= n; ++r) {
            int t = n;
            for (int h = n; h >= 1; --h) {
                if (l < r - lef[h][r] + 1)
                    continue;
                while ((h < t - min(up[t][l], up[t][r]) + 1 || l <= r - lef[t][r] + 1) && h + 2 <= t)
                    --t;
                if (h + 2 <= t)
                    ans = max((r - l - 1) * (t - h - 1) * dp[h][l][r], ans);
            }
        }
    if (ans)
        printf("%d", ans);
    else
        puts("-1");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(N^3)$，其中 $N$ 为网格边长。三重循环枚举列区间 $(l, r)$ 和行 $h$，每层内部 while 循环摊还 $O(1)$。
- 空间复杂度：$O(N^3)$，使用三维数组 $dp$ 记录每个 $(h, l, r)$ 的上矩形最大面积。
