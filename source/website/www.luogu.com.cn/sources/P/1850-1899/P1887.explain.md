# P1887 乘积最大3 题解

## 题目链接

- [洛谷 P1887](https://www.luogu.com.cn/problem/P1887)

## 题目大意

求 M 个和为 N 的正整数，使它们的乘积最大，并输出字典序最小的一种方案。

## 解题思路

根据均值不等式，当和固定时，数之间越接近乘积越大。因此将 N 尽可能平均分给 M 个数。

计算 base = N / M（下取整），剩余 rem = N % M。将 rem 个 1 分配给 rem 个数使其变为 base+1。

为了得到字典序最小的方案，将较小的数放在前面。所以先输出 (M - rem) 个 base，再输出 rem 个 base+1。

## 代码实现

- 读入 N, M，计算 base 和 rem。
- 循环输出 M - rem 个 base，再输出 rem 个 base+1。

## 复杂度分析

- 时间复杂度：O(M)
- 空间复杂度：O(1)
