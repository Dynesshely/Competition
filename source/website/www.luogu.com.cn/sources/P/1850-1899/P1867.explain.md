# P1867 【Mc生存】经验值 题解

## 题目链接

- [洛谷 P1867](https://www.luogu.com.cn/problem/P1867)

## 题目大意

clearman 进行了 n 项操作，初始生命值 10 点，上限 10 点。每次操作付出 x 点生命值（x 可为负，即回血），获得 a 点经验值。若生命值 <= 0 则死亡，后续操作无效。经验升级规则：从 0 级开始，升到 m 级需要累积 2^0 + 2^1 + ... + 2^(m-1) 点经验。求最终等级和剩余经验值。

## 解题思路

模拟操作流程：按顺序执行每项操作，先扣血再判断是否死亡。注意生命值上限为 10，回血不能超过 10。

经验值累加后，通过不断减去当前等级所需的升级经验（2^level）来计算最终等级和剩余经验。

## 代码实现

- hp 初始 10.0（用 double 因为 x 是浮点数），exp 初始 0。
- 每次操作处理：hp -= x，若 hp <= 0 则 break；若 hp > 10 则设为 10；exp += (int)a。
- 计算等级：need 从 1 开始，每次循环 need 左移一位（即乘 2），level 递增，exp 递减。

## 复杂度分析

- 时间复杂度：O(n + level)
- 空间复杂度：O(1)
