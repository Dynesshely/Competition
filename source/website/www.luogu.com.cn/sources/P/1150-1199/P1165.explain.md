# P1165 日志分析 题解

## 题目链接
- [洛谷 P1165](https://www.luogu.com.cn/problem/P1165)

## 题目大意
模拟一个栈的操作，支持三种操作：
1. `0 X`：将重量为 X 的集装箱入栈。
2. `1`：出栈（栈顶元素出栈，若为空则忽略）。
3. `2`：查询当前栈中最大元素重量（若为空则输出 0）。

## 解题思路
使用一个辅助栈来维护当前最大值。每个元素入栈时，栈中同时保存该元素值和当前栈中最大值。

具体地：
- 入栈 `0 X`：若栈空则 push {X, X}，否则 push {X, max(X, top().second)}。
- 出栈 `1`：若非空则 pop。
- 查询 `2`：若非空则输出 top().second，否则输出 0。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int                   n;
stack<pair<int, int>> s;

void input() { scanf("%d", &n); }

void process() {
    for (int i = 0; i < n; i++) {
        int op;
        scanf("%d", &op);
        if (op == 0) {
            int x;
            scanf("%d", &x);
            if (s.empty())
                s.push({x, x});
            else
                s.push({x, max(x, s.top().second)});
        } else if (op == 1) {
            if (!s.empty())
                s.pop();
        } else {
            if (s.empty())
                printf("0\n");
            else
                printf("%d\n", s.top().second);
        }
    }
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N)，每个操作 O(1)。
- 空间复杂度：O(N)。
