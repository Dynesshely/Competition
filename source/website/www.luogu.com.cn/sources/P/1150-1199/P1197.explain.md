# P1197 [JSOI2008] 星球大战 题解

## 题目链接
- [洛谷 P1197](https://www.luogu.com.cn/problem/P1197)

## 题目大意
给定一个 n 个点 m 条边的无向图。依次摧毁 k 个星球（即删除节点及其所有邻边）。每次摧毁后，求剩余星球形成的连通块个数。需要输出初始连通块个数及每次摧毁后的个数。

## 解题思路
逆向处理 + 并查集。

- 正向删除难以维护，改为逆向加点：先标记所有被摧毁的星球，在初始图中只保留未被摧毁的节点，计算连通块数。
- 然后按摧毁顺序的逆序逐个恢复星球（并激活其邻边），每恢复一个节点，连通块数先加一，再减去该节点连接不同连通分量时合并的次数。
- 记录每次恢复后的连通块数，最后正向输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 400010;
const int MAXM = 200010;

int  n, m, k;
int  head[MAXN], nxt[MAXM * 2], to[MAXM * 2], tot;
int  fa[MAXN], sz[MAXN];
bool broken[MAXN];
int  attack[MAXN];
int  ans[MAXN];

void addEdge(int u, int v) {
    nxt[++tot] = head[u];
    to[tot]    = v;
    head[u]    = tot;
}

int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }

bool unite(int x, int y) {
    x = find(x), y = find(y);
    if (x == y)
        return false;
    if (sz[x] < sz[y])
        swap(x, y);
    fa[y] = x;
    sz[x] += sz[y];
    return true;
}

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= m; i++) {
        int u, v;
        scanf("%d%d", &u, &v);
        addEdge(u, v);
        addEdge(v, u);
    }
    scanf("%d", &k);
    for (int i = 1; i <= k; i++) {
        scanf("%d", &attack[i]);
        broken[attack[i]] = true;
    }
}

void process() {
    for (int i = 0; i < n; i++) {
        fa[i] = i;
        sz[i] = 1;
    }
    int comp = 0;
    for (int i = 0; i < n; i++) {
        if (broken[i])
            continue;
        comp++;
        for (int e = head[i]; e; e = nxt[e]) {
            int v = to[e];
            if (!broken[v] && unite(i, v))
                comp--;
        }
    }
    ans[k + 1] = comp;
    for (int i = k; i >= 1; i--) {
        int u     = attack[i];
        broken[u] = false;
        comp++;
        for (int e = head[u]; e; e = nxt[e]) {
            int v = to[e];
            if (!broken[v] && unite(u, v))
                comp--;
        }
        ans[i] = comp;
    }
}

void output() {
    printf("%d\n", ans[1]);
    for (int i = 2; i <= k + 1; i++)
        printf("%d\n", ans[i]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O((n + m) * alpha(n))，并查集近似线性。
- 空间复杂度：O(n + m)。
