# P1166 打保龄球 题解

## 题目链接
- [洛谷 P1166](https://www.luogu.com.cn/problem/P1166)

## 题目大意
给定一局保龄球前若干轮的滚球记录，按照保龄球计分规则计算出每一轮的得分以及累计总分。若某轮的得分暂时无法计算（依赖后续滚球结果），则跳过该轮不输出。

## 解题思路
保龄球一局共十轮，每轮的计分规则如下：
- **全中（Strike）**：第一次滚球击倒全部 10 个柱，本轮得分为 10 + 后续两次滚球击倒柱数之和。
- **补中（Spare）**：两次滚球共击倒 10 个柱，本轮得分为 10 + 后续一次滚球击倒柱数。
- **普通（Open）**：两次滚球未击倒全部 10 个柱，本轮得分即为这两次击倒柱数之和。

输入以空格分隔的若干字符串，每个字符串表示一轮中一次或两次滚球的结果：
- `/` 表示一次击倒全部 10 柱（全中）。
- `数字` 或 `数字/` 或 `数字数字` 表示滚球击倒的柱数。

解析时将每个字符串拆解成若干次滚球（整数 0-10），存入 `rolls` 数组。然后按轮次遍历 `rolls` 数组：
- 若当前球为全中，则取后续两球之和计算得分，指针前移 1。
- 若当前球非全中，则取当前球与下一球之和；若和为 10（补中），则再加下一球得分，指针前移 2。

每轮计算完毕后将得分加入 `frameScores`。输出时依次输出每轮得分与累计总分，得分之间用空格分隔。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> rolls;
vector<int> frameScores;

void parseToken(const string &token) {
    if (token[0] == '/')
        rolls.push_back(10);
    else
        rolls.push_back(token[0] - '0');
    if (token.length() > 1) {
        if (token[1] == '/')
            rolls.push_back(10 - (token[0] - '0'));
        else
            rolls.push_back(token[1] - '0');
    }
}

void input() {
    string line;
    getline(cin, line);
    stringstream ss(line);
    string       token;
    while (ss >> token)
        parseToken(token);
}

void process() {
    int n = rolls.size();
    for (int i = 0, frame = 0; frame < 10 && i < n; ++frame) {
        if (rolls[i] == 10) {
            if (i + 2 < n)
                frameScores.push_back(10 + rolls[i + 1] + rolls[i + 2]);
            ++i;
        } else {
            if (i + 1 >= n)
                break;
            int sum = rolls[i] + rolls[i + 1];
            if (sum == 10) {
                if (i + 2 < n)
                    frameScores.push_back(10 + rolls[i + 2]);
            } else
                frameScores.push_back(sum);
            i += 2;
        }
    }
}

void output() {
    for (int s : frameScores)
        printf("%d ", s);
    printf("\n");
    int total = 0;
    for (int s : frameScores) {
        total += s;
        printf("%d ", total);
    }
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n)，其中 n 为滚球总次数（不超过 21 次）。
- 空间复杂度：O(n)，存储滚球序列与每轮得分。
