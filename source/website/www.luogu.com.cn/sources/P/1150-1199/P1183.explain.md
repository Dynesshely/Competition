# P1183 [洛谷] 多边形的面积 题解

## 题目链接
- [洛谷 P1183](https://www.luogu.com.cn/problem/P1183)

## 题目大意

给定一个简单多边形，所有边都与坐标轴平行（垂直或水平），顶点按逆时针顺序给出。计算该多边形的面积。

## 解题思路

使用**鞋带公式**（Shoelace formula）计算多边形的面积：

$$S = \frac{1}{2} \left| \sum_{i=1}^{n} (x_i y_{i+1} - x_{i+1} y_i) \right|$$

其中 $n$ 为顶点数，$(x_{n+1}, y_{n+1}) = (x_1, y_1)$。

题目虽说明多边形的边与坐标轴平行，但鞋带公式适用于任意简单多边形。由于坐标均为整数且边与坐标轴平行，面积一定是整数，可直接整除 2。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, ans;
int x[105], y[105];

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++)
        scanf("%d%d", &x[i], &y[i]);
}

void process() {
    int s = 0;
    for (int i = 1; i <= n; i++) {
        int j = i % n + 1;
        s += x[i] * y[j] - x[j] * y[i];
    }
    if (s < 0) s = -s;
    ans = s / 2;
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(n)，只需一次遍历所有顶点。
- 空间复杂度：O(n)，存储顶点坐标。
