# P1189 [COI 2001] SEARCH 题解

## 题目链接
- [洛谷 P1189](https://www.luogu.com.cn/problem/P1189)

## 题目大意
给定一个 R x C 的矩形地图，`.` 表示可通行，`X` 表示障碍物不可通行，`*` 表示汽车的初始位置。给定 N 个方向（NORTH、SOUTH、WEST、EAST），在每个方向上汽车可以沿该方向连续移动任意多个可通行的格子（即至少移动一格，可以一直移动到碰壁或障碍物前停下）。求所有可能的最终位置，并在地图上用 `*` 标出。

## 解题思路
使用布尔数组模拟逐方向扩展。记 `cur[r][c]` 表示当前所有可能的位置，初始为起点。对于每个方向指令，计算 `nxt`：

- 在每个方向（北/南/西/东）上，按对应顺序扫描每一行/每一列。
- 维护 `lastSrc` 指向当前行/列中最近一个 `cur` 为 true 的位置。当遇到 `X` 时重置 `lastSrc = -1`。
- 对于北向（从下往上扫）：若当前格非障碍，且 `lastSrc` 存在且当前位置在 `lastSrc` 上方，则该格可达。
- 其余方向同理，仅扫描顺序与判断条件不同。
- 每个方向处理完后，将 `nxt` 拷贝回 `cur`。

最终输出时，`cur` 为 true 的位置输出 `*`，否则输出原地图字符。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int  R, C, N;
char grid[55][55];
bool cur[55][55], nxt[55][55];

void input() {
    scanf("%d%d", &R, &C);
    for (int i = 0; i < R; i++) {
        scanf("%s", grid[i]);
    }
    scanf("%d", &N);
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++) {
            if (grid[i][j] == '*') {
                cur[i][j]  = true;
                grid[i][j] = '.';
            }
        }
    }
}

void process() {
    char dir[10];
    for (int step = 0; step < N; step++) {
        scanf("%s", dir);
        memset(nxt, 0, sizeof(nxt));
        if (dir[0] == 'N') {
            for (int c = 0; c < C; c++) {
                int lastSrc = -1;
                for (int r = R - 1; r >= 0; r--) {
                    if (grid[r][c] == 'X') {
                        lastSrc = -1;
                    } else if (cur[r][c]) {
                        if (lastSrc != -1 && r < lastSrc)
                            nxt[r][c] = true;
                        lastSrc = r;
                    } else if (lastSrc != -1) {
                        nxt[r][c] = true;
                    }
                }
            }
        } else if (dir[0] == 'S') {
            for (int c = 0; c < C; c++) {
                int lastSrc = -1;
                for (int r = 0; r < R; r++) {
                    if (grid[r][c] == 'X') {
                        lastSrc = -1;
                    } else if (cur[r][c]) {
                        if (lastSrc != -1 && r > lastSrc)
                            nxt[r][c] = true;
                        lastSrc = r;
                    } else if (lastSrc != -1) {
                        nxt[r][c] = true;
                    }
                }
            }
        } else if (dir[0] == 'W') {
            for (int r = 0; r < R; r++) {
                int lastSrc = -1;
                for (int c = C - 1; c >= 0; c--) {
                    if (grid[r][c] == 'X') {
                        lastSrc = -1;
                    } else if (cur[r][c]) {
                        if (lastSrc != -1 && c < lastSrc)
                            nxt[r][c] = true;
                        lastSrc = c;
                    } else if (lastSrc != -1) {
                        nxt[r][c] = true;
                    }
                }
            }
        } else {
            for (int r = 0; r < R; r++) {
                int lastSrc = -1;
                for (int c = 0; c < C; c++) {
                    if (grid[r][c] == 'X') {
                        lastSrc = -1;
                    } else if (cur[r][c]) {
                        if (lastSrc != -1 && c > lastSrc)
                            nxt[r][c] = true;
                        lastSrc = c;
                    } else if (lastSrc != -1) {
                        nxt[r][c] = true;
                    }
                }
            }
        }
        memcpy(cur, nxt, sizeof(cur));
    }
}

void output() {
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++) {
            printf("%c", cur[i][j] ? '*' : grid[i][j]);
        }
        printf("\n");
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * R * C)，每个方向对每个格子最多扫描一遍。
- 空间复杂度：O(R * C)。
