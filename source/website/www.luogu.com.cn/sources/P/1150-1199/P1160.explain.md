# P1160 队列安排 题解

## 题目链接
- [洛谷 P1160](https://www.luogu.com.cn/problem/P1160)

## 题目大意
有 N 个同学依次入列，每次新同学 i 插入到已入列的同学 k 的左边或右边。之后进行 M 次删除操作。求最终从左到右的队列顺序。

## 解题思路
使用双向链表模拟队列操作。用数组 l[i] 和 r[i] 分别表示 i 号同学左边和右边的同学编号，维护 head 指向队首。

对于插入操作：根据 p 的值将 i 插入到 k 的左侧或右侧，更新相邻节点的左右指针。
对于删除操作：将被删除的节点的前后节点连接起来，更新 head 指针。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
int       n, m, head;
int       l[MAXN], r[MAXN];
bool      via[MAXN];

void input() {
    scanf("%d", &n);
    head = 1;
    for (int i = 2; i <= n; i++) {
        int k, p;
        scanf("%d%d", &k, &p);
        if (p == 0) {
            l[i] = l[k];
            r[i] = k;
            if (l[k])
                r[l[k]] = i;
            l[k] = i;
            if (head == k)
                head = i;
        } else {
            l[i] = k;
            r[i] = r[k];
            if (r[k])
                l[r[k]] = i;
            r[k] = i;
        }
    }
}

void process() {
    scanf("%d", &m);
    for (int i = 0; i < m; i++) {
        int x;
        scanf("%d", &x);
        if (!via[x]) {
            via[x] = true;
            if (l[x])
                r[l[x]] = r[x];
            if (r[x])
                l[r[x]] = l[x];
            if (head == x)
                head = r[x];
        }
    }
}

void output() {
    for (int i = head; i; i = r[i])
        printf("%d ", i);
    printf("\n");
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N + M)，每个插入和删除操作都是 O(1)。
- 空间复杂度：O(N)。
