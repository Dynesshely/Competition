# P1154 奶牛分厩 题解

## 题目链接
- [洛谷 P1154](https://www.luogu.com.cn/problem/P1154)

## 题目大意
有 N 头奶牛，每头奶牛有唯一编号 S_i。将所有奶牛分配进编号为 0 到 K-1 的厩中，第 i 头奶牛进入 S_i mod K 号厩。要求所有奶牛都住进不同的厩，即 S_i mod K 两两不同。求最小的 K。

## 解题思路
题目要求 K 使所有 S_i mod K 互不相同。等价于对任意 i != j，S_i - S_j 不是 K 的倍数，即 K 不能整除任意两个数的差。

因此，先计算出所有两两差值的绝对值，标记这些差值。然后从小到大枚举 K（从 n 到 maxS），检查 K 的倍数是否都在差值集合中未出现；若 K 的所有倍数均未出现在差值集合中，则 K 即为答案。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int  n, s[5010], ans;
bool hasDiff[1000010];

void input() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d", &s[i]);
}

void process() {
    int maxS = *max_element(s, s + n);
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            hasDiff[abs(s[i] - s[j])] = true;
    for (int k = n; k <= maxS; k++) {
        bool ok = true;
        for (int m = k; m <= maxS; m += k)
            if (hasDiff[m]) {
                ok = false;
                break;
            }
        if (ok) {
            ans = k;
            return;
        }
    }
    ans = maxS + 1;
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N^2 + M * (M/K))，其中 N <= 5000, M <= 1e6。N^2 用于计算差值，后续枚举 K 的倍数每次平均 O(M log M)。
- 空间复杂度：O(M)。
