# P1164 小A点菜 题解

## 题目链接
- [洛谷 P1164](https://www.luogu.com.cn/problem/P1164)

## 题目大意
有 N 种菜（每种只有一份），价格为 a_i，小A有 M 元钱。求刚好花光所有钱的点菜方案数。

## 解题思路
01 背包求方案数。设 dp[j] 表示花费恰好 j 元的方案数，初始化 dp[0] = 1。

对每种菜 i，采用倒序循环 j，转移方程：
dp[j] += dp[j - a[i]]

最终输出 dp[M] 即为方案总数。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;
int a[105];
int dp[10005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++)
        scanf("%d", &a[i]);
}

void process() {
    dp[0] = 1;
    for (int i = 1; i <= n; i++)
        for (int j = m; j >= a[i]; j--)
            dp[j] += dp[j - a[i]];
}

void output() { printf("%d\n", dp[m]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N * M)，N <= 100, M <= 10000。
- 空间复杂度：O(M)。
