# P1181 [洛谷] 数列分段 Section I 题解

## 题目链接
- [洛谷 P1181](https://www.luogu.com.cn/problem/P1181)

## 题目大意

给定一个长度为 N 的正整数数列 A，要将其分成连续的若干段，使得每段元素之和不超过 M（可以等于 M）。求最少能分成多少段。

## 解题思路

贪心：从头遍历数列，尽量让当前段容纳更多元素，当加入下一个元素会导致段和超过 M 时，就新开一段。

- `ans` 记录已分出的段数，初始为 1。
- `cur` 记录当前段的累计和。
- 如果 `cur + a[i] > M`，则新开一段，`ans++`，`cur = a[i]`。
- 否则将 `a[i]` 加入当前段，`cur += a[i]`。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, a[100005];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++i)
        scanf("%d", &a[i]);
}

void process() {
    int ans = 1, cur = 0;
    for (int i = 1; i <= n; ++i) {
        if (cur + a[i] > m) {
            ++ans;
            cur = a[i];
        } else
            cur += a[i];
    }
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(N)，只需要一次遍历。
- 空间复杂度：O(N)，存储数列。
