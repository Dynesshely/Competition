# P1414 又是毕业季II 题解

## 题目链接
- [洛谷 P1414](https://www.luogu.com.cn/problem/P1414)

## 题目大意
给定 $n$ 个学生的能力值，对于每个 $k = 1 \sim n$，求从 $n$ 个数中任选 $k$ 个数的最大公约数的最大值。

## 解题思路
对于每个可能的约数 $d$，统计 $a_i$ 中能被 $d$ 整除的个数 $cnt[d]$。若 $cnt[d] \ge k$，则 $d$ 可以作为 $k$ 个数的公约数。

因此，对于每个 $k$，答案是所有 $cnt[d] \ge k$ 的 $d$ 的最大值。

实现步骤：
1. 统计每个值出现的次数 $freq[v]$。
2. 对每个 $d$，枚举其倍数 $j$，累加 $freq[j]$ 得到 $cnt[d]$。
3. 对每个 $cnt[d] = c$，更新 $best[c] = \max(best[c], d)$。
4. 从大到小递推 $ans[k] = \max(ans[k+1], best[k])$。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 10005;
const int MAXV = 1000005;

int n, maxVal, a[MAXN], freq[MAXV], cnt[MAXV], best[MAXN], ans[MAXN];

void input() {
    scanf("%d", &n);
    maxVal = 0;
    for (int i = 1; i <= n; ++i) {
        scanf("%d", &a[i]);
        if (a[i] > maxVal)
            maxVal = a[i];
        freq[a[i]]++;
    }
}

void process() {
    for (int d = 1; d <= maxVal; ++d)
        for (int j = d; j <= maxVal; j += d)
            cnt[d] += freq[j];
    for (int d = 1; d <= maxVal; ++d)
        if (cnt[d] > 0 && d > best[cnt[d]])
            best[cnt[d]] = d;
    ans[n + 1] = 0;
    for (int k = n; k >= 1; --k)
        ans[k] = max(ans[k + 1], best[k]);
}

void output() {
    for (int k = 1; k <= n; ++k)
        printf("%d\n", ans[k]);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(\maxVal \log \maxVal)$，调和级数枚举倍数。
- 空间复杂度：$O(\maxVal + n)$。
