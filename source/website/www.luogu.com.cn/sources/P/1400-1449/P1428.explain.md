# P1428 小鱼比可爱 题解

## 题目链接

- [洛谷 P1428](https://www.luogu.com.cn/problem/P1428)

## 题目大意

给定 $n$ 只鱼的可爱程度值（从左到右排列），每只鱼只能看到它左边的鱼。对每只鱼，统计其左边有多少只鱼可爱程度小于它。

## 解题思路

数据范围 $n \leq 100$，直接使用双重循环暴力枚举即可。

对于第 $i$ 只鱼，遍历 $j = 1$ 到 $i-1$，统计 $a_j < a_i$ 的个数，即为它眼中不如自己可爱的鱼的数量。每读入一个 $a_i$，就立即计算并输出结果，无需预先存储全部数组（但代码中仍用数组存下了所有值）。

## 代码实现

- 使用 `scanf` / `printf` 进行标准输入输出。
- 外层循环 `i = 1..n`：
  - 读入 $a_i$。
  - 内层循环 `j = 1..i`，统计 $a_j < a_i$ 的个数。
  - 输出当前的统计结果。
- 每只鱼的结果之间以空格分隔，最后输出换行。

## 复杂度分析

- 时间复杂度：$O(n^2)$，数据范围很小，完全可行。
- 空间复杂度：$O(n)$，用于存储所有鱼的可爱程度值。
