# P1418 [TJOI2011] 构造矩阵 题解

## 题目链接
- [洛谷 P1418](https://www.luogu.com.cn/problem/P1418)

## 题目大意
构造一个 $n \times m$ 的 01 矩阵，满足第 $i$ 行有 $r_i$ 个 $1$，第 $j$ 列有 $c_j$ 个 $1$。输出字典序最小的方案（先比较第一行，再比较第二行，依次类推）。

## 解题思路
逐行构造，从左到右决定每个位置是 $0$ 还是 $1$。尽量先放 $0$ 以保字典序最小，若放 $0$ 后剩余部分无解，则改为放 $1$。

判断无解的可行性条件使用 Gale-Ryser 定理的变形：对剩余的矩阵，检查每行剩余需要放的 $1$ 个数是否在列容量的约束范围内。

具体检查方法：
- 对剩余行按还需放置的 $1$ 个数从大到小排序。
- 对当前位置 $(i,j)$，尝试放 $0$，然后检查是否存在可行的填充方案：
  - 对于每个 $k$，前 $k$ 个剩余行的需求之和不能超过各列容量之和的限制。
  - 还需要考虑当前行已放置的 $1$ 数、后面的列容量等约束。

贪心地逐位决定即可得到字典序最小的解。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;
int r[110], c[110];
int remRow[110], remCol[110];
int ans[110][110];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++i) {
        scanf("%d", &r[i]);
        remRow[i] = r[i];
    }
    for (int j = 1; j <= m; ++j) {
        scanf("%d", &c[j]);
        remCol[j] = c[j];
    }
}

void process() {
    for (int i = 1; i <= n; ++i) {
        vector<int> future;
        for (int p = i + 1; p <= n; ++p)
            if (remRow[p] > 0)
                future.push_back(remRow[p]);
        sort(future.begin(), future.end(), greater<int>());
        int         fsz = (int)future.size();
        vector<int> pref(fsz + 1, 0);
        for (int k = 1; k <= fsz; ++k)
            pref[k] = pref[k - 1] + future[k - 1];
        for (int j = 1; j <= m; ++j) {
            int ok = 1;
            if (remRow[i] <= m - j) {
                if (i < n) {
                    for (int k = 1; k <= fsz && ok; ++k) {
                        int rhs = 0;
                        for (int q = 1; q <= m; ++q)
                            rhs += min(k, remCol[q]);
                        if (pref[k] > rhs)
                            ok = 0;
                    }
                    for (int k = 1; k <= fsz + 1 && ok; ++k) {
                        int rowSum = remRow[i] + pref[k - 1];
                        int colCap = 0;
                        for (int q = 1; q <= j; ++q)
                            colCap += min(k - 1, remCol[q]);
                        for (int q = j + 1; q <= m; ++q)
                            colCap += min(k, remCol[q]);
                        if (rowSum > colCap)
                            ok = 0;
                    }
                } else {
                    int colCap = 0;
                    for (int q = j + 1; q <= m; ++q)
                        colCap += min(1, remCol[q]);
                    if (remRow[i] > colCap)
                        ok = 0;
                }
            } else
                ok = 0;
            if (ok)
                ans[i][j] = 0;
            else {
                ans[i][j] = 1;
                --remRow[i];
                --remCol[j];
            }
        }
    }
}

void output() {
    for (int i = 1; i <= n; ++i) {
        for (int j = 1; j <= m; ++j)
            printf("%d", ans[i][j]);
        putchar('\n');
    }
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n^2 m^2)$。
- 空间复杂度：$O(nm)$。
