# P1434 [SHOI2002] 滑雪 题解

## 题目链接
- [洛谷 P1434](https://www.luogu.com.cn/problem/P1434)

## 题目大意
给定一个 $R \times C$ 的二维高度数组，从任意点出发，可以向上下左右四个方向移动，但只能从高处滑向低处（高度严格递减）。求最长的滑坡长度（路径上点的个数）。$R, C \le 100$。

## 解题思路
记忆化搜索（DP + 递归）。定义 $dp[x][y]$ 为从点 $(x, y)$ 出发能滑到的最长路径长度。

对于每个点 $(x, y)$，初始值为 $1$（自身）。枚举四个方向进行转移：
$$dp[x][y] = 1 + \max\{dp[nx][ny] \mid h[nx][ny] < h[x][y]\}$$

递归搜索每个未计算的点，利用记忆化避免重复计算。最终答案为所有 $dp$ 值中的最大值。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int dirs[4][2] = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
int       r, c, h[105][105], dp[105][105];

int dfs(int x, int y) {
    if (dp[x][y])
        return dp[x][y];
    int best = 1;
    for (int i = 0; i < 4; i++) {
        int nx = x + dirs[i][0], ny = y + dirs[i][1];
        if (nx >= 0 && nx < r && ny >= 0 && ny < c && h[nx][ny] < h[x][y])
            best = max(best, dfs(nx, ny) + 1);
    }
    return dp[x][y] = best;
}

void input() {
    scanf("%d%d", &r, &c);
    for (int i = 0; i < r; i++)
        for (int j = 0; j < c; j++)
            scanf("%d", &h[i][j]);
}

void process() {
    for (int i = 0; i < r; i++)
        for (int j = 0; j < c; j++)
            if (!dp[i][j])
                dfs(i, j);
}

void output() {
    int ans = 0;
    for (int i = 0; i < r; i++)
        for (int j = 0; j < c; j++)
            if (dp[i][j] > ans)
                ans = dp[i][j];
    printf("%d\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(R \cdot C)$，每个点只会被计算一次。
- 空间复杂度：$O(R \cdot C)$ 用于存储高度和 DP 数组。
