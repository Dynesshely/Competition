# P1447 [NOI2010] 能量采集 题解

## 题目链接

- [洛谷 P1447](https://www.luogu.com.cn/problem/P1447)

## 题目大意

给定 $n \times m$ 个整点坐标 $(x, y)$（$1 \le x \le n$，$1 \le y \le m$），能量汇集机器在 $(0, 0)$。对于每个点 $(x, y)$，若它与 $(0, 0)$ 的连线上有 $k$ 个其他整点，则能量损失为 $2k + 1$。求所有点的能量损失之和。数据范围：$1 \le n, m \le 10^5$。

## 解题思路

从 $(0, 0)$ 到 $(x, y)$ 的线段上，除端点外的整点个数等于 $\gcd(x, y) - 1$。因此每个点 $(x, y)$ 的能量损失为：

$$2(\gcd(x, y) - 1) + 1 = 2\gcd(x, y) - 1$$

总能量损失为：

$$\sum_{x=1}^{n} \sum_{y=1}^{m} (2\gcd(x, y) - 1) = 2\sum_{x=1}^{n} \sum_{y=1}^{m} \gcd(x, y) - n \cdot m$$

问题转化为计算 $S = \displaystyle\sum_{x=1}^{n} \sum_{y=1}^{m} \gcd(x, y)$。

利用欧拉函数的性质：$\gcd(x, y) = \sum_{d \mid \gcd(x, y)} \varphi(d)$，即 $\sum_{d \mid x, d \mid y} \varphi(d)$。交换求和顺序：

$$S = \sum_{d=1}^{\min(n, m)} \varphi(d) \cdot \left\lfloor \frac{n}{d} \right\rfloor \cdot \left\lfloor \frac{m}{d} \right\rfloor$$

因此可以用线性筛或埃氏筛预处理 $\varphi(1) \sim \varphi(\min(n, m))$，然后枚举 $d$ 求和即可。

最终答案：

$$ans = 2S - n \cdot m$$

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100000;

int       n, m, phi[MAXN + 1];
long long ans;

void input() { scanf("%d%d", &n, &m); }

void process() {
    int lim = min(n, m);
    for (int i = 1; i <= lim; i++)
        phi[i] = i;
    for (int i = 2; i <= lim; i++)
        if (phi[i] == i)
            for (int j = i; j <= lim; j += i)
                phi[j] -= phi[j] / i;
    long long s = 0;
    for (int d = 1; d <= lim; d++)
        s += 1LL * phi[d] * (n / d) * (m / d);
    ans = 2 * s - 1LL * n * m;
}

void output() { printf("%lld\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：$O(\min(n, m) \log \log \min(n, m))$，预处理欧拉函数使用埃氏筛，求和部分 $O(\min(n, m))$。
- 空间复杂度：$O(\min(n, m))$。
