# P1449 后缀表达式 题解

## 题目链接

- [洛谷 P1449](https://www.luogu.com.cn/problem/P1449)

## 题目大意

给定一个后缀表达式（逆波兰表达式）字符串，包含非负整数、运算符 `+ - * /`、分隔符 `.` 和结束符 `@`。计算表达式结果。其中 `/` 运算向 0 取整（与 C++ 整数除法规则一致）。数据范围：表达式长度 $|s| \le 50$，运算过程绝对值不超过 $10^9$。

## 解题思路

后缀表达式的标准求值方法：使用一个栈保存操作数。

从左到右扫描字符串（直到遇到 `@`）：

- 若当前字符是数字，则累加构建操作数。
- 若当前字符是 `.`，表示一个操作数结束，将构建好的数字压入栈中，重置操作数。
- 若当前字符是运算符，则从栈中弹出两个操作数（先弹 $b$，再弹 $a$），计算 $a \ \text{op} \ b$ 后将结果压回栈中。

扫描结束后，栈顶元素即为表达式结果。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

char s[55];
int  stk[55], top;

void input() { scanf("%s", s); }

void process() {
    int num = 0;
    for (int i = 0; s[i] != '@'; i++) {
        char c = s[i];
        if (c >= '0' && c <= '9')
            num = num * 10 + (c - '0');
        else if (c == '.') {
            stk[++top] = num;
            num        = 0;
        } else {
            int b = stk[top--];
            int a = stk[top--];
            if (c == '+')
                stk[++top] = a + b;
            else if (c == '-')
                stk[++top] = a - b;
            else if (c == '*')
                stk[++top] = a * b;
            else
                stk[++top] = a / b;
        }
    }
}

void output() { printf("%d\n", stk[top]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析

- 时间复杂度：$O(|s|)$，每个字符处理一次。
- 空间复杂度：$O(|s|)$，栈深度不超过操作数个数。
