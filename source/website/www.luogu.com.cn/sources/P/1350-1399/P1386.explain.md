# P1386 座位安排 题解

## 题目链接
- [洛谷 P1386](https://www.luogu.com.cn/problem/P1386)

## 题目大意
给 n 个人安排座位，每人有一个目标座位 a_i ∈ [1, n]。第 i 个人先尝试坐 a_i，若被占则依次尝试 a_i+1, a_i+2, ...，直到第 n 个座位。若仍无空位则安排不合法。有 m 个人的目标座位已固定，其余人的编号可自由安排。求合法安排方案数对 M 取模。T 组数据，n ≤ 300，M ≤ 1e9。

## 解题思路
考虑从座位 1 到 n 依次处理。对于每个座位 i，需要保证最终入座的人数不少于 i（即前 i 个座位必须坐满），否则后面的人会找不到座位。

预处理 fixedCnt[i] 表示目标座位固定为 i 的人数，自由人数 free = n - m。

**可行性判断**：扫描座位 i = 1..n，累计 prefFixed 为固定安排到 ≤ i 的人数。若在任何时刻 prefFixed + free < i（即使所有自由人都安排到 ≤ i 仍不够），则无解。

**动态规划**：
- 设 dp[i][j] 表示处理完前 i 个座位，有 j 个自由人已分配到目标座位 ≤ i 的方案数。
- 初始 dp[0][0] = 1。
- 转移：对于座位 i，已有 j-k 人分配到前 i-1 个座位，现从剩余自由人中选 k 人分配到座位 i（目标座位 = i）。剩余自由人数为 free - (j-k)，从中选 k 个的方案数为 C(free - (j - k), k)。
  dp[i][j] = Σ_{k=0}^{j} dp[i-1][j-k] × C(free - (j - k), k)
- 要求 prefFixed + j ≥ i 时才可转移（保证前 i 个座位能被坐满）。

最终答案为 dp[n][free]。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int T, n, m, mod;
int fixedCnt[305];
int dp[305][305];
int C[305][305];

void input() {
    scanf("%d%d%d", &n, &m, &mod);
    memset(fixedCnt, 0, sizeof(fixedCnt));
    for (int i = 0; i < m; i++) {
        int p, q;
        scanf("%d%d", &p, &q);
        fixedCnt[q]++;
    }
}

void process() {
    int free      = n - m;
    int prefFixed = 0;
    for (int i = 1; i <= n; i++) {
        prefFixed += fixedCnt[i];
        if (prefFixed + free < i) {
            printf("NO\n");
            return;
        }
    }

    for (int i = 0; i <= free; i++) {
        C[i][0] = C[i][i] = 1 % mod;
        for (int j = 1; j < i; j++)
            C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % mod;
    }

    memset(dp, 0, sizeof(dp));
    dp[0][0]  = 1;
    prefFixed = 0;

    for (int i = 1; i <= n; i++) {
        prefFixed += fixedCnt[i];
        for (int j = 0; j <= free; j++) {
            if (prefFixed + j < i)
                continue;
            long long sum = 0;
            for (int k = 0; k <= j; k++) {
                int rem = free - (j - k);
                sum     = (sum + (long long)dp[i - 1][j - k] * C[rem][k]) % mod;
            }
            dp[i][j] = sum;
        }
    }

    printf("YES %d\n", dp[n][free]);
}

int main() {
    scanf("%d", &T);
    while (T--) {
        input();
        process();
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n × free^2) = O(n^3)，n ≤ 300 时可接受。
- 空间复杂度：O(n^2)。
