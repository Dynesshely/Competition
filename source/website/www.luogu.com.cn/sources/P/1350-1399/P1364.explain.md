# P1364 洛谷 医院设置 BFS枚举 题解

## 题目链接
- [洛谷 P1364](https://www.luogu.com.cn/problem/P1364)

## 题目大意
给定一棵二叉树，每个结点有一个居民人口数。现需选择一个结点建立医院，使得所有居民到医院的距离之和最小（相邻结点距离为 1）。输出最小距离和。

## 解题思路
由于 n ≤ 100，可以直接枚举每个结点作为医院的位置，对该结点进行 BFS 计算出到其他所有结点的距离，加权求和后取最小值。

具体步骤：
1. 读入二叉树，用邻接表建图。
2. 对于每个结点 i，以 i 为起点进行 BFS，计算出 i 到所有结点的最短距离。
3. 将每个结点的距离乘上该结点的人口数，累加得到以 i 为医院的总距离。
4. 取所有 i 的最小值作为答案输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, w[105];
vector<int> g[105];

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        int u, v;
        scanf("%d%d%d", &w[i], &u, &v);
        if (u) {
            g[i].push_back(u);
            g[u].push_back(i);
        }
        if (v) {
            g[i].push_back(v);
            g[v].push_back(i);
        }
    }
}

int bfs(int s) {
    int dist[105];
    memset(dist, -1, sizeof(dist));
    queue<int> q;
    q.push(s);
    dist[s] = 0;
    int sum = 0;
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        sum += dist[u] * w[u];
        for (int v : g[u]) {
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
    return sum;
}

int process() {
    int ans = 1e9;
    for (int i = 1; i <= n; i++)
        ans = min(ans, bfs(i));
    return ans;
}

int main() {
    input();
    printf("%d\n", process());
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n²)，对每个结点做一次 BFS，每次 BFS 遍历所有结点。
- 空间复杂度：O(n)，存储邻接表和距离数组。
