# P1370 Charlie 的云笔记序列 题解

## 题目链接
- [洛谷 P1370](https://www.luogu.com.cn/problem/P1370)

## 题目大意
给定序列 $a[1 \dots n]$，定义 $F(l, r)$ 为子串 $a[l:r]$ 中本质不同的子序列个数（含空序列）。求 $\sum_{1 \le l \le r \le n} F(l, r) \bmod 998244353$。

## 解题思路
动态规划统计所有子串的本质不同子序列个数之和。考虑以 $i$ 结尾的所有子串 $[l, i]$，递推计算 $sum[i]$ 表示所有以 $i$ 结尾的子串的 $F(l, i)$ 之和。

对于每个位置 $i$，设当前值为 $x = a[i]$，若 $x$ 未出现过，则：
$$
sum[i] = 2 \times (sum[i-1] + 1) \pmod{MOD}
$$
若出现过，设上一次出现位置为 $last[x] = p$，则需要减去重复贡献的部分：
$$
sum[i] = (2 \times (sum[i-1] + 1) - (sum[p-1] + 1)) \bmod MOD
$$
最终答案为 $\sum_{i=1}^{n} sum[i] \bmod MOD$。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD  = 998244353;
const int MAXN = 100005;

int       n, a[MAXN];
long long sum[MAXN];

void input() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++)
        scanf("%d", &a[i]);
}

void process() {
    map<int, int> last;
    sum[0] = 0;
    for (int i = 1; i <= n; i++) {
        int  x  = a[i];
        auto it = last.find(x);
        if (it == last.end()) {
            sum[i] = 2 * (sum[i - 1] + 1) % MOD;
        } else {
            int       p  = it->second;
            long long t1 = 2 * (sum[i - 1] + 1) % MOD;
            long long t2 = (sum[p - 1] + 1) % MOD;
            sum[i]       = (t1 - t2 + MOD) % MOD;
        }
        last[x] = i;
    }
}

void output() {
    long long ans = 0;
    for (int i = 1; i <= n; i++)
        ans = (ans + sum[i]) % MOD;
    printf("%lld\n", ans);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n)$，使用 `map` 记录上次出现位置
- 空间复杂度：$O(n)$
