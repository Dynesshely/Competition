# P1391 洛谷 方阵安排 题解

## 题目链接
- [洛谷 P1391](https://www.luogu.com.cn/problem/P1391)

## 题目大意

给定一个 n × n 的 0/1 方阵，0 代表女生，1 代表男生。每次可以将女生（0）改成男生（1），求最少的修改次数，使得方阵中每个格子四周的男生个数均为偶数（即上下左右四个方向存在的位置中 1 的个数为偶数）。若无解则输出 -1。

## 解题思路

对于格子 (i, j)，其上下左右四个邻居中 1 的个数为偶数，等价于异或和为 0：

$$\text{row}[i-1][j] \oplus \text{row}[i+1][j] \oplus \text{row}[i][j-1] \oplus \text{row}[i][j+1] = 0$$

由此可递推：已知前两行的值，下一行的值可由上两行决定：

$$\text{row}[i+1][j] = \text{row}[i-1][j] \oplus \text{row}[i][j-1] \oplus \text{row}[i][j+1]$$

因此，只要确定了第一行，整个方阵就唯一确定：
- 第二行由第一行决定（对于第一行格子，没有上邻居，约束只包含下、左、右）。
- 后续行由前两行递推得到。

枚举第一行的 2^n 种可能状态，对每种状态：
1. 推导出整个方阵，过程中检查每个位置的值是否 ≥ 初始值（只能将 0 改成 1，不能将 1 改成 0）。
2. 推导完成后检查最后一行是否满足偶数约束（没有下邻居，需验证上、左、右的异或和为 0）。
3. 记录满足条件的方案中修改次数的最小值。

## 代码实现

外层循环枚举第一行的掩码 mask，内层逐行推导。row[0][j] 由 mask 二进制位确定。对于 i ≥ 1，按递推公式计算 row[i][j]，同时检查是否小于初始值 init[i][j]。最后验证最后一行边界条件。ans 记录最小修改次数，初始为 -1 表示无解。

## 复杂度分析

- 时间复杂度：O(n² · 2^n)，n ≤ 18 时 2^18 = 262144，乘以 n² = 324，总运算量约 8500 万次，可以接受。
- 空间复杂度：O(n²)，存储初始方阵和当前行状态。
