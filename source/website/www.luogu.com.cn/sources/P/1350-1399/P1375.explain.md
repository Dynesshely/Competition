# P1375 小猫 题解

## 题目链接
- [洛谷 P1375](https://www.luogu.com.cn/problem/P1375)

## 题目大意
$2n$ 只小猫站成一圈，两两用绳子绑住尾巴连在一起，要求绳子之间不能交叉。求所有可能的配对方案数模 $10^9+7$。

## 解题思路
圆上不相交的弦配对方案数即卡特兰数 $C_n = \frac{1}{n+1} \binom{2n}{n}$。将小猫依次编号，因为绳子不能交叉，所以配对方式与合法括号序列一一对应，方案数为第 $n$ 个卡特兰数。

模数为质数 $10^9+7$，预处理阶乘和逆元阶乘后用公式 $C_n = (2n)! \cdot (n!)^{-2} \cdot (n+1)^{-1} \bmod MOD$ 计算。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD = 1000000007;

int n, ans;

int powMod(int a, int b) {
    int res = 1;
    while (b) {
        if (b & 1)
            res = 1LL * res * a % MOD;
        a = 1LL * a * a % MOD;
        b >>= 1;
    }
    return res;
}

void input() { scanf("%d", &n); }

void process() {
    int m = 2 * n;
    vector<int> fac(m + 1), invFac(m + 1);
    fac[0] = 1;
    for (int i = 1; i <= m; ++i)
        fac[i] = 1LL * fac[i - 1] * i % MOD;
    invFac[m] = powMod(fac[m], MOD - 2);
    for (int i = m; i >= 1; --i)
        invFac[i - 1] = 1LL * invFac[i] * i % MOD;
    int c = 1LL * fac[m] * invFac[n] % MOD * invFac[n] % MOD;
    ans = 1LL * c * powMod(n + 1, MOD - 2) % MOD;
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
