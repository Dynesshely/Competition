# P1385 密令 题解

## 题目链接
- [洛谷 P1385](https://www.luogu.com.cn/problem/P1385)

## 题目大意
给定一个小写字母串 s，每次操作可以选择相邻位置 p 和 p+1，将 s[p] +1 且 s[p+1] -1，或 s[p] -1 且 s[p+1] +1（须保证操作后仍为小写字母 a-z）。求经过任意多次操作后，能得到多少种不同于原串的字符串。T 组数据，|s| ≤ 100，T ≤ 10000，答案对 1e9+7 取模。

## 解题思路
观察操作：每次操作将一个字符 +1、相邻字符 -1，总和不变。因此**整个字符串的字符和（a=0, b=1, ..., z=25）在任意操作下保持不变**。问题转化为：给定长度 n 和字符和 S，有多少种长度为 n 的小写字母串（每个字母在 0~25 之间）其和为 S，再减去原串本身。

这是一个带上限的整数拆分问题。使用**容斥原理**：
- 无上限时，将 S 个不可区分的球放入 n 个盒子（允许空盒），方案数为 C(S + n - 1, n - 1)。
- 减去某个盒子超过 25 的情况：先给该盒子分配 26，剩余 S - 26 再分配，方案数为 C(n, 1) * C(S - 26 + n - 1, n - 1)。
- 加回两个盒子同时超过 25 的情况：C(n, 2) * C(S - 52 + n - 1, n - 1)。
- 以此类推。

即总方案数 = Σ_{k=0}^{min(n, S/26)} (-1)^k * C(n, k) * C(S - 26k + n - 1, n - 1)

最终答案 = 总方案数 - 1（去掉原串）。

预处理阶乘和逆元到 MAXF = 25 * 100 + 100 = 2600，单次查询 O(min(n, S/26))。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MOD  = 1000000007;
const int MAXN = 100;
const int MAXS = 25 * MAXN;
const int MAXF = MAXS + MAXN;

long long fact[MAXF + 1], invFact[MAXF + 1];

long long modPow(long long a, long long b) {
    long long res = 1;
    while (b) {
        if (b & 1)
            res = res * a % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}

void init() {
    fact[0] = 1;
    for (int i = 1; i <= MAXF; i++)
        fact[i] = fact[i - 1] * i % MOD;
    invFact[MAXF] = modPow(fact[MAXF], MOD - 2);
    for (int i = MAXF; i >= 1; i--)
        invFact[i - 1] = invFact[i] * i % MOD;
}

long long C(int n, int k) {
    if (k < 0 || k > n)
        return 0;
    return fact[n] * invFact[k] % MOD * invFact[n - k] % MOD;
}

void input() {
    init();
    int T;
    scanf("%d", &T);
    char s[105];
    while (T--) {
        scanf("%s", s);
        int n   = strlen(s);
        int sum = 0;
        for (int i = 0; i < n; i++)
            sum += s[i] - 'a';
        long long total = 0;
        int       maxK  = min(n, sum / 26);
        for (int k = 0; k <= maxK; k++) {
            long long term = C(n, k) * C(sum - 26 * k + n - 1, n - 1) % MOD;
            if (k & 1)
                total = (total - term + MOD) % MOD;
            else
                total = (total + term) % MOD;
        }
        long long ans = (total - 1 + MOD) % MOD;
        printf("%lld\n", ans);
    }
}

int main() {
    input();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(MAXF + T * n)，MAXF = 2600 为预处理阶乘的开销，每组数据 O(min(n, sum/26)) ≤ O(n)。
- 空间复杂度：O(MAXF)。
