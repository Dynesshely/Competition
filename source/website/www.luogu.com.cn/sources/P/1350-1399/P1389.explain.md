# P1389 洛谷 一个关于序列的游戏 题解

## 题目链接
- [洛谷 P1389](https://www.luogu.com.cn/problem/P1389)

## 题目大意

给定一个长度为 n 的序列 A 和分数数组 V，每次可以删除一个连续段，删除后左右两边合并为相邻元素。删除的连续段需满足：
1. 相邻两个元素相差恰好为 1。
2. 不在最左或最右的元素不能同时小于相邻的左右两个元素（即不能出现山谷）。

每次删除获得分数 V[len]，其中 len 为删除段的长度。求能获得的最大总分。

## 解题思路

这是一个区间 DP 问题，n ≤ 150，且相同数字出现次数不超过 14 次。

定义状态 `f[i][j][k]` 和 `g[i][j][k]` 表示区间 [i, j] 的最大得分，其中 k 与当前正在构建的删除段的长度有关。两种状态分别用于处理递增和递减方向。

预处理 `pre1[i]`、`pre2[i]`、`pre3[i]` 分别表示位置 i 之前最后一个值为 `A[i]-1`、`A[i]+1`、`A[i]` 的位置。

核心转移思路：
- 对于当前区间右端点 j，寻找之前与其值相差 1 的位置 p，将区间 [i, j] 拆分为 [i, p] 和 [p+1, j-1]，如果后者可以被完全删除且前者可以扩展，则可合并形成更长的删除段。
- 利用 `pre3` 链式向前遍历所有相同值的位置（每个值最多 14 次），保证内层遍历的复杂度可控。

最后使用线性 DP 组合各区间结果：`ans[i] = max(ans[i-1], max_{j<i} ans[j] + f[j+1][i][0])`，其中 `f[j+1][i][0]` 表示区间 [j+1, i] 完全删除后的最大得分。

## 代码实现

首先读入 n、分数数组 val 和序列 a。对 a 中每个值进行离散化并记录 pre 数组。然后进行区间 DP 的初始化（单元素区间）和主流程，最后通过线性 DP 合并得到完整序列的答案。

## 复杂度分析

- 时间复杂度：O(n³·C)，其中 C ≤ 14 为相同值的最大出现次数，n ≤ 150，总运算量约五千万次，可以接受。
- 空间复杂度：O(n³)，三维 DP 数组 f 和 g，每个维度 155×155×155。
