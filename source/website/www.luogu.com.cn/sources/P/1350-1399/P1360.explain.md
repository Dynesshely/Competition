# P1360 洛谷 前缀和+哈希 题解

## 题目链接
- [洛谷 P1360](https://www.luogu.com.cn/problem/P1360)

## 题目大意
有 $n$ 天和 $m$ 种能力，每天会给出一个能力提升数字 $x$，其二进制表示中从右往左第 $k$ 位为 $1$ 表示当天第 $k$ 种能力提升一次。求最长的连续区间，使得该区间内每种能力提升的总次数相等。

## 解题思路
设 $pref[k][i]$ 表示前 $i$ 天中第 $k$ 种能力的提升次数。区间 $(l,r]$（即第 $l+1$ 天到第 $r$ 天）内每种能力提升次数相等的充要条件是：对于任意两种能力 $p,q$，有
$$
pref[p][r] - pref[p][l] = pref[q][r] - pref[q][l]
$$
即每种能力的增量相同。等价变形为：
$$
pref[p][r] - pref[0][r] = pref[p][l] - pref[0][l] \quad (\forall p)
$$
因此对每一天 $i$，构造向量 $key_i = (pref[1][i]-pref[0][i],\; pref[2][i]-pref[0][i],\; \dots,\; pref[m-1][i]-pref[0][i])$。用哈希表（C++ 的 `map`）记录每个向量第一次出现的位置，若某向量再次出现，则用当前天数减去该向量首次出现的位置更新答案。

由于 $m\le30$，每个能力提升数字 $x$ 只需 $O(m)$ 时间拆位即可。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;

void input() { scanf("%d%d", &n, &m); }

void process() {
    vector<int>           pref(m, 0);
    map<vector<int>, int> mp;
    vector<int>           key(m, 0);
    mp[key] = 0;
    int ans = 0;
    for (int i = 1; i <= n; i++) {
        int x;
        scanf("%d", &x);
        for (int k = 0; k < m; k++)
            if ((x >> k) & 1)
                pref[k]++;
        for (int k = 0; k < m; k++)
            key[k] = pref[k] - pref[0];
        auto it = mp.find(key);
        if (it != mp.end())
            ans = max(ans, i - it->second);
        else
            mp[key] = i;
    }
    printf("%d\n", ans);
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(nm \log n)$，其中 $n\le10^5$，$m\le30$，`map` 每次操作 $O(\log n)$。
- 空间复杂度：$O(nm)$，哈希表中最多存储 $n$ 个长度为 $m$ 的向量。
