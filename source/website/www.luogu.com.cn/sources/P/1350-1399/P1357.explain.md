# P1357 洛谷 状压DP + 矩阵快速幂 题解

## 题目链接
- [洛谷 P1357](https://www.luogu.com.cn/problem/P1357)

## 题目大意
有一座环形花园，n 个花圃排成一个环（1 和 n 相邻）。每个花圃可以是 C 形或 P 形。要求任意相邻 m 个花圃中 C 形花圃的数量不超过 k 个。求符合条件的花园种数模 10^9+7 的结果。其中 n 最大可达 10^15，m 不超过 5。

## 解题思路
由于 m 不超过 5，考虑用状态压缩 DP 结合矩阵快速幂求解。

1. **状态设计**：用一个 m 位二进制数表示最近 m 个花圃的状态（1 表示 C，0 表示 P）。只有二进制表示中 1 的个数不超过 k 的状态才是合法状态。
2. **状态转移**：对于当前状态 s，下一个花圃可以为 C 或 P，即在 s 后添加一位 0 或 1，并去掉最高位，得到新状态 `ns = ((s << 1) & mask) | b`，其中 b 为 0 或 1。若 ns 为合法状态，则从 s 到 ns 有一条转移。
3. **矩阵建模**：所有合法状态构成矩阵 A，A[i][j] 表示从状态 i 到状态 j 的转移方案数。
4. **环形处理**：因为是环形花园，需要走 n 步后回到起始状态。答案为矩阵 A 的 n 次幂的对角线元素之和，即 `sum(A^n[i][i])`，这表示从每个状态出发走 n 步回到自身的方案总数。
5. **矩阵快速幂**：使用快速幂计算 A^n，时间复杂度 O(S^3 log n)，其中 S 为合法状态数（不超过 C(m, k) 级别）。

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const long long MOD = 1000000007;

long long n;
int       m, k;

struct Mat {
    int                       sz;
    vector<vector<long long>> a;
    Mat(int s)
        : sz(s)
        , a(s, vector<long long>(s, 0)) {}
    Mat operator*(const Mat &o) const {
        Mat r(sz);
        for (int i = 0; i < sz; i++)
            for (int k = 0; k < sz; k++)
                if (a[i][k])
                    for (int j = 0; j < sz; j++)
                        if (o.a[k][j])
                            r.a[i][j] = (r.a[i][j] + a[i][k] * o.a[k][j]) % MOD;
        return r;
    }
};

Mat qpow(Mat a, long long b) {
    Mat r(a.sz);
    for (int i = 0; i < a.sz; i++)
        r.a[i][i] = 1;
    while (b) {
        if (b & 1)
            r = r * a;
        a = a * a;
        b >>= 1;
    }
    return r;
}

void input() { scanf("%lld %d %d", &n, &m, &k); }

void process() {
    int         mask = (1 << m) - 1;
    vector<int> st;
    for (int s = 0; s <= mask; s++)
        if (__builtin_popcount(s) <= k)
            st.push_back(s);
    int sz = st.size();
    int id[32];
    memset(id, -1, sizeof(id));
    for (int i = 0; i < sz; i++)
        id[st[i]] = i;

    Mat A(sz);
    for (int i = 0; i < sz; i++) {
        int s = st[i];
        for (int b = 0; b <= 1; b++) {
            int ns = ((s << 1) & mask) | b;
            if (id[ns] != -1)
                A.a[i][id[ns]]++;
        }
    }

    Mat       An  = qpow(A, n);
    long long ans = 0;
    for (int i = 0; i < sz; i++)
        ans = (ans + An.a[i][i]) % MOD;
    printf("%lld\n", ans);
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(S^3 log n)，其中 S 为合法状态数（不超过 C(m, k)），log n 为矩阵快速幂的幂次。
- 空间复杂度：O(S^2)，用于存储转移矩阵及其幂次结果。
