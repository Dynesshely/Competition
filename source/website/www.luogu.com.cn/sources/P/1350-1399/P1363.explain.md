# P1363 洛谷 幻象迷宫 模意义BFS 题解

## 题目链接
- [洛谷 P1363](https://www.luogu.com.cn/problem/P1363)

## 题目大意
给定一个 N×M 的迷宫矩阵（由 `.` 道路、`#` 墙、`S` 起点组成），该迷宫在无限大的平面上由该矩阵不断重复拼接而成。玩家从 S 出发，可以上下左右移动，不能穿墙。问能否走到距离起点无限远处，即是否存在一条路径使得玩家可以无限远离起点。

## 解题思路
核心思想：将迷宫视为在无限平面上重复铺砌，每个实际坐标 (x, y) 对应到基底迷宫中的坐标 (x mod N, y mod M)。如果能从起点出发，通过"绕圈"到达同一个基底格子但具有不同的实际坐标，则说明可以通过无限次绕圈走到无限远处（相当于找到了一个可无限延伸的路径）。

具体做法：
1. 用 BFS 进行搜索，队列中存储实际坐标（可能会超出 0..N-1, 0..M-1 范围）。
2. 维护两个状态数组：
   - `via[rx][ry]`：标记基底格子 (rx, ry) 在当前测试数据中是否访问过。
   - `rX[rx][ry]` 和 `rY[rx][ry]`：记录第一次访问该基底格子时的实际坐标。
3. BFS 扩展时，计算相邻格子的实际坐标 (nx, ny) 及其对应的基底坐标 (rx, ry)。
4. 如果该基底格子是墙则跳过。
5. 如果该基底格子未访问过，则标记访问，记录实际坐标，入队。
6. 如果该基底格子已访问过，但当前实际坐标 (nx, ny) 与第一次访问时的实际坐标不同，说明找到了一个"绕圈"路径，返回 Yes。
7. 若 BFS 结束仍未找到，返回 No。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1505;
const int DX[] = {0, 0, 1, -1};
const int DY[] = {1, -1, 0, 0};

int  n, m, cur;
char grid[MAXN][MAXN];
int  via[MAXN][MAXN];
int  rX[MAXN][MAXN];
int  rY[MAXN][MAXN];
int  sx, sy;

void input() {
    cur++;
    for (int i = 0; i < n; i++) {
        scanf("%s", grid[i]);
        for (int j = 0; j < m; j++)
            if (grid[i][j] == 'S') {
                sx         = i;
                sy         = j;
                grid[i][j] = '.';
            }
    }
}

bool process() {
    queue<pair<int, int>> q;
    q.push({sx, sy});
    via[sx][sy] = cur;
    rX[sx][sy]  = sx;
    rY[sx][sy]  = sy;
    while (!q.empty()) {
        int x = q.front().first;
        int y = q.front().second;
        q.pop();
        for (int d = 0; d < 4; d++) {
            int nx = x + DX[d];
            int ny = y + DY[d];
            int rx = nx % n;
            if (rx < 0)
                rx += n;
            int ry = ny % m;
            if (ry < 0)
                ry += m;
            if (grid[rx][ry] == '#')
                continue;
            if (via[rx][ry] != cur) {
                via[rx][ry] = cur;
                rX[rx][ry]  = nx;
                rY[rx][ry]  = ny;
                q.push({nx, ny});
            } else if (rX[rx][ry] != nx || rY[rx][ry] != ny)
                return true;
        }
    }
    return false;
}

void output() {
    if (process())
        printf("Yes\n");
    else
        printf("No\n");
}

int main() {
    while (scanf("%d%d", &n, &m) == 2) {
        input();
        output();
    }
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N×M)，每个基底格子至多入队一次，每次出队检查四个方向。
- 空间复杂度：O(N×M)，存储迷宫地图及访问状态数组。
