# P1378 油滴扩展 题解

## 题目链接
- [洛谷 P1378](https://www.luogu.com.cn/problem/P1378)

## 题目大意
给定一个矩形框和框内 $n$ 个点，每次选择一个未放置的点放置油滴，油滴会扩展为圆形，直到碰到矩形边界或其他已扩展的油滴。求所有油滴扩展完毕后，矩形内剩余的最小空间（矩形面积减圆总面积，四舍五入输出整数）。

## 解题思路
$n \le 6$，可直接枚举全排列，对每种顺序模拟油滴扩展过程：
- 对每个油滴，初始可扩展半径为到四条矩形边界的最小距离。
- 再检查所有已扩展油滴，限制为当前点到已扩展油滴圆心的距离减去已扩展油滴的半径（若为负则取 $0$）。
- 取上述限制的最小值作为当前油滴的半径，累加圆面积。
- 取所有排列中总圆面积的最大值，用矩形面积减去即得剩余最小空间。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n;
double px[10], py[10];
double lx, ly, rx, ry;
double ans;

double dist(int a, int b) { return sqrt((px[a] - px[b]) * (px[a] - px[b]) + (py[a] - py[b]) * (py[a] - py[b])); }

void input() {
    double x1, y1, x2, y2;
    scanf("%d", &n);
    scanf("%lf %lf %lf %lf", &x1, &y1, &x2, &y2);
    lx = min(x1, x2);
    rx = max(x1, x2);
    ly = min(y1, y2);
    ry = max(y1, y2);
    for (int i = 0; i < n; i++)
        scanf("%lf %lf", &px[i], &py[i]);
}

void process() {
    double boxArea = (rx - lx) * (ry - ly);
    int ord[10];
    for (int i = 0; i < n; i++)
        ord[i] = i;
    double maxOil = 0;
    do {
        double r[10] = {0};
        double cur = 0;
        for (int i = 0; i < n; i++) {
            int p = ord[i];
            double lim = min(min(px[p] - lx, rx - px[p]), min(py[p] - ly, ry - py[p]));
            for (int j = 0; j < i; j++) {
                int q = ord[j];
                double d = dist(p, q) - r[q];
                if (d < 0) d = 0;
                if (d < lim) lim = d;
            }
            r[p] = lim;
            cur += acos(-1.0) * lim * lim;
        }
        if (cur > maxOil) maxOil = cur;
    } while (next_permutation(ord, ord + n));
    ans = boxArea - maxOil;
}

void output() { printf("%.0f\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n! \cdot n)$，$n \le 6$ 时枚举量很小
- 空间复杂度：$O(n)$
