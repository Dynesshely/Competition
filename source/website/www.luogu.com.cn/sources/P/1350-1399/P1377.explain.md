# P1377 [TJOI2011] 树的序 题解

## 题目链接
- [洛谷 P1377](https://www.luogu.com.cn/problem/P1377)

## 题目大意
给定一个 $1 \sim n$ 的排列作为二叉查找树（BST）的插入序列，求与它生成相同 BST 的所有插入序列中字典序最小的那个。

## 解题思路
BST 的键值按 $1 \sim n$ 顺序插入时，插入顺序（位置）即为节点的权值。问题转化为：已知中序遍历为 $1 \sim n$（键值顺序），按插入顺序（权值）构造笛卡尔树（树根为权值最小的节点），然后输出其前序遍历（键值），即字典序最小的生成序列。

建笛卡尔树可用单调栈 $O(n)$ 实现：
- $pos[x]$ 为键值 $x$ 在插入序列中的位置（权值）。
- 按键值 $1 \sim n$ 顺序（中序）插入，栈中维护权值递增（pos 递增）。
- 当前节点 $v$ 的左孩子为最后一个被弹出的节点，栈顶右孩子设为 $v$。
- 前序遍历用优先队列（小根堆）实现 BFS。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, pos[100005], lch[100005], rch[100005];
int stk[100005], top;

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        int x;
        scanf("%d", &x);
        pos[x] = i;
    }
    int root = 0;
    for (int v = 1; v <= n; v++) {
        int last = 0;
        while (top && pos[stk[top]] > pos[v])
            last = stk[top--];
        lch[v] = last;
        if (top)
            rch[stk[top]] = v;
        else
            root = v;
        stk[++top] = v;
    }
    priority_queue<int, vector<int>, greater<int>> pq;
    pq.push(root);
    bool first = true;
    while (!pq.empty()) {
        int v = pq.top();
        pq.pop();
        if (!first)
            printf(" ");
        printf("%d", v);
        first = false;
        if (lch[v])
            pq.push(lch[v]);
        if (rch[v])
            pq.push(rch[v]);
    }
    printf("\n");
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n)$，优先队列输出
- 空间复杂度：$O(n)$
