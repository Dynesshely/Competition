# P1387 最大正方形 题解

## 题目链接
- [洛谷 P1387](https://www.luogu.com.cn/problem/P1387)

## 题目大意
在一个 n × m 的 0/1 矩阵中，找出一个全由 1 组成的最大正方形，输出其边长。n, m ≤ 100，保证至少有一个 1。

## 解题思路
经典动态规划。设 dp[i][j] 表示以 (i, j) 为右下角的最大全 1 正方形的边长。

若 a[i][j] == 0，则 dp[i][j] = 0。
若 a[i][j] == 1，则 dp[i][j] 取决于上方、左方和左上方的三个 dp 值：
```
dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
```
这表示 (i, j) 能够扩展出的最大正方形边长受限于三个相邻位置中能得到的最大正方形的最小值，再加上自身这一格。

遍历整个矩阵，取 dp[i][j] 的最大值即为答案。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m, ans;
int a[105][105], dp[105][105];

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            scanf("%d", &a[i][j]);
}

void process() {
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j) {
            if (a[i][j] == 1) {
                dp[i][j] = min({dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]}) + 1;
                if (dp[i][j] > ans)
                    ans = dp[i][j];
            }
        }
}

void output() { printf("%d\n", ans); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n × m)，单次遍历矩阵。
- 空间复杂度：O(n × m)，可优化至 O(m) 但 n, m ≤ 100 无需优化。
