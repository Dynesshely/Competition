# P1749 [入门赛 #19] 分饼干 II 题解

## 题目链接

- [洛谷 P1749](https://www.luogu.com.cn/problem/P1749)

## 题目大意

有 T 组测试数据，每组给定饼干总数 N 和小朋友人数 k。要求判断能否将 N 块饼干分给 k 个小朋友，使得每个小朋友至少拿到一块饼干，且每人的饼干数量互不相同。

## 解题思路

要使 k 个小朋友分到的饼干数量互不相同且每人至少一块，最节省饼干的分配方式是让小朋友们分别拿到 1, 2, 3, ..., k 块饼干，所需饼干总数为 1+2+...+k = k(k+1)/2。

因此，只需要判断 N 是否大于等于 k(k+1)/2 即可。若是则输出 Yes，否则输出 No。

由于 N, k 最大可达 1e9，计算 k(k+1)/2 时注意使用 long long 防止溢出。

## 代码实现

- 读入 T，循环处理每组数据。
- 对每组数据判断 k*(k+1)/2 <= N 是否成立，输出对应结果。

## 复杂度分析

- 时间复杂度：O(T)
- 空间复杂度：O(1)
