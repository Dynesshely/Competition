## 题目描述

请你编一程序实现两种不同进制之间的数据转换。

## 输入格式

共三行，第一行是一个正整数，表示需要转换的数的进制 $n\ (2\le n\le 16)$，第二行是一个 $n$ 进制数，若 $n>10$ 则用大写字母 $\verb!A!\sim \verb!F!$ 表示数码 $10\sim 15$，并且该 $n$ 进制数对应的十进制的值不超过 $10^9$，第三行也是一个正整数，表示转换之后的数的进制 $m\ (2\le m\le 16)$。

## 输出格式

一个正整数，表示转换之后的 $m$ 进制数。

## 样例 #1

### 样例输入

```
16
FF
2
```

### 样例输出

```
11111111
```
