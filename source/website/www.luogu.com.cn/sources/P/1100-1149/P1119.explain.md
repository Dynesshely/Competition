# P1119 灾后重建 题解

## 题目链接

- [洛谷 P1119](https://www.luogu.com.cn/problem/P1119)

## 题目大意

给定 N 个村庄（编号 0~N-1），每个村庄在第 t_i 天重建完成。有 M 条双向公路连接村庄，每条公路有长度。只有两端都已重建完成的村庄之间的公路才能通行。Q 次询问，每次给出 (x, y, t)，求在第 t 天从 x 到 y 的最短路径长度（若 x 或 y 未重建或无可达路径则输出 -1）。保证 t_i 和询问的 t 都是不下降的。

## 解题思路

利用 Floyd-Warshall 算法的离线处理。村庄重建时间 t_i 和查询时间 t 均已排序，维护一个指针 k 表示当前已重建的村庄。对于每个询问，先将所有重建时间 ≤ t 的村庄作为中间节点执行 Floyd 松弛，然后检查 dist[x][y] 是否可到达（同时需确认 x 和 y 本身已重建）。

## 代码实现

- 用 `t[i]` 存储每个村庄的重建时间，`dist[u][v]` 存储当前最短距离。
- 对于每个查询 (x, y, curT)，while 循环将重建时间 ≤ curT 的村庄 k 加入 Floyd 松弛过程（三层循环更新距离矩阵）。
- 若 t[x] > curT 或 t[y] > curT 或 dist[x][y] 为 INF，输出 -1；否则输出距离。

## 复杂度分析

- 时间复杂度：O(N^3 + Q)，其中 Floyd 松弛部分每个村庄仅加入一次，每次 O(N^2)。
- 空间复杂度：O(N^2)。
