# P1108 [洛谷] [USACO99FALL] 低价购买 题解

## 题目链接

- [洛谷 P1108](https://www.luogu.com.cn/problem/P1108)

## 题目大意

给定 n 天的股票价格，每次买入的价格必须低于上一次买入的价格。求最长下降子序列的长度，以及该长度的不同下降序列个数（价格序列完全相同视为同一个序列）。

## 解题思路

经典 LIS 变种。设 dp[i] 表示以第 i 天结尾的最长下降子序列长度，cnt[i] 表示对应的方案数。转移时枚举 j < i，若 a[j] > a[i] 则可转移：若 dp[j] + 1 > dp[i] 则更新长度和计数；若 dp[j] + 1 == dp[i] 则累加计数。去重关键：若 a[j] == a[i] 且 dp[j] == dp[i]，则将 cnt[j] 置 0，避免相同价格序列被重复统计。

## 代码实现

- 读入 n 和价格数组 a
- 对每个 i：初始化 dp[i]=1, cnt[i]=1
- 枚举 j < i：
  - 若 a[j] > a[i] 且 dp[j]+1 > dp[i]，更新 dp[i] 和 cnt[i]
  - 若 a[j] > a[i] 且 dp[j]+1 == dp[i]，cnt[i] += cnt[j]
- 再次遍历 j < i，若 a[j]==a[i] 且 dp[j]==dp[i]，将 cnt[j] 置 0
- 统计最大长度和对应方案总数

## 复杂度分析

- 时间复杂度：O(n^2)
- 空间复杂度：O(n)
