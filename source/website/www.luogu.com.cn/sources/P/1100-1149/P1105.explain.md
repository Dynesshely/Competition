# P1105 [洛谷] 平台 题解

## 题目链接

- [洛谷 P1105](https://www.luogu.com.cn/problem/P1105)

## 题目大意

有 N 个平台，每个平台由高度 H、左端点 L、右端点 R 描述。平台视为开区间 (L, R)。对每个平台，分别求出从平台左边缘和右边缘竖直落下后会落到哪个平台（输出平台序号，若下方无平台则输出 0）。若落到两个高度相同的平台均可，则选编号靠前的。

## 解题思路

对于每个平台 i，分别处理左边缘 x = L[i] 和右边缘 x = R[i]。遍历所有高度低于 H[i] 的平台 j，检查 x 是否在 (L[j], R[j]) 区间内。在满足条件的平台中，选择高度最高的（高度相同选编号最小的），即为落点。

## 代码实现

- 读入所有平台的高度和左右端点
- 对每个平台 i：
  - 遍历 j != i，找到高度低于 H[i] 且区间包含左端点的平台，取最高者（编号优先）
  - 同理处理右端点
- 输出每个平台的左右落点编号

## 复杂度分析

- 时间复杂度：O(N^2)
- 空间复杂度：O(N)
