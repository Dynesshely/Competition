# P1127 词链 题解

## 题目链接

- [洛谷 P1127](https://www.luogu.com.cn/problem/P1127)

## 题目大意

给定 n 个单词，若单词 X 的末字母与单词 Y 的首字母相同，则 X 与 Y 可以相连（用 '.' 连接）。求使用所有单词各一次组成的字典序最小的词链（即欧拉通路），若不存在则输出 "***"。

## 解题思路

将每个单词视为从首字母指向末字母的一条有向边，问题转化为在有向图中找一条欧拉通路（恰好经过每条边一次）。

欧拉通路存在的条件：
1. 最多一个节点出度 = 入度 + 1（起点），最多一个节点入度 = 出度 + 1（终点），其余节点入度 = 出度。
2. 图（忽略边的方向后）连通。

使用 Hierholzer 算法找欧拉通路。为得到字典序最小的词链，需要从字典序最小的可能起点出发，并以字典序优先的方式遍历。

实现细节：
- 用 `multiset<string> adj[26]` 存储每个起点对应的所有单词（自动按字典序排序），方便 DFS 时优先选择字典序小的边。
- 根据度数确定起点：若存在出度 > 入度的节点则从该起点开始；否则从含有边的最小字母节点开始尝试所有可能。
- DFS 过程中删除已使用的边，回溯时记录路径，最后反转得到结果。若未能覆盖所有边则无解。

## 代码实现

- `inDeg` 和 `outDeg` 统计每个字母的出入度。
- 判断度数条件是否满足欧拉通路的必要条件。
- 对每个可能的起点调用 `tryStart()`，内部复制邻接表进行 DFS 并检测是否覆盖所有边。
- 所有可能结果中取字典序最小的作为答案。

## 复杂度分析

- 时间复杂度：O(n log n + n * 26)，主要来自 multiset 操作和 DFS 遍历。
- 空间复杂度：O(n)。
