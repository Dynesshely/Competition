# P1144 最短路计数 题解

## 题目链接

- [洛谷 P1144](https://www.luogu.com.cn/problem/P1144)

## 题目大意

给定一个 N 个顶点 M 条边的无向无权图，求从顶点 1 出发到每个顶点的最短路有多少条，答案对 100003 取模。

## 解题思路

BFS 求最短路。因为边权为 1，BFS 的层次遍历天然就是最短路。在 BFS 过程中，如果第一次访问到节点 v（dist[v] == -1），则 cnt[v] = cnt[u]；如果已经访问过且 dist[v] == dist[u] + 1（即当前路径也是最短路），则 cnt[v] = (cnt[v] + cnt[u]) % MOD。

## 代码实现

链式前向星存图。数组模拟队列 BFS。dist 数组初始化为 -1，cnt[1] = 1。在扩展时根据 dist 的状态进行计数累加或初始化。

## 复杂度分析

- 时间复杂度：O(N + M)
- 空间复杂度：O(N + M)
