# P1262 [洛谷] 间谍网络 题解

## 题目链接
- [洛谷 P1262](https://www.luogu.com.cn/problem/P1262)

## 题目大意
有 n 个间谍，其中 p 个可以被收买（各有标价）。若间谍 A 掌握 B 的证据，则收买 A 后可以揭发 B（不花钱）。问能否控制所有间谍，若能则输出最小花费，否则输出无法被控制的间谍中编号最小的。

## 解题思路
使用**Tarjan 强连通分量（SCC）缩点**后转化为 DAG 上的问题：

1. **缩点**：对原图运行 Tarjan 算法，求出每个 SCC。
2. **缩点后处理**：
   - 每个 SCC 的收买代价为其中可收买间谍的最小代价（若无则 INF）。
   - 计算 DAG 的入度。
3. **可达性分析**：
   - 将所有可收买的 SCC（代价不为 INF）入队 BFS，标记可达 SCC。
   - 若有 SCC 不可达，输出其中编号最小的间谍编号。
4. **最小花费**：
   - 若全部可达，答案为所有入度为 0 的 SCC 的最小代价之和（因为入度为 0 的 SCC 无法通过其他 SCC 间接获得，必须直接收买）。
