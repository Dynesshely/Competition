# P1281 [CERC1998] 书的复制 题解

## 题目链接
- [洛谷 P1281](https://www.luogu.com.cn/problem/P1281)

## 题目大意
将 m 本连续的书分给 k 个人抄写，每人抄写的书必须连续，每个人的速度相同。求一种分配方案使抄写页数最多的人用时最短，且让前面的人尽可能少抄。

## 解题思路
二分答案 + 贪心分配。二分最大页数上限 limit，check 函数判断能否在 limit 内分完。确定最小上限后，从后往前贪心分配，让后面的人尽量多抄，从而使得前面的人抄得少。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int m, k, pages[505], ansL[505], ansR[505];

bool check(int limit) {
    int cnt = 1, sum = 0;
    for (int i = 0; i < m; i++) {
        if (pages[i] > limit)
            return false;
        if (sum + pages[i] > limit) {
            cnt++;
            sum = pages[i];
        } else
            sum += pages[i];
    }
    return cnt <= k;
}

void input() {
    scanf("%d%d", &m, &k);
    for (int i = 0; i < m; i++)
        scanf("%d", &pages[i]);
}

void process() {
    int lo = 0, hi = 0;
    for (int i = 0; i < m; i++) {
        if (pages[i] > lo)
            lo = pages[i];
        hi += pages[i];
    }
    while (lo < hi) {
        int mid = (lo + hi) / 2;
        if (check(mid))
            hi = mid;
        else
            lo = mid + 1;
    }
    int maxPages = lo, cur = m - 1;
    for (int i = k - 1; i >= 0; i--) {
        int sum = 0, end = cur;
        while (cur >= i && sum + pages[cur] <= maxPages) {
            sum += pages[cur];
            cur--;
        }
        ansL[i] = cur + 1;
        ansR[i] = end;
    }
}

void output() {
    for (int i = 0; i < k; i++)
        printf("%d %d\n", ansL[i] + 1, ansR[i] + 1);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(m log sum)。
- 空间复杂度：O(m)。
