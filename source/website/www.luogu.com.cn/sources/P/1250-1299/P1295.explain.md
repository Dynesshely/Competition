# P1295 [TJOI2011] 书架 题解

## 题目链接

- [洛谷 P1295](https://www.luogu.com.cn/problem/P1295)

## 题目大意

$N$ 本书各有高度 $h_i$。将它们分成若干连续段，每段总宽度 $\le M$。一段的代价为该段中最大高度。求最小总代价。

## 解题思路

DP + 单调队列 + multiset 优化。

$dp[i]$ = 前 $i$ 本书的最小代价。转移：$dp[i] = \min_{L \le j < i} \{ dp[j] + \max_{j+1 \le k \le i} h_k \}$，其中 $L$ 为满足 $\sum_{t=L+1}^i h_t \le M$ 的最左位置。

用单调递减队列维护区间最大值，multiset 维护每个决策点的 $dp[j] + h_{next}$ 值。每次扫描 $i$ 时：
- 滑动左边界 $L$，将过期的队首弹出。
- 维护单调队列：弹出高度 $\le h_i$ 的队尾（更新 multiset）。
- 将 $dp[back] + h_i$ 插入 multiset。
- $dp[i]$ = min(multiset 最小值, $dp[L] + h_{front}$)。

## 代码实现

- 单调队列 `q` 存下标，multiset `vals` 存决策值。
- 滑动窗口维护宽度约束。

## 复杂度分析

- 时间复杂度：$O(N \log N)$。
- 空间复杂度：$O(N)$。
