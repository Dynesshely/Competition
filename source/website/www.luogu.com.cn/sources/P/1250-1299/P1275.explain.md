# P1275 魔板 题解

## 题目链接
- [洛谷 P1275](https://www.luogu.com.cn/problem/P1275)

## 题目大意
给定两个 n x m 的 01 矩阵（灯泡亮/暗），可以通过两种操作进行变换：
1. 翻转任意一行（所有 0 变 1，1 变 0）。
2. 交换任意两列。
判断能否将第一个矩阵变换为第二个矩阵。

## 解题思路
由于列交换可以任意重排列，关键在于匹配列。枚举第一矩阵的某一列作为目标第一列（匹配第二矩阵的第一列），从而确定每行的翻转状态 r[i]（通过 XOR 比较）。确定行翻转后，对第一矩阵的每一列按 r[i] 进行翻转，将得到的列字符串与第二矩阵的列字符串分别排序后比较是否完全一致。如果存在任一列能匹配成功则输出 YES。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int            k, n, m;
int            a[105][105], b[105][105];
int            r[105];
vector<string> ca, cb;

void input() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &a[i][j]);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            scanf("%d", &b[i][j]);
}

bool check() {
    ca.clear();
    cb.clear();
    for (int j = 0; j < m; j++) {
        string s;
        for (int i = 0; i < n; i++)
            s += '0' + (a[i][j] ^ r[i]);
        ca.push_back(s);
    }
    for (int j = 0; j < m; j++) {
        string s;
        for (int i = 0; i < n; i++)
            s += '0' + b[i][j];
        cb.push_back(s);
    }
    sort(ca.begin(), ca.end());
    sort(cb.begin(), cb.end());
    return ca == cb;
}

bool process() {
    for (int jA = 0; jA < m; jA++) {
        for (int i = 0; i < n; i++)
            r[i] = a[i][jA] ^ b[i][0];
        if (check())
            return true;
    }
    return false;
}

void output(bool ok) { puts(ok ? "YES" : "NO"); }

void solve() {
    input();
    output(process());
}

int main() {
    scanf("%d", &k);
    while (k--)
        solve();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(k * m * n * m log m)。
- 空间复杂度：O(n * m)。
