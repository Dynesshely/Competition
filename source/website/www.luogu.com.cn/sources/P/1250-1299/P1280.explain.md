# P1280 [CHCI 2002 National Competition #2 Juniors] 尼克的任务 题解

## 题目链接
- [洛谷 P1280](https://www.luogu.com.cn/problem/P1280)

## 题目大意
尼克有 n 分钟工作时间，有 k 个任务，每个任务有开始时间 p 和持续时间 t。每个时刻如果有多个任务可选择其一做，其余由同事做；如果只有一个任务则必须由尼克做。求尼克能获得的最大空闲时间。

## 解题思路
倒序 DP。dp[i] 表示从第 i 分钟到第 n 分钟的最大空闲时间。
- 如果第 i 分钟没有任务开始：dp[i] = dp[i+1] + 1（空闲）。
- 如果有任务开始：dp[i] = max(dp[i + t])（选择其中一个任务做，跳过 t 分钟）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         n, k;
vector<int> start[10005];
int         dp[10005];

void input() {
    scanf("%d%d", &n, &k);
    for (int i = 1; i <= k; i++) {
        int p, t;
        scanf("%d%d", &p, &t);
        start[p].push_back(t);
    }
}

void process() {
    for (int i = n; i >= 1; i--) {
        if (start[i].empty())
            dp[i] = dp[i + 1] + 1;
        else
            for (int t : start[i])
                dp[i] = max(dp[i], dp[i + t]);
    }
}

void output() { printf("%d\n", dp[1]); }

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(n + k)。
- 空间复杂度：O(n + k)。
