# P1276 校门外的树（增强版） 题解

## 题目链接
- [洛谷 P1276](https://www.luogu.com.cn/problem/P1276)

## 题目大意
马路上从 0 到 L 每个位置都有一棵树。有两种操作：
- 0 A B：砍掉 [A, B] 范围内的所有树和树苗。
- 1 C D：在 [C, D] 范围内的空穴处补种树苗。
求最终留下的树苗数量，以及种上后被砍掉的树苗数量。

## 解题思路
用状态数组 state[i] 表示位置 i 的状态：
- 1：原始树木
- 2：树苗
- 0：空穴
模拟每次操作即可。砍树时如果遇到树苗则 cut++，种树时只在空穴种树苗。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int         l, n;
vector<int> state;

void output(int remain, int cut) { printf("%d\n%d\n", remain, cut); }

void input() {
    scanf("%d%d", &l, &n);
    state.resize(l + 1, 1);
}

void process() {
    int remain = 0, cut = 0;
    for (int i = 0; i < n; i++) {
        int op, a, b;
        scanf("%d%d%d", &op, &a, &b);
        if (op == 0) {
            for (int j = a; j <= b; j++) {
                if (state[j] == 2)
                    cut++;
                if (state[j] == 1 || state[j] == 2)
                    state[j] = 0;
            }
        } else {
            for (int j = a; j <= b; j++)
                if (state[j] == 0)
                    state[j] = 2;
        }
    }
    for (int i = 0; i <= l; i++)
        if (state[i] == 2)
            remain++;
    output(remain, cut);
}

int main() {
    input();
    process();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(L + N * L)，N 和 L 均较小。
- 空间复杂度：O(L)。
