# P1265 [洛谷] 公路修建 题解

## 题目链接

## 题目大意
n 个城市分布在平面上，每轮每个城市选择与其最近的城市申请修建公路。政府按规则审批：多城申请同一条则共同修建；三个以上申请成环则否决最短的一条。如此反复直到所有城市连通为一整个"城市联盟"。求修建公路总长度。

## 解题思路
题目描述的过程实际上是 **Prim 最小生成树算法** 的具象化描述。每轮每个城市找最近邻，等价于 Prim 中每次选择距离已选集合最近的点加入。因此直接实现 O(n^2) 的 Prim 即可。

算法步骤：
1. 从城市 1 开始，dist[i] 表示城市 i 到当前已选集合的最小距离。
2. 每次选择距离最小的未加入城市 u，加入集合，累加 sqrt(dist[u])。
3. 用新加入的城市 u 更新其他未加入城市的距离（欧几里得距离平方）。
4. 重复 n 轮。

注意 n ≤ 5000，O(n^2) 的平方级别计算在时限内可行。
