# P1833 樱花 题解

## 题目链接

- [洛谷 P1833](https://www.luogu.com.cn/problem/P1833)

## 题目大意

有 n 棵樱花树，每棵树有观赏时间 T[i]、美学值 C[i] 和可观赏次数 P[i]（P[i] = 0 表示无限次，其他正整数表示最多看 P[i] 次）。给定上学前的总剩余时间（不超过 1000 分钟），求能获得的最大美学值。

## 解题思路

这是一道 **混合背包（Mixed Knapsack）** 问题：同一组物品中，有些是 0-1 背包（只能取一次），有些是完全背包（无限次），有些是多重背包（有限次）。

代码根据 P[i] 的值将每棵树分为三类，分别用不同的 DP 方式处理：

1. **P[i] == 0（无限次）**：按完全背包处理，内层容量 j 从 `w[i]` 到 t **正序**遍历。
2. **P[i] == 1（只一次）**：按 0-1 背包处理，内层容量 j 从 t 到 `w[i]` **逆序**遍历。
3. **P[i] >= 2（有限次）**：按多重背包处理，内层容量 j 逆序遍历，再嵌套一层 k 枚举取 k 件。

注意时间输入格式为 `hh:mm`，需要将两个时间点之差转换为分钟数。

## 代码实现

- **`getDeltaTime(minutes)`**：读入起止时间（格式 `hh:mm hh:mm`），计算差值分钟数。公式为 `(c - a - 1) * 60 + d + (60 - b)`，等价于 `(c - a) * 60 + (d - b)`。
- **DP 数组 `f[maxt]`**：`maxt = 1005`，覆盖最大时间范围 1000 分钟。
- **三类背包处理**：
  - `s[i] == 0`：完全背包，`for (int j = w[i]; j <= t; ++j)` 正序。
  - `s[i] == 1`：0-1 背包，`for (int j = t; j >= w[i]; --j)` 逆序。
  - 其余：多重背包，逆序 + 内层 k 枚举件数，注意 `k * w[i] <= j` 条件。
- 最后输出 `f[t]`。

## 复杂度分析

- 时间复杂度：O(n * t * avgP)，对于多重背包部分最坏为 O(n * t * P)，但因 t <= 1000 且多数 P 较小，可以接受。
- 空间复杂度：O(t)，仅需一维 DP 数组。
