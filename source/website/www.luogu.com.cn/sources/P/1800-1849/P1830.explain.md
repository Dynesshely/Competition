# P1830 轰炸III 题解

## 题目链接

- [洛谷 P1830](https://www.luogu.com.cn/problem/P1830)

## 题目大意

有一个 n x m 的城市，经历了 x 次矩形轰炸。每次轰炸给定一个矩形区域（左上角和右下角坐标）。之后有 y 个关键点需要查询，输出每个关键点是否被轰炸过，若被轰炸过则输出被炸次数和最后一次是第几轮轰炸。

## 解题思路

由于 n, m <= 100，可以直接使用二维数组暴力模拟。对于每次轰炸，遍历矩形内的所有格子，更新该格子的被炸次数和最后一次轰炸的轮次编号（即第几次轰炸）。

查询时，根据 cnt 数组判断是否被炸过，输出对应信息。

## 代码实现

- 使用 cnt[r][c] 记录格子被炸次数，last[r][c] 记录最后一次轰炸轮次。
- 读入每次轰炸的矩形坐标，双重循环标记区域内所有格子。
- 读入每个查询点，若 cnt 为 0 输出 N，否则输出 Y、次数和最后一次轮次。

## 复杂度分析

- 时间复杂度：O(x * n * m + y)，最坏 O(100 * 100 * 100)
- 空间复杂度：O(n * m)
