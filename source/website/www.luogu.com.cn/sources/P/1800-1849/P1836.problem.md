## 题目描述

一本书的页码是从 $1\sim n$ 编号的连续整数：$1,2,3,\cdots,n$。请你求出全部页码中所有单个数字的和，例如第 $123$ 页，它的和就是 $ 1+2+3=6$。

## 输入格式

一行一个整数 $n$。

## 输出格式

一行，代表所有单个数字的和。

## 样例 #1

### 样例输入

```
3456789
```

### 样例输出

```
96342015
```

## 提示

$1\le n\le 10^9$
