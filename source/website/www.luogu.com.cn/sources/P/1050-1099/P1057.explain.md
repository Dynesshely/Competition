# P1057 [NOIP 2008 普及组] 传球游戏 题解

## 题目链接

- [洛谷 P1057](https://www.luogu.com.cn/problem/P1057)

## 题目大意

$n$ 个同学站成一圈（$1 \sim n$），球从 $1$ 号开始，每次可以向左或向右传（即传给相邻的同学）。求传了 $m$ 次后球恰好回到 $1$ 号的方案数。

## 解题思路

### DP 状态定义

设 $dp[i][j]$ 表示传了 $i$ 次后，球在 $j$ 号同学手里的方案数。

### 状态转移

对于第 $i$ 次传球后球在 $j$ 号，则第 $i-1$ 次传球后球必然在 $j$ 的左侧或右侧邻居手中。考虑圆圈首尾相接：

$$dp[i][j] = dp[i-1][j-1] + dp[i-1][j+1]$$

其中左侧 $j-1$ 若等于 $0$ 则回到 $n$（即 $j=1$ 时左邻居是 $n$），右侧 $j+1$ 若等于 $n+1$ 则回到 $1$（即 $j=n$ 时右邻居是 $1$）。

代码中以三元条件表达式实现：

```
dp[i][j] = dp[i-1][j == 1 ? n : j - 1] + dp[i-1][j == n ? 1 : j + 1];
```

### 边界条件

$dp[0][1] = 1$，表示第 $0$ 次传球（初始状态）球在 $1$ 号手中且方案数为 $1$。其余 $dp[0][j] = 0$。

### 答案

$dp[m][1]$ 即为传 $m$ 次后球回到 $1$ 号的方案数。

## 代码实现

- 用二维数组 `dp[35][35]` 存储状态，$n, m \le 30$ 足够。
- 双重循环：外层 $i$ 从 $1$ 到 $m$（传球次数），内层 $j$ 从 $1$ 到 $n$（当前位置）。
- 转移时处理环的首尾相接。
- 输出 `dp[m][1]`。

## 复杂度分析

- 时间复杂度：$O(n \times m)$，状态数 $n \times m \le 900$，每次转移 $O(1)$。
- 空间复杂度：$O(n \times m)$，可优化为 $O(n)$ 的滚动数组但 $n, m \le 30$ 不需要。
