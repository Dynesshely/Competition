# P1062 [NOIP 2006 普及组] 数列 题解

## 题目链接

- [洛谷 P1062](https://www.luogu.com.cn/problem/P1062)

## 题目大意

序列由 $k$ 的方幂及有限个互不相等的 $k$ 的方幂之和构成，按递增排列。求序列的第 $N$ 项。

例如 $k=3$：$1, 3, 4, 9, 10, 12, 13, \ldots$

## 解题思路

观察序列的结构：每一项对应二进制位为 $1$ 的位置取 $k$ 的相应次幂再求和。本质上，第 $N$ 项就是把 $N$ 的二进制表示看作 $k$ 进制表达。

即：$a_N = \sum_{i} k^i \cdot [N \text{ 的第 } i \text{ 位为 } 1]$

例如 $k=3, N=6$（二进制 $110$）：$a_6 = 3^2 + 3^1 = 12$。

## 代码实现

```cpp
long long ans = 0, pw = 1;
while (N) {
    if (N & 1) ans += pw;
    pw *= k;
    N >>= 1;
}
```

## 复杂度分析

- 时间复杂度：$O(\log N)$，$N \le 1000$。
- 空间复杂度：$O(1)$。
