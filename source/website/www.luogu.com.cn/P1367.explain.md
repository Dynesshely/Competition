# P1367 蚂蚁 题解

## 题目链接
- [洛谷 P1367](https://www.luogu.com.cn/problem/P1367)

## 题目大意
有 $n$ 只蚂蚁在无限长的木棍上，每只蚂蚁有初始位置 $a_i$ 和初始朝向 $b_i$（$1$ 表示向右，$-1$ 表示向左）。蚂蚁以每秒 $1$ 单位速度移动，相遇时立即掉头。求 $t$ 秒后每只蚂蚁的位置与朝向（$0$ 表示正在转向）。

## 解题思路
经典蚂蚁问题：蚂蚁相遇掉头等价于蚂蚁互相穿过但交换身份。按此思路：
1. 将蚂蚁按初始位置排序，计算每只蚂蚁若不掉头（互相穿过）$t$ 秒后的"鬼魂"位置。
2. 将鬼魂位置排序，位置顺序与初始蚂蚁顺序对应——由于蚂蚁不可区分，穿过后的位置顺序与初始顺序一致。
3. 碰撞检测：若某位置与相邻位置重合，则该处蚂蚁正在转向，朝向为 $0$（即两只蚂蚁在 $t$ 时刻恰好相遇）。
4. 通过鬼魂位置对应的原始方向确定最终朝向（若未碰撞则保持原方向，因为穿过等价于交换方向）。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Ant {
    int id, pos, dir, finalPos, finalDir;
};

int         n, t;
vector<Ant> ants;

void input() {
    scanf("%d%d", &n, &t);
    ants.resize(n);
    for (int i = 0; i < n; ++i) {
        scanf("%d%d", &ants[i].pos, &ants[i].dir);
        ants[i].id = i;
    }
}

void process() {
    sort(ants.begin(), ants.end(), [](const Ant &a, const Ant &b) { return a.pos < b.pos; });
    vector<pair<int, int>> ghostPairs(n);
    for (int i = 0; i < n; ++i)
        ghostPairs[i] = {ants[i].pos + ants[i].dir * t, i};
    sort(ghostPairs.begin(), ghostPairs.end());
    for (int i = 0; i < n; ++i) {
        ants[i].finalPos = ghostPairs[i].first;
        if ((i > 0 && ghostPairs[i].first == ghostPairs[i - 1].first) || (i < n - 1 && ghostPairs[i].first == ghostPairs[i + 1].first))
            ants[i].finalDir = 0;
        else
            ants[i].finalDir = ants[ghostPairs[i].second].dir;
    }
    sort(ants.begin(), ants.end(), [](const Ant &a, const Ant &b) { return a.id < b.id; });
}

void output() {
    for (int i = 0; i < n; ++i)
        printf("%d %d\n", ants[i].finalPos, ants[i].finalDir);
}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(n \log n)$，排序为主
- 空间复杂度：$O(n)$
