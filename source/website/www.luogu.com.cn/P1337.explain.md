# P1337 [JSOI2004] 平衡点 / 吊打XXX 题解

## 题目链接
- [洛谷 P1337](https://www.luogu.com.cn/problem/P1337)

## 题目大意
给定平面上 $n$ 个点（洞口），每个点有一个重量 $w_i$，求一个点 $x$（绳结位置）使得 $\sum w_i \times \text{dist}(x, p_i)$ 最小，即求带权费马点（加权几何中位数）。

## 解题思路
本题是经典的**模拟退火**求几何中位数的问题。物理模型为：每个重物通过绳子穿过桌面上的洞，最终在绳结处达到平衡，此时系统总势能最小，等价于求加权距离和的最小值点。

解题步骤如下：

1. **初始化**：以所有点的加权平均坐标作为初始解。
2. **模拟退火**：在初始解附近随机扰动，以概率 $\exp(-\Delta E / T)$ 接受更差的解，逐步降低温度 $T$，迭代多次以跳出局部最优。
3. **迭代精化**：模拟退火结束后，使用类 Weiszfeld 迭代法进一步逼近精确解。每次迭代以当前点向各点的加权方向移动，直至收敛。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int    n;
double xs[1005], ys[1005], wt[1005];
double ax, ay;

double calc(double x, double y) {
    double res = 0;
    for (int i = 0; i < n; i++) {
        double dx = x - xs[i], dy = y - ys[i];
        res += sqrt(dx * dx + dy * dy) * wt[i];
    }
    return res;
}

void input() {
    scanf("%d", &n);
    double sumw = 0;
    for (int i = 0; i < n; i++) {
        scanf("%lf%lf%lf", &xs[i], &ys[i], &wt[i]);
        ax += xs[i] * wt[i];
        ay += ys[i] * wt[i];
        sumw += wt[i];
    }
    ax /= sumw;
    ay /= sumw;
}

void process() {
    double ae = calc(ax, ay);
    for (int run = 0; run < 4; run++) {
        double t = 10000, cx = ax, cy = ay, ce = ae;
        while (t > 1e-14) {
            double nx    = cx + t * (rand() * 2.0 / RAND_MAX - 1);
            double ny    = cy + t * (rand() * 2.0 / RAND_MAX - 1);
            double ne    = calc(nx, ny);
            double delta = ne - ce;
            if (delta < 0) {
                cx = nx;
                cy = ny;
                ce = ne;
            } else if (exp(-delta / t) * RAND_MAX > rand()) {
                cx = nx;
                cy = ny;
                ce = ne;
            }
            if (ne < ae) {
                ax = nx;
                ay = ny;
                ae = ne;
            }
            t *= 0.999;
        }
    }
    for (int i = 0; i < 100000; i++) {
        double nx = 0, ny = 0, sumw = 0;
        for (int j = 0; j < n; j++) {
            double d = sqrt((ax - xs[j]) * (ax - xs[j]) + (ay - ys[j]) * (ay - ys[j]));
            if (d < 1e-12) {
                nx   = ax;
                ny   = ay;
                sumw = 1;
                break;
            }
            double w = wt[j] / d;
            nx += xs[j] * w;
            ny += ys[j] * w;
            sumw += w;
        }
        double px = nx / sumw, py = ny / sumw;
        if (fabs(px - ax) < 1e-12 && fabs(py - ay) < 1e-12)
            break;
        ax = px;
        ay = py;
    }
}

void output() { printf("%.3lf %.3lf\n", ax, ay); }

int main() {
    srand(time(0));
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：$O(T \cdot n + K \cdot n)$，其中 $T$ 为模拟退火迭代次数（约数万次），$K$ 为精化迭代次数（最多 $10^5$ 次），$n$ 为点数。实际运行在数据范围内很快。
- 空间复杂度：$O(n)$，仅存储各点坐标和权值。
