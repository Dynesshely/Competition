# P1168 中位数 题解

## 题目链接
- [洛谷 P1168](https://www.luogu.com.cn/problem/P1168)

## 题目大意
给定一个长度为 N 的非负整数序列 A，对前 1、3、5、... 个奇数项分别求中位数并输出。

## 解题思路
采用对顶堆（双堆）维护中位数，这是一种在线处理数据流的经典方法：
- 大根堆 `left` 维护较小的一半数。
- 小根堆 `right` 维护较大的一半数。

依次读入每个数 x：
1. 若 `left` 为空或 x 小于等于 `left` 堆顶，则插入 `left`，否则插入 `right`。
2. 平衡两个堆：确保 `left` 的大小要么等于 `right`，要么比 `right` 多一个元素。
3. 当已读入奇数个元素（下标从 1 开始，i 为奇数）时，`left` 的堆顶即为当前的中位数，直接输出。

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int n;

void input() { scanf("%d", &n); }

void process() {
    priority_queue<int>                            left;
    priority_queue<int, vector<int>, greater<int>> right;
    for (int i = 1; i <= n; i++) {
        int x;
        scanf("%d", &x);
        if (left.empty() || x <= left.top())
            left.push(x);
        else
            right.push(x);
        if (left.size() < right.size()) {
            left.push(right.top());
            right.pop();
        }
        if (left.size() > right.size() + 1) {
            right.push(left.top());
            left.pop();
        }
        if (i % 2 == 1)
            printf("%d\n", left.top());
    }
}

void output() {}

int main() {
    input();
    process();
    output();
    return 0;
}
```

## 复杂度分析
- 时间复杂度：O(N log N)，每个元素插入堆操作 O(log N)。
- 空间复杂度：O(N)，存储全部元素的两堆。
