# 1065 奇数求和 题解

## 题目链接

- [一本通 1065](https://ybt.ssoier.cn/problem_show.php?pid=1065)

## 题目大意

计算从 m 到 n（包括端点）之间所有奇数的和，其中 0 <= m <= n <= 300。

## 解题思路

从 m 遍历到 n，对每个数判断是否为奇数（不能被2整除），若为奇数则累加到总和中。

## 代码实现

变量 `m`、`n` 为区间端点，`sum` 初始化为0。用 `for` 循环从 `i = m` 遍历到 `i = n`，若 `i % 2 != 0` 则 `sum += i`。最后输出 `sum`。

## 复杂度分析

- 时间复杂度：O(n - m)，需要遍历区间内的每个数。
- 空间复杂度：O(1)，仅使用常数个变量。
