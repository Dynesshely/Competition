# 2018 【例4.3】输出奇偶数之和 题解

## 题目链接

- [一本通 2018](https://ybt.ssoier.cn/problem_show.php?pid=2018)

## 题目大意

输入 n，分别求 1 到 n 之间所有偶数的和与奇数的和。

## 解题思路

使用 for 循环遍历 1 到 n，根据 `i % 2` 判断奇偶，分别累加到奇数和与偶数和变量中。

## 代码实现

- 定义变量 `jsum` 记录奇数和、`osum` 记录偶数和，初始化为 0。
- 循环 `for (int i = 1; i <= n; ++i)`：
  - 若 `i % 2 == 0`，则 `osum += i`。
  - 否则 `jsum += i`。
- 输出 `osum` 和 `jsum`，用空格隔开。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。
