## 题目描述

一个最简单的计算器，支持<code>+</code>,<code>-</code>,<code>*</code>,<code>/</code> 四种运算。仅需考虑输入输出为整数的情况，数据和运算结果不会超过<code>int</code>表示的范围。然而：\n1. 如果出现除数为0的情况，则输出：<code>Divided by zero!</code>\n2. 如果出现无效的操作符(即不为 <code>+</code>,<code>-</code>,<code>*</code>,<code>/</code> 之一），则输出：<code>Invalid operator!</code>

## 输入格式

输入只有一行，共有三个参数，其中第$1$、$2$个参数为整数，第$3$个参数为操作符（<code>+</code>,<code>-</code>,<code>*</code>,<code>/</code>）。

## 输出格式

输出只有一行，一个整数，为运算结果。然而：\n1.如果出现除数为$0$的情况，则输出：<code>Divided by zero!</code>\n2.如果出现无效的操作符(即不为<code>+</code>,<code>-</code>,<code>*</code>,<code>/</code>之一），则输出：<code>Invalid operator!</code>

## 样例 #1

### 样例输入

```
1 2 +
```

### 样例输出

```
3
```
