# 2064 【例2.1】交换值 题解

## 题目链接

- [一本通 2064](https://ybt.ssoier.cn/problem_show.php?pid=2064)

## 题目大意

输入两个正整数 a 和 b，交换它们的值后输出。

## 解题思路

使用临时变量 c 暂存 a 的值，然后将 b 赋给 a，再将 c（即原 a）赋给 b。

## 代码实现

- 定义 `int a, b, c`，读入 a 和 b。
- 执行 `c = a; a = b; b = c;` 完成交换。
- 输出 a 和 b。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
