# 1293 买书 题解

## 题目链接

- [一本通 1293](https://ybt.ssoier.cn/problem_show.php?pid=1293)

## 题目大意

小明有 $n$ 元钱（$0 \le n \le 1000$），全部用来买书。书的价格只有 10 元、20 元、50 元、100 元四种，每种书可以购买多本。求有多少种不同的买书方案。

## 解题思路

**完全背包求方案数**问题。

- 四种书的价格相当于四种物品，每种可以无限取。
- 状态定义：`f[j]` 表示凑出 $j$ 元的方案数。
- 边界：`f[0] = 1`，表示花 0 元有 1 种方案（什么都不买）。
- 由于每种书可以无限购买，内层循环需要**从小到大**遍历（与完全背包一致）：
  - 对每种价格 $c[i]$，循环 $j = c[i]..n$，执行 `f[j] += f[j - c[i]]`。
- 最终答案：`f[n]`。

注意：当 $n = 0$ 时，输出 0（因为要"全部用来买书"，花 0 元不算方案）。

## 代码实现

- `c[10000]`：存储四种书的价格（10, 20, 50, 100）。
- `f[10000]`：DP 数组。
- 初始化 `f[0] = 1`。
- 若 `v == 0` 直接输出 0。
- 外层循环 $i=1..4$ 枚举四种价格，内层循环 $j = c[i]..v$ 累加方案数。
- 输出 `f[v]`。

## 复杂度分析

- 时间复杂度：$O(4 \times n)$，$n \le 1000$。
- 空间复杂度：$O(n)$。
