# 1007 计算(a+b)×c的值 题解

## 题目链接

- [一本通 1007](https://ybt.ssoier.cn/problem_show.php?pid=1007)

## 题目大意

给定三个整数 a, b, c，计算表达式 (a + b) * c 的值。

## 解题思路

按要求直接计算即可。先计算 a + b，再乘以 c，C++ 中可用括号控制运算顺序。

## 代码实现

定义三个 `int` 变量 `a`, `b`, `c`，用 `cin` 读入后，使用 `cout` 输出 `(a + b) * c` 的计算结果。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
