# 1301 大盗阿福 题解

## 题目链接

- [一本通 1301](https://ybt.ssoier.cn/problem_show.php?pid=1301)

## 题目大意

一条街上有 $N$ 家店铺，每家店有若干现金。不能同时洗劫相邻的两家店铺。求在不惊动警察的情况下能得到的最大现金总数。

## 解题思路

经典 **打家劫舍（House Robber）DP** 问题。

- 状态定义：`dp[i]` 表示前 $i$ 家店铺能获得的最大现金。
- 状态转移：对于第 $i$ 家店，有两种选择：
  - 不抢第 $i$ 家，则收益为 `dp[i-1]`。
  - 抢第 $i$ 家，则收益为 `dp[i-2] + arr[i]`。
  - 取二者最大值：`dp[i] = max(dp[i-2] + arr[i], dp[i-1])`。
- 边界：`dp[1] = arr[1]`。
- 答案：所有 `dp[i]` 中的最大值（或直接取 `dp[n]`）。

## 代码实现

- `arr[100010]`：存储每家店铺的现金。
- `dp[100010]`：DP 数组。
- `term()` 函数处理每组数据：
  - 读入第一家，初始化 `dp[1] = arr[1]` 和 `ans`。
  - 从 $i=2$ 开始边读入边计算 `dp[i] = max(dp[i-2] + arr[i], dp[i-1])`，同时更新 `ans = max(ans, dp[i])`。
- 每组装毕输出 `ans`。

## 复杂度分析

- 时间复杂度：$O(T \times N)$，$T \le 50, N \le 100000$。
- 空间复杂度：$O(N)$。
