# 1051 分段函数 题解

## 题目链接

- [一本通 1051](https://ybt.ssoier.cn/problem_show.php?pid=1051)

## 题目大意

根据输入的浮点数 N（0 <= N < 20），按三个区间分别计算分段函数值并保留三位小数输出。

## 解题思路

题目给出了三个区间上的分段函数表达式：
- 0 <= x < 5：y = -x + 2.5
- 5 <= x < 10：y = 2 - 1.5(x - 3)^2
- 10 <= x < 20：y = x/2 - 1.5

通过 `if-else if` 判断 N 落在哪个区间，代入对应公式计算结果即可。

## 代码实现

定义 `double` 变量 `N` 和 `y`。用多重分支判断 N 的范围：
- `N >= 0 && N < 5` 时，`y = 2.5 - N`（等价于 `-N + 2.5`）
- `N >= 5 && N < 10` 时，`y = 2 - 1.5 * (N - 3) * (N - 3)`
- `N >= 10 && N < 20` 时，`y = N / 2 - 1.5`

最后使用 `printf("%0.3f", y)` 保留三位小数输出。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次运算。
- 空间复杂度：O(1)，仅使用常数个变量。
