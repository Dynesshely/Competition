# 1019 浮点数向零舍入 题解

## 题目链接

- [一本通 1019](https://ybt.ssoier.cn/problem_show.php?pid=1019)

## 题目大意

输入一个单精度浮点数，将其向零舍入到整数（正数向下取整，负数向上取整）。

## 解题思路

C++ 中将 `float` 赋值给 `int` 时会自动截断小数部分，行为刚好是向零舍入（正数向下，负数向上）。因此直接使用强制类型转换即可。

## 代码实现

用 `float` 读入浮点数 `in`，将其赋值给 `int` 变量 `i`（自动截断小数部分）。判断 `in` 的正负后直接输出 `i`。当 `in` 为正数时，`i` 即为向下取整结果；当 `in` 为负数时，`i` 即为向上取整结果；当 `in` 为 0 时直接输出 0。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
