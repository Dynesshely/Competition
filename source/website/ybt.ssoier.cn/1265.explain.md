# 1265 【例9.9】最长公共子序列 题解

## 题目链接

- [一本通 1265](https://ybt.ssoier.cn/problem_show.php?pid=1265)

## 题目大意

给定两个由大写字母构成的字符串 $X$ 和 $Y$（长度 $\le 1000$），求它们的最长公共子序列（LCS）的长度。

## 解题思路

经典 **动态规划（DP）** LCS 问题。

- 状态定义：`_map[i][j]` 表示 $X$ 的前 $i$ 个字符与 $Y$ 的前 $j$ 个字符的最长公共子序列长度。
- 状态转移：
  - 如果 $X[i-1] == Y[j-1]$（注意字符串下标从 0 开始），则 `_map[i][j] = _map[i-1][j-1] + 1`。
  - 否则 `_map[i][j] = max(_map[i-1][j], _map[i][j-1])`。
- 结果：`_map[al][bl]`，其中 `al`、`bl` 分别为两字符串长度。

## 代码实现

- 读入字符串 `a, b`，获取长度 `al, bl`。
- 定义二维数组 `_map[al+1][bl+1]`，初始化为 0。
- 二重循环 $i=1..al, j=1..bl$：
  - 取 `up = _map[i-1][j]`、`left = _map[i][j-1]`、`upleft = _map[i-1][j-1]`。
  - 若 `a[i-1] == b[j-1]` 则 `_map[i][j] = upleft + 1`，否则取 `max(up, left)`。
- 最后输出 `_map[al][bl]`。

## 复杂度分析

- 时间复杂度：$O(al \times bl) = O(n^2)$，其中 $n$ 为字符串长度（最大 1000）。
- 空间复杂度：$O(al \times bl) = O(n^2)$。
