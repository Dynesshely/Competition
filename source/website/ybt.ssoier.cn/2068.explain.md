# 2068 【例2.6】鸡兔同笼 题解

## 题目链接

- [一本通 2068](https://ybt.ssoier.cn/problem_show.php?pid=2068)

## 题目大意

已知头共 x 个，脚共 y 只，求鸡和兔各有多少只。

## 解题思路

设鸡有 j 只，兔有 t 只。由 j + t = x 和 2j + 4t = y 解得 t = (y - 2x) / 2，j = x - t。

## 代码实现

- 定义 `double x, y`，读入头数和脚数。
- 计算 `t = (y - 2 * x) / 2`，`j = x - t`。
- 输出 j 和 t。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
