# 1043 整数大小比较 题解

## 题目链接

- [一本通 1043](https://ybt.ssoier.cn/problem_show.php?pid=1043)

## 题目大意

输入两个整数 x 和 y，比较它们的大小，输出 >、= 或 <。

## 解题思路

使用 if-else 分支结构对 x 和 y 进行大小比较，分别处理三种情况。注意 x 的范围为 [0, 2^32)，需要使用 `long long` 类型存储。

## 代码实现

- 定义 `long long` 类型变量 `x` 和 `y` 存储输入的整数。
- 使用 `if(x > y)` 输出 `>`，`else if(x < y)` 输出 `<`，否则输出 `=`。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
