## 题目描述

小明手里有$n$元钱全部用来买书，书的价格为$10$元，$20$元，$50$元，$100$元。\n问小明有多少种买书方案？（每种书可购买多本）

## 输入格式

一个整数 $n$，代表总共钱数。（$0 \\le n \\le 1000$）

## 输出格式

一个整数，代表选择方案种数。

## 样例 #1

### 样例输入

```
20
```

### 样例输出

```
2
```
