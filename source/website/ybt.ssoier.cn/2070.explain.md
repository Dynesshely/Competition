# 2070 【例2.13】数字对调 题解

## 题目链接

- [一本通 2070](https://ybt.ssoier.cn/problem_show.php?pid=2070)

## 题目大意

输入一个三位数，将百位数与个位数对调后输出。

## 解题思路

通过整除和取模运算分别取出个位、十位、百位数字，然后重新组合。

## 代码实现

- 读入整数 a。
- `b = a / 100` 取出百位，`c = a / 10 % 10` 取出十位，`d = a % 10` 取出个位。
- 计算 `n = d * 100 + c * 10 + b`。
- 输出 n。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
