# 1031 反向输出一个三位数 题解

## 题目链接

- [一本通 1031](https://ybt.ssoier.cn/problem_show.php?pid=1031)

## 题目大意

输入一个三位数，将其反向输出。

## 解题思路

通过取模和整除运算分离出个位、十位和百位数字，然后按个位、十位、百位的顺序输出即可。

## 代码实现

- 定义变量 `a` 存储输入的三位数，`ge`、`shi`、`bai` 分别存储个位、十位、百位。
- `ge = a % 10` 获取个位，`shi = a / 10 % 10` 获取十位，`bai = a / 100` 获取百位。
- 按顺序 `cout << ge << shi << bai` 输出反向结果。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
