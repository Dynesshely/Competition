## 题目描述

给定非负整数$n$，求$2^n$的值，即$2$的$n$次方。

## 输入格式

一个整数$n$。$0 ≤ n < 31$。

## 输出格式

一个整数，即$2$的$n$次方。

## 样例 #1

### 样例输入

```
3
```

### 样例输出

```
8
```
