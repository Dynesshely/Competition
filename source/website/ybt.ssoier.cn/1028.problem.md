## 题目描述

给定一个字符，用它构造一个对角线长5个字符，倾斜放置的菱形。

## 输入格式

输入只有一行， 包含一个字符。

## 输出格式

该字符构成的菱形。

## 样例 #1

### 样例输入

```
*
```

### 样例输出

```
*
 ***
*****
 ***
  *
```
