# 1147 最高分数的学生姓名 题解

## 题目链接

- [一本通 1147](https://ybt.ssoier.cn/problem_show.php?pid=1147)

## 题目大意

输入 N 名学生的分数和姓名，找出最高分学生的姓名并输出。数据保证最高分只有一位同学。

## 解题思路

用结构体存储学生信息（姓名和分数），将学生按分数从高到低排序，排序后第一个元素即为最高分学生，输出其姓名。

## 代码实现

定义结构体 `stu` 包含 `name`（`string`）和 `grades`（`int`），定义比较函数 `cmp` 按 `grades` 降序排列。读入 N 名学生的信息后调用 `sort(students, students+N, cmp)` 排序，输出 `students[0].name`。

## 复杂度分析

- 时间复杂度：O(N log N)，主要取决于排序的时间。
- 空间复杂度：O(N)，需要存储 N 名学生的信息。
