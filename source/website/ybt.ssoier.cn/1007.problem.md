## 题目描述

给定$3$个整数$a、b、c$，计算表达式$(a+b)×c$的值。

## 输入格式

输入仅一行，包括三个整数$a、b、c$, 数与数之间以一个空格分开。$(－10,000<a,b,c<10,000)$

## 输出格式

输出一行，即表达式的值。

## 样例 #1

### 样例输入

```
2 3 5
```

### 样例输出

```
25
```
