# 1317 【例5.2】组合的输出 题解

## 题目链接

- [一本通 1317](https://ybt.ssoier.cn/problem_show.php?pid=1317)

## 题目大意

从 $1..n$ 中选出 $r$ 个数组成组合（不分顺序），按字典序输出所有组合。每个数输出宽度占 3 个字符。

## 解题思路

**DFS 递归生成组合**。

- 使用数组 `q[22]` 存储当前已选的数字。
- 递归函数 `search(now, t)`：表示当前选了数字 `now` 作为第 $t$ 个元素。
  - 如果已经选了 $r$ 个数（`t == r`），则输出当前组合。
  - 否则从 `now + 1` 到 $n$ 枚举下一个要选的数字，递归调用 `search(i, t+1)`。
- 主函数枚举第一个数字 $i = 1..n-r+1$，调用 `search(i, 1)`。

由于要求输出宽度 3 且右对齐，使用 `printf("%3d", ...)` 格式化输出。

## 代码实现

- `q[22]`：存储当前组合的选择。
- `search(int now, int t)`：递归生成组合。将 `now` 存入 `q[t]`；若 `t == r` 则输出；否则枚举更大的数递归。
- 主函数：读入 $n, r$，对第一个数枚举 $i=1..n-r+1$ 开始搜索。

## 复杂度分析

- 时间复杂度：$O(C(n, r))$，即组合数。
- 空间复杂度：$O(r)$。
