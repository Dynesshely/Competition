# 1015 计算并联电阻的阻值 题解

## 题目链接

- [一本通 1015](https://ybt.ssoier.cn/problem_show.php?pid=1015)

## 题目大意

输入两个电阻的阻值 r1 和 r2，计算并联后的总电阻，保留 2 位小数。

## 解题思路

并联电阻公式为 R = 1 / (1/r1 + 1/r2)。使用 `double` 类型进行浮点运算以避免整数除法导致的精度丢失。

## 代码实现

读入两个 `double` 变量 `r1`, `r2`，按照公式 `R = 1 / (1/r1 + 1/r2)` 计算并联电阻值，使用 `printf("%0.2f", R)` 保留 2 位小数输出。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
