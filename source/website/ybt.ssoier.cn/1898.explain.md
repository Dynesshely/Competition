# 1898 【16NOIP提高组】愤怒的小鸟 题解

## 题目链接

- [一本通 1898](https://ybt.ssoier.cn/problem_show.php?pid=1898)

## 题目大意

在平面第一象限中有 n 只小猪，每只小猪坐标为 (xᵢ, yᵢ)。从 (0,0) 发射小鸟，飞行轨迹为 y = ax² + bx（a ≤ 0）的抛物线。求消灭所有小猪至少需要发射多少只小鸟。

## 解题思路

**状态压缩动态规划（DP + 位运算）。** 当 n ≤ 18 时，可以用二进制位表示每只小猪是否已被消灭。

标准解法：
1. 枚举任意两只小猪（或单独一只），计算它们确定的抛物线（a < 0），检查该抛物线能覆盖哪些小猪，用一个掩码（bitmask）表示。
2. 设 `dp[mask]` 表示当前已消灭状态为 mask 时还需要的最少小鸟数。
3. 从空状态 `dp[0] = 0` 开始，每次选择一条抛物线覆盖一批小猪，更新 DP 状态。
4. 最终答案为 `dp[(1<<n)-1]`。

## 代码实现

当前 `1898.cpp` 为一个框架性实现：
- 读入关卡数 `turns`。
- 对每个关卡读入小猪数量 n 和神秘指令类型 m。
- 将 n 只小猪坐标存入 `pigs[]` 数组。
- 使用 `switch(m)` 分支处理三种指令类型，但各分支均为空，未实现实际逻辑。
- 最终输出 `ans`（初始为 0）。

代码缺少完整的抛物线预处理和 DP 搜索实现，需要补充以下内容：
1. 枚举任意两点计算抛物线参数 a、b，生成覆盖掩码。
2. 初始化 DP 数组为无穷大，`dp[0] = 0`。
3. 对每个状态 mask，枚举可用抛物线进行状态转移。
4. 输出 `dp[(1<<n)-1]`。

## 复杂度分析

- 时间复杂度：O(2ⁿ × n²)，其中 n ≤ 18。
- 空间复杂度：O(2ⁿ)，DP 数组大小。
