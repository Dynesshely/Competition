# 1059 求平均年龄 题解

## 题目链接

- [一本通 1059](https://ybt.ssoier.cn/problem_show.php?pid=1059)

## 题目大意

输入 n 个学生的年龄（整数），求他们的平均年龄，结果保留两位小数。

## 解题思路

用一个变量累加所有年龄之和，最后除以人数 n 得到平均值。注意做除法时需将整数转换为浮点数以保留小数部分。

## 代码实现

变量 `n` 存储人数，`sum` 累加年龄总和，`tmp` 暂存每次读入的年龄。`for` 循环读入 n 个年龄并累加，最后用 `printf("%0.2f", sum * 1.0 / (n * 1.0))` 输出保留两位小数的平均值。`* 1.0` 是为了将整数转换为浮点数再做除法。

## 复杂度分析

- 时间复杂度：O(n)，需要遍历 n 个年龄求和。
- 空间复杂度：O(1)，仅使用常数个变量。
