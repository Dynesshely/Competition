## 题目描述

给定一个序列$a_1,a_2,…,a_n$，如果存在$i < j$并且$a_i > a_j$，那么我们称之为逆序对，求逆序对的数目。

## 输入格式

第一行为$n$,表示序列长度，接下来的$n$行，第$i+1$行表示序列中的第$i$个数。

## 输出格式

所有逆序对总数。

## 样例 #1

### 样例输入

```
4
3
2
3
2
```

### 样例输出

```
3
```
