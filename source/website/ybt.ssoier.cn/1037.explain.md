# 1037 计算2的幂 题解

## 题目链接

- [一本通 1037](https://ybt.ssoier.cn/problem_show.php?pid=1037)

## 题目大意

给定非负整数 n（0 <= n < 31），求 2^n 的值。

## 解题思路

通过循环累乘计算 2 的 n 次方。当 n = 0 时直接输出 1，否则从 2 开始循环乘以 2，共乘以 n-1 次。

## 代码实现

- 读入整数 `n`。
- 特判 n = 0 的情况，直接输出 1。
- 定义 `unsigned long long sum = 2`，循环 `for (int i = 0; i < n - 1; i++)` 将 sum 乘以 2。
- 输出 sum。

## 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(1)
