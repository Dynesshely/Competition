# 1176 谁考了第k名 题解

## 题目链接

- [一本通 1176](https://ybt.ssoier.cn/problem_show.php?pid=1176)

## 题目大意

输入 n 名学生的学号和成绩（各不相同），求成绩第 k 名的学生的学号和成绩。

## 解题思路

将学生信息按成绩从高到低排序，排序后第 k-1 个元素（0 索引）即为第 k 名。代码使用冒泡排序实现降序排列。

## 代码实现

定义结构体 `date`（含 `num` 学号和 `result` 成绩），读入 n 名学生的信息。使用冒泡排序（双重 `for` 循环）按 `result` 从大到小排序：外层控制轮数，内层相邻比较，若前一个小于后一个则交换两者的 `result` 和 `num`。排序后输出索引 `k-1` 的学号和成绩。代码末尾注释了使用 `sort` 和 `cmp` 的替代实现方案。

## 复杂度分析

- 时间复杂度：O(n^2)，使用冒泡排序；若采用 sort 则为 O(n log n)。
- 空间复杂度：O(n)，需要存储 n 名学生的信息。
