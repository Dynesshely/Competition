# 1291 数字组合 题解

## 题目链接

- [一本通 1291](https://ybt.ssoier.cn/problem_show.php?pid=1291)

## 题目大意

给定 $n$ 个正整数（$n \le 20$）和一个目标 $t$（$t \le 1000$），求有多少种不同的组合方式使得选出的数之和等于 $t$。

## 解题思路

本题属于 **子集和问题**，可以用 0/1 背包的方案数 DP 求解。

- 状态定义：`f[j]` 表示凑出和为 $j$ 的组合方案数。
- 边界：`f[0] = 1`，表示凑出和为 0 有一种方案（不选任何数）。
- 对于每个数 $c[i]$，从 $t$ 向下遍历到 $c[i]$，执行 `f[j] += f[j - c[i]]`。
- 最终答案：`f[t]`。

注意内层循环需要**从大到小**遍历，这与 0/1 背包一致，确保每个数只被用一次（即组合中的元素不重复选取）。

## 代码实现

- `c[1000]`：存储 $n$ 个正整数。
- `f[1000]`：DP 数组，`f[j]` 表示凑出 $j$ 的方案数。
- 初始化 `f[0] = 1`。
- 外层循环枚举每个数 $i=1..m$，内层循环从 $v$ 向下到 $c[i]$ 更新方案数。
- 输出 `f[v]`。

## 复杂度分析

- 时间复杂度：$O(n \times t)$，$n \le 20, t \le 1000$。
- 空间复杂度：$O(t)$。
