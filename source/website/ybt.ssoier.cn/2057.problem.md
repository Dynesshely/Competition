## 题目描述

根据输入的表示星期几的数字，对应输出它的英文名称。\n1、星期一：<code>Monday</code>\n2、星期二：<code>Tuesday</code>\n3、星期三：<code>Wednesday</code>\n4、星期四：<code>Thursday</code>\n5、星期五：<code>Friday</code>\n6、星期六：<code>Saturday</code>\n7、星期日：<code>Sunday</code>\n错误输入请输出：<code>input error!</code>

## 输入格式

输入的表示星期几的数字。

## 输出格式

对应输出它的英文名称。

## 样例 #1

### 样例输入

```
1
```

### 样例输出

```
Monday
```
