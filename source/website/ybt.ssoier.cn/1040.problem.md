## 题目描述

输入一个浮点数，输出这个浮点数的绝对值，保留到小数点后两位。

## 输入格式

输入一个浮点数，其绝对值不超过10000。

## 输出格式

输出这个浮点数的绝对值，保留到小数点后两位。

## 样例 #1

### 样例输入

```
-3.14
```

### 样例输出

```
3.14
```
