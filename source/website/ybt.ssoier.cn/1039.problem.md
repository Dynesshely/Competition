## 题目描述

给定一个整数$N$，判断其正负。如果$N>0$，输出<code>positive</code>；如果$N=0$，输出<code>zero</code>；如果$N<0$，输出<code>negative</code>。

## 输入格式

一个整数$N(-10^9 ≤ N ≤ 10^9)$。

## 输出格式

如果$N > 0$, 输出<code>positive</code>;\n如果$N = 0$, 输出<code>zero</code>;\n如果$N < 0$, 输出<code>negative</code>。

## 样例 #1

### 样例输入

```
1
```

### 样例输出

```
positive
```
