## 题目描述

读入一个单精度浮点数，保留3位小数输出这个浮点数。

## 输入格式

只有一行，一个单精度浮点数。

## 输出格式

也只有一行，读入的单精度浮点数。

## 样例 #1

### 样例输入

```
12.34521
```

### 样例输出

```
12.345
```
