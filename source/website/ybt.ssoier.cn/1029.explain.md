# 1029 计算浮点数相除的余 题解

## 题目链接

- [一本通 1029](https://ybt.ssoier.cn/problem_show.php?pid=1029)

## 题目大意

给定两个双精度浮点数 a 和 b，计算 a 除以 b 的余数 r，满足 a = k * b + r，其中 k 是整数，0 <= r < b。

## 解题思路

根据余数定义，先计算整数商 k = floor(a / b)，再计算余数 r = a - k * b。注意 k 取整时直接赋值给整型变量即可实现向下取整。

## 代码实现

- 定义 `double` 类型的变量 `a`、`b`、`r` 和 `int` 类型变量 `k`。
- 使用 `scanf` 读入 a 和 b，计算 `k = a / b`（自动取整），再计算 `r = a - k * b`。
- 使用 `printf("%g", r)` 输出余数，`%g` 自动选择合适的小数位数。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
