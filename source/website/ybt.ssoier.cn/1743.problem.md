## 题目描述

小明每天晚上都在数羊。\n对于每只羊$i$，都有一个吵闹程度$a[i]$，每只羊的吵闹程度都不同。\n小明要数的是对于羊$i,j,k(i<j<k)$满足$a[i]<a[k]$而且$a[k]<a[j]$的羊的$3$元排列$(i,j,k)$组数。\n现在小明想请你帮他数这样的羊的组数。

## 输入格式

第一行有一个正整数$N$，即羊的总数。\n接下来一行有$N$ 个不同的正整数，第$i$个数表示第$i$头羊的吵闹程度。

## 输出格式

有且仅有一个整数,即要求的羊的组数。

## 样例 #1

### 样例输入

```
3
1 3 2
```

### 样例输出

```
1
```
