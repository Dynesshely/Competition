# 1035 等差数列末项计算 题解

## 题目链接

- [一本通 1035](https://ybt.ssoier.cn/problem_show.php?pid=1035)

## 题目大意

给出一个等差数列的前两项 a1 和 a2，求第 n 项的值。

## 解题思路

先计算公差 d = a2 - a1，再根据等差数列通项公式 an = a1 + (n - 1) * d 计算第 n 项。本题代码使用递归方式实现，每次递归加上公差并递增计数器，直到计数器等于 n。

## 代码实现

- 读入 a1、a2、n，计算公差 `d = a2 - a1`。
- 定义递归函数 `getn(a1, d, n)`，全局变量 `now` 记录当前项数，递归到 `now == n` 时返回当前项值减去一个公差（因为递归多加了）。
- 主函数调用 `getn(a1, d, n)` 并输出结果。

## 复杂度分析

- 时间复杂度：O(n)，递归 n 次
- 空间复杂度：O(n)，递归调用栈深度为 n
