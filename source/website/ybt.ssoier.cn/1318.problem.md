## 题目描述

任何一个大于1的自然数n，总可以拆分成若干个小于n的自然数之和。\n当n=7共14种拆分方法：\n<pre>7=1+1+1+1+1+1+1\n7=1+1+1+1+1+2\n7=1+1+1+1+3\n7=1+1+1+2+2\n7=1+1+1+4\n7=1+1+2+3\n7=1+1+5\n7=1+2+2+2\n7=1+2+4\n7=1+3+3\n7=1+6\n7=2+2+3\n7=2+5\n7=3+4\ntotal=14</pre>

## 输入格式

输入n。

## 输出格式

按字典序输出具体的方案。

## 样例 #1

### 样例输入

```
7=1+1+1+1+1+1+1\n7=1+1+1+1+1+2\n7=1+1+1+1+3\n7=1+1+1+2+2\n7=1+1+1+4\n7=1+1+2+3\n7=1+1+5\n7=1+2+2+2\n7=1+2+4\n7=1+3+3\n7=1+6\n7=2+2+3\n7=2+5\n7=3+4\ntotal=14
```

### 样例输出

```
7
```
