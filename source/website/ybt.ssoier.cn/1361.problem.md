## 题目描述

给出一个整数n（n≤2000）和k个变换规则（k≤15）。规则：\n① 1个数字可以变换成另1个数字；\n② 规则中，右边的数字不能为零。\n例如：n=234，k=2规则为\n2 → 5\n3 → 6\n上面的整数234经过变换后可能产生出的整数为（包括原数）234，534，264，564共4种不同的产生数。\n求经过任意次的变换（0次或多次），能产生出多少个不同的整数。仅要求输出不同整数个数。

## 输入格式

$\\begin{matrix}n\\\\k\\\\x_1&y_1\\\\x_2&y_2\\\\…&…\\\\x_n&y_n\\end{matrix}$

## 输出格式

格式为一个整数（满足条件的整数个数）。

## 样例 #1

### 样例输入

```
234
2
2 5
3 6
```

### 样例输出

```
4
```
