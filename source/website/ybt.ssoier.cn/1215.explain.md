# 1215 迷宫 题解

## 题目链接

- [一本通 1215](https://ybt.ssoier.cn/problem_show.php?pid=1215)

## 题目大意

给定 $n \times n$ 的迷宫矩阵，`.` 表示可通行，`#` 表示障碍。从起点 $(ha, la)$ 到终点 $(hb, lb)$，每次可上下左右移动一格，问能否到达。坐标从 0 开始计数。

## 解题思路

使用 **DFS（深度优先搜索）** 判断路径是否存在。

- 用字符串数组 `str` 存储迷宫。
- 用 `vis` 二维数组标记已访问的格子，防止重复搜索。
- 边界检查函数 `pan_duan(x, y)` 判断坐标是否在合法范围内。
- 搜索过程：
  - 标记当前格子已访问。
  - 如果当前格子是终点标记 `'T'`，则返回 `true`。
  - 向四个方向尝试，如果邻居可通行（不是 `'#'`）、未访问且在边界内，则递归搜索。
  - 如果任一方向找到路径，返回 `true`；否则返回 `false`。

注意：起点或终点本身就是 `'#'` 时直接判定无法到达。为了便于判断终点，代码将起点和终点分别替换为 `'S'` 和 `'T'`。

## 代码实现

- `go[4][2]`：四个方向的方向向量。
- `pan_duan(int x, int y)`：判断坐标是否越界。
- `dfs(int x, int y)`：深度优先搜索。先标记 `vis[x][y] = true`，若当前位置为 `'T'` 则返回 `true`；否则枚举四个方向递归。只要有一个方向成功就返回 `true`。
- 主函数：读入组数 $t$，对每组数据先读 $n$ 和矩阵，读入起点终点坐标。如果起点或终点为 `'#'` 则直接输出 `"NO"`；否则将起点标为 `'S'`、终点标为 `'T'`，调用 DFS。每次结束后重置 `vis` 数组。

## 复杂度分析

- 时间复杂度：$O(k \times n^2)$，其中 $k$ 为测试组数，最坏情况下每个格子访问一次。
- 空间复杂度：$O(n^2)$，用于存储迷宫和访问标记。
