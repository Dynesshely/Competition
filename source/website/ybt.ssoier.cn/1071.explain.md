# 1071 菲波那契数 题解

## 题目链接

- [一本通 1071](https://ybt.ssoier.cn/problem_show.php?pid=1071)

## 题目大意

输入正整数 k（1 <= k <= 46），输出菲波那契数列的第 k 项，其中数列第 1、2 项均为 1，后续每项为前两项之和。

## 解题思路

用数组预先生成菲波那契数列的前 50 项，然后根据输入的 k 直接输出第 k 项。递推公式为 `f[i] = f[i-1] + f[i-2]`。

## 代码实现

定义数组 `f[50]`，初始化 `f[1] = 1, f[2] = 1`。用 `for` 循环从 `i = 3` 到 50 递推计算 `f[i] = f[i-1] + f[i-2]`。读入 `k` 后直接输出 `f[k]`。

## 复杂度分析

- 时间复杂度：O(1)，预计算 50 项为常数时间，查询为 O(1)。
- 空间复杂度：O(1)，仅使用固定大小的数组。
