## 题目描述

读入$n$($1≤n≤10000$)个整数，求它们的和与均值。

## 输入格式

输入第一行是一个整数$n$，表示有$n$个整数。\n第$2\\sim n+1$行每行包含$1$个整数。每个整数的绝对值均不超过$10000$。

## 输出格式

输出一行，先输出和，再输出平均值(保留到小数点后$5$位)，两个数间用单个空格分隔。

## 样例 #1

### 样例输入

```
4
344
222
343
222
```

### 样例输出

```
1131 282.75000
```
