# 1966 【14NOIP普及组】比例简化 题解

## 题目链接

- [一本通 1966](https://ybt.ssoier.cn/problem_show.php?pid=1966)

## 题目大意

给定 A、B 和上限 L，在 A' <= L 且 B' <= L 且 A' 与 B' 互质的前提下，找到一组 (A', B') 使得 A'/B' >= A/B 且 A'/B' - A/B 的值尽可能小。

## 解题思路

由于 L <= 100，可以直接枚举 A' 和 B' 从 1 到 L 的所有组合，检查 A'/B' >= A/B 且 A' 与 B' 互质，记录使差值最小的组合。这是一个典型的暴力枚举题。

## 代码实现

（注：当前代码为框架代码，未实现具体逻辑。）

标准实现步骤：
1. 读入 A, B, L。
2. 初始化最优解 `bestA = 1, bestB = 1`。
3. 两层循环枚举 A' 从 1 到 L，B' 从 1 到 L：
   - 计算 `A' * B >= B' * A`（交叉相乘避免浮点误差）。
   - 计算差值，若当前差值更小，则更新最优解。
4. 输出最优的 A' 和 B'。

## 复杂度分析

- 时间复杂度：O(L^2)，L <= 100，完全可行。
- 空间复杂度：O(1)。
