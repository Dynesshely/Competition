# 2020 【例4.5】第几项 题解

## 题目链接

- [一本通 2020](https://ybt.ssoier.cn/problem_show.php?pid=2020)

## 题目大意

输入 m，求最小的 n 使得 1 + 2 + 3 + ... + n > m。

## 解题思路

从 1 开始累加，每次累加后判断和是否超过 m，一旦超过则输出当前的项数 n。由于 m 范围不大，可以直接使用循环累加模拟。

## 代码实现

（注：当前目录下的 2020-4.cpp 为另一题目的代码，与此题无关。）

标准实现步骤：
1. 读入 m。
2. 定义变量 `sum = 0`，`n = 0`。
3. 循环 `while (sum <= m)`：`n++`，`sum += n`。
4. 输出 n。

## 复杂度分析

- 时间复杂度：O(sqrt(m))，因为 1+2+...+n ≈ n^2/2。
- 空间复杂度：O(1)。
