# 1061 求整数的和与均值 题解

## 题目链接

- [一本通 1061](https://ybt.ssoier.cn/problem_show.php?pid=1061)

## 题目大意

输入 n 个整数，求它们的总和以及平均值（保留五位小数）。

## 解题思路

用一个 `double` 变量累加所有整数之和，最后分别输出整数和（强转为 `int`）和浮点数平均值（保留五位小数）。

## 代码实现

变量 `n` 为整数个数，`sum`（`double` 类型）累加所有数的总和，`tmp` 暂存每次读入的值。循环读入并累加后，用 `cout` 输出 `(int)sum` 作为整数和，用 `fixed << setprecision(5)` 控制格式输出 `sum / (n * 1.0)` 作为平均值。

## 复杂度分析

- 时间复杂度：O(n)，需要遍历 n 个数求和。
- 空间复杂度：O(1)，仅使用常数个变量。
