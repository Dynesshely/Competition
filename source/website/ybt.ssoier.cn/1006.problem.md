## 题目描述

大部分的在线题库，都会将A+B问题作为第一题，以帮助新手熟悉平台的使用方法。\nA+B问题的题目描述如下：给定两个整数A和B，输出A+B的值。保证A、B及结果均在整型范围内。现在请你解决这一问题。

## 输入格式

一行，包含两个整数A，B，中间用单个空格隔开。A和B均在整型范围内。

## 输出格式

一个整数，即A+B的值。保证结果在整型范围内。

## 样例 #1

### 样例输入

```
1 2
```

### 样例输出

```
3
```
