## 题目描述

传说古代的叙拉古国王海伦二世发现的公式，利用三角形的三条边长来求取三角形面积。已知△ABC中的三边长分别为a,b,c，求△ABC的面积。\n提示：海伦公式\n$s=\\sqrt{ p(p-a)(p-b)(p-c)}$,其中$p=\\frac{a+b+c}{2}$。

## 输入格式

三角形的三条边长。

## 输出格式

面积。（保留3位小数）

## 样例 #1

### 样例输入

```
2.1 3.1 4.1
```

### 样例输出

```
3.179
```
