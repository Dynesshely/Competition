# 1060 均值 题解

## 题目链接

- [一本通 1060](https://ybt.ssoier.cn/problem_show.php?pid=1060)

## 题目大意

输入 n 个浮点数，计算它们的平均值并保留四位小数输出。

## 解题思路

循环读入所有浮点数并累加求和，最后除以 n 得到均值。由于数据本就是浮点数，除法直接得到浮点结果。

## 代码实现

变量 `n` 为数据个数，`sum` 为浮点累加和，`tmp` 暂存每次读入的数值。用 `for` 循环读入并累加，然后用 `printf("%0.4f", sum / n)` 保留四位小数输出。

## 复杂度分析

- 时间复杂度：O(n)，需要遍历 n 个数据求和。
- 空间复杂度：O(1)，仅使用常数个变量。
