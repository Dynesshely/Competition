# 2071 【例2.14】平均分 题解

## 题目链接

- [一本通 2071](https://ybt.ssoier.cn/problem_show.php?pid=2071)

## 题目大意

x 位男生平均分 87，y 位女生平均分 85，求全体同学平均分。

## 解题思路

全体总分 = 87x + 85y，总人数 = x + y，平均分 = 总分 / 总人数。

## 代码实现

- 定义 `double x, y, sum`。
- 读入 x, y。
- `sum = 87 * x + 85 * y`。
- 输出 `sum / (x + y)`。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
