## 题目描述

已知某班有男同学x位，女同学y位，x位男生平均分是87分，y位女生的平均分是85，问全体同学平均分是多少分？

## 输入格式

男女同学人数。

## 输出格式

平均分（保留4位小数）。

## 样例 #1

### 样例输入

```
2 3
```

### 样例输出

```
85.8000
```
