# 1063 最大跨度值 题解

## 题目链接

- [一本通 1063](https://ybt.ssoier.cn/problem_show.php?pid=1063)

## 题目大意

给定一个长度为 n 的非负整数序列，求最大值与最小值的差（即最大跨度值）。

## 解题思路

同时维护最大值和最小值两个变量，遍历序列的每个数进行更新，最后两者相减即为跨度值。

## 代码实现

变量 `low` 初始化为 `0x3f`（一个较大的数），`high` 初始化为 `-0x3f`（一个较小的数），`tmp` 暂存每次读入的数。循环中通过 `low = min(low, tmp)` 和 `high = max(high, tmp)` 分别维护最小值和最大值，最后输出 `high - low`。

## 复杂度分析

- 时间复杂度：O(n)，需要遍历 n 个数。
- 空间复杂度：O(1)，仅使用常数个变量。
