# 1008 计算(a+b)/c的值 题解

## 题目链接

- [一本通 1008](https://ybt.ssoier.cn/problem_show.php?pid=1008)

## 题目大意

给定三个整数 a, b, c，计算表达式 (a + b) / c 的值，其中 c 不为 0。

## 解题思路

直接进行整数运算即可。由于 a, b, c 都是整数，C++ 中整数除法会向下取整，符合题目对整数商的要求。

## 代码实现

定义三个 `int` 变量 `a`, `b`, `c`，用 `cin` 读入后，使用 `cout` 输出 `(a + b) / c` 的计算结果。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
