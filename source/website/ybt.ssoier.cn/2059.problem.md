## 题目描述

期末来临了，班长小Q决定将剩余班费$x$元钱，用于购买若干支钢笔奖励给一些学习好、表现好的同学。已知商店里有三种钢笔，它们的单价为$6$元、$5$元和$4$元。小Q想买尽量多的笔（鼓励尽量多的同学），同时他又不想有剩余钱。请您编一程序，帮小Q制订出一种买笔的方案。

## 输入格式

一个正整数$x$（剩余班费）。

## 输出格式

一行，依次为6元、5元和4元钱笔的数目，用一个空格隔开。

## 样例 #1

### 样例输入

```
10
```

### 样例输出

```
1 0 1
```
