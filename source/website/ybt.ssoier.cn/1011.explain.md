# 1011 甲流疫情死亡率 题解

## 题目链接

- [一本通 1011](https://ybt.ssoier.cn/problem_show.php?pid=1011)

## 题目大意

给定确诊数和死亡数，计算甲流死亡率，以百分数形式输出，精确到小数点后 3 位。

## 解题思路

死亡率等于死亡数除以确诊数，再乘以 100% 得到百分数。使用 `double` 浮点类型进行计算，用 `printf` 控制输出格式。

## 代码实现

用 `double` 读入确诊数 `a` 和死亡数 `b`，计算死亡率 `c = b / a`。使用 `printf("%0.3f%%", c * 100)` 输出，其中 `%0.3f` 保留 3 位小数，`%%` 输出一个百分号。将 `c` 乘以 100 是将小数转换为百分数。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
