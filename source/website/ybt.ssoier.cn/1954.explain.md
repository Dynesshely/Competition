# 1954 【11NOIP普及组】统计单词数 题解

## 题目链接

- [一本通 1954](https://ybt.ssoier.cn/problem_show.php?pid=1954)

## 题目大意

给定一个单词和一篇文章（含空格），不区分大小写，统计该单词在文章中作为独立单词出现的次数和第一次出现的位置（首字母位置，从 0 开始）。若未出现则输出 -1。

## 解题思路

先将文章中的大写字母转换为小写字母，然后在文章中遍历查找。对于每个位置，判断是否与目标单词的首字母匹配，且该位置是单词开头（前一个字符为空格或是字符串开头），然后逐个字符比较后续字符，同时检查单词结束后是否为空格或字符串结尾。

## 代码实现

- 读入目标单词 `input` 和文章 `tline`。
- 将 `tline` 中的大写字母转为小写（循环遍历，对 'A'-'Z' 范围字符加 32）。
- 遍历 `tline` 的每个字符：
  - 若当前字符等于 `input[0]` 且是单词开头（`i==0` 或 `tline[i-1]==32`），则进行匹配检查。
  - 匹配时逐个比较字符（考虑大小写差异），同时确保匹配结束后是空格或末尾。
- 若匹配成功，则计数 `count++`，并记录第一次出现的位置 `location`。
- 输出 `count` 和 `location`。

## 复杂度分析

- 时间复杂度：O(n * m)，其中 n 为文章长度，m 为单词长度。
- 空间复杂度：O(1)，仅使用常数额外空间。
