# 1821 【00NOIP提高组】乘积最大 题解

## 题目链接

- [一本通 1821](https://ybt.ssoier.cn/problem_show.php?pid=1821)

## 题目大意

给定一个长度为 N 的数字串，要求使用 K 个乘号将其分成 K+1 个部分，使这 K+1 个部分乘积最大，输出最大乘积。

## 解题思路

**动态规划。** 本题是经典的区间 DP / 划分 DP 问题。

定义状态 `dp[i][j]` 表示在前 i 位数字中插入 j 个乘号所能得到的最大乘积。转移方程为：
- `dp[i][j] = max(dp[k][j-1] * num(k+1, i))`，其中 j ≤ k < i，`num(k+1, i)` 表示数字串第 k+1 位到第 i 位组成的数字。

边界条件：`dp[i][0]` 为前 i 位数字本身（不插入乘号）。

由于 N ≤ 40，K ≤ 6，结果可能非常大，需要使用高精度计算或 64 位整数（C++ 中可能超过 long long 范围，需用高精度）。

## 代码实现

当前 `1821.cpp` 为一个框架性实现，仅包含了输入部分：
- 读入 N、K 以及数字串（存储为整数 `num`）。
- 缺少 DP 计算和输出逻辑的完整实现。

完整的实现需要：
1. 将数字串按位拆分存入数组。
2. 预处理任意区间 [l, r] 对应的数字值。
3. 按乘号个数从小到大进行 DP 递推。
4. 输出 `dp[N][K]` 作为最大乘积。

## 复杂度分析

- 时间复杂度：O(N² × K)，枚举位置和乘号个数。
- 空间复杂度：O(N × K)，DP 数组。
