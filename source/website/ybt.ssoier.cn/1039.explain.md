# 1039 判断数正负 题解

## 题目链接

- [一本通 1039](https://ybt.ssoier.cn/problem_show.php?pid=1039)

## 题目大意

给定一个整数 N，判断其正负。若 N > 0 输出 positive；N = 0 输出 zero；N < 0 输出 negative。

## 解题思路

使用 if-else 分支结构对 N 进行正负判断，分别处理大于 0、小于 0、等于 0 三种情况。

## 代码实现

- 使用 `long long` 类型变量 `N` 存储输入的整数。
- 使用 `if(N > 0) ... else if(N < 0) ... else ...` 进行判断并输出对应字符串。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
