# 2054 【例3.4】适合晨练 题解

## 题目链接

- [一本通 2054](https://ybt.ssoier.cn/problem_show.php?pid=2054)

## 题目大意

输入温度 t，若 25 <= t <= 30 输出 "ok!"，否则输出 "no!"。

## 解题思路

使用 if-else 判断温度是否在 [25, 30] 范围内。

## 代码实现

- 定义 `double t`，读入温度。
- 若 `t >= 25.0 && t <= 30.0` 输出 "ok!"，否则输出 "no!"。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
