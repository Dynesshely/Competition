# 1055 判断闰年 题解

## 题目链接

- [一本通 1055](https://ybt.ssoier.cn/problem_show.php?pid=1055)

## 题目大意

输入一个年份 a（0 < a < 3000），判断是否为闰年，是则输出 "Y"，否则输出 "N"。

## 解题思路

闰年的判断规则：能被400整除，或者能被4整除但不能被100整除。封装一个 `isLeap` 函数实现这一逻辑，主函数中读入年份并调用该函数判断输出。

## 代码实现

`isLeap` 函数返回 `year % 400 == 0 || (year % 4 == 0 && year % 100 != 0)` 的布尔值。主函数中用 `scanf` 读入年份，根据 `isLeap` 的返回值输出 `"Y"` 或 `"N"`。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次取模运算。
- 空间复杂度：O(1)，仅使用常数个变量。
