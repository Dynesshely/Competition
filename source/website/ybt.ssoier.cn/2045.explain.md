# 2045 【例5.13】蛇形填数 题解

## 题目链接

- [一本通 2045](https://ybt.ssoier.cn/problem_show.php?pid=2045)

## 题目大意

在 n x n 的方阵中填入 1 到 n^2，按照蛇形（从右上角开始，向下、向左、向上、向右循环移动）的方式填充。

## 解题思路

从右上角（第 1 行第 n 列）开始，按照下、左、上、右四个方向依次移动，当遇到边界或已填数字时改变方向。这种方法也称为"螺旋填数"的变体。

## 代码实现

- 定义二维数组 `arr[30][30]` 初始化为 0。
- 从 `(row=1, col=n)` 开始填入数字 1。
- 使用 while 循环直到填满 `n * n` 个数字：
  - 向下移动：`while (row + 1 <= n && !arr[row + 1][col])` 先自增行再填数。
  - 向左移动：`while (col - 1 >= 1 && !arr[row][col - 1])` 先自减列再填数。
  - 向上移动：`while (row - 1 >= 1 && !arr[row - 1][col])` 先自减行再填数。
  - 向右移动：`while (col + 1 <= n && !arr[row][col + 1])` 先自增列再填数。
- 每个方向移动前检查是否越界和是否已填。
- 最后按行输出方阵。

## 复杂度分析

- 时间复杂度：O(n^2)，需填充 n^2 个格子。
- 空间复杂度：O(n^2)。
