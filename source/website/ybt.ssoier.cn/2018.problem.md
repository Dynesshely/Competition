## 题目描述

利用for循环，分别输出$1∼n$之间的所有奇数的和、偶数的和。

## 输入格式

输入$n$。

## 输出格式

输出为一行，两个数(用一个空格隔开)，偶数之和与奇数之和。

## 样例 #1

### 样例输入

```
10
```

### 样例输出

```
30 25
```
