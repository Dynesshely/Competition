# 1050 骑车与走路 题解

## 题目链接

- [一本通 1050](https://ybt.ssoier.cn/problem_show.php?pid=1050)

## 题目大意

输入行走距离（米），分别计算骑车总耗时（找车开锁27秒 + 骑行距离/3.0 + 锁车23秒）和步行总耗时（距离/1.2），比较两者快慢并输出 "Bike"、"Walk" 或 "All"。

## 解题思路

根据题目给的固定参数直接计算两种方式所需时间：
- 骑车耗时 = 27 + 23 + 距离 / 3.0
- 步行耗时 = 距离 / 1.2

然后比较两个浮点数的大小，按题目要求输出对应的结果字符串。

## 代码实现

定义 `double` 类型变量 `dis` 存储距离，分别计算 `bike` 和 `walk` 的时间。通过 `if-else if-else` 分支判断 `bike` 与 `walk` 的大小关系：若骑车时间更短则输出 `"Bike"`，步行时间更短输出 `"Walk"`，相等则输出 `"All"`。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次运算。
- 空间复杂度：O(1)，仅使用常数个变量。
