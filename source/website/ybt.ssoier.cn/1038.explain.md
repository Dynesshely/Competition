# 1038 苹果和虫子 题解

## 题目链接

- [一本通 1038](https://ybt.ssoier.cn/problem_show.php?pid=1038)

## 题目大意

一箱 n 个苹果，一条虫子每 x 小时吃掉一个苹果（吃完一个才吃下一个）。经过 y 小时后，还有多少个完整的苹果？

## 解题思路

计算虫子吃掉的苹果数。y 小时内虫子能吃 y/x 个苹果（整数除法自动向下取整），但如果有余数需要额外吃掉一个（因为虫子开始吃下一个但没吃完，该苹果已经不完整）。注意边界情况：y = 0 时所有苹果完整；吃掉数量超过 n 时结果为 0。

## 代码实现

- 读入 n、x、y。
- 若 y = 0，输出 n 并返回。
- 若 y/x > n，输出 0 并返回。
- 剩余完整苹果数 `left = n - y/x + (y % x == 0 ? 0 : -1)`，即去掉已吃完的和正在吃的不完整的那个。
- 输出 left。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
