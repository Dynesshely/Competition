# 2034 【例5.1】反序输出 题解

## 题目链接

- [一本通 2034](https://ybt.ssoier.cn/problem_show.php?pid=2034)

## 题目大意

输入 n 个数，按输入相反的顺序打印这 n 个数。

## 解题思路

将输入的数字存入数组，然后从数组末尾到开头逆序输出即可。

## 代码实现

（注：当前代码存在变量名错误（使用了未定义的 `i`），标准实现如下。）

标准实现步骤：
1. 定义数组 `nums[100]` 和索引变量 `index = 0`。
2. 循环读入数字直到输入结束（EOF 或先输入 n 再循环 n 次）。
3. 从 `index-1` 到 0 逆序遍历数组，输出每个元素。
4. 注意输出格式控制（空格）。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(n)。
