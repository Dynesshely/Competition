# 1058 求一元二次方程 题解

## 题目链接

- [一本通 1058](https://ybt.ssoier.cn/problem_show.php?pid=1058)

## 题目大意

输入一元二次方程 ax^2 + bx + c = 0 的三个系数 a、b、c（浮点数，a != 0），求方程的根并精确到小数点后5位输出。分相等实根、不等实根和无实根三种情况处理。

## 解题思路

使用判别式 d = b^2 - 4ac 判断根的情况：
- d > 0：两个不等实根，用求根公式计算，按从小到大顺序输出
- d == 0：两个相等实根，直接用公式 x = -b / (2a) 输出两次
- d < 0：无实根，输出 "No answer!"

注意输出时使用 `min` 和 `max` 确保小根在前、大根在后。

## 代码实现

读入 a、b、c，计算判别式 d。在 d > 0 的分支中直接用公式 `x1 = (-b + sqrt(b*b - 4*a*c)) / (2*a)` 和 `x2 = (-b - sqrt(b*b - 4*a*c)) / (2*a)` 计算，输出时 `min(x1,x2)` 在前、`max(x1,x2)` 在后。d == 0 时直接计算 `x1 = -b / (2*a)`，输出 `"x1=x2=..."` 格式。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次运算和一次开方。
- 空间复杂度：O(1)，仅使用常数个变量。
