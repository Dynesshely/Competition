## 题目描述

NCL是一家专门从事计算器改良与升级的实验室，最近该实验室收到了某公司所委托的一个任务：需要在该公司某型号的计算器上加上解一元一次方程的功能。实验室将这个任务交给了一个刚进入的新手ZL先生。为了很好的完成这个任务,ZL先生首先研究了一些一元一次方程的实例： \n4+3x=8 \n6a-5+1=2-2a \n-5+12Y=0 \nZL先生被主管告之，在计算器上键入的一个一元一次方程中，只包含整数、小写字母 及十、一、＝这三个数学符号（当然，符号“一”既可作减号，也可作负号）。方程中并没有括号，也没有除号，方程中的字母表示未知数。

## 输入格式

输入一个一元一次方程,可认为输入的一元一次方程均为合法的，且有唯一实数解。

## 输出格式

将解方程的结果（精确至小数点后三位）输出。

## 样例 #1

### 样例输入

```
6a-5+1=2-2a
```

### 样例输出

```
a=0.750
```
