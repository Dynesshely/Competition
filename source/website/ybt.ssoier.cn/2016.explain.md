# 2016 【例4.1】for循环求和 题解

## 题目链接

- [一本通 2016](https://ybt.ssoier.cn/problem_show.php?pid=2016)

## 题目大意

输入 n，计算 1 + 2 + 3 + ... + n 的值。

## 解题思路

使用 for 循环从 1 遍历到 n，累加每个数到 sum 变量中，最后输出 sum。

## 代码实现

- 定义变量 `n` 和 `sum`，`sum` 初始化为 0。
- 使用 `scanf` 读入 n。
- 执行 `for (int i = 1; i <= n; ++i) sum += i;` 累加求和。
- 输出 `sum`。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。
