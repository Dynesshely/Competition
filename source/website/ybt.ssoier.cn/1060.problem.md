## 题目描述

给出一组样本数据，包含$n$个浮点数，计算其均值，精确到小数点后$4$位。

## 输入格式

输入有两行，第一行包含一个整数$n$（$n$小于$100$），代表样本容量；第二行包含$n$个绝对值不超过$1000$的浮点数，代表各个样本数据。

## 输出格式

输出一行，包含一个浮点数，表示均值，精确到小数点后$4$位。

## 样例 #1

### 样例输入

```
2
1.0 3.0
```

### 样例输出

```
2.0000
```
