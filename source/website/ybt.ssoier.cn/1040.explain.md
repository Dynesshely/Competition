# 1040 输出绝对值 题解

## 题目链接

- [一本通 1040](https://ybt.ssoier.cn/problem_show.php?pid=1040)

## 题目大意

输入一个浮点数，输出其绝对值，保留两位小数。

## 解题思路

使用三目运算符判断浮点数的正负：如果非负则保持不变，否则取相反数即为绝对值。

## 代码实现

- 定义 `double` 类型变量 `in` 存储输入浮点数。
- 使用三目运算 `in = (in >= 0 ? in : in * -1.0)` 计算绝对值。
- 使用 `printf("%0.2f", in)` 保留两位小数输出。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
