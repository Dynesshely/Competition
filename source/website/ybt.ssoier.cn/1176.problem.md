## 题目描述

在一次考试中，每个学生的成绩都不相同，现知道了每个学生的学号和成绩，求考第k名学生的学号和成绩。

## 输入格式

第一行有两个整数，分别是学生的人数$n（1≤n≤100）$，和求第k名学生的$k（1≤k≤n）$。\n其后有$n$行数据，每行包括一个学号（整数）和一个成绩（浮点数），中间用一个空格分隔。

## 输出格式

输出第$k$名学生的学号和成绩，中间用空格分隔。（注：请用$\\%g$输出成绩）

## 样例 #1

### 样例输入

```
5 3
90788001 67.8
90788002 90.3
90788003 61
90788004 68.4
90788005 73.9
```

### 样例输出

```
90788004 68.4
```
