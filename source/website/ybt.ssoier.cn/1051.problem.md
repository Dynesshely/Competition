## 题目描述

编写程序，计算下列分段函数$y=f(x)$的值。结果保留到小数点后三位。\n     $y=-x+2.5; \\quad \\quad 0≤x<5$\n     $y=2-1.5(x-3)(x-3); \\quad\\quad 5≤x<10$\n     $y=\\frac{x}{2}-1.5; \\quad\\quad 10≤x<20$

## 输入格式

一个浮点数$N(0 ≤ N < 20)$。

## 输出格式

输出 $N$ 对应的分段函数值：$f(N)$。结果保留到小数点后三位。

## 样例 #1

### 样例输入

```
1.0
```

### 样例输出

```
1.500
```
