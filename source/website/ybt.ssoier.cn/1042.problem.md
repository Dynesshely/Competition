## 题目描述

任意输入一个字符，判断其$ASCII$是否是奇数，若是，输出$YES$，否则，输出$NO$。例如，字符$A$的$ASCII$值是$65$，则输出$YES$，若输入字符$B$($ASCII$值是$66$)，则输出$NO$。

## 输入格式

输入一个字符。

## 输出格式

如果其$ASCII$值为奇数，则输出$YES$，否则，输出$NO$。

## 样例 #1

### 样例输入

```
A
```

### 样例输出

```
YES
```
