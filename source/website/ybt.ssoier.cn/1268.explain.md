# 1268 【例9.12】完全背包问题 题解

## 题目链接

- [一本通 1268](https://ybt.ssoier.cn/problem_show.php?pid=1268)

## 题目大意

给定一个容量为 $M$ 的背包和 $n$ 种物品，每种物品有重量 $W_i$ 和价值 $C_i$，每种物品可以无限次选取。求能获得的最大总价值。

## 解题思路

**完全背包问题**，使用一维 DP（优化空间）。

- 状态定义：`f[j]` 表示容量为 $j$ 的背包能获得的最大价值。
- 与 0/1 背包不同，完全背包中每种物品可以多次选取，因此容量循环需要**从小到大**遍历，这样才能反复将当前物品放入。
- 状态转移：对于每个物品，从容量 $j = W_i$ 到 $M$，执行 `f[j] = max(f[j], f[j-W_i] + C_i)`。
- 最终答案：`f[M]`，输出格式带前缀 `max=`。

## 代码实现

- `f[N]`：一维 DP 数组，初始全 0。
- `qread()`：快速读入函数（处理整数输入）。
- 读入 $m$（容量）和 $n$（物品种数）。
- 外层循环 $i=1..n$ 枚举物品种类，读入重量 `wi` 和价值 `ci`。
- 内层循环 $j = wi..m$（从小到大），尝试将当前物品放入背包。
- 最后按格式 `printf("max=%d\n", f[m])` 输出。

## 复杂度分析

- 时间复杂度：$O(n \times m)$，其中 $n \le 30, m \le 200$。
- 空间复杂度：$O(m)$，一维 DP 数组。
