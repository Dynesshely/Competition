# 1070 人口增长 题解

## 题目链接

- [一本通 1070](https://ybt.ssoier.cn/problem_show.php?pid=1070)

## 题目大意

输入当前人口基数 x（亿）和年数 n，按每年 0.1% 的增长率计算 n 年后的人口数，保留四位小数。

## 解题思路

每年人口增长 0.1%，即每年人口变为上一年的 `(1 + 0.001)` 倍。通过循环 n 次累乘该系数即可得到 n 年后的结果。

## 代码实现

变量 `x`（浮点数）为人口基数，`n` 为年数。用 `for` 循环执行 n 次 `x += x * 0.001`（等价于 `x *= 1.001`）。最后用 `printf("%0.4f", x)` 保留四位小数输出。

## 复杂度分析

- 时间复杂度：O(n)，需要循环 n 次计算增长。
- 空间复杂度：O(1)，仅使用常数个变量。
