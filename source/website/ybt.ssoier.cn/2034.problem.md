## 题目描述

输入$n$个数,要求程序按输入时的逆序把这$n$个数打印出来，已知整数不超过$100$个。也就是说，按输入相反顺序打印这$n$个数。

## 输入格式

输入一行共有$n$个数，每个数之间用空格隔开。

## 输出格式

如题要求：一行，共有$n$个数，每个数之间用一个空格隔开。

## 样例 #1

### 样例输入

```
1 7 3 4 5
```

### 样例输出

```
5 4 3 7 1
```
