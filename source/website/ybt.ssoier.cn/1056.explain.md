# 1056 点和正方形的关系 题解

## 题目链接

- [一本通 1056](https://ybt.ssoier.cn/problem_show.php?pid=1056)

## 题目大意

输入一个点的坐标 (x, y)，判断该点是否在以 (-1,-1)、(1,-1)、(1,1)、(-1,1) 为顶点的正方形内（包括边界），是则输出 "yes"，否则输出 "no"。

## 解题思路

正方形的四个顶点坐标确定了 x 和 y 的取值范围都是 [-1, 1]。因此只需检查 x 是否在 -1 到 1 之间（含边界），且 y 是否在 -1 到 1 之间（含边界）。

## 代码实现

读入整数 x, y，通过 `if` 判断 `x >= -1 && x <= 1 && y >= -1 && y <= 1` 是否成立，成立输出 `"yes"`，否则输出 `"no"`。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次比较。
- 空间复杂度：O(1)，仅使用常数个变量。
