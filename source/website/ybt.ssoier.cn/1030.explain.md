# 1030 计算球的体积 题解

## 题目链接

- [一本通 1030](https://ybt.ssoier.cn/problem_show.php?pid=1030)

## 题目大意

给定球的半径 r，使用公式 V = (4/3) * pi * r^3 计算球的体积，其中 pi = 3.14，结果保留两位小数。

## 解题思路

直接套用球体积公式计算。注意除法运算中使用 `4.0/3.0` 而非 `4/3`，以避免整数除法截断。使用 `const` 定义圆周率 PI。

## 代码实现

- 定义 `double` 类型变量 `r` 和 `V`。
- 定义 `const double PI = 3.14` 作为圆周率常量。
- 计算 `V = (4.0 / 3.0) * PI * r * r * r`，用 `printf("%0.2f", V)` 保留两位小数输出。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
