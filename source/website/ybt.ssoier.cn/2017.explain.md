# 2017 【例4.2】输出偶数 题解

## 题目链接

- [一本通 2017](https://ybt.ssoier.cn/problem_show.php?pid=2017)

## 题目大意

输入 n，按从小到大顺序输出 1 到 n 之间的所有偶数，空格隔开。

## 解题思路

使用 for 循环遍历 1 到 n，对每个数用取模运算 `i % 2 == 0` 判断是否为偶数，是则输出。

## 代码实现

- 读入 n。
- 循环 `for (int i = 1; i <= n; ++i)`，判断 `i % 2 == 0` 时输出 `i`。
- 末尾输出换行符。

## 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。
