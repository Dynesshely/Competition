# 1971 【15NOIP普及组】求和 题解

## 题目链接

- [一本通 1971](https://ybt.ssoier.cn/problem_show.php?pid=1971)

## 题目大意

纸带有 n 个格子，每个格子有颜色 color_i 和数字 number_i。定义三元组 (x, y, z) 满足 x < y < z, y - x = z - y, color_x = color_z。分数为 (x + z) * (number_x + number_z)。求所有满足条件的三元组分数之和模 10007。

## 解题思路

条件 y - x = z - y 意味着 x 和 z 关于 y 对称，即 x + z = 2y，所以 x 和 z 的奇偶性相同。因此可以将格子按颜色和奇偶性分组，对每组分别计算贡献。标准解法是利用数学推导，每组的贡献为：组内所有 (number_x + number_z) * (x + z) 之和，可以拆解为前缀和优化到 O(n)。

## 代码实现

（注：当前代码为三层循环的暴力枚举，复杂度 O(n^3)，仅适用于小数据。）

代码实现分析：
- 定义结构体 `block` 存储颜色 `color` 和数字 `num`。
- 读入每个格子的数字 `num` 和颜色 `color`。
- 三层循环枚举 x, y, z，检查条件 `y - x == z - y` 且 `color_x == color_z`。
- 若满足条件，累加 `(x + z) * (number_x + number_z)` 到 `score`。
- 最终输出 `score % 10007`。

## 复杂度分析

- 时间复杂度：O(n^3)（暴力枚举），优化后可为 O(n)。
- 空间复杂度：O(n)。
