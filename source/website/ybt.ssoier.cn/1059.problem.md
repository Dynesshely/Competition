## 题目描述

班上有学生若干名，给出每名学生的年龄（整数），求班上所有学生的平均年龄，保留到小数点后两位。

## 输入格式

第一行有一个整数$n$（$1\\le n \\le 100$），表示学生的人数。其后$n$行每行有$1$个整数，表示每个学生的年龄，取值为$15$到$25$。

## 输出格式

输出一行，该行包含一个浮点数，为要求的平均年龄，保留到小数点后两位。

## 样例 #1

### 样例输入

```
2
18
17
```

### 样例输出

```
17.50
```
