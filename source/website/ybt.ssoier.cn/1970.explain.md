# 1970 【15NOIP普及组】扫雷游戏 题解

## 题目链接

- [一本通 1970](https://ybt.ssoier.cn/problem_show.php?pid=1970)

## 题目大意

给定 n 行 m 列的雷区，`*` 表示地雷，`?` 表示非地雷格。输出每个非地雷格周围（八个方向）的地雷数量，地雷格仍输出 `*`。

## 解题思路

对于每个格子，如果是地雷则直接标记为 `*`，否则统计其八个相邻格子中地雷的个数。实现时分别处理内部格子、边界格子和四个角上的格子，或者统一用方向数组遍历八个方向。

## 代码实现

- 读入 n, m 和雷区地图 `map`，用整数数组 `nums` 存储结果。
- 常量 `boomSign = 42` 对应字符 `'*'` 的 ASCII 码。
- 处理内部格子（`i>0 && i<n-1 && j>0 && j<m-1`）：
  - 若 `map[i][j] != '*'`，用方向数组 `xs[8]` 和 `ys[8]` 遍历八个方向，统计地雷数量。
  - 否则标记为 `boomSign`。
- 单独处理四条边（不含角）和四个角上的格子，分别统计周围地雷数量。
- 输出时，值为 `boomSign` 的输出 `*`，否则输出数字。

## 复杂度分析

- 时间复杂度：O(n * m)，每个格子至多检查 8 个方向。
- 空间复杂度：O(n * m)。
