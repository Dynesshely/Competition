# 1052 计算邮资 题解

## 题目链接

- [一本通 1052](https://ybt.ssoier.cn/problem_show.php?pid=1052)

## 题目大意

输入邮件重量（克）和是否加急（y/n），按规则计算邮费：1000克以内基本费8元，超过部分每500克加收4元（不足500克按500克计），加急多收5元。

## 解题思路

分两部分计算：
1. 基础邮费：若重量不超过1000克，直接取8元；否则按公式 `8 + ceil((重量 - 1000) / 500) * 4` 计算超出部分的费用。
2. 加急附加费：若字符为 `'y'`，则额外增加5元。

计算超出部分需要向上取整（不足500克按500克计），通过条件表达式判断余数是否为0来实现。

## 代码实现

变量 `mess` 存储重量，`price` 存储邮费，`type` 存储加急标识。先判断重量是否大于1000，若大于则计算 `(mess - 1000) / 500 + ((mess - 1000) % 500 == 0 ? 0 : 1)` 来得到需要加收的500克份数，再乘以4加上基本费8元。若 `type == 'y'` 则 `price += 5`。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次运算。
- 空间复杂度：O(1)，仅使用常数个变量。
