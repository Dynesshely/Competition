# 2073 【例2.16】三角形面积 题解

## 题目链接

- [一本通 2073](https://ybt.ssoier.cn/problem_show.php?pid=2073)

## 题目大意

输入三角形三边长 a, b, c，使用海伦公式求面积，保留 3 位小数。

## 解题思路

使用海伦公式：p = (a + b + c) / 2，面积 s = sqrt(p(p-a)(p-b)(p-c))。

## 代码实现

- 定义 `double a, b, c, s, p`。
- 读入三边长。
- 计算 `p = (a + b + c) / 2`。
- 计算 `s = sqrt(p * (p - a) * (p - b) * (p - c))`。
- 用 `printf("%0.3f", s)` 保留 3 位小数输出。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
