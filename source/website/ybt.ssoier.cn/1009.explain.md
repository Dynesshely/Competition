# 1009 带余除法 题解

## 题目链接

- [一本通 1009](https://ybt.ssoier.cn/problem_show.php?pid=1009)

## 题目大意

给定被除数和除数（非零），求整数商和余数。

## 解题思路

使用 C++ 的整除运算符 `/` 求商，取模运算符 `%` 求余数。题目要求使用默认的整除和取余运算。

## 代码实现

定义两个 `int` 变量 `a`（被除数）和 `b`（除数），用 `cin` 读入。使用 `cout` 输出 `a / b`（商）和 `a % b`（余数），中间用空格分隔。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
