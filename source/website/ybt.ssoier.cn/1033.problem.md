## 题目描述

已知线段的两个端点的坐标$A(X_a,Y_a)$，$B(X_b，Y_b)$，求线段$AB$的长度，保留到小数点后$3$位。

## 输入格式

第一行是两个实数$X_a，Y_a$，即$A$的坐标。\n第二行是两个实数$X_b，Y_b$，即$B$的坐标。\n输入中所有实数的绝对值均不超过$10000$。

## 输出格式

一个实数，即线段$AB$的长度，保留到小数点后$3$位。

## 样例 #1

### 样例输入

```
1 1
2 2
```

### 样例输出

```
1.414
```
