# 1046 判断一个数能否同时被3和5整除 题解

## 题目链接

- [一本通 1046](https://ybt.ssoier.cn/problem_show.php?pid=1046)

## 题目大意

给定一个整数 n，判断它能否同时被 3 和 5 整除。能输出 YES，否则输出 NO。

## 解题思路

使用模运算判断：若 n % 3 == 0 且 n % 5 == 0，则 n 能同时被 3 和 5 整除。

## 代码实现

- 定义 `long long` 类型变量 `x` 读入整数。
- 使用 `if(x % 3 == 0 && x % 5 == 0)` 判断，输出 "YES" 或 "NO"。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
