# 1064 奥运奖牌计数 题解

## 题目链接

- [一本通 1064](https://ybt.ssoier.cn/problem_show.php?pid=1064)

## 题目大意

输入 n 天中每天获得的金、银、铜牌数，分别累计金牌总数、银牌总数、铜牌总数以及总奖牌数并输出。

## 解题思路

用三个变量分别累加金牌、银牌、铜牌数量，循环读入 n 行数据并累加，最后总奖牌数等于三者之和。

## 代码实现

变量 `gm`、`sm`、`rm` 分别累加金、银、铜牌数，`t1`、`t2`、`t3` 暂存每天的数据。用 `while(n--)` 循环读入并累加，最后输出 `gm`、`sm`、`rm` 以及 `gm + sm + rm`。

## 复杂度分析

- 时间复杂度：O(n)，需要遍历 n 天的数据。
- 空间复杂度：O(1)，仅使用常数个变量。
