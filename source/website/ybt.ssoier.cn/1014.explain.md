# 1014 与圆相关的计算 题解

## 题目链接

- [一本通 1014](https://ybt.ssoier.cn/problem_show.php?pid=1014)

## 题目大意

给定圆的半径 r，输出直径、周长和面积，每个数保留 4 位小数。圆周率取 3.14159。

## 解题思路

根据圆的几何公式计算：
- 直径 d = 2 * r
- 周长 C = 2 * pi * r
- 面积 S = pi * r * r

使用 `double` 类型运算，用 `printf` 控制输出格式。

## 代码实现

读入 `double` 类型半径 `r`，定义常量 `PI = 3.14159`。用 `printf("%0.4f %0.4f %0.4f", 2*r, 2*PI*r, PI*r*r)` 一次性输出三个值，均保留 4 位小数，中间以空格分隔。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
