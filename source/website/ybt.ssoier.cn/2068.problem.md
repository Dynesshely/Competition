## 题目描述

数学中经典的“鸡兔同笼”问题，已知头共x个，脚共y只，问笼中的鸡和兔各有多少只?

## 输入格式

头和脚的数量。

## 输出格式

鸡和兔各自数量。一个空格隔开。

## 样例 #1

### 样例输入

```
30 90
```

### 样例输出

```
15 15
```
