## 题目描述

读入一个整数，若这个数大于1并且小于$100$，则输出<code>yes</code>。

## 输入格式

## 输出格式

满足条件输出<code>yes</code>，否则不输出。

## 样例 #1

### 样例输入

```
76
```

### 样例输出

