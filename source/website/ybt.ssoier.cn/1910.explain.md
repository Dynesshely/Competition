# 1910 【00NOIP普及组】计算器的改良 题解

## 题目链接

- [一本通 1910](https://ybt.ssoier.cn/problem_show.php?pid=1910)

## 题目大意

输入一个合法的一元一次方程（含整数、小写字母、+、-、=号，无括号和除号），求方程的解，精确到小数点后三位。

## 解题思路

**字符串解析 + 线性方程求解。** 将方程看作字符串，从左到右依次解析，提取出未知数的系数项和常数项，分别求和，然后计算 `常数和 / 系数和` 得到解。

关键技巧：
1. 先扫描一遍字符串，标记所有运算符的位置（包括 `=` 号），并记录未知数字母的位置。
2. 再扫描一遍，区分系数项（含有字母的项）和常数项（纯数字项），根据项在等号左侧还是右侧决定符号的正负：
   - 在等号左侧的项，移到等号右侧需要变号，因此代码中将其取相反数。
   - 在等号右侧的项，同样处理。
3. 最终 `numSum` 为常数项合并结果，`charSum` 为系数项合并结果，解为 `numSum / charSum`。

## 代码实现

- `isChar()` 判断字符是否为字母，`isNum()` 判断字符是否为数字。
- `operators[]` 标记每个位置是否为运算符（非字母非数字），`equ` 记录 `=` 号的位置，`ccc` 记录未知数字母的位置。
- 第一遍扫描：标记运算符位置，记录 `=` 和字母位置。
- 第二遍扫描：按项解析。
  - 当遇到字母时，向左找到最近的运算符或开头，提取系数（默认系数为 1 或 -1），存入 `chars[]`。若该项在等号左侧则取相反数。
  - 当遇到运算符（且前一个字符不是字母，表示这是一个常数项）或到达字符串末尾时，向左提取数字，存入 `nums[]`。若该项在等号右侧则取相反数。
- 分别求和得到 `numSum`（常数和）和 `charSum`（系数和）。
- 计算 `res = numSum / charSum`，按格式输出 `未知数=结果`。

## 复杂度分析

- 时间复杂度：O(n)，其中 n 为方程字符串长度，每个字符遍历常数次。
- 空间复杂度：O(n)，存储运算符标记和系数/常数数组。
