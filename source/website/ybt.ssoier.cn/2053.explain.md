# 2053 【例3.3】三个数 题解

## 题目链接

- [一本通 2053](https://ybt.ssoier.cn/problem_show.php?pid=2053)

## 题目大意

输入三个整数，按从大到小的顺序输出。

## 解题思路

将三个数放入数组，使用 `sort` 函数从小到大排序，再逆序输出。

## 代码实现

- 定义数组 `ints[3]`，读入三个整数。
- 调用 `sort(ints, ints + 3)` 从小到大排序。
- 从 `i = 2` 到 `0` 逆序输出数组元素。

## 复杂度分析

- 时间复杂度：O(1)（固定 3 个元素）。
- 空间复杂度：O(1)。
