## 题目描述

读入一个双精度浮点数，分别按输出格式“%f”，“%f”保留5位小数，“%e”和“%g”的形式输出这个整数，每次在单独一行上输出。

## 输入格式

一个双精度浮点数。

## 输出格式

第一行是按“%f”输出的双精度浮点数；\n第二行是按“%f”保留5位小数输出的双精度浮点数；\n第三行是按“%e”输出的双精度浮点数；\n第四行是按“%g”输出的双精度浮点数。

## 样例 #1

### 样例输入

```
12.3456789
```

### 样例输出

```
12.345679
12.34568
1.234568e+001
12.3457
```
