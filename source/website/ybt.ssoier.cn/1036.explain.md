# 1036 A x B 问题 题解

## 题目链接

- [一本通 1036](https://ybt.ssoier.cn/problem_show.php?pid=1036)

## 题目大意

输入两个正整数 A 和 B（1 <= A, B <= 50000），求 A * B 的值。

## 解题思路

直接计算乘积。注意数据范围，A 和 B 最大为 50000，乘积最大为 2.5e9，超过 int 范围（约 2.1e9），需要使用 `unsigned long long` 存储。

## 代码实现

- 定义 `unsigned long long` 类型变量 `A` 和 `B` 确保乘积不溢出。
- 读入后直接 `cout << A * B` 输出结果。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
