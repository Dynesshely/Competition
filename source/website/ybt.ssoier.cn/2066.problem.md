## 题目描述

已知小明有n元，他买了一本书，这本书原价为m元，现在打8折出售。求小明还剩多少钱(保留2位小数)。

## 输入格式

输入n,m。

## 输出格式

小明还剩多少钱(保留2位小数)。

## 样例 #1

### 样例输入

```
100 100
```

### 样例输出

```
20.00
```
