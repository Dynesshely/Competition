# 1054 三角形判断 题解

## 题目链接

- [一本通 1054](https://ybt.ssoier.cn/problem_show.php?pid=1054)

## 题目大意

输入三个正整数作为三条线段长度，判断它们能否构成一个三角形，能则输出 "yes"，否则输出 "no"。

## 解题思路

判断三条线段能否构成三角形的充要条件是：任意两边之和大于第三边，且任意两边之差小于第三边。代码中同时检查了这两个条件，实际上在正数范围内，两边之和大于第三边已经可以推出两边之差小于第三边，但显式写出两个条件更加严谨。

## 代码实现

定义数组 `bian[3]` 读入三条边长。用复合条件判断：
- `bian[0] + bian[1] > bian[2] && bian[1] + bian[2] > bian[0] && bian[0] + bian[2] > bian[1]`（两边之和大于第三边）
- 同时满足 `bian[0] - bian[1] < bian[2] && bian[1] - bian[2] < bian[0] && bian[0] - bian[2] < bian[1]`（两边之差小于第三边）

都满足则输出 `"yes"`，否则输出 `"no"`。

## 复杂度分析

- 时间复杂度：O(1)，仅进行常数次比较运算。
- 空间复杂度：O(1)，仅使用固定大小的数组。
