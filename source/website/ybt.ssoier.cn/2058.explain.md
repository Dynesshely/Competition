# 2058 【例3.10】简单计算器 题解

## 题目链接

- [一本通 2058](https://ybt.ssoier.cn/problem_show.php?pid=2058)

## 题目大意

输入两个数和一个运算符（+、-、*、/），输出运算结果。处理除数为 0 和无效运算符的情况。

## 解题思路

使用 switch 语句根据运算符进行分支处理，注意除法时检查除数是否为零。

## 代码实现

- 定义 `double a, b` 和 `char oper`。
- 读入 a, b 和 oper。
- 使用 `switch(oper)` 进行分支：
  - `'+'`：输出 a + b。
  - `'-'`：输出 a - b。
  - `'*'`：输出 a * b。
  - `'/'`：若 b == 0 输出 "Divided by zero!"，否则输出 a / b。
  - `default`：输出 "Invalid operator!"。

## 复杂度分析

- 时间复杂度：O(1)。
- 空间复杂度：O(1)。
