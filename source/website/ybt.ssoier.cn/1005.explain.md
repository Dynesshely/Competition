# 1005 地球人口承载力估计 题解

## 题目链接

- [一本通 1005](https://ybt.ssoier.cn/problem_show.php?pid=1005)

## 题目大意

已知现有资源加上新生资源可供 x 亿人生活 a 年，或供 y 亿人生活 b 年，求地球最多能养活多少亿人（即资源恰好不枯竭的人口数）。

## 解题思路

设地球现有资源为 S，每年新生资源为 R，每人每年消耗 1 单位资源。则有：

- S + a * R = x * a
- S + b * R = y * b

两式相减消去 S 得：(b - a) * R = b * y - a * x，即每年新生资源 R = (b * y - a * x) / (b - a)。

最多养活的人口数即为 R（每年新生资源的供应能力）。需要注意输入保证 `x > y`，`a < b` 且 `a * x < b * y`。

## 代码实现

读入四个正整数 `x`, `a`, `y`, `b`（注意代码中变量顺序与题目输入一致）。计算公式 `hold = (b * y - a * x) / (b - a)`，用 `double` 类型存储结果，最后用 `printf` 保留两位小数输出。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
