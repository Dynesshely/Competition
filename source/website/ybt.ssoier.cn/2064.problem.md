## 题目描述

输入两个正整数a和b，试交换a、b的值（使a的值等于b，b的值等于a）。

## 输入格式

输入两个正整数a和b。

## 输出格式

输出a与b交换值后的结果。

## 样例 #1

### 样例输入

```
2 3
```

### 样例输出

```
3 2
```
