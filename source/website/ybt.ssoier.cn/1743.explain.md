# 1743 数羊 题解

## 题目链接

- [一本通 1743](https://ybt.ssoier.cn/problem_show.php?pid=1743)

## 题目大意

给定一个长度为 N 的序列 a（所有元素互不相同），统计满足 i < j < k 且 a[i] < a[k] < a[j] 的三元组 (i, j, k) 的个数。

## 解题思路

**三重循环暴力枚举。** 由于没有明确的题目数据范围限制，代码直接使用三重循环遍历所有可能的下标组合 (i, j, k)，当满足条件 `a[i] < a[k] && a[k] < a[j]` 时累加计数器。

这是最直观的暴力枚举方法，适用于 N 较小的情况。如果 N 较大，可以考虑使用树状数组或线段树进行优化。

## 代码实现

- `a[]` 数组存储输入的序列，索引从 1 开始。
- 三层 `for` 循环分别枚举 i（最左端）、j（中间）、k（最右端），满足 i < j < k。
- 当条件 `a[i] < a[k] && a[k] < a[j]` 成立时，`ans` 自增。
- 最后输出 `ans`。

## 复杂度分析

- 时间复杂度：O(N³)，三重循环遍历所有三元组。
- 空间复杂度：O(N)，存储序列。
