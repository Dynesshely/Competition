# 1003 对齐输出 题解

## 题目链接

- [一本通 1003](https://ybt.ssoier.cn/problem_show.php?pid=1003)

## 题目大意

读入三个整数，按每个整数占 8 个字符宽度右对齐输出，整数之间以一个空格分隔。

## 解题思路

使用 C 语言的 `printf` 函数可以方便地控制输出格式。格式说明符 `%8d` 表示输出一个整数，占据至少 8 个字符宽度，不足部分在左侧用空格补齐，即右对齐。

## 代码实现

使用 `cin` 读入三个整数 `a`, `b`, `c`。然后用 `printf("%8d %8d %8d", a, b, c)` 按格式要求输出，每个整数占 8 个字符宽度并右对齐，中间以空格分隔。

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
